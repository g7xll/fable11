Vendored fonts — SIL Open Font License 1.1
==========================================

The fonts in this folder are bundled locally so the project runs fully offline,
with no calls to Google Fonts or any CDN at runtime. They are loaded via the JS
FontFace API in `src/main.tsx`.

1. Instrument Serif  (InstrumentSerif-Regular.woff2, InstrumentSerif-Italic.woff2)
   - Designer: Rodrigo Fuenzalida, Jordan Egstad
   - Source: https://fonts.google.com/specimen/Instrument+Serif
   - License: SIL Open Font License, Version 1.1
   - Used for the hero's `.instrument` display class (the italic "Beautiful"
     / "Shader" wordmark), matching the original 21st.dev component.

2. Inter  (Inter-Variable.woff2)
   - Designer: Rasmus Andersson
   - Source: https://fonts.google.com/specimen/Inter
   - License: SIL Open Font License, Version 1.1
   - Used as the UI/body typeface for the hero copy and the lab chrome.

Full SIL OFL 1.1 text: https://openfontlicense.org/open-font-license-official-text/

These subsetted woff2 files were downloaded from the Google Fonts CSS API
(latin subset). No modifications were made to the glyph outlines.
