# GRIDWRIGHT — NEO-STRUCTURALIST CODING-SAAS LANDING PAGE

BUILD A FULL, MULTI-SECTION, FULLY RESPONSIVE MARKETING LANDING PAGE FOR A FICTIONAL DEVELOPER-INFRASTRUCTURE SAAS NAMED **GRIDWRIGHT** — "CONSTRUCTING DIGITAL REALITY THROUGH CODE." THE NAMED DESIGN LANGUAGE IS **"NEO-STRUCTURALIST BLUEPRINT"**: A PRECISE, PAPER-LIGHT, ARCHITECTURAL AESTHETIC THAT FEELS LIKE AN ENGINEER'S DRAFTING TABLE FUSED WITH A CODE EDITOR. THE WHOLE PAGE IS LITERALLY DRAWN ON A RULED STRUCTURAL GRID — VERTICAL COLUMN LINES RUN EDGE TO EDGE BEHIND EVERY SECTION, AND A FAINT DIAGONAL CROSS-HATCH DUSTS THE BACKGROUND LIKE GRAPH PAPER. THE MOOD IS CONFIDENT, RIGID, AND HONEST: EVERY MARGIN IS DELIBERATE, EVERY CORNER IS SHARP (ZERO BORDER-RADIUS), AND COLOR IS USED SPARINGLY AS STRUCTURE. STRICTLY AVOID GENERIC-AI-SLOP: NO PURPLE GRADIENTS, NO GLASSMORPHISM, NO NEON, NO SOFT BLURRED DROP SHADOWS — ONLY HARD, OFFSET "BRUTAL" SHADOWS WITH A SOLID FILL AND ZERO BLUR.

## AESTHETIC IDENTITY

- NAMED LANGUAGE: **NEO-STRUCTURALIST BLUEPRINT** — DRAFTING-TABLE PRECISION MEETS MONOSPACE CODE LOGIC.
- EVERYTHING SITS ON A SLIGHTLY-OFF-WHITE PAPER CANVAS (NEVER PURE WHITE), DUSTED WITH A FAINT DIAGONAL CROSS-HATCH (TWO REPEATING 45° / -45° LINEAR GRADIENTS AT ~7% BLACK, SOFTLY BLURRED ~1.5PX SO IT READS AS PAPER TEXTURE, NOT A HARD GRID).
- A CONTAINER OF FULL-HEIGHT VERTICAL GRID LINES (1PX, LIGHT NEUTRAL) SPANS THE PAGE: A CENTERED FRAME AT ~94% WIDTH, MAX 1800PX, WITH LEFT/RIGHT BORDERS AND 5 INTERIOR VERTICAL RULES SPLITTING IT INTO 6 COLUMNS ON DESKTOP (4 ON TABLET, FEWER ON MOBILE). THESE LINES SIT BEHIND CONTENT AND ALIGN EVERYTHING.
- HARD-EDGED EVERYWHERE: BORDER-RADIUS IS 0 (OR AT MOST 2PX). BUTTONS AND PANELS USE A SIGNATURE OFFSET SHADOW (E.G. `4px 4px 0 #1A1A1A` OR `8px 8px 0 #107A4D`) THAT COLLAPSES ON HOVER AS THE ELEMENT NUDGES DOWN-RIGHT.

- DISPLAY / HEADINGS: **SPACE GROTESK** (WEIGHTS 300–700) — USED FOR ALL H1/H2/H3, STATS, PRICE NUMBERS, AND THE BRAND WORDMARK. HEADINGS RUN VERY LARGE (HERO H1 UP TO ~8XL / ~7REM) WITH TIGHT TRACKING AND A COMPRESSED LINE-HEIGHT (~0.9).

## MOTION & INTERACTION SPEC

- SCROLL REVEALS VIA INTERSECTIONOBSERVER: ELEMENTS START HIDDEN (OPACITY 0 + A TRANSFORM — UP 40PX, RIGHT -40PX, OR SCALE 0.9 DEPENDING ON THE ELEMENT) AND EASE IN WITH A CUBIC-BEZIER(0.16, 1, 0.3, 1) OVER ~0.8S; STAGGER VIA TRANSITION-DELAY ON GROUPED ITEMS.
- HERO TERMINAL: TYPEWRITER EFFECT THAT TYPES THE LOG LINES SEQUENTIALLY WITH A PERSISTENT BLINKING `_` CURSOR.
- STATS: COUNT-UP ANIMATION TRIGGERED ON FIRST VISIBILITY.
- STATUS BAR: ~50 SMALL SQUARES SNAPPED TO A 10PX GRID, EACH BLINKING WITH A RANDOM DELAY/DURATION (SCALE 0.8↔1, OPACITY 0.1↔0.8).
- OFFSET-SHADOW BUTTONS/PANELS: ON HOVER, TRANSLATE +2PX/+2PX AND DROP THE SHADOW TO ZERO (THE "PRESS" EFFECT). FEATURE CARDS SWAP BORDER TO GREEN OR GAIN THE OFFSET SHADOW.
- ENVELOPE: FLAP OPENS (ROTATEX 0→180°, TRANSFORM-ORIGIN TOP) THEN THE POSTCARD ANIMATES UP AND SCALES IN WITH A SPRINGY EASE; ALSO RE-TRIGGERS ON FORM SUBMIT.
- RESPECT `prefers-reduced-motion`: DISABLE TYPEWRITER, COUNT-UP, BLINKING, AND TRANSFORM REVEALS (SHOW FINAL STATE).
