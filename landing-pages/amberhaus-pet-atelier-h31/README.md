# Amberhaus

A multi-section landing page for **Amberhaus**, a fictional high-end pet boutique and care atelier, built in a "Warm Amber Boutique" design language — a quiet, editorial look that reads like a printed lifestyle monograph about pampered pets rather than a generic pet-store site. Everything sits on crisp warm-white paper punctuated by a single saturated amber-gold accent (`#D97706`) and a butter-cream tint, with generous rounded radii and hairline borders.

Type mixes a chunky Bricolage Grotesque display, Inter body, and a Playfair Display italic flourish. Signature touches include a custom amber paw-print cursor page-wide and a faint texture overlay. Key sections: a triptych hero with auto-crossfading image sliders and parallax center text, a horizontal-scroll story track (300vh sticky), and vertical marquee testimonials with alternating up/down columns. Interactions cover a spring-eased floating menu, per-letter footer wordmark jumps, and scroll-reveal fades.

## Run

This is a static project — open `index.html` in a browser, or serve the folder:

```sh
python3 -m http.server 8000
```

See `prompt.md` for the full build spec; `demo.mp4` shows it in motion.
