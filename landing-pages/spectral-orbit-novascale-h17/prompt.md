## MOTION / ANIMATION / INTERACTION SPEC

- ORBS: CONTINUOUS LINEAR `ROTATE` LOOPS (PRIMARY 18S, SECONDARY 22S REVERSE, FEATURE-CARD ORBS 10–15S).
- MARQUEES: DUPLICATE CONTENT IN JS, TRANSLATE-X FROM 0 TO -50% INFINITELY (WORDS ~25S, LOGOS ~30S); TESTIMONIALS ~50S; PAUSE ON HOVER.
- SCROLL REVEAL: INTERSECTION-OBSERVER FADE-UP (OPACITY 0→1, TRANSLATEY 20PX→0, CUBIC-BEZIER(0.2,0.8,0.2,1), ~0.8S) WITH OPTIONAL PER-ELEMENT STAGGER DELAYS.
- CTA BUTTONS: HOVER LIFT (`-translate-y` 2PX) + CORAL GLOW SHADOW.
- FAQ: SINGLE-OPEN ACCORDION, ANIMATED MAX-HEIGHT, ICON ROTATION.
- STAT TILES & PROCESS CARDS: HOVER LIFT + GLOW INTENSIFY.
- MOBILE NAV: SLIDE-DOWN FULL-SCREEN OVERLAY, ANIMATED HAMBURGER-TO-X.
- RESPECT `PREFERS-REDUCED-MOTION` BY EASING DOWN OR DISABLING CONTINUOUS SPINS.
