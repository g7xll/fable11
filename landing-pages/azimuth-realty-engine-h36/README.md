# Azimuth

A multi-section landing page for **Azimuth**, a fictional real-estate marketing and growth agency, built in a "High-Altitude Precision" design language — a midnight mission-control deck crossed with a luxury architecture prospectus. A deep navy void (`#0E1726`), a single charged electric-cobalt accent (`#2F6BFF`), oversized editorial display type, and crisp monospace technical labels create an engineered, expensive, confident feel.

Type spans four families: Bricolage Grotesque display, Inter Tight body, JetBrains Mono labels, and a Playfair Display italic accent. Highlights include a floating pill navbar with a wide mega menu, a 38/62 split hero with a grayscale-to-color media panel, an ecosystem bento grid, a horizontally scrollable portfolio of property cards, a cobalt-section "method" bento, and a stacked-card final CTA whose layers fan out on scroll. Motion uses two flavors of IntersectionObserver reveal, mega-menu transitions, native smooth horizontal scroll, and a custom cobalt scrollbar, respecting `prefers-reduced-motion`.

## Run

This is a static project — open `index.html` in a browser, or serve the folder:

```sh
python3 -m http.server 8000
```

See `prompt.md` for the full build spec; `demo.mp4` shows it in motion.
