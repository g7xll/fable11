## MOTION & INTERACTION SPEC

- SCROLL-REVEAL: ELEMENTS FADE + RISE (OR SLIDE LEFT/RIGHT FOR JOURNEY CARDS)
  VIA INTERSECTIONOBSERVER WITH A SOFT CUBIC-BEZIER EASE; REVEAL ONCE.
- HERO ENTRANCE: HEADLINE CLUSTER SLIDES DOWN/IN ON LOAD WITH STAGGER.
- DRIFTING BLOBS: TWO BLURRED ORGANIC SHAPES ANIMATE POSITION/SCALE SLOWLY
  (~18S) ON A LOOP BEHIND THE HERO.
- MARQUEES: HERO IMAGE CARDS + MOSS TICKER + ASSURANCE STRIP USE CSS
  TRANSLATE LOOPS; HERO MARQUEE PAUSES ON HOVER.
- HEADER: CONDENSES PADDING + GAINS FROSTED BG/BORDER AFTER 20PX SCROLL.
- PACKAGES + FAQ + MOBILE MENU + FORM: ALL VANILLA JS, NO FRAMEWORK.
- RESPECT `PREFERS-REDUCED-MOTION`: DISABLE BLOB DRIFT, MARQUEE, AND REVEAL
  TRANSFORMS.

## RESPONSIVE BEHAVIOR

- MOBILE-FIRST. NAV COLLAPSES TO A FULL-SCREEN DRAWER. ALL MULTI-COLUMN GRIDS
  COLLAPSE TO ONE COLUMN. HERO TYPE SCALES WITH `CLAMP()`. STICKY JOURNEY
  CARDS REMAIN READABLE STACKED ON SMALL SCREENS. TOUCH TARGETS ≥ 44PX.
