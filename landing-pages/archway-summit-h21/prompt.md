# ARCHWAY SUMMIT — "GALLERY ARCH" TECH-CONFERENCE LANDING PAGE

## NAMED AESTHETIC IDENTITY

BUILD A FULL, MULTI-SECTION MARKETING LANDING PAGE FOR A FICTIONAL DEVELOPER CONFERENCE NAMED **ARCHWAY SUMMIT** ("THE FUTURE OF CODE, UNDER ONE ARCH"). THE NAMED DESIGN LANGUAGE IS **"GALLERY ARCH"**: A NEAR-BLACK EDITORIAL GALLERY WHERE EACH HEADLINE SPEAKER IS FRAMED INSIDE A TALL, PASTEL, MUSEUM-STYLE ARCH — LIKE PORTRAITS HUNG IN A MODERN ART HALL. THE MOOD IS CONFIDENT, CURATED, AND EXPENSIVE: A PITCH-BLACK ROOM, A FEW SATURATED PASTEL BLOCKS, OVERSIZED UPPERCASE DISPLAY TYPE, AND CRISP MONOSPACE TIMESTAMPS. AVOID GENERIC-AI-SLOP: NO PURPLE GRADIENTS, NO GLASSMORPHISM, NO STOCK BLUR. CONTRAST COMES FROM HARD BLACK NEGATIVE SPACE AGAINST CANDY-PASTEL ARCH CARDS.

## MOTION / ANIMATION / INTERACTION SPEC

- SCROLL-REVEAL: ELEMENTS START AT OPACITY 0 / `translateY(20px)` AND ANIMATE TO VISIBLE VIA AN INTERSECTIONOBSERVER (THRESHOLD ~0.05), TRANSITION ~0.8S EASE-OUT, EACH OBSERVED ONCE.
- ARCH CARDS: `transform` + `filter` TRANSITION ~0.3S; LIFT ON HOVER; PORTRAIT GRAYSCALE→COLOR ON HOVER WITH A 0.5S PORTRAIT SLIDE.
- SCHEDULE TOGGLE: INSTANT CONTENT SWAP WITH A SHORT FADE-IN ON THE NEWLY SHOWN DAY; ACTIVE BUTTON GAINS A GLOW SHADOW.
- ARROW BUTTONS / CTAS: SMOOTH COLOR INVERSION AND ICON ROTATION ON HOVER; PILL CTAS SCALE TO 1.05 ON HOVER.
- CUSTOM THIN DARK SCROLLBAR. SMOOTH-SCROLL ANCHOR NAVIGATION.
