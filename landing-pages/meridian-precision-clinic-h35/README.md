# Meridian — Precision-Medicine Clinic

A single-page landing site for Meridian, a fictional bespoke precision-medicine clinic for longevity and executive health. The named aesthetic is "Architectural Clinical Editorial": the calm restraint of a luxury architecture monograph fused with the precision of a medical journal — flat, matte, sharp-cornered, hairline-divided, in Meridian deep navy and warm bone with whisper-quiet brass.

Sections include a transparent-to-solid fixed header, a split full-viewport hero whose navy panel slides in on load, a credentials marquee, a split philosophy editorial, a hairline discipline grid, a numbered method process, a deep-navy contact form, and footer. Vanilla HTML/CSS (custom properties) + JS, with on-load hero cascade, IntersectionObserver scroll reveals with directional variants, hover inversions, and `prefers-reduced-motion` support. Self-contained and offline-runnable.

## Run

This is a static project — open `index.html` in a browser, or serve the folder:

```sh
python3 -m http.server 8000
```

See `prompt.md` for the full build spec; `demo.mp4` shows it in motion.
