# ASTRAL VEIL — COSMIC NOCTURNE SAAS LANDING PAGE

## NAMED AESTHETIC IDENTITY

BUILD A FULL, MULTI-SECTION MARKETING LANDING PAGE FOR A FICTIONAL MODERN SOFTWARE
PLATFORM CALLED **"NOVELA"**, EXPRESSED IN A DESIGN LANGUAGE NAMED **"ASTRAL VEIL"**:
A DEEP, NEAR-BLACK COSMIC NOCTURNE WHERE SOFT VIOLET AND COBALT NEBULA GLOWS BLEED
THROUGH A VEIL OF FROSTED GLASS. THE MOOD IS QUIET, PREMIUM, AND SLIGHTLY CELESTIAL —
LIKE A HIGH-END OBSERVATORY DASHBOARD. EVERYTHING FLOATS; NOTHING SHOUTS. EDITORIAL
SERIF HEADLINES SIT INSIDE A CALM ENGINEERING GRID, PUNCTUATED BY GLASSMORPHIC CARDS,
A STARFIELD, AND DRIFTING UI FRAGMENTS.

## DESIGN LANGUAGE & MOOD

- DARK-FIRST, CINEMATIC, AND RESTRAINED. THE CANVAS IS ALMOST BLACK SO THAT COLORED
  NEBULA GLOWS AND THIN HAIRLINE BORDERS DO ALL THE WORK.
- LARGE NEGATIVE SPACE; GENEROUS VERTICAL RHYTHM (SECTIONS ~6–7REM PADDING).
- CONTRAST COMES FROM A SERIF/SANS PAIRING AND FROM A SINGLE BRIGHT-WHITE PILL CTA
  AGAINST THE DARK.
- TEXTURE: A FAINT STARFIELD, A SUBTLE GRID OVERLAY, AND TWO OVERSIZED RADIAL GLOWS
  (VIOLET TOP-CENTER, COBALT BOTTOM-RIGHT) THAT NEVER FULLY RESOLVE — A "VEIL".

## EXACT COLOR PALETTE

- CANVAS / INK: `#07070B` (NEAR-BLACK), DEEPER PANEL `#0B0B12`.
- SURFACE / GLASS: `rgba(255,255,255,0.04)` FILLS WITH `rgba(255,255,255,0.08)`
  HAIRLINE BORDERS.
- TEXT: PRIMARY `#F4F4F7`, SECONDARY `#A1A1AD`, MUTED `#6C6C78`.
- ACCENT VIOLET: `#A78BFA` (GLOW `#7C3AED`).
- ACCENT COBALT: `#60A5FA` (GLOW `#2563EB`).
- ACCENT AURORA: `#34D399` (LIVE/STATUS DOTS), WARM SPARK `#FBBF24` (RATING STARS).
- GRADIENT HEADLINE TEXT: VERTICAL `#FFFFFF → #FFFFFF → #6C6C78` CLIPPED TO TEXT.

## TYPOGRAPHY

- SERIF DISPLAY: **PLAYFAIR DISPLAY** (WEIGHTS 400/500/600, PLUS ITALIC 400) FOR ALL
  HEADINGS, LOGO WORDMARK, AND STAT NUMBERS. USE ITALIC FOR THE SECOND LINE OF THE
  HERO HEADLINE ("THAT SEES THE WHOLE SKY") AS AN EDITORIAL FLOURISH.
- SANS UI: **INTER** (WEIGHTS 300/400/500/600) FOR BODY, NAV, BUTTONS, LABELS.
- SECTION EYEBROWS: INTER, 11PX, UPPERCASE, LETTER-SPACING ~0.22EM, MUTED COLOR.
- HERO HEADLINE SCALE: CLAMP FROM ~3REM (MOBILE) TO ~7REM (DESKTOP), LINE-HEIGHT 1.05.
- VENDOR BOTH FONTS LOCALLY (SELF-HOSTED WOFF2 + @font-face); NO CDN AT RUNTIME.

## HERO COMPOSITION DETAIL

- LAYERED BACKGROUND (BACK→FRONT): SOLID NEAR-BLACK → A CSS STARFIELD (MANY TINY
  RADIAL-GRADIENT DOTS WITH VARIED OPACITY, GENTLY TWINKLING) → A FAINT 1PX GRID
  (REPEATING LINEAR GRADIENTS, MASK-FADED TOWARD EDGES) → TWO LARGE BLURRED RADIAL
  GLOWS (VIOLET TOP-CENTER, COBALT BOTTOM-RIGHT) → CONTENT.
- CONTENT IS CENTER-ALIGNED, MAX-WIDTH ~64REM, WITH STAGGERED ENTRANCE ANIMATIONS.

## MOTION / ANIMATION / INTERACTION SPEC

- **ENTRANCE**: ON LOAD, HERO ELEMENTS RISE + FADE IN A STAGGERED SEQUENCE
  (PILL → HEADLINE → SUBHEAD → CTAS → CARDS), EACH ~0.7S EASE-OUT, ~90MS APART.
- **SCROLL REVEAL**: AN INTERSECTIONOBSERVER FADES + SLIDES UP EACH SECTION BLOCK
  (OPACITY 0→1, TRANSLATEY 24PX→0) ONCE 12% VISIBLE; REVEAL ONCE, THEN UNOBSERVE.
- **FLOATING**: HERO UI FRAGMENTS AND INTEGRATION CAPSULES USE INFINITE
  `float` KEYFRAMES (TRANSLATEY ±10–20PX) WITH STAGGERED DELAYS/DURATIONS.
- **STARFIELD TWINKLE**: SUBTLE OPACITY OSCILLATION ON THE STAR LAYER.
- **NAVBAR**: SHRINK/INTENSIFY BLUR + BORDER AFTER ~24PX SCROLL.
- **HOVER**: CARD BORDERS BRIGHTEN, GRADIENT-BORDER CARDS INTENSIFY, BUTTONS SHIFT
  SHADE, PORTRAIT DE-SATURATES TO COLOR, GLYPHS ROTATE SLIGHTLY.
- **MARQUEE**: CONTINUOUS LINEAR TRANSLATE LOOP, PAUSE-FREE, EDGE-MASKED.
- RESPECT `prefers-reduced-motion`: DISABLE FLOAT/MARQUEE/TWINKLE AND REVEAL
  INSTANTLY.

## RESPONSIVE BEHAVIOR

- DESKTOP (≥1280PX): FULL BENTO ASYMMETRY, EDGE FLOATING FRAGMENTS VISIBLE,
  2-COLUMN SPLITS, 4-COLUMN FOOTER.
- TABLET (768–1279PX): GRIDS COLLAPSE TO 2 COLUMNS, FLOATING FRAGMENTS HIDDEN,
  HEADLINE SCALES DOWN.
- MOBILE (<768PX): SINGLE COLUMN THROUGHOUT, HAMBURGER NAV WITH GLASS DROPDOWN,
  STACKED CTAS, REDUCED GLOW SIZES, COMFORTABLE TAP TARGETS.
