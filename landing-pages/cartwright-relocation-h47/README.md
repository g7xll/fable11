# Cartwright & Co

A multi-section landing page for **Cartwright & Co**, a premium craft-led relocation and logistics company, built in a "Warm Artisan Logistics" aesthetic — a soft, editorial take that feels like a boutique hospitality brand rather than a freight warehouse. Terracotta warmth (`#D9603B`), cream paper, generous whitespace, rounded geometry, and one sharp serif accent word per headline say "your belongings are in careful hands."

Type pairs Inter body with DM Sans display and a Fraunces italic serif accent. Sections include a sticky glass navbar with a services mega-menu, a two-column hero with opposing vertical image tickers, a partner marquee, soft service cards, a dark stats band with count-up figures, a 3-step process timeline that fills on scroll, an auto-cycling stacked testimonial deck, and a split contact/quote panel with an inline success state. Buttons use a terracotta wipe-fill hover; counters and timeline animate via IntersectionObserver.

## Run

This is a static project — open `index.html` in a browser, or serve the folder:

```sh
python3 -m http.server 8000
```

See `prompt.md` for the full build spec; `demo.mp4` shows it in motion.
