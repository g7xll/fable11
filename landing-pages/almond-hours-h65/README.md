# Almond Hours

A landing page for **Almond Hours**, a fictional luxury senior-living and dignified-care brand, built in a "Warm Clinical Editorial" design language — the quiet restraint of a printed lifestyle monograph fused with the trust of a bespoke private clinic. Everything sits on crisp near-white paper anchored by almond-ink neutrals, warmed by a butter-cream tint, with generous rounded radii and hairline borders.

Type pairs Plus Jakarta Sans display headings with Inter body and Inter Tight uppercase labels, plus an Instrument Serif italic flourish. Signature interactions include a sticky mega-menu pill nav with vertical flip hovers, an infinite image marquee, a horizontal programs accordion that morphs width on hover, a single-open FAQ with a morphing plus/minus icon, and a cursor-reveal wordmark whose radial mask follows a crosshair cursor to reveal color-cycling text. Motion is driven by IntersectionObserver reveals and a left-to-right text wipe.

## Run

This is a static project — open `index.html` in a browser, or serve the folder:

```sh
python3 -m http.server 8000
```

See `prompt.md` for the full build spec; `demo.mp4` shows it in motion.
