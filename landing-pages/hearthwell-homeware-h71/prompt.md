## HERO COMPOSITION DETAILS

- HERO TOP PADDING MUST CLEAR THE FIXED NAV + ANNOUNCEMENT BAR (~180PX TOP).
- HEADLINE: `clamp`-SCALED FRAUNCES, ~48PX MOBILE → ~88PX DESKTOP, LEADING TIGHT, TRACKING SLIGHTLY NEGATIVE.
- THE HERO IMAGE FRAME USES `object-cover` SLIDES; ACTIVE SLIDE `opacity:1`, OTHERS `opacity:0`, 1000MS EASE.
- A FAINT GRAIN/NOISE OR VIGNETTE IS OPTIONAL BUT KEEP IT EXTREMELY SUBTLE.

## MOTION / ANIMATION / INTERACTION SPEC

- **SCROLL REVEAL**: ELEMENTS WITH A `reveal` CLASS START AT `opacity:0; translateY(28px)` AND ANIMATE TO NEUTRAL VIA AN `IntersectionObserver` WHEN ~12% VISIBLE, USING `cubic-bezier(0.16,1,0.3,1)` OVER ~0.8S. SUPPORT STAGGERED `transition-delay` ON GROUPED ITEMS (E.G. HERO STACK, GRID CARDS).
- **HERO SLIDER**: AUTO CROSS-FADE EVERY 4S; DOTS REFLECT + SET ACTIVE SLIDE; PAUSE WHILE THE USER HOVERS THE FRAME.
- **EDITORIAL SPLIT SLIDER**: AUTO-ADVANCE EVERY ~5S WITH FADE+SLIDE; DOTS CONTROL.
- **MARQUEE**: CSS `@keyframes` INFINITE TRANSLATE, DUPLICATED CONTENT FOR SEAMLESS LOOP, `animation-play-state: paused` ON HOVER.
- **NAV SCROLL STATE**: ADD A `scrolled` CLASS PAST ~40PX THAT TIGHTENS PADDING + DEEPENS SHADOW.
- **HOVER MICRO-INTERACTIONS**: IMAGE ZOOM ON CARDS, BUTTON `translateY(-2px)` LIFT, ARROW NUDGE ON "VIEW ALL", COLOR SHIFT ON PRODUCT TITLES.
- RESPECT `prefers-reduced-motion`: DISABLE AUTOPLAY + REVEAL TRANSFORMS, SHOWING CONTENT STATICALLY.

## RESPONSIVE BEHAVIOR

- MOBILE-FIRST. NAV CENTER LINKS HIDE UNDER `md`; A HAMBURGER REVEALS A ROUNDED DROPDOWN.
- COLLECTIONS: 1 COL → 2 COL (`sm`) → 4 COL (`lg`). PRODUCTS: 2 COL → 3 COL (`md`) → 4 COL (`lg`). EDITORIAL SPLIT STACKS TO ONE COLUMN ON MOBILE (TEXT ABOVE IMAGE).
- HERO OVERLAY "FEATURED ITEM" CARD AND SLIDE-DOTS ARE DESKTOP-ONLY.
- ALL TYPE USES `clamp()` SO IT SCALES FLUIDLY; GENEROUS SECTION PADDING THAT TIGHTENS ON SMALL SCREENS.
