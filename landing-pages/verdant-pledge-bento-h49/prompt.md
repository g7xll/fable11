## MOTION / ANIMATION / INTERACTION SPEC

- SCROLL-REVEAL: EVERY MAJOR BLOCK STARTS AT OPACITY 0, TRANSLATED DOWN ~20PX, AND FADES/RISES INTO PLACE VIA AN INTERSECTIONOBSERVER (0.8S EASE-OUT, STAGGERED PER CARD WITHIN A GROUP).
- HERO GLASS OVERLAYS: GENTLE FADE-IN-UP ON LOAD; THE AMBER MAP DOT PULSES CONTINUOUSLY.
- STAT NUMBERS COUNT UP FROM 0 TO TARGET (~1.6S, EASE-OUT) ONCE THE IMPACT BAND IS VISIBLE.
- HOVER: TABLE ROWS REVEAL A ROTATED IMAGE PREVIEW + TRAILING ARROW; PROBLEM IMAGES SCALE 1.1; ARROW BADGES INVERT; THE LIME WORDMARK ARROW SLIDES RIGHT; BUTTONS LIFT WITH SHADOW.
- FAQ PLUS ICONS ROTATE 45° AND GO LIME ON OPEN.
- MOBILE MENU SLIDES/FADES OPEN, SWAPPING HAMBURGER ↔ CLOSE ICON.
- RESPECT `PREFERS-REDUCED-MOTION`: DISABLE TRANSFORMS/COUNTERS, SHOW FINAL STATE.
