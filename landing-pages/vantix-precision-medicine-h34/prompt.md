BUILD A SINGLE, SELF-CONTAINED, FULLY RESPONSIVE MARKETING LANDING PAGE FOR A
NEXT-GENERATION PRECISION-MEDICINE INSTITUTE CALLED **VANTIX**. THE PAGE MUST
FEEL LIKE A HIGH-TRUST CLINICAL-INTELLIGENCE LAB — PRECISE, DATA-DRIVEN,
SURGICAL, AND QUIETLY LUXURIOUS — NOT GENERIC HOSPITAL OR SAAS SLOP. SHIP IT AS
PLAIN STATIC FILES (`INDEX.HTML` + `STYLES.CSS` + `MAIN.JS`) WITH ALL ASSETS
(FONTS, IMAGES) VENDORED LOCALLY SO IT RUNS FULLY OFFLINE.

## MOTION / ANIMATION / INTERACTION SPEC

- **SCROLL REVEALS**: VIA `INTERSECTIONOBSERVER` — ELEMENTS RISE 30PX AND FADE
  IN WITH `CUBIC-BEZIER(0.16, 1, 0.3, 1)` OVER ~1S, WITH STAGGERED PER-ELEMENT
  DELAYS DRIVEN BY DATA ATTRIBUTES. ALSO SUPPORT A "SCALE" VARIANT (0.95 → 1)
  FOR THE SPECIALIST CARDS.
- **HERO METRICS COUNT-UP**: NUMERALS ANIMATE FROM 0 TO THEIR TARGET ON LOAD
  (RESPECTING DECIMALS / SUFFIXES LIKE `%`, `M`, AND TEXT LIKE `TOP 1%`).
- **GRAYSCALE → COLOR**: ALL EDITORIAL PHOTOS START `GRAYSCALE(100%)` AND
  TRANSITION TO FULL COLOR ON HOVER OVER ~0.7S.
- **HOVER MICRO-INTERACTIONS**: CTA GLOW, ARROW TRANSLATE/ROTATE, CARD LIFT WITH
  ACCENT BORDER, SPECIALIST PORTRAIT ZOOM, NAV-LINK COLOR SHIFT.
- **RAPID-RESPONSE RING**: A GENTLE INFINITE PULSE ANIMATION ON THE STRIP ICON.
- **RESPECT `PREFERS-REDUCED-MOTION`**: DISABLE COUNT-UP AND REVEAL TRANSFORMS,
  RENDERING THE FINAL STATE IMMEDIATELY.
