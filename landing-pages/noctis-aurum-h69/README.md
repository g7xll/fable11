# Noctis Aurum — Fine Jewelry Maison

A multi-section e-commerce landing page for Noctis Aurum, a fictional fine-jewelry maison (atelier de luxe, est. 1989). The named aesthetic is "Obsidian & Champagne" — a deep, nocturnal, old-money luxury language where champagne gold glints against ink-dark obsidian and cool pearl, with serif headlines the size of doorways and generous negative space.

Sections run from an announcement banner and sticky blurred header through a split hero with a 3-image cross-fade slideshow and floating "current focus" card, a circular category rail, a gold marquee, an 8-product grid, a dark heritage feature, a men's 3-card strip, a bespoke atelier block, and footer. Vanilla HTML/CSS (custom properties) + JS for the slideshow, scroll reveals, marquee pause, mobile drawer, and a quick-add simulation. Cormorant Garamond + Inter vendored locally; self-contained and offline-runnable.

## Run

This is a static project — open `index.html` in a browser, or serve the folder:

```sh
python3 -m http.server 8000
```

See `prompt.md` for the full build spec; `demo.mp4` shows it in motion.
