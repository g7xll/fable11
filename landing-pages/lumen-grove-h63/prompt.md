# LUMEN GROVE — BOTANICAL BATHHOUSE & FOREST-SPA LANDING PAGE

## NAMED AESTHETIC IDENTITY

BUILD A FULL, MULTI-SECTION, FULLY SELF-CONTAINED MARKETING LANDING PAGE FOR **"LUMEN GROVE"**, A FICTIONAL BOTANICAL BATHHOUSE AND FOREST-SPA STUDIO. THE NAMED AESTHETIC IS **"VERDANT APOTHECARY"** — A WARM, ORGANIC, EDITORIAL WELLNESS LANGUAGE THAT FEELS LIKE A SUN-WARMED CONSERVATORY CROSSED WITH A MODERN HERBAL APOTHECARY. THE MOOD IS GROUNDED, LUSH, EXPENSIVE, AND QUIET — DEEP FOREST GREEN INK ON WARM BONE PAPER, PUNCTUATED BY A SINGLE LIVING CHARTREUSE ACCENT. THE SIGNATURE GESTURE IS A LARGE, ROUNDED "PANEL-WITHIN-A-PANEL" HERO CARD THAT FLOATS ON THE CANVAS WITH GENEROUS INNER PADDING, SPLIT BETWEEN TYPE AND A FULL-BLEED PHOTOGRAPH. STRICTLY AVOID GENERIC SAAS GRADIENTS, GLASSMORPHISM BLUR CARDS, NEON, EMOJI, AND COLD BLUE-GREY. EVERY CORNER IS SOFTLY ROUNDED, EVERY SURFACE IS MATTE, EVERYTHING BREATHES.

## COLOR SYSTEM

USE A WARM, EARTHY, BOTANICAL PALETTE — NO PURE BLACK, NO COLD GREY:

- BASE / CANVAS: WARM BONE `#EDE7DA`
- SURFACE / CARD FILL: SOFT IVORY `#F7F3EA`
- FOREST INK / PRIMARY: DEEP PINE `#1E2A20`
- MID FOREST (SECONDARY PANELS, FOOTER): `#28392C`
- TEXT SECONDARY: MUTED OLIVE-GREY `#4A5247`
- TEXT MUTED: `#8B8A7E`
- ACCENT / LIVING CHARTREUSE (BUTTONS, DOTS, HIGHLIGHTS): `#C6F03B`
- ACCENT DEEP (HOVER): `#B0DB1F`
- LIGHT TEXT ON DARK: `#F2EFE2`
- HAIRLINE BORDERS: FOREST INK AT 8–12% OPACITY

PANELS ARE FLAT MATTE FILLS WITH LARGE BORDER-RADIUS (16–28PX) AND NO HARD DROP SHADOWS — AT MOST A VERY SOFT, LOW-OPACITY AMBIENT SHADOW. PHOTOGRAPHS SIT INSIDE ROUNDED MASKS, SLIGHTLY WARM-GRADED, WITH A SUBTLE DARK GRADIENT SCRIM WHERE LIGHT TEXT OVERLAPS THEM.

## MOTION / ANIMATION / INTERACTION SPEC

- **REVEAL ON SCROLL**: ELEMENTS WITH A REVEAL CLASS START AT `opacity:0, translateY(24px)` AND EASE TO REST VIA AN INTERSECTION OBSERVER ON A SMOOTH CUBIC-BEZIER (`0.16, 1, 0.3, 1`, ~0.8S), WITH STAGGERED TRANSITION-DELAYS BETWEEN SIBLINGS.
- **CUSTOM CURSOR**: A SMALL CHARTREUSE DOT THAT TRAILS THE POINTER ON DESKTOP (HIDDEN ON TOUCH), SUBTLY SCALING UP OVER INTERACTIVE ELEMENTS.
- **MARQUEE**: PURE-CSS INFINITE TRANSLATE, PAUSES ON HOVER.
- **ACCORDION**: CLICKING A RITUAL EXPANDS ITS DESCRIPTION (ANIMATED MAX-HEIGHT) AND CROSS-FADES THE PAIRED IMAGE; ONLY ONE OPEN AT A TIME.
- **HOVERS**: NAV PILLS FADE IN A SOFT BACKGROUND; CTA ARROWS NUDGE; AVATARS / TILES LIFT GENTLY; CHARTREUSE BUTTONS DEEPEN ON HOVER.
- **SMOOTH ANCHOR SCROLL** FOR IN-PAGE NAV LINKS.
