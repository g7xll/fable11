## MOTION / INTERACTION SPEC

- ON LOAD: HEADER FADES IN; HERO IMAGE KEN-BURNS ZOOMS FROM SCALE 1.1 → 1.0; THE INFO CARD SLIDES UP WITH A DELAY; THE GREEN HEADLINE HIGHLIGHT WIPES IN VIA CLIP-PATH (LEFT-TO-RIGHT); THE DIVIDER GROWS FROM WIDTH 0; PARAGRAPH AND BUTTON FADE IN ON STAGGERED DELAYS.
- THE HERO CAPABILITY MARQUEE LOOPS INFINITELY VIA A DUPLICATED TRACK TRANSLATED -50%.
- SCROLL REVEALS: AN INTERSECTIONOBSERVER ADDS A "VISIBLE" CLASS THAT FADES + TRANSLATES ELEMENTS UP (WITH OPTIONAL ZOOM/LEFT VARIANTS) USING A `cubic-bezier(0.16,1,0.3,1)` EASE; PER-ELEMENT DELAY VIA A DATA ATTRIBUTE FOR STAGGERED GRIDS.
- STAT NUMBERS COUNT UP WITH AN EASE-OUT-CUBIC OVER ~2.5S WHEN THEIR CONTAINER ENTERS THE VIEWPORT.
- ALL HOVERS ARE 300–700MS COLOR / SCALE / TRANSFORM TRANSITIONS. ZERO BLUR ON SHADOWS WHERE POSSIBLE; SHADOWS ARE SOFT BUT RESTRAINED.
- RESPECT `prefers-reduced-motion`: DISABLE KEN-BURNS, MARQUEE, AND TRANSFORM REVEALS, KEEPING CONTENT VISIBLE.
