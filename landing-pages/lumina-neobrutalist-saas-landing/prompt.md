# LUMINA — NEO-BRUTALIST SAAS LANDING PAGE

> BUILD A SINGLE-PAGE, SELF-CONTAINED LANDING PAGE FOR A FICTIONAL CREATOR-OS SAAS CALLED **LUMINA**, REPRODUCED SAME-TO-SAME (PIXEL-FAITHFUL) FROM THE REFERENCE DRAFT. SHIP IT AS PLAIN STATIC HTML + CSS + A SMALL AMOUNT OF VANILLA JS, RUNNABLE OFFLINE WITH ALL FONTS VENDORED LOCALLY. NO BUILD STEP REQUIRED.

## SUMMARY

A VIBRANT NEO-BRUTALIST SAAS LANDING PAGE UTILIZING A DOMINANT `#FFE17C` YELLOW, DEEP CHARCOAL `#171E19`, AND SAGE `#B7C6C2` ACCENTS. IT FEATURES HIGH-CONTRAST BLACK BORDERS, HARD OFFSET SHADOWS, AND BOLD GEOMETRIC TYPOGRAPHY TO CONVEY PROFESSIONAL CONFIDENCE WITH A PLAYFUL EDGE.

## STYLE

THE STYLE IS A MODERN TAKE ON NEO-BRUTALISM. TYPOGRAPHY PAIRS THE HIGH-IMPACT **CABINET GROTESK** (WEIGHTS 400–900) FOR HEADINGS WITH **SATOSHI** (WEIGHTS 400–900) FOR READABILITY. THE PALETTE USES `#FFE17C` AS A PRIMARY BACKGROUND, BALANCED BY `#171E19` CHARCOAL AND `#B7C6C2` SAGE. VISUAL ELEMENTS ARE DEFINED BY 2PX SOLID BLACK BORDERS AND 4PX–8PX HARD SHADOWS (NO BLUR). MICRO-INTERACTIONS INVOLVE `TRANSLATE` EFFECTS ON HOVER WHERE BUTTONS MOVE 4PX TO 'FILL' THEIR SHADOW SPACE.

### SPEC

CREATE A DESIGN BASED ON NEO-BRUTALIST PRINCIPLES.

- **COLORS:** PRIMARY `#FFE17C` (YELLOW), BACKGROUND `#171E19` (CHARCOAL), ACCENT `#B7C6C2` (SAGE), DARK GRAY `#272727`, UI `#FFFFFF` (WHITE), TEXT `#000000` (BLACK), STAR YELLOW `#FFBC2E`.
- **TYPOGRAPHY:** HEADINGS IN CABINET GROTESK (EXTRABOLD/BLACK 800–900, `TRACKING-TIGHTER`); HERO H1 RENDERS AT ~96PX, WEIGHT 900, LETTER-SPACING ≈ `-4.8PX`. BODY IN SATOSHI (MEDIUM, 500).
- **UI ELEMENTS:** USE 2PX SOLID BLACK BORDERS ON ALL CARDS, BUTTONS, AND SECTIONS.
- **HARD SHADOWS:** `BOX-SHADOW: 4PX 4PX 0PX 0PX #000000` FOR STANDARD ELEMENTS AND `8PX 8PX 0PX 0PX #000000` FOR LARGE CONTAINERS; THE BROWSER MOCKUP USES `12PX 12PX 0PX #000`.
- **BUTTONS:** HOVER STATE `TRANSFORM: TRANSLATE(4PX, 4PX)` AND REMOVES THE SHADOW TO SIMULATE A PHYSICAL PRESS.
- **DOT PATTERN:** INCLUDE A 32PX × 32PX RADIAL DOT PATTERN (OPACITY ~10%) ON PRIMARY YELLOW BACKGROUNDS.
