## MOTION / ANIMATION / INTERACTION SPEC

- **LETTER-BY-LETTER HEADING REVEAL:** EVERY MAJOR HEADING SPLITS INTO PER-CHARACTER SPANS THAT FADE + RISE (`TRANSLATEY(0.25EM)` → 0) WITH A 25MS STAGGER PER CHAR, TRIGGERED VIA INTERSECTIONOBSERVER.
- **SCROLL REVEAL:** GENERIC BLOCKS START AT OPACITY 0 / `TRANSLATEY(20PX)` AND EASE IN ON ENTER; METHODOLOGY COLUMNS SLIDE IN FROM ±80PX X.
- **MARQUEE:** FORM HEADER AND ANY TICKERS LOOP VIA A `-50%` TRANSLATEX LINEAR INFINITE.
- **GRID PULSE:** CTA BACKGROUND LINES SCALE X/Y SLOWLY (8S EASE-IN-OUT INFINITE) WITH STAGGERED DELAYS.
- **FLOAT LOOP:** METRIC NUMBERS DRIFT VERTICALLY A FEW PIXELS.
- **HOVER:** CARD IMAGES ZOOM, CTAS SCALE 1.05, THE CONFIRM BUTTON FILL WIPES.
- RESPECT `PREFERS-REDUCED-MOTION` BY SHORT-CIRCUITING ANIMATIONS.
