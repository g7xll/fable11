# VELLUM & VERDIGRIS — AN ARCHIVAL APOTHECARY WELLNESS LANDING PAGE

## NAMED AESTHETIC IDENTITY

BUILD A FULL, MULTI-SECTION MARKETING LANDING PAGE FOR A FICTIONAL HIGH-END
WELLNESS SANCTUARY & APOTHECARY CALLED **"VELLUM & VERDIGRIS"**. THE NAMED
AESTHETIC IS **"ARCHIVAL APOTHECARY"** — A VINTAGE, SEPIA-TONED, MUSEUM-CATALOGUE
FEEL THAT READS LIKE A LEATHER-BOUND LEDGER OF FORGOTTEN HEALING RITUALS. THE
MOOD IS QUIET, HEAVY, SLOW, AND EXPENSIVE: AGED PAPER, PRESSED BOTANICALS, INK
STAMPS, CATALOGUE CODES, AND THREE-FOOT STONE WALLS. NOTHING SHOUTS. THE PAGE
SHOULD FEEL LIKE TURNING THE PAGES OF A PRESERVED VOLUME. STRICTLY AVOID GENERIC
SAAS GRADIENTS, GLASSMORPHISM, NEON, ROUNDED PILLS, AND DROP-SHADOW CARD SOUP.
EVERY DECISION LEANS PRINT-EDITORIAL: HAIRLINE RULES, CATALOGUE NUMBERING,
TRACKED-OUT SMALL CAPS, AND OVERSIZED SERIF DISPLAY TYPE.

## COLOR SYSTEM

A WARM, LOW-CONTRAST VINTAGE-PAPER PALETTE. NO PURE BLACK, NO PURE WHITE.

- `--cream` (PRIMARY BACKGROUND / VELLUM): `#F4ECD8`
- `--paper` (SECONDARY PANEL / AGED LEAF): `#E7D9BC`
- `--ink` (PRIMARY TEXT / DARK SEPIA BROWN): `#3A2C1E`
- `--ink-soft` (BODY TEXT): `#5A4632`
- `--verdigris` (ACCENT — AGED COPPER GREEN): `#5C6B54`
- `--rust` (SECONDARY ACCENT — BURNT SIENNA): `#9A5B34`
- `--gilt` (HAIRLINE / FAINT GOLD): `#B9A074`

TEXT SELECTION: BACKGROUND `--ink`, COLOR `--cream`. CUSTOM SCROLLBAR: TRACK
`--cream`, THUMB `--ink`. A SUBTLE PAPER-GRAIN / FILM-NOISE OVERLAY SITS ABOVE
THE WHOLE PAGE AT LOW OPACITY (SVG FRACTAL TURBULENCE), AND ALL PHOTOGRAPHY IS
TREATED WITH A SEPIA + LOW-CONTRAST FILTER SO IT BLENDS INTO THE PAPER.

## TYPOGRAPHY

- DISPLAY / HEADINGS: **PLAYFAIR DISPLAY** (SERIF), WEIGHTS 400/700, REGULAR &
  ITALIC. USED HUGE (UP TO ~8.5REM) WITH TIGHT LEADING (~0.9).
- EDITORIAL ACCENT / PULL-QUOTES: **INSTRUMENT SERIF** (SERIF), ITALIC. USED FOR
  LYRICAL SUB-LINES AND TESTIMONIALS AT LARGE SIZES.
- BODY / LABELS / CATALOGUE CODES: **COURIER PRIME** (MONOSPACE TYPEWRITER).
  ALWAYS UPPERCASE FOR LABELS, WITH WIDE LETTER-SPACING (0.3em–0.4em).

ALL FONTS MUST BE VENDORED LOCALLY AS WOFF2 (NO GOOGLE FONTS CDN AT RUNTIME).

## MOTION / ANIMATION / INTERACTION SPEC

- **HERO ENTRANCE**: TEXT ELEMENTS FADE-RISE (TRANSLATEY 20PX → 0) IN A STAGGER
  (DELAYS ~0.3S/0.5S/0.7S/0.9S/1.2S) WITH CUBIC-BEZIER(0.22,1,0.36,1). THE IMAGE
  REVEAL-OVERLAY SLIDES UP (SCALE-Y 1 → 0, ~1.5S) WHILE THE IMAGE ZOOMS OUT.
- **SCROLL REVEAL**: AN INTERSECTIONOBSERVER FADE-RISES EACH SECTION BLOCK ON
  FIRST ENTRY (THRESHOLD ~0.12, 1S EASE), WITH OPTIONAL PER-CHILD STAGGER.
- **HAIRLINE GROW**: SECTION HEADER UNDER-RULES ANIMATE WIDTH 0 → 100% WHEN
  REVEALED.
- **CUSTOM CURSOR**: A SMALL HOLLOW INK RING THAT FOLLOWS THE POINTER ON
  POINTER-FINE DEVICES AND ENLARGES OVER INTERACTIVE ELEMENTS. HIDE ON TOUCH.
- **HOVER**: RITUAL IMAGES SLOW-ZOOM (~1S), TITLES ITALICIZE, BUTTONS SHIFT
  BETWEEN INK / VERDIGRIS / RUST. RESPECT `prefers-reduced-motion` (DISABLE
  TRANSFORMS/ZOOM, KEEP CONTENT VISIBLE).

## RESPONSIVE BEHAVIOR

- MOBILE-FIRST. HERO COLLAPSES TO A SINGLE COLUMN (TEXT THEN IMAGE), HEADER
  BOOKING BUTTON HIDES BEHIND THE HAMBURGER, GRIDS COLLAPSE TO ONE COLUMN.
- DISPLAY TYPE SCALES FLUIDLY WITH `clamp()`. CUSTOM CURSOR & FLOATING
  MARGINALIA CARDS HIDE ON SMALL / TOUCH SCREENS.
- KEEP GENEROUS VERTICAL RHYTHM (SECTION PADDING SCALING ~6REM → ~10REM).
