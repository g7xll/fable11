# TERRA SIENA — HANDCRAFTED CERAMICS ATELIER LANDING PAGE

BUILD A FULL, MULTI-SECTION LANDING PAGE FOR A FICTIONAL CERAMICS STUDIO NAMED **TERRA SIENA**. THE NAMED AESTHETIC IDENTITY IS **"SUN-FIRED EARTH"** — A WARM, EDITORIAL, ARTISANAL E-COMMERCE BRAND PAGE THAT FEELS LIKE A PRINTED CRAFT CATALOG: HAND-THROWN STONEWARE, KILN SMOKE, RAW CLAY, AND MEDITERRANEAN LIGHT. THE MOOD IS TACTILE, UNHURRIED, AND EXPENSIVE — INDUSTRIAL PRECISION SOFTENED BY THE IMPERFECTION OF THE HUMAN HAND.

## DESIGN LANGUAGE & MOOD

- A LUXURY ARTISAN STOREFRONT THAT READS LIKE AN ATELIER MONOGRAPH. GENEROUS NEGATIVE SPACE, LARGE EDITORIAL SERIF DISPLAY TYPE, AND SMALL ALL-CAPS LABELS WITH WIDE LETTER-SPACING.
- EVERYTHING FEELS WARM AND EARTHEN — TERRACOTTA, OCHRE, BURNT SIENA, AND BONE-CREAM, NEVER COLD OR DIGITAL.
- TEXTURE IS A CHARACTER: SUBTLE PAPER/GRAIN BACKGROUND, SOFT DROP SHADOWS, ROUNDED "POTTERY" CORNERS, AND ARCH MOTIFS THAT ECHO KILN OPENINGS.

## TYPOGRAPHY

- DISPLAY / HEADINGS: A HIGH-CONTRAST EDITORIAL SERIF (E.G. **FRAUNCES** OR **PLAYFAIR DISPLAY**), WEIGHTS 400–700, WITH ITALIC USED FOR ACCENT WORDS AND STAT FIGURES. TIGHT TRACKING ON LARGE SIZES, LINE-HEIGHT ~0.95–1.05.
- BODY / UI: A CLEAN GROTESQUE SANS (E.G. **INTER TIGHT** OR **SATOSHI**), WEIGHTS 300–600.
- MICRO-LABELS: SANS, 11PX, BOLD, UPPERCASE, LETTER-SPACING 0.2EM, OFTEN IN TERRACOTTA.

## MOTION / ANIMATION / INTERACTION SPEC

- **WORD-BY-WORD TEXT REVEAL**: HEADINGS SPLIT INTO WORD SPANS THAT FADE + SLIDE IN FROM THE LEFT WITH A STAGGERED ~40MS DELAY WHEN SCROLLED INTO VIEW.
- **SCROLL REVEALS**: GENERIC ELEMENTS FADE + RISE 40PX VIA INTERSECTION OBSERVER (CUBIC-BEZIER 0.2,0,0.2,1, ~1.2S).
- **HERO MARQUEE & FOOTER MARQUEE**: SEAMLESS LINEAR INFINITE LOOPS; HERO IMAGE STRIP PAUSES ON HOVER.
- **DECK SPREAD**: ONE-SHOT SPREAD ANIMATION ON ENTER (0.9S EASE), PER-CARD HOVER LIFT/STRAIGHTEN/SCALE WITH OVERLAY REVEAL.
- **FLIP CARDS**: 0.8S `rotateY(180deg)` PRESERVE-3D ON HOVER, CLICK-TO-FLIP FALLBACK ON TOUCH.
- **FAQ**: ANIMATED MAX-HEIGHT + ICON ROTATION, AUTO-CLOSE SIBLINGS.
- MENU OVERLAY FADES/SCALES IN; ALL HOVER STATES USE WARM TERRACOTTA TRANSITIONS.
