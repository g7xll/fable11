## MOTION / ANIMATION / INTERACTION SPEC

- **SCROLL REVEALS**: ELEMENTS START HIDDEN (OPACITY 0, TRANSLATEY 30PX OR TRANSLATEX ±100PX FOR LEFT/RIGHT VARIANTS) AND ANIMATE TO REST VIA AN INTERSECTIONOBSERVER (THRESHOLD ~0.12), TRANSITION ~0.8–1S EASE-OUT, WITH STAGGERED `transition-delay` ACROSS GRID ITEMS.
- **STAT COUNTERS**: COUNT UP FROM 0 TO TARGET WHEN THE ABOUT SECTION ENTERS VIEW, EASING TOWARD THE TARGET, OBSERVED ONCE.
- **MARQUEE**: INFINITE LINEAR TRANSLATE OF A DUPLICATED TRACK (-50% LOOP), ~28S, PAUSING ON HOVER; SPINNING STAR GLYPHS (~12S LINEAR).
- **BUTTON HOVER**: PRIMARY INDIGO BUTTONS DARKEN TOWARD INK; THE CIRCULAR ARROW CHIP SLIDES/ROTATES; UNDERLINE LINKS WIDEN THEIR ICON GAP.
- **CARD HOVER**: SUBJECT/HERO CARDS LIFT (`translateY(-6/-8px)`) WITH DEEPER SHADOW; TILTED CARDS RETURN TO 0°; IMAGES SCALE TO ~1.05–1.10 OVER 700MS.
- **FLOATING DOODLES**: A FEW SPARKLE/STAR SVGS GENTLY FLOAT (±12PX, ~4S EASE-IN-OUT) IN THE HERO.
- RESPECT `prefers-reduced-motion`: DISABLE FLOATS/MARQUEE MOTION AND REVEAL TRANSFORMS WHEN REQUESTED.
