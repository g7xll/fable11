# Jack — 3D Creator Portfolio

Dark-themed portfolio landing page for "Jack", a 3D creator. Built with
React 18, TypeScript, Tailwind CSS, and Framer Motion (Vite toolchain,
Kanit typeface).

Sections: full-viewport hero with magnetic portrait → scroll-driven dual
marquee → about with character-by-character text reveal → white services
list → sticky-stacking project cards.

The originating prompt for this experiment is preserved in
[`prompt.txt`](./prompt.txt).

## Commands

```bash
npm install
npm run dev       # local dev server
npm run build     # type-check + production build
npm run preview   # serve the production build
```
