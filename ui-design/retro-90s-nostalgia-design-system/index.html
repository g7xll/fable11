WEBMASTER 95 is living documentation for a Retro / 90s Nostalgia design system — every token, pattern and component rendered the way it would’ve looked on a Windows 95 machine. It is deliberately anti-modern: zero border-radius, no soft shadows, and not one drop of minimalism.

Bright icon boxes with 2px-stroke glyphs and beveled frames — the building blocks of the system, one feature per tile.

The signature 4-value border-color plus inset shadows. Outset pops, inset sinks.

Garish Palette

Pure-saturation system colors. No subtle grays allowed past #808080.

Table Layouts

07 · Method

SHIP IT IN 4 STEPS

An alternating-row checklist — the classic database look.

1

Centralize the tokens

Drop the palette, bevels and type stacks into :root. One source of truth.

CORE
2

Bevel everything

Wrap controls and panels in .bevel-out / .bevel-in. Non-negotiable.

STYLE
3

Add the loud bits

Marquee, rainbow text, hit counter, pulsing badges. Embrace the cheese.

FUN
4

Honor reduced motion

Freeze the marquee, stop the rainbow, kill the pulse. Keep the bevels.
