# Velorah — Hero Landing

Cinematic single-page hero: fullscreen looping background video, liquid-glass
navigation and CTAs, Instrument Serif display type over a deep navy theme.

Built with React + Vite + TypeScript + Tailwind CSS + shadcn/ui.

## Run

```sh
npm install
npm run dev      # dev server
npm run build    # type-check + production build
npm run preview  # serve the production build
```

See `prompt.md` for the original experiment prompt.
