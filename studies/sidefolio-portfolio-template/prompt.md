## SUMMARY

SIDEFOLIO IS A CLEAN, LIGHT-THEME DEVELOPER PORTFOLIO BUILT AROUND A FIXED LEFT
SIDEBAR (AVATAR + NAME + ROLE, A NAV WITH TABLER-STYLE OUTLINE ICONS, A SOCIALS
GROUP, AND A DARK "READ RESUME" PILL BUTTON WITH AN ANIMATED SKY-GLOW HOVER) AND
A WHITE, ROUNDED CONTENT PANEL ON A LIGHT-GRAY PAGE. THE CONTENT AREA IS A
CENTERED `max-w-4xl` COLUMN. HEADINGS USE THE CAL SANS DISPLAY FONT WITH A SUBTLE
DARK-GRAY GRADIENT CLIP; BODY COPY IS INTER. THE BUILD COVERS ALL 14 PAGES OF THE
TEMPLATE: HOME, ABOUT, PROJECTS (LIST), 4 PROJECT DETAILS, ARTICLES (BLOG LIST),
4 BLOG ARTICLES, CONTACT, AND RESUME.
