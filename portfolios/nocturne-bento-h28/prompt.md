# NOCTURNE BENTO — "MIDNIGHT ATELIER" DESIGNER PORTFOLIO

## NAMED AESTHETIC IDENTITY

BUILD A SINGLE-PAGE, FORCED-DARK PERSONAL PORTFOLIO CALLED **"NOCTURNE BENTO"** FOR A FICTIONAL INDEPENDENT PRODUCT DESIGNER NAMED **MIRA VOSS**. THE NAMED AESTHETIC IS **"MIDNIGHT ATELIER"**: A NEAR-BLACK STUDIO CANVAS, A SLOWLY DRIFTING PERSPECTIVE GRID FLOOR THAT RECEDES TOWARD A HORIZON, OVERSIZED EDITORIAL SERIF DISPLAY TYPE PAIRED WITH A TIGHT GROTESQUE, AND A SINGLE ELECTRIC ACCENT (ACID-LIME) USED SPARINGLY AGAINST GREYSCALE PHOTOGRAPHY. THE MOOD IS CONFIDENT, NOCTURNAL, GALLERY-LIKE, AND PRECISE — A DESIGN STUDIO SEEN AT 2 A.M. THROUGH A GLASS WALL. THE SIGNATURE STRUCTURAL DEVICE IS A **BENTO-GRID HERO**: SEVERAL ASYMMETRIC GLASS PANELS OF DIFFERENT SIZES THAT TOGETHER COMPOSE THE ABOVE-THE-FOLD, ONE OF WHICH IS A SOLID ACCENT-COLOURED CALL-TO-ACTION TILE.

## TECH STACK

- PURE STATIC SITE: A SINGLE `INDEX.HTML`, ONE VENDORED `STYLES.CSS`, AND ONE VENDORED `MAIN.JS`. NO BUILD STEP, NO FRAMEWORK, NO CDN AT RUNTIME.
- ALL ASSETS VENDORED LOCALLY UNDER `ASSETS/` — FONTS (SELF-HOSTED WOFF2), PHOTOGRAPHY (LOCAL JPG/WEBP), AND ANY ICONS AS INLINE SVG. THE SITE MUST RUN FULLY OFFLINE.
- VANILLA JS ONLY: INTERSECTIONOBSERVER SCROLL REVEALS, A POINTER-PARALLAX HERO, A MOBILE MENU, AND A FAKE CONTACT FORM THAT VALIDATES AND SHOWS A SUCCESS STATE.

## COLOR PALETTE

- `--ink` BACKGROUND BASE: `#070707` (NEAR-BLACK).
- `--surface` PANEL FILL: `#101012` AT ~45% OPACITY OVER GLASS BLUR.
- `--line` HAIRLINE BORDERS: `RGBA(255,255,255,0.10)`.
- `--accent` ELECTRIC ACID-LIME: `#C6F24E`.
- `--accent-dim` HOVER/PRESS: `#B2E03A`.
- `--text` PRIMARY: `#F4F4F2`. `--muted` SECONDARY: `#8B8B8F`.
- TEXT ON ACCENT TILES: `#0A0A0A`.
- SELECTION: ACCENT BACKGROUND, INK TEXT.

## TYPOGRAPHY

- DISPLAY / HEADINGS: A HIGH-CONTRAST EDITORIAL SERIF (E.G. "FRAUNCES" OR "PLAYFAIR DISPLAY"), USED AT LARGE SIZES WITH TIGHT LEADING (~0.9), MIXING ROMAN AND **ITALIC** — KEY WORDS SET IN ITALIC AND TINTED ACCENT.
- BODY / UI / LABELS: A TIGHT GROTESQUE SANS (E.G. "INTER TIGHT" / "INTER"), WEIGHTS 300/400/600. UPPERCASE EYEBROWS AND NAV LINKS WITH WIDE LETTER-SPACING (~0.18EM).
- MONO ACCENTS: A MONOSPACE (E.G. "JETBRAINS MONO" / "IBM PLEX MONO") FOR STATUS PILLS, TAGS, COORDINATES, AND DATA LABELS.
- SELF-HOST ALL FONT FILES; DEFINE `@FONT-FACE` WITH `FONT-DISPLAY: SWAP` AND ROBUST SYSTEM FALLBACKS.

## BACKGROUND SYSTEM (FIXED, BEHIND EVERYTHING)

1. SOLID INK BASE LAYER.
2. AN INFINITE ANIMATED **PERSPECTIVE GRID**: A LARGE GRID OF THIN WHITE LINES (~`RGBA(255,255,255,0.16)`, 60PX CELLS) TILTED WITH `PERSPECTIVE(800PX) ROTATEX(~22DEG)` SO IT READS AS A FLOOR RECEDING TO A HORIZON, SLOWLY PANNING VIA A `BACKGROUND-POSITION` KEYFRAME LOOP (~5–8S, SEAMLESS). RESPECT `PREFERS-REDUCED-MOTION`.
3. A SOFT RADIAL VIGNETTE THAT DARKENS THE EDGES AND KEEPS THE CENTER LEGIBLE.
4. A SINGLE BLURRED ACCENT ORB BLEEDING IN FROM ONE CORNER FOR DEPTH.

## LAYOUT & SECTION BREAKDOWN

CONTENT IS CENTERED IN A `MAX-WIDTH ~1400PX` COLUMN. SECTIONS BELOW THE HERO SHARE VERTICAL HAIRLINE BORDERS (LEFT/RIGHT) SO THE WHOLE PAGE READS AS ONE RULED SHEET.

1. **HEADER (FIXED):** LOGOTYPE "MIRA V." IN SERIF AT LEFT; CENTER/RIGHT NAV (`WORK`, `STUDIO`, `CONTACT`) AS UPPERCASE TRACKED LINKS THAT TINT ACCENT ON HOVER. USE `MIX-BLEND-MODE: DIFFERENCE` SO THE HEADER STAYS LEGIBLE OVER ANY PANEL. MOBILE: A HAMBURGER OPENING A FULL-SCREEN OVERLAY MENU WITH LARGE SERIF LINKS.

2. **HERO — BENTO GRID:** A 12-COLUMN GRID OF ASYMMETRIC GLASS PANELS:
   - **NAME PANEL (LARGE, ~8 COLS):** A LIVE "ONLINE" DOT + MONO LABEL, AN "EST." BADGE, THEN THE GIANT NAME `MIRA` / *VOSS* (SECOND LINE ITALIC + ACCENT), AND A ONE-LINE POSITIONING STATEMENT SEPARATED BY A HAIRLINE.
   - **PORTRAIT PANEL (TALL, ~4 COLS):** A GREYSCALE PHOTO THAT DESATURATES → COLOUR ON HOVER AND SCALES SLIGHTLY; A BOTTOM GRADIENT WITH AN ITALIC CAPTION.
   - **QUOTE PANEL (~5 COLS):** A SHORT DESIGN APHORISM WITH ONE WORD UNDERLINED IN ACCENT, PLUS A MONO "EX:" CREDIT LINE.
   - **CTA TILE (~4 COLS):** SOLID ACCENT FILL, "AVAILABLE" PILL, A SERIF "START A PROJECT" HEADING, AND AN ARROW THAT ROTATES 45° ON HOVER; WHOLE TILE CLICKABLE WITH AN ACCENT GLOW THAT INTENSIFIES ON HOVER.
   - **STATS TILE (~3 COLS):** A BIG NUMBER (`40+` PROJECTS) AND A LOCATION (`BERLIN`) STACKED WITH A DIVIDER.
   - PANELS REVEAL WITH A STAGGERED FADE-UP ON LOAD AND RESPOND SUBTLY TO POINTER PARALLAX.

3. **SELECTED WORK:** SECTION EYEBROW + GIANT SERIF TITLE WITH A "VIEW ARCHIVE →" LINK; THEN A 2×2 BORDERED GRID OF FOUR PROJECT CARDS (IMAGE THAT ZOOMS ON HOVER, TITLE THAT TINTS ACCENT, YEAR PILL, SHORT DESCRIPTION, AND UPPERCASE TAG ROW). CELLS DIVIDED BY HAIRLINES; HOVER LIFTS THE CELL BACKGROUND.

4. **STUDIO / ABOUT:** TWO COLUMNS — LEFT: A SERIF PHILOSOPHY HEADLINE ("THE USER IS THE ONLY REALITY." STYLE) ABOVE A TALL GREYSCALE PORTRAIT WITH A MONO CAPTION; RIGHT: SHORT BIO PARAGRAPHS, AN "EXPERIENCE" LIST (COMPANY · ROLE · DATE ROWS THAT NUDGE RIGHT AND TINT THEIR BORDER ON HOVER), AND A TEXT LINK WITH AN ARROW.

5. **SERVICES:** A STICKY "SERVICES" LABEL IN THE LEFT COLUMN; RIGHT, THREE NUMBERED CARDS (`01/02/03`) EACH WITH A CIRCULAR INDEX BADGE, A TITLE, AND A DESCRIPTION.

6. **CONTACT:** A GLOWING ACCENT HALO BEHIND TWO COLUMNS — LEFT: GIANT SERIF "READY TO START THE *PROJECT?*", AVAILABILITY COPY, EMAIL, AND SOCIAL LINKS; RIGHT: A GLASS-CARD CONTACT FORM (NAME, EMAIL, MESSAGE) WITH ACCENT-BORDER FOCUS STATES, A SOLID ACCENT SUBMIT BUTTON, AND A CLIENT-SIDE VALIDATED SUCCESS STATE.

7. **FOOTER:** LOGOTYPE + STUDIO BLURB, A SITEMAP COLUMN, A CONNECT COLUMN, A HAIRLINE RULE, AND A COPYRIGHT/LOCATION ROW IN SMALL UPPERCASE MONO.

## MOTION / INTERACTION SPEC

- ON-LOAD: HERO PANELS FADE-UP AND STAGGER (CUBIC-BEZIER(0.16,1,0.3,1), ~0.8–1S).
- ON-SCROLL: `[DATA-REVEAL]` ELEMENTS START AT `OPACITY 0 / TRANSLATEY(30PX)` AND ANIMATE IN VIA INTERSECTIONOBSERVER (~0.1 THRESHOLD), ONCE.
- POINTER PARALLAX: HERO PANELS AND THE ACCENT ORB SHIFT A FEW PIXELS WITH CURSOR MOVEMENT (DISABLED ON TOUCH / REDUCED MOTION).
- HOVERS: PHOTOS GRAYSCALE→COLOUR + SCALE; CTA ARROW ROTATES 45°; EXPERIENCE ROWS NUDGE RIGHT; CARD BACKGROUNDS LIFT; LINKS TINT ACCENT.
- FORM: VALIDATE REQUIRED FIELDS + EMAIL FORMAT, THEN SWAP TO A SUCCESS MESSAGE.
- ALL MOTION GATES ON `PREFERS-REDUCED-MOTION: REDUCE`.

## RESPONSIVE BEHAVIOR

- DESKTOP (≥1024PX): FULL BENTO GRID AND MULTI-COLUMN SECTIONS AS DESCRIBED.
- TABLET (768–1023PX): BENTO COLLAPSES TO 2 COLUMNS; WORK GRID STAYS 2×2; SERVICES STACK.
