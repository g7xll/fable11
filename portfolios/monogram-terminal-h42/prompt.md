# MONOGRAM TERMINAL — A MONOCHROME EDITORIAL DEVELOPER PORTFOLIO

## NAMED AESTHETIC IDENTITY

BUILD A SINGLE-PAGE, FORCED-DARK PERSONAL PORTFOLIO CALLED **"MONOGRAM TERMINAL"**
FOR A FICTIONAL FULL-STACK / SYSTEMS DEVELOPER NAMED **REN OKADA**. THE NAMED
DESIGN LANGUAGE IS **"MONOCHROME TERMINAL EDITORIAL"** — A STRICTLY GREYSCALE,
NEAR-BLACK CANVAS THAT READS LIKE A CODE EDITOR AND A FINELY-SET PRINT MAGAZINE
FUSED TOGETHER. THE MOOD IS DISCIPLINED, CONFIDENT, WARM, AND PRECISE — PART
HACKER-TERMINAL, PART HIGH-END EDITORIAL — AND NEVER NEON, NEVER GAMER-RGB,
NEVER GENERIC-SAAS-GRADIENT. THERE IS **NO HUE ANYWHERE**: THE ENTIRE PALETTE IS
BLACK, WHITE, AND GREY, WITH WHITE ITSELF AS THE ONLY "ACCENT." TYPOGRAPHY AND
SPACING DO ALL THE TALKING.

## DESIGN LANGUAGE & MOOD

- A SINGLE NARROW (MAX ~1150PX) CENTERED COLUMN ON A VOID BACKGROUND, FRAMED BY
  HAIRLINE VERTICAL BORDERS DOWN BOTH EDGES SO THE PAGE FEELS LIKE A BOUND
  EDITORIAL SPREAD OR A TERMINAL PANE.
- DEEP MATTE BLACK EVERYWHERE; PANELS AND CARDS ARE BARELY-LIFTED NEAR-BLACK
  SURFACES SEPARATED ONLY BY 1PX TRANSLUCENT-WHITE HAIRLINES, NEVER BY DROP
  SHADOWS WITH COLOR.
- MONOSPACE SECTION LABELS FORMATTED LIKE FILE PATHS / TERMINAL BREADCRUMBS,
  E.G. `... /about me ...`, `~/work`, `cd projects/`.
- ITALIC SERIF IS USED SPARINGLY FOR EMPHASIS WORDS INSIDE OTHERWISE SANS/MONO
  PARAGRAPHS, GIVING AN EDITORIAL "PULL-QUOTE" FEEL.
- SUBTLE FAINT GRID / SCANLINE TEXTURE AND A VERY LOW-OPACITY FILM-GRAIN OVERLAY
  OVER THE WHOLE PAGE TO EVOKE A CRT TERMINAL WITHOUT BEING LOUD.

## COLOR PALETTE (STRICT GREYSCALE)

- BACKGROUND VOID: `#050505`
- SURFACE / PANEL: `#0F0F0F`
- CARD: `#111111`
- RAISED ACCENT CHIP: `#1A1A1A`
- HAIRLINE DIM: `RGBA(255,255,255,0.06)`
- HAIRLINE LIGHT: `RGBA(255,255,255,0.12)`
- TEXT PRIMARY: `#FFFFFF`
- TEXT SECONDARY: `#9CA3AF`
- TEXT MUTED: `#6B7280`
- "ACCENT" = PURE WHITE FILLS ON BLACK TEXT FOR PRIMARY BUTTONS/PILLS.

## TYPOGRAPHY

- **DISPLAY / HEADINGS & UI LABELS:** `JetBrains Mono` (MONO), WEIGHTS 400/500/700,
  TIGHT TRACKING, USED FOR THE HERO HEADLINE, SECTION TITLES, NAV, AND TAGS.
- **BODY:** `Inter`, WEIGHTS 300/400/500/600, LIGHT WEIGHT FOR PARAGRAPHS.
- **EMPHASIS SERIF:** `Merriweather` ITALIC (300 ITALIC / 700) FOR PULL-QUOTE
  WORDS AND THE FOOTER WORDMARK.
- HERO HEADLINE IS HUGE (CLAMP ~3.25REM → 6.5REM), MONO, BOLD, TRACKING-TIGHT,
  LINE-HEIGHT ~1.05, BREAKING ACROSS TWO LINES.

## LAYOUT & SECTION BREAKDOWN (TOP TO BOTTOM)

1. **STICKY HEADER** — A `name.dev`-STYLE MONO WORDMARK ON THE LEFT; A
   MONO NAV (`about / projects / articles`) PLUS A WHITE PILL `contact` BUTTON
   ON THE RIGHT. THE HEADER IS STICKY WITH A BLURRED, TRANSLUCENT BLACK
   BACKDROP. ON MOBILE THE NAV COLLAPSES TO A HAMBURGER THAT OPENS A
   FULL-SCREEN SLIDE-IN OVERLAY MENU.
2. **HERO** — TWO-LINE MONO HEADLINE (`Full-stack` / `Systems Developer`),
   A LIGHT INTER SUBPARAGRAPH WITH ONE OR TWO ITALIC-SERIF EMPHASIS PHRASES
   ("WRITE MAINTAINABLE, CLEAN AND UNDERSTANDABLE CODE"), A PRIMARY WHITE
   "PROJECTS" PILL NEXT TO A CIRCULAR ARROW ICON BUTTON, AND A WRAPPED GRID OF
   SMALL SOCIAL CHIPS (GITHUB / LINKEDIN / X / INSTAGRAM / TELEGRAM) RENDERED AS
   BORDERED RAISED CHIPS WITH INLINE SVG ICONS.
3. **FEATURED SLIDER** — A LARGE ROUNDED (~1.5REM RADIUS) CINEMATIC PANEL THAT
   CROSS-FADES BETWEEN 3 FEATURED ARTICLES. EACH SLIDE HAS A DIMMED FULL-BLEED
   BACKGROUND IMAGE (GRAYSCALE), A BOTTOM-UP BLACK GRADIENT SCRIM, A MONO TITLE,
   A SHORT DECK, AND `Read more` + ARROW BUTTONS. BELOW IT: A PREV CIRCLE BUTTON,
   ANIMATED PILL DOTS (ACTIVE DOT WIDENS), AND A NEXT CIRCLE BUTTON. AUTO-ADVANCE
   ON A TIMER, PAUSE ON HOVER, AND THE IMAGE SLOWLY SCALES ON HOVER.
4. **ABOUT** — TWO-COLUMN: LEFT IS A MONO PATH LABEL `... /about me ...` ABOVE A
   LARGE LIGHT STATEMENT WITH ITALIC-SERIF EMPHASIS ("I'M A *full-stack
   developer* WITH MORE THAN *5 years* EXPERIENCE") PLUS A SUPPORTING
   PARAGRAPH; RIGHT IS A SQUARE GRAYSCALE PORTRAIT IN A ROUNDED FRAME THAT
   DESATURATES → SATURATES (ACTUALLY: GOES FROM GRAYSCALE TO FULL ON HOVER) AND
   HAS A SOFT BOTTOM GRADIENT.
5. **STACK CARDS** — A FOUR-UP RESPONSIVE GRID OF NEAR-BLACK CARDS (FRONT-END /
   STYLES / BACK-END / DEVOPS), EACH WITH A MONO TITLE, A SMALL CIRCULAR ICON
   BADGE, AND A SLASH-SEPARATED LIST OF TECHNOLOGIES IN TINY MONO TEXT.
6. **WORK TIMELINE** — A `~/work` TITLE OVER A CENTERED MAX-WIDTH COLUMN WITH A
   VERTICAL HAIRLINE TIMELINE: EACH ROW HAS A RIGHT-ALIGNED MONO YEAR, A BOLD
   COMPANY NAME, A DURATION, AND A MONO ROLE LINE; ROWS UNDERLINE-HIGHLIGHT ON
   HOVER. ENDS WITH A RIGHT-ALIGNED ITALIC-SERIF TOTAL ("4 years 9 months").
7. **PROJECTS** — A `cd projects/` LABEL OVER A TWO-UP GRID OF PROJECT CARDS.
   EACH CARD: MONO PROJECT NAME, TAG PILLS, A LIGHT DESCRIPTION, A ROW OF
   CIRCULAR ICON BUTTONS (GITHUB + A DIAGONAL "OPEN" ARROW THAT FILLS WHITE),
   AND A 4:3 GRAYSCALE PREVIEW IMAGE THAT BRIGHTENS ON HOVER.
8. **ARTICLES** — AN `Articles` TITLE OVER A THREE-UP GRID OF TEXT-ONLY ARTICLE
   CARDS (MONO TITLE THAT UNDERLINES ON HOVER, A SHORT DECK, A WHITE `Read more`
   PILL + ARROW). PAGINATED ACROSS TWO PAGES WITH NUMBERED PILL CONTROLS
   (1 / 2 / NEXT) THAT FADE-SWAP THE PAGES.
9. **FOOTER / CONTACT** — A TOP HAIRLINE, A `... /contacts ...` LABEL, A
   FOUR-COLUMN ROW: A MONO LINK NAV, A SMALL "SITE" CREDITS BLOCK, AND A
   RIGHT-ALIGNED LARGE SERIF + MONO NAME WORDMARK ("REN" SERIF / "OKADA" MONO)
   WITH A "FULL-STACK DEVELOPER" CAPTION. BELOW: A SIX-UP GRID OF FULL-WIDTH
   SOCIAL CHIP BUTTONS.

## MOTION / ANIMATION / INTERACTION SPEC

- ON LOAD, THE HERO FADES + RISES IN (TRANSLATE-Y 10PX → 0, OPACITY 0 → 1, ~0.5S
  EASE-OUT). SECTIONS REVEAL ON SCROLL VIA INTERSECTION OBSERVER WITH THE SAME
  FADE-RISE.
- SLIDER: AUTO-ADVANCE EVERY ~5S, CROSS-FADE 500MS, PAUSE ON HOVER, CLICKABLE
  PREV/NEXT AND DOTS; ACTIVE DOT ANIMATES FROM A 8PX CIRCLE TO A 16PX PILL.
- BUTTONS: WHITE PILLS SCALE TO ~1.05 ON HOVER; CIRCULAR ICON BUTTONS INVERT
  (BLACK BG → WHITE BG, WHITE ICON → BLACK ICON) ON HOVER WITH A SMOOTH
  TRANSITION.
- PORTRAIT + PROJECT IMAGES: GRAYSCALE → COLOR / DIM → BRIGHT ON HOVER OVER
  ~0.7S.
- MOBILE MENU: SLIDES IN FROM THE RIGHT (TRANSLATE-X 100% → 0) OVER 300MS,
  LOCKS BODY SCROLL, CLOSES ON LINK CLICK OR THE X BUTTON.
- A FILM-GRAIN / SCANLINE OVERLAY DRIFTS VERY SUBTLY; CUSTOM THIN GREY
  SCROLLBAR.
- RESPECT `prefers-reduced-motion`: DISABLE AUTO-ADVANCE AND ENTRANCE TRANSFORMS.

## RESPONSIVE BEHAVIOR

- DESKTOP: NARROW CENTERED COLUMN WITH SIDE HAIRLINES; HERO, ABOUT ARE TWO
  COLUMN; STACK CARDS FOUR-UP; PROJECTS TWO-UP; ARTICLES THREE-UP.
- TABLET: GRIDS COLLAPSE TO TWO-UP; HERO/ABOUT REMAIN STACKABLE.
- MOBILE: SINGLE COLUMN THROUGHOUT; HAMBURGER NAV; SOCIAL CHIPS WRAP TWO-UP;
  HERO HEADLINE SCALES DOWN VIA CLAMP; SLIDER SHRINKS HEIGHT.
