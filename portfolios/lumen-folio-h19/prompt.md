# LUMEN FOLIO — SOFT-MINIMAL PERSONAL PORTFOLIO

## AESTHETIC IDENTITY

BUILD A SINGLE-PAGE PERSONAL PORTFOLIO CALLED **"LUMEN FOLIO"** FOR A FICTIONAL INDEPENDENT DESIGNER NAMED **NOA WILDE**. THE NAMED AESTHETIC IS **"SOFT EDITORIAL MINIMALISM"**: A BRIGHT, AIRY, PAPER-WHITE CANVAS WITH OVERSIZED ROUNDED CARDS, GENEROUS WHITESPACE, A FLOATING GLASS NAVIGATION PILL, AND CALM MICRO-INTERACTIONS. THE MOOD IS HUMAN, WARM, CONFIDENT, AND UNHURRIED — THE OPPOSITE OF A DARK, NEON, MAXIMALIST PORTFOLIO. EVERYTHING SHOULD FEEL LIKE A WELL-PRINTED DESIGN MAGAZINE TRANSLATED TO THE WEB.

## TECH STACK

- PLAIN STATIC SITE: A SINGLE `INDEX.HTML`, ONE `STYLES.CSS`, AND ONE `MAIN.JS`. NO BUILD STEP, NO FRAMEWORK, NO CDN AT RUNTIME.
- ALL ASSETS VENDORED LOCALLY UNDER `ASSETS/` (FONTS AS WOFF2, IMAGES AS LOCALLY GENERATED SVG/RASTER). NO REMOTE HOTLINKS.
- VANILLA JAVASCRIPT ONLY: `INTERSECTIONOBSERVER` FOR SCROLL REVEALS, A SMALL TYPEWRITER ROTATOR, A HORIZONTAL DRAG-SCROLL CAROUSEL, AND A FAKE CONTACT-FORM HANDLER.
- ICONS ARE HAND-WRITTEN INLINE SVG (STROKE STYLE, 1.75 WIDTH, ROUND CAPS/JOINS) — NO ICON LIBRARY.

## COLOR SYSTEM

- CANVAS / PAGE BG: WARM PAPER WHITE `#FBFAF8`.
- CARD SURFACE: SOFT OFF-WHITE `#F4F2EE` WITH HAIRLINE BORDER `#ECE9E3`.
- INK / PRIMARY TEXT: NEAR-BLACK `#1A1916`.
- MUTED TEXT: `#7A766E`.
- ACCENT (WARM TERRACOTTA/CLAY): `#C2603F` USED SPARINGLY FOR HIGHLIGHTS, LINKS, AND THE ROTATING WORD.
- SECONDARY ACCENT (SAGE): `#6E8B6A` FOR THE "AVAILABLE" STATUS PILL.
- DARK INVERTED PANEL (TESTIMONIAL / CTA): `#1A1916` BG WITH PAPER TEXT.
- SOFT TINTED ICON CHIPS USE 8–12% TINTS OF CLAY, SAGE, A DUSTY BLUE `#5B7290`, AND A MUTED PLUM `#7C5E7A`.

## TYPOGRAPHY

- DISPLAY / HEADINGS: A HIGH-CONTRAST EDITORIAL SERIF (E.G. "FRAUNCES" OR "INSTRUMENT SERIF" STYLE) FOR H1/H2 AND THE ROTATING WORD — LARGE, TIGHT LEADING, SLIGHTLY OPTICAL.
- BODY / UI: A CLEAN HUMANIST SANS (E.G. "INTER" OR "GENERAL SANS" STYLE), WEIGHTS 400/500/600.
- WEIGHTS: BODY 400, EMPHASIS 500, NAV/BUTTONS 500, HEADINGS 500–600 SERIF.
- HERO H1 USES `CLAMP()` FLUID SIZING FROM ~2.6REM TO ~5REM. TIGHT TRACKING ON HEADINGS (`-0.02EM`), RELAXED BODY LEADING (1.6).

## MOTION / ANIMATION / INTERACTION SPEC

- ON-LOAD: NAV PILL FADES + SLIDES DOWN; HERO ELEMENTS FADE-UP IN STAGGER.
- SCROLL REVEALS: EVERY SECTION STARTS `OPACITY 0` + `TRANSLATEY(28PX)` AND ANIMATES TO REST VIA `INTERSECTIONOBSERVER` (THRESHOLD ~0.12), WITH 0/100/200MS STAGGER DELAYS, EASE-OUT, ~0.7S.
- ROTATING WORD: TYPEWRITER DELETE/RETYPE LOOP THROUGH THE WORD LIST WITH A BLINKING CARET, IN THE CLAY ACCENT.
- CAROUSEL: SNAP-SCROLL + CLICK-AND-DRAG (POINTER) HORIZONTAL SCROLLING WITH MOMENTUM FEEL; HIDDEN SCROLLBAR.
- HOVER MICRO-INTERACTIONS: IMAGE ZOOMS, CARD LIFTS, POLAROID STRAIGHTEN + COLOR, BUTTON SCALE/INVERT, STATUS-DOT PULSE.
- RESPECT `PREFERS-REDUCED-MOTION`: DISABLE TRANSFORMS/TYPEWRITER AND SHOW CONTENT STATICALLY.
