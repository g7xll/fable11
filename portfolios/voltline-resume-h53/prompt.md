# VOLTLINE — NEO-BRUTALIST ENGINEER RESUME PORTFOLIO

> A SELF-CONTAINED GENERATION PROMPT FOR A SINGLE-PAGE, MULTI-SECTION
> PERSONAL PORTFOLIO / DIGITAL RESUME FOR A SENIOR SOFTWARE ENGINEER,
> STAGED AS A LOUD, PRINTABLE NEO-BRUTALIST DOCUMENT ON WARM CREAM PAPER.

## AESTHETIC IDENTITY

BUILD A PORTFOLIO CALLED **"VOLTLINE"** — A RAW, CONFIDENT, NEO-BRUTALIST
DIGITAL RESUME THAT READS LIKE A BOLD PRINTED BROADSHEET. THE WHOLE PAGE
SITS ON WARM CREAM "PAPER" WITH A FAINT ORANGE DOT-GRID PRINTED ACROSS IT.
EVERYTHING IS DEFINED BY THICK INK-BLACK OUTLINES, HARD OFFSET DROP
SHADOWS (NO BLUR), HIGH-CONTRAST ORANGE-ON-INK BLOCKS, AND A FEW DELIBERATELY
SKEWED BANDS THAT TILT THE LAYOUT JUST ENOUGH TO FEEL HAND-SET. THE MOOD IS
ELECTRIC, INDUSTRIAL, AND PROUDLY ANALOG-MEETS-DIGITAL — A RESUME THAT
SHOUTS WITHOUT BECOMING NOISE. NO SOFT GRADIENTS ON SURFACES, NO ROUNDED
PILLOWY CARDS, NO DROP-SHADOW BLUR — ONLY CRISP, STACKED, OFFSET GEOMETRY.

THE DOT-GRID BACKGROUND IS A RADIAL-GRADIENT DOT PATTERN IN ORANGE AT
~3% OPACITY, 24PX TILE. EVERY HARD SHADOW IS A SOLID OFFSET BLOCK
(E.G. `8px 8px 0` INK, OR `12px 12px 0` ORANGE) — NEVER BLURRED.

## TYPOGRAPHY

- DISPLAY (HUGE HEADLINES, NAME, SECTION TITLES): **BEBAS NEUE**, A TALL
  CONDENSED ALL-CAPS GROTESQUE. USE FOR THE HERO NAME AT CLAMPED
  ~`9rem`, SECTION HEADINGS AT ~`5rem`, AND BIG STAT NUMBERS.
- PRIMARY / LABELS / BUTTONS: **SPACE GROTESK** 500–700, UPPERCASE WITH
  WIDE LETTER-SPACING FOR NAV, TAGS, ROLE LINES, AND CTA TEXT.
- BODY: **INTER** 400–600 FOR PARAGRAPHS AND BULLET DETAIL.

SELF-HOST ALL THREE FONTS LOCALLY (WOFF2) AND REFERENCE THEM VIA
`@font-face` WITH RELATIVE PATHS — NO GOOGLE FONTS CDN AT RUNTIME.

PADDING, IN THIS ORDER:

   MONOGRAM LOGO (`AV.`) IN ORANGE BEBAS NEUE ON THE LEFT, UPPERCASE
   SPACE GROTESK NAV LINKS (WORK / STACK / SCHOOL) IN THE CENTER-RIGHT,
   AND A SOLID INK "DOWNLOAD" BUTTON ON THE RIGHT THAT TRIGGERS
   `window.print()`.

2. **HERO** — FULL-VIEWPORT, A 12-COLUMN GRID. LEFT (7 COLS): AN
   "AVAILABLE FOR HIRE" ORANGE TAG, A TWO-LINE GIANT BEBAS NAME, A ROLE

   SUMMARY PARAGRAPH WITH HARD-NUMBER CLAIMS, A ROW OF THREE STAT BLOCKS
   (EACH A LEFT COLORED BORDER + BIG BEBAS NUMBER + UPPERCASE CAPTION:
   "8+ YEARS EXP", "2M+ USERS SERVED", "40% PERFORMANCE ↑"), AND TWO
   CTAS WITH DUAL-OFFSET BRUTALIST BORDERS. RIGHT (5 COLS): AN INK
   CONTACT CARD WITH A THICK ORANGE BORDER AND A `12px 12px 0` ORANGE
   HARD SHADOW, LISTING EMAIL / PHONE / LOCATION WITH GLYPH ICONS AND A
   ROW OF SQUARE SOCIAL BUTTONS (IN / GH / ↓).

3. **EXPERIENCE** — A SKEWED BAND (`skewY(-2deg)` ON THE SECTION, CONTENT

   EACH CARD: THICK INK BORDER ON WHITE, A "CURRENT" TAG ON THE TOP ROLE,
   ROLE TITLE IN BEBAS, COMPANY IN COBALT, LOCATION IN GREY, A BOXED
   DATE CHIP WITH ITS OWN COLORED BORDER, AND A TWO-COLUMN GRID OF
   ACHIEVEMENT BULLETS EACH WITH A COLORED LEFT BORDER. ON HOVER THE CARD

4. **STACK / SKILLS** — FOUR LARGE COLOR-BLOCK CARDS (ORANGE, COBALT,

5. **SCHOOL / EDUCATION** — ANOTHER SKEWED PALE-PEACH BAND: A DEGREE CARD

6. **CLOSING CTA** — CENTERED ON CREAM WITH A FAINT ORANGE→COBALT WASH:

7. **FOOTER** — INK SLAB WITH AN 8PX ORANGE TOP BORDER: MONOGRAM +

## SIGNATURE BRUTALIST DETAILS

- **DUAL-OFFSET BORDER**: INTERACTIVE BLOCKS HAVE A 3PX INK BORDER PLUS A
  PSEUDO-ELEMENT 3PX ORANGE BORDER OFFSET `8px / 8px`; ON HOVER THE
  ORANGE LAYER SLIDES TO `12px / 12px`.
- **SKEWED BANDS**: ALTERNATE SECTIONS TILT `-2deg` WITH CONTENT
  COUNTER-SKEWED SO TEXT STAYS LEVEL.
- **DIAGONAL ACCENT** BEHIND THE HERO: A LARGE ROTATED ORANGE→AMBER WEDGE
  AT ~8% OPACITY.

## MOTION & INTERACTION SPEC

- **SCROLL REVEAL**: ELEMENTS TAGGED FOR ANIMATION START AT `opacity:0`,
  `translateY(30px)` AND EASE IN (`cubic-bezier(0.16,1,0.3,1)`, ~0.6S) AS
  THEY ENTER VIEW VIA INTERSECTIONOBSERVER, STAGGERED PER SECTION.
- **STAT COUNT-UP**: THE THREE HERO STAT NUMBERS COUNT UP FROM ZERO TO
  THEIR TARGET ONCE WHEN FIRST VISIBLE.
- **SMOOTH ANCHOR SCROLL** WITH FIXED-HEADER OFFSET FOR NAV LINKS.
- HOVER STATES: CARD-LIFT SHADOWS, ICON SCALE, COLOR INVERSIONS ON CHIPS
  AND SOCIAL BUTTONS.
- A SUBTLE PARALLAX DRIFT ON THE HERO DIAGONAL WEDGE WHILE SCROLLING.
- RESPECT `prefers-reduced-motion`: DISABLE COUNT-UP, PARALLAX, AND
  TRANSFORM-BASED REVEALS.

## RESPONSIVE BEHAVIOR

- DESKTOP ≥1024PX: 12-COL HERO (7/5 SPLIT), 2-COL CARD GRIDS, FULL DISPLAY
  TYPE.
- TABLET: HERO STACKS TO ONE COLUMN (CONTACT CARD BELOW), CARD GRIDS STAY
  TWO-UP WHERE THEY FIT.
- MOBILE ≤640PX: SINGLE COLUMN THROUGHOUT, REDUCED DISPLAY SIZES VIA
  `clamp()`, SKEW ANGLES SOFTENED, STAT ROW STAYS THREE-ACROSS BUT
  COMPACT, NAV LINKS COLLAPSE TO THE CTA ONLY.

## PRINT BEHAVIOR

PROVIDE A `@media print` MODE: HIDE THE DOT-GRID, HEADER, FOOTER, AND
OFFSET PSEUDO-SHADOWS; FLATTEN SKEWS; KEEP THE CONTENT AS A CLEAN
PRINTABLE RESUME SO "DOWNLOAD" PRODUCES A USABLE PDF.

## TECH

PLAIN HTML + CSS + VANILLA JS, NO BUILD STEP, NO RUNTIME CDNS. ALL FONTS
AND ASSETS VENDORED LOCALLY. THE RESULT MUST BE A FULLY SELF-CONTAINED,
OFFLINE-RUNNABLE STATIC PAGE.
