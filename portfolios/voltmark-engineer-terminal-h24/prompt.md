## MOTION / ANIMATION / INTERACTION SPEC

- SCROLL-REVEAL: ELEMENTS FADE + RISE 30PX VIA INTERSECTIONOBSERVER (0.8S EASE-OUT), ONCE.
- HERO BLOB: TRACKS MOUSE WITH EASED TRANSLATION; GRID MASKED BY RADIAL FADE.
- FLOAT: CODE CHIPS AND TERMINAL FLOAT GENTLY (6S EASE-IN-OUT, ONE DELAYED) ON A ±12PX Y CYCLE.
- TERMINAL TYPEWRITER: TYPES THE BOOT SCRIPT CHAR-BY-CHAR (~40MS + JITTER) WHEN SCROLLED INTO VIEW; PERSISTENT BLINKING BLOCK CURSOR.
- COUNTERS: COUNT UP FROM 0 TO TARGET ON FIRST VIEW.
- MARQUEE: SEAMLESS INFINITE TRANSLATE LOOP (DUPLICATED TRACK), ~25S LINEAR.
- PROJECT FILTERS: TOGGLE ACTIVE PILL STYLING AND SHOW/HIDE CARDS.
- ACCORDIONS (EXPERIENCE + FAQ): MAX-HEIGHT TRANSITION WITH ROTATING ICON.
- ALL INTERACTIVE ELEMENTS HAVE HOVER/ACTIVE TRANSITIONS; CTAS SCALE SLIGHTLY ON HOVER AND COMPRESS ON ACTIVE.
- RESPECT `prefers-reduced-motion`.
