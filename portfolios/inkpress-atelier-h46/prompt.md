# INKPRESS ATELIER — A RISOGRAPH-PRINT GRAPHIC DESIGNER PORTFOLIO

BUILD A SINGLE-PAGE, FULLY RESPONSIVE PORTFOLIO SITE FOR A FICTIONAL INDEPENDENT GRAPHIC DESIGNER / ART DIRECTOR NAMED **MARGOT ELLERY**, STAGED AS A PRINTED RISOGRAPH ZINE SPREAD. THE NAMED AESTHETIC IDENTITY IS **"INKPRESS ATELIER"** — A RETRO-MODERN SWISS-PRINT / RISOGRAPH LANGUAGE THAT FEELS LIKE A HAND-PULLED POSTER COLLECTION BOUND INTO A SINGLE FOLDED MAGAZINE. THE WHOLE PAGE LIVES INSIDE A NARROW CENTERED "PAPER FRAME" (LIKE A PHYSICAL PRINTED OBJECT FLOATING ON A WORKBENCH), AND EACH SECTION READS LIKE A SEPARATE PAGE OF THE ZINE.

## DESIGN LANGUAGE & MOOD

- RETRO PRINT, RISOGRAPH, AND SWISS EDITORIAL — CONFIDENT, LOUD-BUT-TASTEFUL, TACTILE, HANDMADE. THE OPPOSITE OF GENERIC SAAS: NO PURPLE GRADIENTS, NO GLASSMORPHISM, NO NEON GLOW.
- THE ENTIRE EXPERIENCE IS A SINGLE CENTERED COLUMN (MAX-WIDTH ~860PX) OF WARM CREAM PAPER, WITH A HEAVY OUTER DROP SHADOW, SITTING ON A SATURATED MARIGOLD-YELLOW DESK BACKGROUND. A FAINT PAPER-GRAIN / HALFTONE TEXTURE OVERLAYS THE WHOLE THING AT LOW OPACITY TO SELL THE PRINTED FEEL.
- A 4-COLOR PRINT BAR (BLUE / RED / YELLOW / GREEN) RUNS ACROSS THE TOP EDGE OF THE PAPER LIKE A REGISTRATION STRIP.
- HEAVY USE OF "PRINT ARTIFACTS": HARD OFFSET BLOCK SHADOWS (NO BLUR), MIX-BLEND-MULTIPLY COLOR OVERLAPS, JAGGED / SAWTOOTH SECTION EDGES, DOTTED LEADER LINES, REGISTRATION CORNER MARKS, ROTATED STICKERS, AND DECORATIVE GEOMETRIC SHAPES (CIRCLES, DIAMONDS, HALF-CIRCLES, TRIANGLES, CHEESE-WEDGES) THAT PEEK OVER SECTION SEAMS.

## EXACT COLOR PALETTE

- PAPER CREAM: `#F3EFE0` (PRIMARY BACKGROUND OF THE FRAME)
- DESK MARIGOLD: `#FDB913` (PAGE BACKGROUND BEHIND THE FRAME, AND ACCENT)
- INK RED / VERMILION: `#E94E33` (PRIMARY ACCENT)
- PRESS BLUE: `#005EB8` (SECONDARY BLOCK COLOR)
- LEAF GREEN: `#008F4C` (TERTIARY BLOCK COLOR)
- INK BLACK: `#1A1A1A` (TEXT + HEAVY OUTLINES)
- USE THE FOUR PRINT COLORS AS FLAT FILLS FOR FULL-BLEED BLOCKS; NEVER AS SOFT GRADIENTS.

## TYPOGRAPHY

- DISPLAY: **ANTON** (SINGLE WEIGHT, TIGHT TRACKING, ALL-CAPS) FOR EVERY HEADLINE, THE LOGO, SECTION TITLES, AND POSTER WORDMARKS — OVERSIZED AND PUNCHY.
- BODY / UI: **ROBOTO CONDENSED** (400 + 700), ALWAYS UPPERCASE FOR LABELS/NAV/META WITH WIDE LETTER-SPACING; SENTENCE CASE FOR LONGER PARAGRAPHS.
- VENDOR BOTH FONTS LOCALLY (WOFF2) — NO REMOTE GOOGLE-FONTS LINKS AT RUNTIME.

## MOTION / ANIMATION / INTERACTION SPEC

- **ENTRANCE:** HEADER NAV SLIDES DOWN; LOGO PLATE POPS IN WITH AN OVERSHOOT (SCALE 0.8 → 1.05 → 1) EASE.
- **SCROLL REVEAL:** AN INTERSECTIONOBSERVER ADDS AN `IS-VISIBLE` CLASS THAT FADES + RISES ELEMENTS (TRANSLATEY 30PX → 0) WITH A SLIGHT SPRINGY CUBIC-BEZIER, STAGGERED VIA DELAY UTILITIES (100/200/300MS).
- **CONTINUOUS:** SLOW-SPIN BADGES (~10S LINEAR), GENTLE BOUNCE STICKERS, AND THE MARQUEE (~25S LINEAR).
- **HOVER MICRO-INTERACTIONS:** EVENT ROWS TRANSLATE RIGHT; POSTER CARD STRAIGHTENS + SCALES + GROWS ITS SHADOW; MENU ROWS RECOLOR RED; PROJECT CARDS SCALE/ROTATE/RECOLOR THEIR THUMBNAILS, INVERT TAG COLORS, AND REVEAL HIDDEN PRICES; SOCIAL DIAMONDS COUNTER-ROTATE THEIR GLYPHS.
- RESPECT `PREFERS-REDUCED-MOTION`: DISABLE THE INFINITE/CONTINUOUS ANIMATIONS AND SOFTEN REVEALS.

## RESPONSIVE BEHAVIOR

- THE PAPER FRAME STAYS CENTERED WITH OUTER PADDING ON ALL SCREENS; ON MOBILE IT GOES NEARLY FULL-WIDTH.
- TWO-COLUMN BLOCKS (EXHIBITIONS, SERVICES) STACK VERTICALLY ON SMALL SCREENS; THE GREEN SERVICES SIDEBAR BECOMES A HORIZONTAL BAR.
- THE 3-COLUMN WORKS GRID COLLAPSES TO 1 COLUMN ON MOBILE; DISPLAY TYPE SCALES DOWN FLUIDLY; CENTER NAV LINKS COLLAPSE GRACEFULLY.
