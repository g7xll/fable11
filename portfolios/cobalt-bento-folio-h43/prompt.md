# COBALT BENTO FOLIO — "DAYLIGHT WORKBENCH" DEVELOPER PORTFOLIO

## NAMED AESTHETIC IDENTITY

BUILD A SINGLE-PAGE, FORCED-LIGHT PERSONAL PORTFOLIO CALLED **"COBALT BENTO FOLIO"** FOR A FICTIONAL FULL-STACK ENGINEER NAMED **NORA AVELINE**. THE NAMED AESTHETIC IS **"DAYLIGHT WORKBENCH"**: A SOFT, AIRY, OVERCAST-MORNING CANVAS BUILT ENTIRELY FROM A BENTO GRID OF FROSTED-GLASS PANELS FLOATING OVER A COOL PALE-GRAY BACKDROP. THE MOOD IS CALM, OPTIMISTIC, TACTILE, AND PRODUCT-DESIGNER-PRECISE — A TIDY DESK BY A NORTH-FACING WINDOW. ONE COMMANDING, SATURATED **ROYAL-COBALT** PANEL ANCHORS THE COMPOSITION (THE COLOSSAL "PORTFOLIO / 2025" HERO TILE) WHILE EVERYTHING ELSE STAYS WHITE-ON-GLASS WITH NEAR-BLACK INK PANELS USED AS PUNCTUATION. THE SIGNATURE STRUCTURAL DEVICE IS THE **ASYMMETRIC BENTO LAYOUT**: A 12-COLUMN GRID WHERE PANELS OF DIFFERENT SPANS (3/9, 5/7, 4/4/4) INTERLOCK INTO A SINGLE CONTINUOUS DASHBOARD, EVERY PANEL ROUNDED, BORDERED IN HAIRLINE WHITE, AND LIFTED ON A SOFT SHADOW.

## TECH STACK

- PURE STATIC SITE: A SINGLE `INDEX.HTML`, ONE VENDORED `STYLES.CSS`, AND ONE VENDORED `MAIN.JS`. NO BUILD STEP, NO FRAMEWORK, NO RUNTIME CDN.
- ALL ASSETS VENDORED LOCALLY UNDER `ASSETS/` — FONTS (SELF-HOSTED WOFF2), PHOTOGRAPHY (LOCAL JPG/WEBP), ALL ICONS AS INLINE SVG. THE SITE MUST RUN FULLY OFFLINE.
- VANILLA JS ONLY: INTERSECTIONOBSERVER STAGGERED PANEL REVEALS, A POINTER-PARALLAX HERO, A REQUESTANIMATIONFRAME COUNT-UP FOR STATS, AN ANIMATED SVG SKILL-RING, A TYPEWRITER ROTATOR, AND A FAKE CONTACT FORM WITH VALIDATION + SUCCESS STATE.

## MOTION / INTERACTION SPEC

- ON LOAD: PANELS FADE + RISE IN A STAGGERED CASCADE (TOP-LEFT TO BOTTOM-RIGHT) VIA INTERSECTIONOBSERVER, EACH WITH A SHORT DELAY.
- HERO COBALT PANEL: POINTER PARALLAX ON THE ROTATING RING + A CURSOR-TRACKING RADIAL SPECULAR HIGHLIGHT; THE BACKGROUND RING ROTATES SLOWLY AND CONTINUOUSLY.
- TYPEWRITER ROTATOR IN THE HERO CYCLES ROLES (TYPE IN, PAUSE, DELETE, NEXT).
- SKILL RING ANIMATES ITS STROKE WHEN REVEALED; TECH BADGES BOB ON A LOOPED FLOAT KEYFRAME.
- COUNT-UP STATS RUN ONCE WHEN THE STRIP ENTERS VIEW.
- PANEL HOVER: LIFT + SHADOW DEEPEN; CONTACT TILES + EDUCATION CARDS SCALE; THE HERO ARROW ROTATES 45°.
- NAV PILL SCROLL-SPY HIGHLIGHTS THE CURRENT SECTION; CLICKS SMOOTH-SCROLL.
- ALL MOTION RESPECTS `PREFERS-REDUCED-MOTION` (DISABLE PARALLAX, TYPEWRITER, BOB, AND COUNT-UP ANIMATION DURATION).

## RESPONSIVE BEHAVIOR

- DESKTOP (≥1024PX): FULL 12-COLUMN BENTO AS DESCRIBED, PILL NAV VISIBLE.
- TABLET (640–1023PX): PANELS REFLOW TO 2-COLUMN OR FULL-WIDTH SPANS; THE THREE RÉSUMÉ COLUMNS STACK TO TWO THEN ONE; HERO STAYS PROMINENT.
- MOBILE (<640PX): EVERY PANEL FULL WIDTH AND STACKED IN READING ORDER; PILL NAV COLLAPSES TO A HAMBURGER GLASS SHEET; FLOATING ACTION BUTTON APPEARS; HERO WORDMARK SCALES DOWN VIA CLAMP BUT STAYS COMMANDING.
