# Super Travel — Luxury Editorial Travel Landing Page (HTML, CSS, League Spartan)

A single-page luxury travel landing built on a strict 12-column grid with bold editorial typography, a warm off-white and dusty-rose palette, and grayscale-to-color hover photography — designed to feel like a high-end print spread brought to life on the web. The hero fills the full viewport with a ~15vw headline and a floating concierge badge on a 4s bounce, a three-column services grid sits on a secondary cream background, and a staggered two-column portfolio grid offsets right-column items 100px down for architectural rhythm. Navigation uses glassmorphism (`rgba(253, 248, 243, 0.8)` + `blur(12px)`) and all motion uses `cubic-bezier(0.16, 1, 0.3, 1)` at 1s for a weighted, premium feel. Generated with Claude Fable 5.

## Run

This is a static project — open `index.html` in a browser, or serve the folder:

```sh
python3 -m http.server 8000
```

See `prompt.md` for the full build spec; `demo.mp4` shows it in motion.
