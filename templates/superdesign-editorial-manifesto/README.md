# Modern Editorial Manifesto — Premium Dark Editorial Launch Page (HTML + CSS + Vanilla JS)

A premium editorial launch page template for storytelling-first brands, creative studios, and product launches that prize typographic impact over conventional SaaS UI — ideal as a manifesto page, a long-form brand statement, or a prestige announcement site. Built around massive Fraunces variable serif headlines (up to 15vw, 0.8 line-height, with optical sizing), Manrope body copy, a high-contrast palette of charcoal (`#0A0A0C`) and warm cream (`#E8E6E1`) with a gold-cream accent (`#D4C5A2`), and a global SVG fractal-noise grain overlay at ~0.04 opacity for paper-stock texture. The page unfolds as ten thematic chapters: a full-screen typographic hero cover with a vertical side label, an intro paragraph, a sticky two-column essay with a bordered pull-quote, a vertical narrative list of full-width hover rows, a light-background manifesto grid, gradient-clipped editorial testimonials, a backdrop-blur CTA card, a final statement, and a footer — with no box-shadows, rounded buttons, icon libraries, or images anywhere. All motion is slow and deliberate (≥1s eased), with reveal-on-scroll fades and a staggered hero entrance; reduced-motion is respected. Delivered as self-contained static HTML with no build step. Generated with Claude Fable 5.

## Run

This is a static project — open `index.html` in a browser, or serve the folder:

```sh
python3 -m http.server 8000
```

See `prompt.md` for the full build spec; `demo.mp4` shows it in motion.
