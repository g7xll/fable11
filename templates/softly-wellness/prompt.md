# SUMMARY

BUILD **SOFTLY** — A WARM, MOBILE-FIRST LANDING PAGE FOR A DIGITAL-WELLNESS APP, RECREATED PIXEL-FOR-PIXEL FROM THE REFERENCE DESIGN. THE PAGE SHOULD FEEL LIKE A "DIGITAL LIVING ROOM": MINIMALIST, TACTILE, AND INTENTIONALLY SLOW. A SINGLE VERTICAL COLUMN FLOWS FROM A SPACIOUS HERO THROUGH INTERACTIVE SCROLLS AND PHONE MOCKUPS INTO A HIGH-CONTRAST WAITLIST CONVERSION. TAB TITLE: "SOFTLY - FIND YOUR RHYTHM".

# STYLE

DIGITAL MINIMALISM MEETS GEN-Z LIFESTYLE. WARM DESATURATED PASTELS, A PERSISTENT PAPER-GRAIN OVERLAY, HIGH-RADIUS BLURRED BACKGROUND BLOBS, AND FLUID LOW-VELOCITY MOTION. TYPOGRAPHY PAIRS THE CLEAN ROUNDED **OUTFIT** SANS-SERIF (WEIGHTS 300/400/500/600) WITH THE WHIMSICAL **REENIE BEANIE** CURSIVE USED ONLY FOR EXPRESSIVE EMPHASIS (LOAD BOTH FROM GOOGLE FONTS). ALL HEADINGS ARE SENTENCE-CASE WITH TIGHT TRACKING.

## ANIMATIONS

- **REVEAL-ON-SCROLL**: ELEMENTS START `OPACITY:0; TRANSLATEY(30PX)`, TRANSITION `0.8S EASE-OUT` TO `OPACITY:1; TRANSLATEY(0)`; TOGGLED VIA INTERSECTIONOBSERVER (`THRESHOLD:0.1`, `ROOTMARGIN:0PX 0PX -50PX 0PX`) ADDING AN `IS-VISIBLE` CLASS, THEN UNOBSERVING. STAGGER WITH DELAY-100/200 UTILITIES.
