# HEAVYWEIGHT — BRUTALIST INDUSTRIAL DESIGN SYSTEM

> THIS PROMPT REPRODUCES A SAME-TO-SAME UI OF THE REFERENCE DRAFT AT
> `https://p.superdesign.dev/draft/7d25a4cc-c30e-48b7-a282-7384de4bfeed`.
> THE ORIGINAL PROMPT IS PRESERVED VERBATIM BELOW, FOLLOWED BY THE EXPANDED
> IMPLEMENTATION SPEC AND THE CORE CODE NEEDED TO REGENERATE THE SAME OUTPUT.

## SUMMARY

HEAVYWEIGHT IS A MINIMALIST YET HIGH-IMPACT DESIGN SYSTEM BUILT ON BRUTALIST PRINCIPLES. IT UTILIZES MASSIVE UPPERCASE DISPLAY TYPE, A MONOSPACE AESTHETIC FOR UTILITY TEXT, AND AN INTENTIONAL USE OF NEGATIVE SPACE. THE UI FEELS INDUSTRIAL AND STRUCTURAL, MIMICKING THE 'HEAVY' NATURE OF THE PRODUCTS IT SHOWCASES.

## STYLE

THE STYLE IS DEFINED BY A 'CONCRETE' (#F2F2F2) AND 'INK' (#0A0A0A) PALETTE WITH HIGH-CONTRAST INTERACTIONS. TYPOGRAPHY PAIRS 'ANTON' FOR MASSIVE, AGGRESSIVE HEADLINES WITH 'SPACE GROTESK' FOR TECHNICAL, MONOSPACE-STYLE METADATA. KEY CHARACTERISTICS INCLUDE A PERSISTENT 4% OPACITY GRAIN OVERLAY, GRAYSCALE IMAGERY THAT REVEALS COLOR ON HOVER, AND THE USE OF 'MIX-BLEND-DIFFERENCE' TO ENSURE TEXT LEGIBILITY ACROSS VARYING BACKGROUNDS.

### SPEC

CREATE A DESIGN WITH A BACKGROUND OF #F2F2F2 (CONCRETE) AND PRIMARY TEXT IN #0A0A0A (INK). APPLY A GLOBAL GRAINY TEXTURE USING A NOISE SVG AT 0.04 OPACITY. FOR TYPOGRAPHY: USE 'ANTON' FOR H1-H3 HEADERS (UPPERCASE, TRACKING-TIGHTER, LEADING-0.85) AND 'SPACE GROTESK' FOR BODY AND METADATA (TRACKING-WIDE). INTERACTIVE LINKS SHOULD USE A 'HOVER:LINE-THROUGH' EFFECT WITH A DECORATION THICKNESS OF 2PX. IMAGES SHOULD DEFAULT TO GRAYSCALE WITH A 700MS TRANSITION TO FULL COLOR AND A SLIGHT 1.05X SCALE ON HOVER. USE AN ACCENT COLOR OF #DC2626 (RED) SPARINGLY FOR LIVE STATUS INDICATORS OR NOTIFICATIONS. SHADOWS SHOULD BE SHARP AND HIGH-INTENSITY, USED ONLY ON FLOATING ELEMENTS LIKE THE CART ICON (#000000 OPACITY 0.1).

## LAYOUT & STRUCTURE

THE LAYOUT IS ASYMMETRICAL AND INTENTIONALLY 'UNBALANCED' TO CREATE VISUAL INTEREST. IT UTILIZES A FIXED HEADER WITH ELEMENTS PUSHED TO EXTREME CORNERS AND A MAIN CONTENT AREA THAT MIXES LARGE-SCALE HERO IMAGERY WITH DENSE, TECHNICAL INFORMATION BLOCKS.

### NON-STANDARD HEADER

A FIXED HEADER (Z-INDEX 40) USING `MIX-BLEND-DIFFERENCE` TO APPEAR WHITE ON DARK BACKGROUNDS AND BLACK ON LIGHT. THE LOGO (ANTON FONT, 4XL) SITS TOP-LEFT WITH A SUB-LABEL (EST. 2024 / NYC) IN 10PX TRACKING-WIDEST. NAVIGATION IS A VERTICAL COLUMN OF LINKS SHIFTED RIGHT (MR-32) TO PREVENT OVERLAP WITH CENTRAL HERO CONTENT. LINKS ARE UPPERCASE, 12PX, WITH LINE-THROUGH HOVER EFFECTS.

### ASYMMETRIC HERO SECTION

A SPLIT LAYOUT. LEFT SIDE (42% WIDTH) CONTAINS A MASSIVE HEADLINE (TEXT-9XL) POSITIONED LOW TO AVOID HEADER ELEMENTS. INCLUDE A SUB-PARAGRAPH (MAX-WIDTH 320PX) AND A MINIMALIST UNDERLINE CTA. RIGHT SIDE (65% WIDTH) IS AN ABSOLUTE-POSITIONED IMAGE WITH A PARTIAL OFF-SCREEN CROP, GRAYSCALE, AND OBJECT-LEFT ALIGNMENT. INCLUDE A VERTICAL SECTION LABEL ('COLLECTION 004') ROTATED -90 DEGREES AT THE BOTTOM-LEFT OF THE IMAGE.

### HORIZONTAL PRODUCT STRIP

A HORIZONTAL SCROLL SECTION FOR 'ESSENTIALS'. CARDS HAVE AN ASPECT RATIO OF 3:4 WITH A SECONDARY IMAGE THAT FADES IN ON HOVER (HOVER:OPACITY-100). METADATA SITS BELOW THE IMAGE WITH A BOLD UPPERCASE TITLE AND A MONOSPACE PRICE ($140). USE `SNAP-X` AND HIDE SCROLLBARS.

### TECHNICAL INFO GRID

A 12-COLUMN GRID LAYOUT. THE LEFT 4 COLUMNS ARE STICKY, CONTAINING MASSIVE HEADERS AND TECHNICAL 'ARCHITECTURE' DESCRIPTIONS WITH LEFT-BORDER ACCENTS (BORDER-L-2). THE RIGHT 8 COLUMNS FEATURE A VARIETY OF IMAGE ASPECT RATIOS (4:5 AND 1:1) SHOWCASING MACRO DETAILS OF PRODUCT CONSTRUCTION.

### LIVE INVENTORY GRID

AN ASYMMETRIC 'BROKEN' GRID FOR PRODUCTS. USE VARYING COLUMN SPANS (COL-SPAN-6, COL-SPAN-3) AND NEGATIVE MARGINS (MT-24) TO CREATE A STAGGERED, EDITORIAL LOOK. EACH ITEM INCLUDES SIZE AVAILABILITY IN A STRIKE-THROUGH MONOSPACE FONT FOR OUT-OF-STOCK ITEMS.

### MASONRY SOCIAL PROOF

A MASONRY COLUMN LAYOUT (COLUMNS-3). CARDS ARE EITHER SOLID BACKGROUND (#F2F2F2) OR OUTLINED (1PX INK). EACH CARD IS SLIGHTLY ROTATED (ROTATE-1 OR ROTATE-2) TO LOOK LIKE POSTERS PASTED ON A WALL. USE MONOSPACE FONT FOR QUOTES AND A SMALL DOT ICON FOR USER HANDLES.

### UTILITY FOOTER

A DEEP #0A0A0A BACKGROUND WITH #FFFFFF TEXT. ORGANISE LINKS INTO A 6-COLUMN GRID. THE FINAL FOOTER ELEMENT SHOULD BE A MASSIVE, OVERSIZED 'HEAVY' DISPLAY TEXT (10VW) IN LOW-OPACITY WHITE (TEXT-WHITE/10) AT THE BOTTOM, ACTING AS A STRUCTURAL ANCHOR.

## SPECIAL COMPONENTS

### FLOATING ACTION CART

AN ISOLATED, HIGH-Z-INDEX CART BUTTON.

A CIRCULAR BUTTON (64PX X 64PX) FIXED AT THE TOP-RIGHT CORNER. BACKGROUND IS WHITE, ICON IS INK. FEATURES A NOTIFICATION BADGE IN RED (#DC2626) WITH WHITE 10PX TEXT. INCLUDES A HOVER:SCALE-110 TRANSITION AND A SHARP DROP SHADOW.

### DETAIL BADGE

TECHNICAL SPECS OVERLAY ON IMAGES.

A SMALL, SHARP-CORNERED WHITE RECTANGLE (PADDING: 4PX 8PX) PLACED AT THE TOP-RIGHT OR BOTTOM-LEFT OF PRODUCT IMAGES. CONTAINS TECHNICAL DATA LIKE '500GSM' OR 'GARMENT DYED' IN BOLD, 10PX UPPERCASE FONT.

## EXPANDED IMPLEMENTATION SPEC (SAME-TO-SAME REPRODUCTION)

BUILD AS A SELF-CONTAINED STATIC SITE: `index.html` + `styles.css` + `script.js`,
WITH ALL FONTS, IMAGES, AND THE NOISE TEXTURE VENDORED LOCALLY UNDER `assets/`.
NO BUILD STEP, NO CDN AT RUNTIME — IT MUST RUN OFFLINE FROM A STATIC SERVER.

### FONTS (VENDOR LOCALLY)

- **ANTON** — WEIGHT 400 ONLY — DISPLAY HEADLINES, LOGO, BIG NUMBERS, GHOST FOOTER WORD.
- **SPACE GROTESK** — WEIGHTS 400 / 500 / 700 — BODY, NAV, METADATA, PRICES, BADGES, QUOTES.
- DOWNLOAD THE WOFF2 FILES FROM GOOGLE FONTS (`fonts.gstatic.com`) INTO `assets/fonts/`
  AND DECLARE THEM VIA `@font-face` WITH `font-display: swap`.

### DESIGN TOKENS (EXACT)

```
--concrete: #f2f2f2;   /* page background */
--ink:      #0a0a0a;   /* primary text + dark sections */
--white:    #ffffff;   /* floating cart bg, footer text, badges bg */
--red:      #dc2626;   /* live dots, cart badge, notifications ONLY */
--hairline: rgba(10,10,10,0.12);  /* 1px dividers / outlined cards */
```

GLOBAL: `body { background: var(--concrete); color: var(--ink);
font-family: 'Space Grotesk'; letter-spacing: 0.02em; -webkit-font-smoothing: antialiased; }`
ALL HEADERS H1–H3: `font-family:'Anton'; text-transform:uppercase; letter-spacing:-0.03em; line-height:0.85;`

### GRAIN OVERLAY (PERSISTENT, GLOBAL)

A FIXED FULL-VIEWPORT `::before`-LESS DEDICATED `.grain` LAYER, `position:fixed; inset:0;
pointer-events:none; z-index:60; opacity:0.04;` USING A LOCALLY-SAVED `assets/noise.svg`
(`feTurbulence type='fractalNoise' baseFrequency='0.85' numOctaves='3' stitchTiles='stitch'`)
TILED VIA `background-image:url(assets/noise.svg); background-size:180px;`. IT SITS ABOVE
CONTENT BUT BELOW THE CART. ADD A SUBTLE `@keyframes grainShift` THAT TRANSLATES THE LAYER
A FEW PX EVERY ~8 STEPS (STEP TIMING, `animation:grainShift 0.6s steps(1) infinite;`)
FOR A LIVE FILM-GRAIN FLICKER.

### IMAGE BEHAVIOUR (GLOBAL RULE)

`img { filter: grayscale(1) contrast(1.05); transition: filter 700ms ease, transform 700ms ease; }`
ON HOVER OF THE IMAGE WRAPPER: `filter: grayscale(0); transform: scale(1.05);`
WRAPPERS ARE `overflow:hidden` SO THE SCALE CROPS CLEANLY.

### LINK HOVER (GLOBAL RULE)

`a.link { text-decoration: none; }` ON HOVER → `text-decoration: line-through;
text-decoration-thickness: 2px;` NAV / UTILITY LINKS ARE `text-transform:uppercase;
font-size:12px; letter-spacing:0.08em;`.

### HEADER

`position:fixed; top:0; inset-inline:0; z-index:40; mix-blend-mode:difference; color:#fff;`
(BECAUSE OF DIFFERENCE BLEND, WHITE TEXT RENDERS BLACK OVER THE CONCRETE BG AND WHITE OVER
DARK IMAGERY). PADDING ~32PX. LEFT: LOGO `HEAVYWEIGHT®` IN ANTON ~`clamp(28px,4vw,40px)`,
WITH A 10PX `letter-spacing:0.2em` SUB-LABEL `EST. 2024 — NYC`. RIGHT: A VERTICAL FLEX
COLUMN OF NAV LINKS (`INDEX / ESSENTIALS / ARCHIVE / CONTACT`) PUSHED IN WITH `margin-right:8rem`
(MR-32) SO IT CLEARS THE FLOATING CART AND CENTRAL HERO COPY.

### FLOATING CART

`position:fixed; top:24px; right:24px; z-index:50; width:64px; height:64px; border-radius:9999px;
background:#fff; color:var(--ink); display:grid; place-items:center;
box-shadow:0 10px 30px rgba(0,0,0,0.1); transition:transform .25s ease;`
HOVER → `transform:scale(1.1);`. CONTAINS AN INLINE SVG BAG ICON + AN ABSOLUTE RED BADGE
(`#dc2626`, WHITE 10PX, TOP-RIGHT, MIN 18PX CIRCLE) SHOWING THE ITEM COUNT (START `03`).
THE COUNT INCREMENTS WHEN ANY "ADD" / INVENTORY ITEM IS CLICKED (SEE JS), WITH A QUICK
SCALE-PULSE ON THE BADGE.

### HERO

FULL-HEIGHT (`min-height:100vh`) RELATIVE SECTION. RIGHT-SIDE IMAGE IS `position:absolute;
right:0; top:0; width:65%; height:100%; object-fit:cover; object-position:left;` CROPPED OFF
THE RIGHT EDGE. LEFT COPY SITS IN A `width:42%` BLOCK PUSHED LOW (`align-items:flex-end;
padding-bottom:8vh`): HEADLINE `HEAVY / DUTY / GEAR` STACKED, `font-size:clamp(64px,12vw,180px)`
(TEXT-9XL SCALE), LEADING 0.85. BELOW: A `max-width:320px` SPACE-GROTESK PARAGRAPH, THEN A
MINIMALIST CTA = UPPERCASE 12PX LABEL WITH A 1PX UNDERLINE THAT GROWS ON HOVER (`→ ENTER INDEX`).
A VERTICAL LABEL `COLLECTION 004` IS `position:absolute` AT THE IMAGE'S BOTTOM-LEFT,
`transform:rotate(-90deg); transform-origin:left bottom;` MONOSPACE 10–12PX TRACKING-WIDEST,
WITH A SMALL RED LIVE DOT.

### HORIZONTAL PRODUCT STRIP ("ESSENTIALS")

A `display:flex; overflow-x:auto; scroll-snap-type:x mandatory;` ROW WITH HIDDEN SCROLLBARS
(`scrollbar-width:none; ::-webkit-scrollbar{display:none}`). EACH CARD: `flex:0 0 clamp(260px,28vw,360px);
scroll-snap-align:start;` IMAGE WRAPPER `aspect-ratio:3/4; position:relative; overflow:hidden;`
HOLDING A PRIMARY IMG + A SECONDARY IMG (`position:absolute; inset:0; opacity:0; transition:opacity .5s;`)
THAT FADES TO `opacity:1` ON CARD HOVER. METADATA ROW BELOW: BOLD UPPERCASE TITLE (SPACE GROTESK 700)
ON THE LEFT, MONOSPACE PRICE (`$140`) ON THE RIGHT, PLUS A SMALL `+ ADD` LINK THAT INCREMENTS THE CART.
A DETAIL BADGE (e.g. `500GSM`) SITS TOP-RIGHT ON THE IMAGE.

### TECHNICAL INFO GRID

`display:grid; grid-template-columns:repeat(12,1fr); gap:24px;`. LEFT COLUMN = `grid-column:span 4;
position:sticky; top:120px; align-self:start;` WITH A MASSIVE ANTON HEADER (`CON— / STRUC— / TION`),
AND A STACK OF `border-left:2px solid var(--ink); padding-left:16px;` "ARCHITECTURE" SPEC BLOCKS
(LABEL + SHORT TECHNICAL PARAGRAPH). RIGHT = `grid-column:span 8;` AN INNER GRID OF IMAGES WITH
MIXED `aspect-ratio:4/5` AND `1/1`, EACH WITH A BOTTOM-LEFT DETAIL BADGE (`GARMENT DYED`, `YKK®`, etc.).

### LIVE INVENTORY GRID ("BROKEN" GRID)

`grid-template-columns:repeat(12,1fr);` ITEMS USE VARYING SPANS (`span 6`, `span 3`) AND PER-ITEM
`margin-top` OFFSETS (e.g. `mt-24` = 6rem ON SELECTED ITEMS) FOR A STAGGERED EDITORIAL RHYTHM.
EACH ITEM: IMAGE + TITLE + MONOSPACE PRICE + A ROW OF SIZE PILLS `S M L XL`; OUT-OF-STOCK SIZES
RENDER WITH `text-decoration:line-through; opacity:0.4;` IN MONOSPACE. A `● LIVE` RED INDICATOR
AND A LIVE STOCK COUNTER ("12 LEFT") THAT TICKS DOWN OCCASIONALLY VIA JS.

### MASONRY SOCIAL PROOF

`columns:3; column-gap:24px;` (RESPONSIVE DOWN TO 1). EACH CARD `break-inside:avoid;
margin-bottom:24px;` ALTERNATING SOLID (`background:var(--concrete); border:1px solid var(--hairline)`)
AND OUTLINED (`background:transparent; border:1px solid var(--ink)`). EVERY CARD IS SLIGHTLY ROTATED
(`transform:rotate(-1deg)` / `rotate(2deg)` ALTERNATING) LIKE PASTED POSTERS; HOVER STRAIGHTENS TO
`rotate(0)`. QUOTE TEXT IS MONOSPACE; HANDLE ROW = SMALL `●` DOT + `@handle` MONOSPACE.

### FOOTER

`background:var(--ink); color:#fff;` PADDING-TOP ~96PX. A `grid-template-columns:repeat(6,1fr)`
LINK GRID (INDEX / SUPPORT / LEGAL / SOCIAL COLUMNS + A NEWSLETTER INPUT WITH AN UNDERLINE SUBMIT).
THE LAST ELEMENT IS A SINGLE WORD `HEAVY` IN ANTON AT `font-size:10vw; line-height:0.8;
color:rgba(255,255,255,0.1);` SPANNING FULL WIDTH AS A STRUCTURAL ANCHOR, SLIGHTLY CLIPPED AT
THE BOTTOM EDGE.

### MICRO-INTERACTIONS / JS (`script.js`)

1. CART COUNT STATE — CLICKING ANY `[data-add]` (STRIP `+ ADD`, INVENTORY ITEMS) INCREMENTS THE
   BADGE NUMBER (ZERO-PADDED TO 2 DIGITS) AND TRIGGERS A `transform:scale(1.4)` PULSE THAT EASES BACK.
2. SCROLL REVEAL — `IntersectionObserver` ADDS `.in` TO SECTIONS / CARDS FOR A
   `translateY(24px)→0 + opacity 0→1` ENTRANCE (DURATION ~700MS, `cubic-bezier(.16,1,.3,1)`),
   STAGGERED PER CHILD VIA `transition-delay`.
3. LIVE STOCK TICKER — A `setInterval` (~4s) RANDOMLY DECREMENTS ONE INVENTORY ITEM'S "N LEFT"
   COUNTER AND BRIEFLY FLASHES THE RED `● LIVE` DOT.
4. HORIZONTAL STRIP — SUPPORTS DRAG-TO-SCROLL (POINTER DOWN/MOVE) IN ADDITION TO NATIVE WHEEL/SNAP.
5. RESPECT `prefers-reduced-motion` BY DISABLING THE GRAIN FLICKER + SCROLL-REVEAL TRANSFORMS.

### IMAGERY

VENDOR ~8–10 EDITORIAL APPAREL / TECHNICAL-GEAR PHOTOS LOCALLY INTO `assets/img/`
(HERO, ESSENTIALS PRIMARY + SECONDARY, CONSTRUCTION MACROS, INVENTORY). ALL RENDER GRAYSCALE
BY DEFAULT AND REVEAL COLOR ON HOVER PER THE GLOBAL IMAGE RULE.
