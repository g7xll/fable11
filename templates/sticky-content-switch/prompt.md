## EXPANDED SPECIFICATION

### CONCEPT

A TWO-COLUMN, SCROLL-DRIVEN EDITORIAL SECTION. THE **LEFT COLUMN IS STICKY** (PINNED, VERTICALLY CENTERED IN THE VIEWPORT) AND HOLDS THE ACTIVE PROJECT'S TEXT BLOCK: AN INDEX NUMBER (`01 / 06`), A LARGE DISPLAY HEADING, A KICKER/CATEGORY LABEL, A SHORT PARAGRAPH, AND A META ROW. THE **RIGHT COLUMN IS A TALL STACK OF FULL-HEIGHT IMAGE PANELS** THAT SCROLL PAST. WHENEVER A NEW IMAGE CROSSES THE VERTICAL CENTER OF THE VIEWPORT, THE STICKY LEFT-SIDE TEXT SWAPS TO THAT IMAGE'S CONTENT WITH A CROSSFADE + VERTICAL SLIDE TRANSITION. A THIN PROGRESS BAR AND ANIMATED INDEX TRACK THE ACTIVE ITEM.

### TECH / CONSTRAINTS

- PLAIN, SELF-CONTAINED **HTML + CSS + VANILLA JS** (NO BUILD STEP, NO FRAMEWORK). SERVES AS A STATIC `INDEX.HTML`.
- ALL ASSETS VENDORED LOCALLY UNDER `ASSETS/` (FONTS + IMAGES). FULLY OFFLINE.
- DESKTOP-FIRST, RESPONSIVE DOWN TO MOBILE (BELOW 900PX THE TWO COLUMNS STACK; THE HEADING SITS ABOVE EACH IMAGE AS A SIMPLE STACKED FEED, STILL UPDATING ON SCROLL).

### DESIGN LANGUAGE

- COLORS: NEAR-BLACK INK `#0E0E0C` ON A WARM PAPER `#F4F1EA` BASE; MUTED GRAPHITE `#6B6862` FOR META; A SINGLE ACCENT `#C4622D` (BURNT ORANGE) FOR THE PROGRESS FILL, ACTIVE INDEX, AND HOVER UNDERLINES.
- TYPE: A TIGHT GROTESQUE/ANTON-STYLE DISPLAY FACE FOR THE HEADING (VENDORED **ANTON**) AND **INTER** FOR BODY/META. HEADING IS UPPERCASE, ~6.5VW, LINE-HEIGHT 0.92, LETTER-SPACING -0.02EM.
- A FIXED ~4% SVG FRACTAL-NOISE GRAIN OVERLAY ACROSS THE PAGE.
- MOTION SIGNATURE: `CUBIC-BEZIER(0.16, 1, 0.3, 1)` FOR ALL EASES.

### LAYOUT

1. **FIXED TOP NAV** — A SLIM ROW: WORDMARK "STUDIO INDEX" LEFT, A CENTER LABEL "SELECTED WORK — 2026", AND A RIGHT MENU PILL. MIX-BLEND-DIFFERENCE SO IT STAYS LEGIBLE OVER PAPER AND IMAGES.
2. **INTRO HERO** (100VH) — GIANT "STICKY / CONTENT SWITCH" SPLIT HEADLINE WITH A PER-LETTER RISE-REVEAL ON LOAD, A ONE-LINE SUBHEAD, AND AN ANIMATED "SCROLL" PILL THAT BOBS.
3. **THE SWITCH SECTION** — A FLEX/GRID ROW:
   - LEFT (40%): `POSITION: STICKY; TOP: 0; HEIGHT: 100VH;` FLEX-CENTERED CONTENT. THE TEXT BLOCK IS ABSOLUTELY STACKED; ONLY THE ACTIVE ITEM IS VISIBLE (OPACITY 1, `TRANSLATEY(0)`), OTHERS ARE `OPACITY 0, TRANSLATEY(±24PX)`. THE SWAP TRANSITION IS ~0.6S.
   - RIGHT (60%): A COLUMN OF SIX `100VH` IMAGE PANELS. EACH IMAGE IS `OBJECT-FIT: COVER` INSIDE A ROUNDED (28PX) FRAME WITH A SUBTLE INNER PARALLAX (IMAGE SCALES 1.08 → 1.0 AS IT ENTERS, VIA SCROLL-LINKED TRANSFORM) AND GRAYSCALE-TO-COLOR ON ACTIVE.
   - A FIXED VERTICAL PROGRESS RAIL ON THE FAR LEFT EDGE: SIX TICKS + A FILLING ACCENT BAR; THE ACTIVE TICK GROWS AND TURNS ACCENT.
4. **OUTRO / FOOTER** — A LARGE "INDEX COMPLETE" CTA AND A GIANT GHOSTED WORDMARK.

### THE CORE INTERACTION (LOAD-BEARING JS)

THE SWITCH IS DRIVEN BY AN `INTERSECTIONOBSERVER` WHOSE `ROOTMARGIN` COLLAPSES THE VIEWPORT TO A 1PX BAND AT THE VERTICAL CENTER, SO AN IMAGE BECOMES "ACTIVE" EXACTLY WHEN ITS MIDPOINT CROSSES SCREEN CENTER:

```js
const panels = [...document.querySelectorAll('.panel')];
const items  = [...document.querySelectorAll('.switch-item')];
const ticks  = [...document.querySelectorAll('.rail-tick')];
let active = -1;

function setActive(i){
  if (i === active) return;
  active = i;
  items.forEach((el,n)=> el.classList.toggle('is-active', n===i));
  panels.forEach((el,n)=> el.classList.toggle('is-active', n===i));
  ticks.forEach((el,n)=> el.classList.toggle('is-active', n<=i));
  document.querySelector('.rail-fill').style.transform =
    `scaleY(${(i+1)/panels.length})`;
}

// activate when a panel's midpoint crosses viewport center
const io = new IntersectionObserver((entries)=>{
  entries.forEach(e=>{
    if (e.isIntersecting) setActive(panels.indexOf(e.target));
  });
}, { rootMargin: '-50% 0px -50% 0px', threshold: 0 });
panels.forEach(p=> io.observe(p));
```

A SECOND `RAF` SCROLL LOOP APPLIES THE PER-PANEL PARALLAX SCALE BASED ON EACH PANEL'S DISTANCE FROM CENTER, AND DRIVES THE STICKY HEADLINE'S SUBTLE Y-DRIFT. THE INDEX NUMBER ANIMATES BY SWAPPING `01 → 02 …` IN SYNC WITH `SETACTIVE`.

### SUBTLE DETAILS

- HEADLINE SWAP: OLD TEXT FADES/SLIDES UP (`-24PX`), NEW TEXT RISES FROM `+24PX`, 0.6S `CUBIC-BEZIER(0.16,1,0.3,1)`, STAGGERED ~60MS PER SUB-ELEMENT (INDEX, KICKER, TITLE, BODY).
- ACTIVE IMAGE: `FILTER: GRAYSCALE(0)`, INACTIVE: `GRAYSCALE(0.55) BRIGHTNESS(0.85)`, 0.6S TRANSITION.
- PROGRESS FILL USES `TRANSFORM: SCALEY()` WITH `TRANSFORM-ORIGIN: TOP` SO IT GROWS SMOOTHLY.
- LOAD: PER-LETTER HERO REVEAL, THEN A SOFT FADE-IN OF NAV AND SCROLL PILL.
- RESPECT `PREFERS-REDUCED-MOTION`: DISABLE PARALLAX AND LETTER STAGGER, KEEP INSTANT SWAPS.
- VENDOR ANTON + INTER WOFF2 LOCALLY; VENDOR SIX HIGH-QUALITY IMAGES LOCALLY (PROCEDURALLY GENERATED IF DOWNLOADS UNAVAILABLE).
