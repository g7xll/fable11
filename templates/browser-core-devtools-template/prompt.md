- A SIMULATED UI WINDOW CONTAINS THREE STACKED `.card` ELEMENTS. A CSS-DRIVEN FAKE CURSOR MOVES BETWEEN THEM ON A LOOP WITH `ease-in-out`, AND A DARK INSPECTOR TOOLTIP (#111827) REVEALS CSS PROPS IN CYAN MONO AS IT LANDS ON EACH CARD.
```css

  0%   {top:48px;  left:40px}
  22%  {top:48px;  left:60%}     /* hover card 1 */
  33%  {top:48px;  left:60%}
  55%  {top:150px; left:55%}     /* hover card 2 */
```
