# Browser-Core DevTools

A high-density inspector that turns any tab into a workspace. Built around a strict 4px / 8px grid, monospaced data labels, and sharp cyan highlights.

## Features

  • Pattern-grid canvas with live pixel-dimension readouts
  • Property inspector: typography, colors, layout
  • Simulated selection box with element tags
  • Command palette + 24 keyboard bindings

## Install

Add the extension, then drop the loader into your project:

$ npm i @browser-core/devtools

Technical FAQ

Questions, answered.

How does the inspector measure elements?+
It reads the layout box via getBoundingClientRect() and snaps overlays to the 20px pattern grid. Dimension labels update on every reflow, so the readout always matches the live DOM.
Does it work offline?+
Yes. The workspace is a fully self-contained static bundle — HTML, CSS, vanilla JS, and locally vendored fonts. No CDN or build step required.
What browsers are supported?+
Any Chromium-based browser plus modern Firefox and Safari. Custom ::-webkit-scrollbar styling is progressive — Firefox falls back to scrollbar-width.
Can I theme the accent color?+
All colors flow from CSS custom properties. Override --accent in your root and every active state, ring, and highlight updates at once.
