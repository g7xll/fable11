# Superdesign Editorial Grid — Editorial Tech UI Template (HTML + CSS + Vanilla JS)

A sophisticated editorial-tech landing page template suited to premium SaaS products, design tools, and technical agencies that want a high-fashion, editorial look over a muted organic palette. Playfair Display serif handles large expressive headers, Space Grotesk sans covers body text, and Space Mono fills all technical labels and buttons — over cream (`#F7F6F2`), charcoal (`#1C1C1C`), and forest-green accent (`#3D7068`). A fixed background layer renders vertical guides at 25/50/75% width plus a 40px square grid masked with a radial fade. Sections include a floating-to-fixed blurred nav, a 9vw serif hero with a pulse-dot badge and tracking-expanding CTA, a three-column statistics grid, a scroll-synced word-reveal section (per-span opacity 0.15→1.0), an interactive workflow with a sticky grayscale card and a `#3B82F6` scan-line progress bar, use-case tabs with a 240px ghost glyph background, and a bottom-bordered contact form. All animations use `cubic-bezier(0.16, 1, 0.3, 1)` at 700–1000ms; border-radius never exceeds 2px. Delivered as self-contained static HTML with no build step. Generated with Claude Fable 5.

## Run

This is a static project — open `index.html` in a browser, or serve the folder:

```sh
python3 -m http.server 8000
```

See `prompt.md` for the full build spec; `demo.mp4` shows it in motion.
