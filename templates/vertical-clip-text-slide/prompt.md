## THE CORE MOTION — VERTICAL CLIP, LETTER BY LETTER

EACH HEADLINE LINE IS SPLIT INTO INDIVIDUAL LETTER SPANS BY JS ON LOAD. WHITESPACE
BECOMES A FIXED-WIDTH GAP. EACH LETTER IS WRAPPED IN A **CLIP COLUMN**:

- THE OUTER WRAPPER (`.CLIP`) IS `DISPLAY: INLINE-BLOCK; OVERFLOW: HIDDEN;
  VERTICAL-ALIGN: TOP;` — THIS IS THE VERTICAL MASK. ITS HEIGHT IS THE LINE'S
  CAP HEIGHT (`1EM`), SO NOTHING IS VISIBLE ABOVE OR BELOW IT.
- THE INNER GLYPH (`.GLYPH`) STARTS TRANSLATED **UP AND OUT OF THE MASK**
  (`TRANSFORM: TRANSLATEY(-110%)`) AND ANIMATES **DOWN** TO `TRANSLATEY(0)`.
  BECAUSE THE WRAPPER CLIPS OVERFLOW, THE LETTER APPEARS TO POUR/SLIDE DOWN INTO
  ITS SLOT FROM THE TOP EDGE.
- STAGGER: EACH LETTER'S `ANIMATION-DELAY` = `BASE + INDEX * 70MS`, COUNTED
  CONTINUOUSLY ACROSS ALL LINES (LINE 2 CONTINUES THE COUNT FROM LINE 1), SO THE
  WHOLE HEADLINE READS AS ONE LEFT-TO-RIGHT WAVE.
- PER-LETTER DURATION: `0.62S`. EASING: `CUBIC-BEZIER(0.22, 1, 0.36, 1)`
  (AN EXPO-OUT — FAST DROP, SOFT SETTLE).
- A SUBTLE SECONDARY TOUCH: THE GLYPH ALSO FADES `OPACITY 0 → 1` OVER THE FIRST
  60% OF ITS SLIDE, AND CARRIES A HAIR OF BLUR (`FILTER: BLUR(6PX) → BLUR(0)`)
  THAT CLEARS AS IT SETTLES.
- AFTER THE LAST LETTER LANDS, THE SUB-LINE AND BOTTOM CAPTION FADE/SLIDE UP
  (`OPACITY 0→1`, `TRANSLATEY(12PX)→0`, `0.6S`, SAME EASING).

### LOOP / REPLAY BEHAVIOR

- ON LOAD THE CASCADE PLAYS ONCE.
- IT THEN **AUTO-LOOPS**: AFTER THE LAST LETTER SETTLES, HOLD ~`2.2S`, THEN
  REWIND (CLEAR THE PLAYED STATE) AND REPLAY THE WHOLE CASCADE — INDEFINITELY.
- THE "REPLAY" BUTTON RESTARTS THE CASCADE IMMEDIATELY AND RESETS THE LOOP TIMER.
- RESPECT `PREFERS-REDUCED-MOTION`: IF SET, SKIP THE SLIDE/BLUR AND JUST SHOW THE
  HEADLINE STATICALLY (NO LOOP).

## CORE CODE (ESSENTIAL SNIPPETS)

### CLIP + GLYPH MARKUP (PRODUCED BY JS PER LETTER)

```html
<span class="line">
  <!-- one of these per letter; data-i is the global stagger index -->
  <span class="clip"><span class="glyph" style="--i:3">M</span></span>
</span>
```

### THE KEYFRAMES + MASK CSS

```css
.clip {                 /* the vertical mask column */
  display: inline-block;
  overflow: hidden;
  vertical-align: top;
  line-height: 0.92;
}
.glyph {
  display: inline-block;
  will-change: transform, opacity, filter;
  transform: translateY(-110%);
  opacity: 0;
  filter: blur(6px);
}
.play .glyph {
  animation: clip-drop 0.62s cubic-bezier(0.22, 1, 0.36, 1) both;
  animation-delay: calc(var(--base, 0.2s) + var(--i) * 70ms);
}
@keyframes clip-drop {
  0%   { transform: translateY(-110%); opacity: 0; filter: blur(6px); }
  60%  { opacity: 1; }
  100% { transform: translateY(0);     opacity: 1; filter: blur(0); }
}
```

### SPLIT + STAGGER + LOOP (VANILLA JS)

```js
let gi = 0;                                   // global letter index
document.querySelectorAll('.line').forEach((line) => {
  const text = line.textContent;
  line.textContent = '';
  [...text].forEach((ch) => {
    if (ch === ' ') { line.append(Object.assign(document.createElement('span'),
      { className: 'space', textContent: ' ' })); return; }
    const clip = document.createElement('span'); clip.className = 'clip';
    const g = document.createElement('span'); g.className = 'glyph';
    g.style.setProperty('--i', gi++); g.textContent = ch;
    clip.append(g); line.append(clip);
  });
});
const TOTAL = gi, LAST = 200 + TOTAL * 70 + 620;       // ms until settled
const stage = document.querySelector('.stage');
function play() { stage.classList.remove('play'); void stage.offsetWidth;
  stage.classList.add('play'); }
function loop() { play(); setTimeout(loop, LAST + 2200); }
loop();
document.querySelector('.replay').addEventListener('click', () => { /* reset timer + */ play(); });
```
