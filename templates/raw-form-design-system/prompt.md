# RAW FORM — BRUTALIST E-COMMERCE DESIGN SYSTEM

## SUMMARY

BUILD **RAW FORM** (BRAND: **STUDIO/RAW**), A HIGH-CONTRAST, BRUTALIST E-COMMERCE STOREFRONT / DESIGN SYSTEM THAT PRIORITIZES TYPOGRAPHIC DRAMA AND ATMOSPHERIC DEPTH. IT REPLACES TRADITIONAL CONTAINER-HEAVY UI WITH MASSIVE TYPE BREAKS, SOFT ORGANIC GRADIENT BLOBS, AND A SHARP, GRID-BASED HIERARCHY. THE FINAL RENDER MUST BE SAME-TO-SAME (PIXEL/LAYOUT PARITY) WITH THE REFERENCE DESIGN.

THE PAGE IS A SINGLE, VERTICALLY-SCROLLING STORE LANDING PAGE WITH THESE SECTIONS IN ORDER: FIXED NAV → TYPOGRAPHIC HERO → "OUTERWEAR" CATEGORY DIVIDER → BORDERLESS PRODUCT GRID (WITH "FULL ARCHIVE" TILE) → "DESIGNED FOR MOVEMENT" CAMPAIGN BLOCK → "OBJECTS" CATEGORY DIVIDER → DARK FOOTER.

## STYLE

THE STYLE BLENDS THE RAW ENERGY OF SWISS BRUTALISM WITH A REFINED, WARM COLOR PALETTE. ANIMATIONS ARE SMOOTH, USING CUBIC-BEZIER CURVES FOR A PREMIUM FEEL.

## LAYOUT & STRUCTURE

### NAVIGATION (FIXED)

- FIXED TOP, FULL WIDTH, Z-50, HORIZONTAL PADDING ~1.5–2REM, VERTICAL PADDING ~1.5REM. TRANSPARENT OVER THE BACKGROUND.
- LEFT: BRAND WORDMARK **STUDIO/RAW** IN CLASH DISPLAY, BOLD, UPPERCASE, ~XL/2XL, TRACKING TIGHT.
- CENTER: TEXT LINKS **COLLECTION · CAMPAIGN · ARCHIVE** IN SATOSHI MEDIUM, ~14PX, UPPERCASE, LETTER-SPACING ~0.1EM. HIDDEN ON MOBILE.
- RIGHT: ICON SET — SEARCH AND SHOPPING BAG (LUCIDE, 20PX). THE BAG SHOWS A SMALL ACCENT-RED COUNT DOT (`-top-1 -right-1`).

### HERO SECTION

- FULL VIEWPORT HEIGHT (`min-h-screen`), FLEX, CONTENT VERTICALLY CENTERED, RELATIVE, OVERFLOW HIDDEN.
- BACKGROUND: THREE LARGE BLURRED BLOBS (SEE ANIMATED GRADIENT BLOBS) IN `#DB4A2B` AND `#F8A348`, BEHIND THE TYPE, `mix-blend-mode: multiply`, SLOW PULSE.
- HEADLINE: TWO LINES IN CLASH DISPLAY, BOLD (700), UPPERCASE, `font-size: 18vw` (~259PX AT 1440W), `line-height: 0.75`, TRACKING `-0.05em` (≈ -12.96PX). LINE 1 = **RAW**; LINE 2 = **FORM.** INDENTED `margin-left: 10vw` (`md:15vw`) AS ITS OWN BLOCK. THE PERIOD IN "FORM." IS PART OF THE WORD.
- SUBTEXT: SATOSHI, ~20PX, MAX-WIDTH ~400PX: "FUNCTIONALITY STRIPPED TO ITS CORE. NO EXCESS. JUST PURE UTILITY FOR THE MODERN NOMAD."
- CTAS (ROW): PRIMARY = INTERACTIVE CTA BUTTON LABELLED **SHOP NOW**; SECONDARY = TEXT LINK **EXPLORE LOOKBOOK** IN `#DB4A2B` WITH A TRAILING ARROW ICON.

### CATEGORY DIVIDERS ("OUTERWEAR" AND "OBJECTS")

ALTERNATE ALIGNMENT (E.G. OUTERWEAR LEFT-ALIGNED, OBJECTS RIGHT/CENTER) FOR RHYTHM, MATCHING THE REFERENCE (BOTH RENDER ROUGHLY CENTER-FILLING THE VIEWPORT WIDTH).

## SPECIAL COMPONENTS

### INTERACTIVE CTA BUTTON

A HIGH-CONTRAST BUTTON WITH A DIRECTIONAL FILL ANIMATION. RECTANGULAR, NO BORDER-RADIUS. BACKGROUND `#1E1E1E`, TEXT `#E4E2DD`. ON HOVER, A WHITE (`#FFFFFF`) BACKGROUND SLIDES LEFT-TO-RIGHT (`transform: scaleX(0)`→`scaleX(1)` FROM THE LEFT, OR TRANSLATE -100%→0) BEHIND THE TEXT; TEXT COLOR TRANSITIONS TO `#DB4A2B`. DURATION 0.3S EASE-OUT.

### ANIMATED GRADIENT BLOBS

SOFT ORGANIC SHAPES PROVIDING DEPTH BEHIND TEXT. ABSOLUTELY-POSITIONED DIVS, `width: 60vw; height: 60vw` (REFERENCE USES VARIED SIZES UP TO ~50VW), FULLY ROUNDED, COLORS `#DB4A2B` AND `#F8A348`, `filter: blur(140px)` (100PX+), `mix-blend-mode: multiply`. ANIMATE A SLOW PULSE (OPACITY ~0.6→0.9) WITH SLIGHT TRANSLATION OVER 10–15S. POSITIONS ROUGHLY: `top-[10%] left-[20%]`, `top-[40%] right-[10%]`, `bottom-[10%] left-[40%]`.
