# Raw Form — Brutalist E-Commerce Design System (Vanilla HTML, CSS, Clash Display)

A high-contrast brutalist e-commerce storefront and design system for a fictional apparel label, **Studio/Raw**, that replaces container-heavy UI with typographic drama, atmospheric gradient blobs, and a sharp grid-based hierarchy. The warm palette pairs base cream (`#E4E2DD`) and ink (`#1E1E1E`) with an accent red-orange (`#DB4A2B`), warm orange, and soft pink. Clash Display drives all headings — hero at `18vw`, 0.75 line-height, tight tracking — while Satoshi handles body, nav, and metadata. The vertically-scrolling page covers a fixed transparent nav, a blob-backdrop typographic hero, two category dividers, a borderless product grid, a campaign block, and a dark footer with a `10vw` background year, all with an 0.8s `cubic-bezier(0.16, 1, 0.3, 1)` slide-up entrance. Generated with Claude Fable 5.

## Run

This is a static project — open `index.html` in a browser, or serve the folder:

```sh
python3 -m http.server 8000
```

See `prompt.md` for the full build spec; `demo.mp4` shows it in motion.
