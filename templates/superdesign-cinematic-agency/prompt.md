# SUPERDESIGN — CINEMATIC AGENCY DESIGN SYSTEM

## SUMMARY

BUILD **SUPERDESIGN** (BRAND WORDMARK: **SUPERDESIGN**), AN IMMERSIVE, DARK-MODE CREATIVE-AGENCY LANDING PAGE / DESIGN SYSTEM THAT USES 3D SPATIAL DEPTH AND HIGH-CONTRAST EDITORIAL TYPOGRAPHY TO CREATE A THEATRICAL, CINEMATIC EXPERIENCE. IT BLENDS BRUTALIST LAYOUT STRUCTURES (HEAVY BORDERS, MASSIVE TYPE) WITH FLUID, HIGH-TECH MOTION. THE FINAL RENDER MUST BE SAME-TO-SAME (PIXEL/LAYOUT PARITY) WITH THE REFERENCE DESIGN.

THE PAGE IS A SINGLE, VERTICALLY-SCROLLING SITE WITH THESE SECTIONS IN ORDER: FIXED BLEND-MODE NAV → 3D ROLODEX CUBE HERO (WITH MASSIVE "SUPERDESIGN" BACKGROUND TYPE) → "CASE STUDIES" BENTO PROJECT GRID → "PLANS THAT GROW WITH YOU" THREE-TIER PRICING (DARK PURPLE, CENTRAL GLOW) → "STARTUPS: FIRST YEAR IS FREE" BANNER → "ESTIMATE YOUR GROWTH PLAN PRICE" GLASS CALCULATOR → MASSIVE-SCALE EDITORIAL FOOTER ("LET'S BUILD THE IMPOSSIBLE.").

## STYLE

A BASE OF NEAR-BLACK `#050505` FOR DEPTH. TYPOGRAPHY IS THE HERO: **ASPEKTA** (WEIGHTS 300 / 500 / 700 / 900), `font-weight: 900` FOR DISPLAY HEADINGS, `font-weight: 700` WITH WIDE TRACKING (`0.2em`) FOR SECONDARY LABELS. ALL HEADINGS ARE UPPERCASE. COLOR ACCENTS ARE MINIMAL BUT PUNCHY: A CYAN→PINK GRADIENT FOR PRIMARY BUTTONS, AND A PURPLE GLOW FOR THE PRICING SECTION. INTERACTIVE ELEMENTS USE `backdrop-filter: blur(8px)` AND `1px solid rgba(255,255,255,0.1)` BORDERS — **NO DROP SHADOWS**; DEPTH COMES FROM BORDERS + BLUR.

### FONTS (VENDOR LOCALLY)

- **ASPEKTA** — WEIGHTS 300, 500, 700, 900 — USED FOR EVERYTHING. VENDOR THE WOFF2 FILES LOCALLY UNDER `assets/fonts/` AND SERVE VIA `@font-face`; NO REMOTE FONT HOTLINKS IN THE FINAL BUILD. FALLBACK STACK: `Aspekta, -apple-system, "system-ui", "Segoe UI", Roboto, sans-serif`.

- BODY BACKGROUND `#050505`, TEXT `#FFFFFF`, FONT-FAMILY ASPEKTA.
- ALL HEADINGS UPPERCASE FOR THE CINEMATIC EDITORIAL FEEL.

## LAYOUT & STRUCTURE

### NAVIGATION (FIXED, BLEND-MODE)

- FIXED `top: 0`, FULL WIDTH, Z-50, PADDING `1.5rem 2rem`. NO SOLID BACKGROUND.
- **ESSENTIAL:** APPLY `mix-blend-difference` TO THE ENTIRE NAV CONTAINER SO IT STAYS LEGIBLE OVER SHIFTING BACKGROUND COLORS.
- LEFT: WORDMARK **SUPERDESIGN** (ASPEKTA 900, UPPERCASE) PRECEDED BY A SMALL **RED-500 PULSING DOT** (`#EF4444`, gentle opacity/scale pulse) FOR A "LIVE" STATUS FEEL.
- CENTER: NAV LINKS **HOME · ABOUT · PAGES · PORTFOLIO** (UPPERCASE, ~14PX, TRACKING ~0.1EM). HIDDEN ON MOBILE.
- RIGHT: PILL CTA **GET STARTED** WITH THE CYAN→PINK ACCENT GRADIENT BACKGROUND, ROUNDED-FULL, DARK/WHITE TEXT, SCALE 1.02 ON HOVER.

### 3D ROLODEX CUBE HERO

- FULL VIEWPORT HEIGHT (`100vh`), RELATIVE, OVERFLOW HIDDEN, `perspective: 2000px` ON THE HERO CONTAINER (KEEP THIS — IT DRIVES THE 3D DEPTH).
- BACKGROUND: MASSIVE CENTERED WORDMARK **SUPERDESIGN** SET IN ASPEKTA 900, UPPERCASE, FILLING THE VIEWPORT WIDTH (~`11vw`–`12vw`, EFFECTIVELY ONE GIANT LINE), `#FFFFFF` AT LOW OPACITY (~0.3, INCREASING/FADING ON SCROLL), SITTING BEHIND THE CUBE.
- FOREGROUND: A 3D **ROLODEX CUBE** — A `transform-style: preserve-3d` "CUBE-WRAPPER" CENTERED IN THE VIEWPORT, ~`50vh × 50vh`. FOUR FACES, EACH ROTATED ON THE X-AXIS AND PUSHED OUT ON Z:
  - FRONT: `rotateX(0deg) translateZ(30vh)`
  - BOTTOM: `rotateX(-90deg) translateZ(30vh)`
  - BACK: `rotateX(-180deg) translateZ(30vh)`
  - TOP: `rotateX(90deg) translateZ(30vh)`
  - (THE REFERENCE USES A LARGER `translateZ(55vw)` AT WIDE BREAKPOINTS — SCALE THE Z OFFSET RESPONSIVELY SO THE FACES STAY EDGE-TO-EDGE OF THE CUBE.)
- EACH FACE: A HIGH-QUALITY `object-fit: cover` IMAGE WITH A CENTERED UPPERCASE TITLE OVERLAY (ASPEKTA 900, TRACKING `0.2em`). FACE LABELS: **STRATEGY · DEVELOP · DESIGN · MARKETING** (DEVELOP IS THE FRONT FACE IN THE REFERENCE).
- MOTION: A CONSTANT, SMOOTH **INFINITE ROTATION ON THE X-AXIS** — `@keyframes { to { transform: rotateX(360deg); } }` ON THE CUBE-WRAPPER (CSS ANIMATION OR GSAP), SO THE FACES TUMBLE LIKE A ROLODEX.
- BELOW THE CUBE: A THIN VERTICAL LINE + **SCROLL DOWN** LABEL (UPPERCASE, TRACKED).
