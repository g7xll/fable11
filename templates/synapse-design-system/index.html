// Core Principles

A system engineered for contrast and motion

Every surface, type pairing, and accent is tuned for a premium high-tech feel — built on a vantablack foundation.

Ambient Lighting

Floating violet and cyan orbs drift behind every surface, casting soft glowing blurs across the canvas.

Editorial Type

Instrument Serif for high-impact headings paired with crisp Inter for body and interface readability.

Neon Accents

A disciplined violet and cyan accent system delivers extreme contrast against the solid black base.

Glass Surfaces

High-blur glassmorphism — 16px and above — keeps chrome legible even over bright ambient orbs.

Motion Tokens

Snappy cubic-bezier easing replaces generic transitions for a technical, deliberate feel throughout.

Status Signals

Emerald operational indicators with pulsing dots communicate live system health at a glance.

// Integration

Drop the tokens straight into your stack

One config file defines the entire Synapse language — colors, fonts, and motion — ready to consume.

synapse.config.ts 
// Synapse design tokens — vantablack foundation
import { defineSystem } from '@synapse/core'

export const theme = defineSystem({
  colors: {
    bg:      '#030303',  // solid black
    violet:  '#8b5cf6',
    cyan:    '#06b6d4',
    emerald: '#10b981',  // status only
  },
  fonts: {
    heading: 'Instrument Serif',
    body:    'Inter',
  },
  motion: {
    ease:  'cubic-bezier(0.23, 1, 0.32, 1)',
    blur:  16,           // minimum glass blur
  },
})
