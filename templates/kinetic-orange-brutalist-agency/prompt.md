# KINETIC ORANGE — DIGITAL BRUTALIST AGENCY LANDING PAGE

## SUMMARY

BUILD **KINETIC ORANGE** (BRAND: **SUPERDESIGN®**), A DIGITAL-FIRST BRUTALIST AGENCY / PORTFOLIO LANDING PAGE THAT USES HEAVY TYPOGRAPHY AND CONTINUOUS MOTION TO CREATE URGENCY AND TECHNICAL SOPHISTICATION. IT RELIES ON A STRICT THREE-COLOR SYSTEM (ELECTRIC ORANGE, BLACK, WHITE), THICK 2PX BORDERS, SKEWED SECTIONS, AND LOOPING MARQUEES. THE FINAL RENDER MUST BE SAME-TO-SAME (PIXEL/LAYOUT PARITY) WITH THE REFERENCE DESIGN.

THE PAGE IS A SINGLE, VERTICALLY-SCROLLING LANDING PAGE WITH THESE SECTIONS IN ORDER: FLOATING PILL NAV → TYPOGRAPHIC HERO (ON ORANGE) → SKEWED BLACK MARQUEE BAND → DARK VERTICAL SERVICE LIST → GIANT "LET'S TALK" CTA (ON ORANGE) → FOOTER.

## STYLE

A HIGH-IMPACT DIGITAL BRUTALIST THEME. IT PAIRS A HEAVY BLACK-WEIGHT DISPLAY FACE FOR HEADLINES WITH THE TECHNICAL PRECISION OF SPACE MONO FOR METADATA AND UI LABELS. THE PRIMARY SURFACE IS VIBRANT ELECTRIC ORANGE. VISUALS ARE DEFINED BY THICK BORDERS, A SKEWED MARQUEE BAND, AND CONTINUOUS LINEAR ANIMATIONS. NO GRADIENTS, NO SOFT PASTELS.

### FONTS (LOAD FROM GOOGLE FONTS, OR VENDOR LOCALLY)

- **DISPLAY / HEADLINES**: A HEAVY BLACK-WEIGHT SANS — USE **ARCHIVO BLACK** (OR INTER / ARCHIVO NARROW AT WEIGHT 900) — ALWAYS UPPERCASE, `font-weight: 900`, `letter-spacing: -0.04em` (TRACKING TIGHTER), `line-height: 0.85–0.9`.
- **METADATA / UI LABELS / NAV / TAGS**: **SPACE MONO** — `letter-spacing: -0.02em`, OFTEN UPPERCASE.
- **BODY**: **INTER**.
- IMPORT: `https://fonts.googleapis.com/css2?family=Archivo+Black&family=Inter:wght@300;400;500;600;700;800;900&family=Space+Mono:wght@400;700&display=swap`.

### COLOR TOKENS (EXACT HEX)

- BRAND ORANGE (PRIMARY BG / ACCENT): `#FF4D00`
- SOLID BLACK (TEXT / DARK SECTIONS): `#000000`
- PURE WHITE (HIGH-CONTRAST TEXT ON BLACK): `#FFFFFF`

### GLOBAL RULES

- DEFAULT BODY BACKGROUND `#FF4D00`, DEFAULT TEXT `#000000`.
- SELECTION COLOR: `background: #000000; color: #FF4D00;`.
- HIDE SCROLLBARS (`scrollbar-width: none`, `::-webkit-scrollbar { display: none }`).
- BORDERS ARE SHARP: `2px solid #000000` FOR SECTION DIVIDERS AND BUTTONS; `1px` WHITE @ 20% FOR LIST-ITEM DIVIDERS ON DARK. NO ROUNDED CORNERS EXCEPT PILL-SHAPED NAV AND BUTTONS (`rounded-full`).
- MOTION: LINEAR INFINITE MARQUEES FOR TEXT BANDS; A LINEAR `spin` (360° OVER 12S) FOR THE CIRCULAR SCROLL INDICATOR.
- INTERACTIONS: HOVER STATES USE HORIZONTAL TRANSLATION (`translate-x-4`, +16PX) AND SCALE (`scale-110`). NO DROP SHADOWS EXCEPT A SUBTLE ONE FOR NAV DEPTH.

## LAYOUT & STRUCTURE

### FLOATING NAVIGATION (FIXED)

- FIXED TOP, FULL WIDTH, HIGH Z-INDEX, PADDING ~1.5–2REM. SITS OVER THE ORANGE HERO.
- LEFT: BRAND WORDMARK **SUPERDESIGN®** IN A HEAVY DISPLAY WEIGHT, BLACK TEXT.
- CENTER: A FLOATING BLACK **PILL** (`background:#000000; border-radius:9999px`, SUBTLE SHADOW) HOUSING THE LINKS **WORK · SERVICES · ABOUT · CONTACT** IN WHITE SPACE MONO, ~12PX, UPPERCASE. ON HOVER A LINK BECOMES BLACK TEXT ON A WHITE PILL/BACKGROUND.
- RIGHT: TWO SOCIAL / UTILITY ICON BUTTONS (LUCIDE, ~18–20PX) IN SMALL BLACK SQUARES OR OUTLINED CHIPS.

### TYPOGRAPHIC HERO (ON ORANGE)

- FULL VIEWPORT HEIGHT (`min-h-screen`), CONTENT CENTERED, BACKGROUND `#FF4D00`.
- HEADLINE: **SUPER DESIGN** ON TWO LINES, CENTERED, ARCHIVO BLACK, UPPERCASE, `font-size: 16vw`, `line-height: 0.85`, TRACKING `-0.04em`, BLACK TEXT. LINE 1 = **SUPER**, LINE 2 = **DESIGN**.
- BELOW THE HEADLINE: A FULL-WIDTH `2px solid #000000` HORIZONTAL BORDER SEPARATING THE HERO FROM A METADATA ROW.
- METADATA ROW (THREE COLUMNS, SPACE MONO, UPPERCASE, ~12PX): LEFT = **BASED IN DIGITAL VOID** (WITH A SMALL LEADING ICON/DOT); CENTER = THE ROTATING CIRCULAR **SCROLL DOWN** INDICATOR (SEE SPECIAL COMPONENTS); RIGHT = TWO-LINE LABEL **AI PRODUCT DESIGNER** / **SINCE 2024**, RIGHT-ALIGNED.

### SKEWED MARQUEE BAND

- FULL-WIDTH SECTION, BACKGROUND `#000000`, ROTATED `transform: skewY(-2deg)` (CONTENT COUNTER-SKEWED SO TEXT READS STRAIGHT IF NEEDED), OVERFLOW HIDDEN, WITH `2px` BLACK EDGES BLENDING INTO ADJACENT SECTIONS.
- ROW 1 (SCROLLS LEFT): ORANGE `#FF4D00` TEXT, ARCHIVO BLACK, UPPERCASE, `font-size: 10vw`, REPEATING THE STRING **`STRATEGY • DESIGN • AI • • `** (REPEAT 8×+ FOR A SEAMLESS LOOP).
- ROW 2 (SCROLLS RIGHT / REVERSE): WHITE TEXT @ ~80% OPACITY, SAME TYPE/SIZE, REPEATING **`FUTURE • CREATIVE • CODE • • `**.
- BOTH ROWS LOOP INFINITELY WITH A LINEAR TIMING FUNCTION; DUPLICATE THE CONTENT TRACK AND TRANSLATE -50% FOR A GAPLESS LOOP.

### VERTICAL SERVICE LIST (ON BLACK)

- SECTION BACKGROUND `#000000`, TEXT `#FFFFFF`, LARGE VERTICAL PADDING.
- SECTION HEADER: **SERVICES** IN WHITE ARCHIVO BLACK UPPERCASE (~7–8VW) WITH A SOLID `#FF4D00` STAR ICON TO ITS RIGHT.
- FOUR LIST ITEMS, EACH FULL-WIDTH AND SEPARATED BY A `1px` WHITE @ 20% TOP BORDER. EACH ITEM IS A ROW: **[INDEX (ORANGE, SPACE MONO)] [TITLE + TAGS] [ARROW ICON (HIDDEN, ORANGE)]**.
  - TITLE: ARCHIVO BLACK, UPPERCASE, `font-size: 7vw`, WHITE.
  - TAGS: A ROW OF PILL-SHAPED OUTLINED CHIPS IN SPACE MONO UPPERCASE (~10–12PX), THIN WHITE/20% BORDER.
- ITEMS (INDEX / TITLE / TAGS):
  1. **(01)** — **AI STRATEGY** — CONSULTING · IMPLEMENTATION · LLM INTEGRATION
  2. **(02)** — **INTERFACE** — UI/UX DESIGN · DESIGN SYSTEMS · PROTOTYPING
  3. **(03)** — **DEVELOPMENT** — REACT · TYPESCRIPT · NEXT.JS · WEBGL
  4. **(04)** — **BRANDING** — IDENTITY · MOTION · 3D VISUALS
- HOVER (SEE BRUTALIST SERVICE CARD): ROW BACKGROUND SHIFTS TO WHITE @ ~5%, TITLE TRANSLATES +16PX RIGHT, AND THE TRAILING `#FF4D00` ARROW BECOMES OPACITY 1 AND ROTATES 45°.

### GIANT CTA + FOOTER

- CTA SECTION: BACKGROUND `#FF4D00`, CENTERED. HEADING **LET'S TALK** IN BLACK ARCHIVO BLACK UPPERCASE (~14VW). BELOW IT A LARGE `rounded-full` BUTTON, BACKGROUND `#000000`, WHITE TEXT, LABEL **HELLO@SUPERDESIGN.AI**, THAT SCALES UP (`scale-110`) ON HOVER.
- FOOTER: SEPARATED BY A `2px solid #000000` TOP BORDER, ON ORANGE. LEFT: COPYRIGHT **© 2024 SUPERDESIGN INC.** IN SPACE MONO ~12PX. RIGHT: HORIZONTAL TEXT LINKS **INSTAGRAM · TWITTER · LINKEDIN · GITHUB** IN SPACE MONO ~12PX, UPPERCASE.

## SPECIAL COMPONENTS

### ROTATING SCROLL INDICATOR

A CIRCULAR SVG WITH TEXT THAT ROTATES INFINITELY. CREATE A **144PX** DIAMETER CIRCLE. INSIDE, USE AN SVG `<textPath>` ON A CIRCULAR PATH TO REPEAT THE STRING **`SCROLL DOWN • `** 3–4 TIMES, IN SPACE MONO, ~9PX, WEIGHT BOLD, UPPERCASE, WIDE TRACKING. PLACE A STATIC LUCIDE **ARROW-DOWN** ICON IN THE CENTER. ANIMATE THE SVG TEXT CONTAINER WITH A LINEAR 360° ROTATION OVER **12S**, INFINITE.

### BRUTALIST SERVICE CARD

AN INTERACTIVE FULL-WIDTH LIST ITEM WITH A TOP BORDER. LAYOUT: `[INDEX (ORANGE, MONO)] [TITLE + TAGS (WHITE, ARCHIVO)] [ARROW ICON (HIDDEN, ORANGE)]`. ON HOVER: BACKGROUND SHIFTS TO WHITE @ 5% OPACITY; TITLE TRANSLATES +16PX HORIZONTALLY; THE ARROW OPACITY BECOMES 1 AND ROTATES 45°. SMOOTH TRANSITIONS (~0.3S EASE).

## TECH / DELIVERY

- BUILD AS A SELF-CONTAINED PROJECT (PLAIN HTML + TAILWIND-STYLE UTILITIES OR A VITE SETUP). FONTS VIA GOOGLE FONTS OR VENDORED UNDER `assets/`. LUCIDE ICONS FOR ARROW-DOWN, ARROW (DIAGONAL), STAR, AND SOCIAL/UTILITY ICONS. NO REMOTE IMAGE HOTLINKS.
- ALL MARQUEES, THE 12S SPIN, AND HOVER TRANSFORMS MUST RUN SMOOTHLY AND LOOP SEAMLESSLY. RESPONSIVE DOWN TO MOBILE (NAV PILL COLLAPSES, TYPE SCALES VIA VW UNITS, SERVICE TAGS WRAP).

## SPECIAL NOTES

- MUST: MAINTAIN EXTREMELY HIGH CONTRAST. MUST: USE ONLY UPPERCASE FOR ARCHIVO BLACK HEADERS. MUST: KEEP ALL BORDERS SHARP (NO ROUNDED CORNERS EXCEPT PILL NAV AND BUTTONS).
- DO NOT: USE GRADIENTS, DROP SHADOWS (EXCEPT SUBTLE NAV DEPTH), OR SOFT PASTELS. DO NOT: USE STANDARD SANS-SERIFS FOR HEADLINES.
- ACCEPTANCE CRITERION: THE RENDERED OUTPUT MATCHES THE REFERENCE DESIGN SAME-TO-SAME IN LAYOUT, TYPE SCALE, COLOR, SECTION ORDER, AND COPY.
