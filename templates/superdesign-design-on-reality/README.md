# Superdesign Design on Reality — Poster-Modernist Developer Tool Landing Page (HTML + CSS)

A poster-modernist, reality-first marketing landing page template for a developer tool — ideal for developer tools, design systems, and technical SaaS products that want a typographic, anti-gradient aesthetic. Built with a strict 12-column grid, a cream base (`#E3E2DE`), a single cobalt-blue accent (`#1351AA`), jet-black text, hard 1px dividers, zero border-radius, and zero shadows; General Sans handles all typography, with a monospace stack reserved for tiny index numbers. The entire page sits inside a centered `max-w-[1600px]` framed column with visible left/right borders. Five sections — Manifesto, System, Workflow, Why Different, and Access — each feature a sticky 3-column label sidebar alongside oversized headline content in the right 9 columns, separated by hairline rules; the only motion is fast linear 0.3s color transitions on hover. Delivered as a self-contained static page with no build step. Generated with Claude Fable 5.

## Run

This is a static project — open `index.html` in a browser, or serve the folder:

```sh
python3 -m http.server 8000
```

See `prompt.md` for the full build spec; `demo.mp4` shows it in motion.
