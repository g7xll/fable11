# Superdesign Modern Atmospheric — AI Design Tool Marketing Landing Page (HTML + CSS + Vanilla JS)

A single-page marketing landing page template for AI tools, SaaS products, and developer platforms that want a futuristic, premium "atmospheric" aesthetic — deep slate backgrounds (`#0F172A`), clean off-white feature surfaces (`#F8F9FA`), glassmorphism panels with 12px blur, vibrant indigo accents (`#6366F1`), and animated glow gradients. Typography pairs Lora (serif) for large expressive headlines, Inter (sans) for UI and body, and Space Grotesk for the wordmark and technical accents. The page is a tiered vertical scroll: sticky nav, a 110vh atmospheric dark hero with a glowing "vibe input box," a floating glass integration bar with framework logos and integration icons, a light-mode feature scroll-spy section (sticky sidebar tracked via IntersectionObserver), a dark masonry glass testimonials grid, a light FAQ accordion, and a footer. Motion is subtle but intentional: a blinking typing cursor, a 180° logo rotation on hover, a multi-color outer glow that intensifies on focus, and 500ms sticky-nav state transitions. Delivered as self-contained static HTML with no build step. Generated with Claude Fable 5.

## Run

This is a static project — open `index.html` in a browser, or serve the folder:

```sh
python3 -m http.server 8000
```

See `prompt.md` for the full build spec; `demo.mp4` shows it in motion.
