# SUMMARY

A SOPHISTICATED, VIBE-CENTRIC DESIGN SYSTEM THAT BLENDS DEEP SLATE ATMOSPHERIC BACKGROUNDS WITH A CLEAN, GRID-BASED FEATURE LAYOUT. IT USES A MIX OF 'INTER' (SANS) FOR UTILITY, 'LORA' (SERIF) FOR EMOTIONAL HEADLINES, AND 'SPACE GROTESK' FOR A TECHNICAL EDGE. THE VISUAL IDENTITY RELIES ON GLASS PANELS (BLUR 12PX), VIBRANT INDIGO ACCENTS (#6366F1), AND ANIMATED GLOW GRADIENTS TO CREATE A FUTURISTIC, PREMIUM FEEL.

# STYLE

THE STYLE IS 'MODERN ATMOSPHERIC'. IT PAIRS DARK, DEEP BACKGROUNDS (#0F172A) IN THE HERO AND TESTIMONIALS WITH CLEAN, OFF-WHITE SURFACES (#F8F9FA) FOR DETAILED FEATURES. TYPOGRAPHY IS A CRITICAL PILLAR, USING A SERIF FONT (LORA) FOR LARGE, EXPRESSIVE HEADLINES AND A HIGH-LEGIBILITY SANS (INTER) FOR BODY TEXT. ANIMATIONS ARE SUBTLE BUT INTENTIONAL: TYPING CURSORS, 180-DEGREE LOGO ROTATIONS ON HOVER, AND 500MS SMOOTH TRANSITIONS FOR STICKY NAVIGATION STATES.

- TYPING-CURSOR ANIMATION ON THE HERO SUBHEADING (BLINKING CARET, OPTIONAL TYPE-IN OF ROTATING PHRASES).

- STICKY NAV STATE TRANSITIONS USE 500MS EASE-IN-OUT (BACKGROUND/BLUR/SHADOW SHIFT ON SCROLL).
- INPUT GLOW INTENSIFIES ON HOVER/FOCUS; FAQ ACCORDION SMOOTH HEIGHT 0 → MAX-CONTENT WITH CHEVRON ROTATION; FEATURE NAV DOT/COLOR SHIFTS ON SCROLL INTERSECTION (500MS).

STICKY/ABSOLUTE TOP NAV. THREE ZONES:
- **LEFT**: LOGO = ICON (ROTATES 180° ON HOVER) + BOLD LOWERCASE WORDMARK **"superdesign"** (SPACE GROTESK).
- **CENTER**: A CAPSULE-SHAPED PILL `BG-WHITE/10 BACKDROP-BLUR-MD` (ROUNDED-FULL, 1PX WHITE/10 BORDER) CONTAINING TEXT LINKS (E.G. FEATURES, TESTIMONIALS, FAQ). HIDDEN ON MOBILE.

NAV BECOMES MORE OPAQUE/BLURRED AFTER SCROLLING PAST THE HERO (500MS TRANSITION).

## FEATURE SCROLL-SPY

LIGHT-MODE SECTION, BACKGROUND `#F8F9FA`, SLATE-900 TEXT. TWO-COLUMN LAYOUT:
- **LEFT (≈25%)**: A `STICKY TOP-32` SIDEBAR WITH VERTICAL FEATURE LINKS.
