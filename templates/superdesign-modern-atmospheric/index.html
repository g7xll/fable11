01 — Generation

From a sentence to a screen.

Type what you want in plain language. Superdesign reads intent, infers structure, and renders a working, responsive interface in seconds — no blank canvas, no mockup detour. 

// prompt
design("a pricing page with 3 tiers,
        a glassy hero, dark theme")
  .withMotion("subtle")
  .render()  // → live UI
