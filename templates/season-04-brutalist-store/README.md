# Season 04 — High-Fashion Brutalist E-Commerce Storefront (HTML, CSS, Clash Grotesk, SVG noise grain)

A single-page high-fashion brutalist e-commerce storefront that blends luxury editorial sensibilities with harsh technical-utilitarian industrial design, featuring a sticky blend-mode nav, a full-screen split-indented hero, a manifesto grid, asymmetric product grid, category dividers, and a multi-column dark newsletter footer. The raw aesthetic runs on a neutral beige base (`#E3E2DE`) with burnt-red and neon acid-green micro-interactions, all under a persistent SVG fractal-noise grain overlay with `mix-blend-mode: multiply`. Typography pairs Clash Grotesk for massive tight-tracked display headings with General Sans for body and a monospace for technical metadata and prices — all vendored locally for full offline use. Generated with Claude Fable 5.

## Run

This is a static project — open `index.html` in a browser, or serve the folder:

```sh
python3 -m http.server 8000
```

See `prompt.md` for the full build spec; `demo.mp4` shows it in motion.
