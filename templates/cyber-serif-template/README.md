# Cyber Serif — Premium Avant-Garde Design System Template (Vanilla HTML + CSS + JS, Glassmorphism)

A premium, avant-garde static landing page that pairs Newsreader serif display type with Space Grotesk monospace labels and a surgical emerald accent (`#10B981`) on a deep black base (`#050505`). Glassmorphism cards (`rgba(255,255,255,0.02)`, 12px blur) feature shimmer-border animations and radial-gradient spotlight effects that track the cursor. Morphing blurred background glows provide atmospheric depth. Sections include a fixed navbar that turns to blurred glass on scroll, a 100vh split hero with floating parallax UI mockup, a 3-column spotlight feature grid, a benchmark table with count-up numbers, and an animated gradient-text CTA. All motion uses a weighted `cubic-bezier(0.16, 1, 0.3, 1)`: scroll reveals, shimmer sweeps, cursor spotlights, and pulse-glow on pill buttons. Generated with Claude Fable 5.

## Run

This is a static project — open `index.html` in a browser, or serve the folder:

```sh
python3 -m http.server 8000
```

See `prompt.md` for the full build spec; `demo.mp4` shows it in motion.
