# Red Sun — Coral and Ink Editorial Template (Vanilla HTML, CSS, JS)

A modern, premium editorial landing page template, **Red Sun**, pairing bold coral (`#EF4623`) with a deep ink-blue foundation (`#2D3B42`) across an asymmetrical, modular layout. Large-scale Instrument Serif headlines (italic for emphasis, hero at `clamp(4rem, 11vw, 10rem)`), Manrope body and UI type, soft peach (`#FDF1EE`) accents, ambient blur circles, and large corner radii (minimum 24px — no sharp 90° corners anywhere) define the visual language. The page flows from a glassmorphism-on-scroll nav with a rotating coral logo mark, through a bento value-prop with a floating animated UI simulator (skeleton loaders + "Code Generated" status bar), a three-column features bento, a full-width coral CTA with a dot-grid pattern overlay, and a deep-ink five-column footer. The signature "fade-up" reveal starts every element at `translateY(28px) rotate(2deg)` on `cubic-bezier(0.16, 1, 0.3, 1)`. Generated with Claude Fable 5.

## Run

This is a static project — open `index.html` in a browser, or serve the folder:

```sh
python3 -m http.server 8000
```

See `prompt.md` for the full build spec; `demo.mp4` shows it in motion.
