# SUMMARY

THE RED SUN SYSTEM USES A BOLD CORAL PRIMARY COLOR AGAINST A DEEP INK-BLUE FOUNDATION TO CREATE A MODERN, EDITORIAL AESTHETIC. IT RELIES ON LARGE-SCALE SERIF TYPOGRAPHY, GENEROUS WHITE SPACE, AND SMOOTH, NON-LINEAR ANIMATIONS (CUBIC-BEZIER) TO CONVEY A SENSE OF PREMIUM CRAFTSMANSHIP AND TECHNOLOGICAL SPEED.

# STYLE

THE STYLE IS DEFINED BY ITS HIGH-CONTRAST COLOR DUO: CORAL (#EF4623) AND INK (#2D3B42). TYPOGRAPHY FOLLOWS AN EDITORIAL HIERARCHY: 'INSTRUMENT SERIF' FOR EXPRESSIVE, LARGE-SCALE HEADLINES (ITALICIZED FOR EMPHASIS) AND 'MANROPE' (WEIGHTS 300-700) FOR FUNCTIONAL UI ELEMENTS AND BODY COPY. UI ELEMENTS USE LARGE CORNER RADII (ROUNDED-3XL TO ROUNDED-[3REM]) AND SUBTLE DEPTH THROUGH 'SOFT PEACH' (#FDF1EE) BACKGROUND ACCENTS AND SOFT BLURS.

## SPEC

APPLY A 'CORAL AND INK' EDITORIAL STYLE. PRIMARY COLOR: #EF4623 (CORAL); SECONDARY COLOR: #2D3B42 (INK); BACKGROUND ACCENT: #FDF1EE (SOFT PEACH). TYPOGRAPHY: USE 'INSTRUMENT SERIF' FOR HEADINGS (SIZES 60PX TO 160PX, TRACKING-TIGHT) AND 'MANROPE' FOR BODY (18PX, LEADING-RELAXED). FOR ANIMATIONS, USE A CUSTOM CUBIC-BEZIER(0.16, 1, 0.3, 1) CURVE FOR 'FADE-UP' EFFECTS THAT INCLUDE A STARTING 2-DEGREE ROTATION. NAVIGATION MUST BE A GLASSMORPHISM HEADER (BACKDROP-BLUR 12PX, WHITE/80 OPACITY). BUTTONS SHOULD HAVE A 30PX CORNER RADIUS AND INCLUDE A SHADOW-LG SHADOW-PRIMARY/20. USE AMBIENT BACKGROUND BLURS (#EF4623 AT 10% OPACITY) WITH 100PX-120PX BLUR RADII TO CREATE DEPTH.

# IMPLEMENTATION NOTES (CORE CODE + SMALL DETAILS)

## TECH STACK

- SINGLE SELF-CONTAINED STATIC SITE: `INDEX.HTML` + `STYLES.CSS` + `MAIN.JS` (NO BUILD STEP, RUNS OFFLINE).
- TAILWIND-FREE: ALL UTILITY-LIKE STYLING WRITTEN AS HAND-AUTHORED CSS USING CSS CUSTOM PROPERTIES.
- FONTS VENDORED LOCALLY UNDER `ASSETS/FONTS/` ('INSTRUMENT SERIF' REGULAR + ITALIC, 'MANROPE' VARIABLE) AND LOADED VIA `@FONT-FACE` WITH `FONT-DISPLAY: SWAP`. NO REMOTE FONT CDN AT RUNTIME.
- ICONS ARE INLINE SVGS (LUCIDE PATHS): SPARKLES, ARROW-RIGHT, ZAP, LAYERS, SHIELD, CHECK-CIRCLE, GLOBE, CODE, GIT-BRANCH, AND SOCIAL GLYPHS.

## DESIGN TOKENS (`:ROOT`)

```css
--coral: #EF4623;        /* primary */
--ink:   #2D3B42;        /* secondary / text */
--peach: #FDF1EE;        /* soft background accent */
--paper: #FBFAF8;        /* page background (warm off-white) */
--white: #FFFFFF;
--ease:  cubic-bezier(0.16, 1, 0.3, 1);   /* used on EVERY entry + hover */
```

- BODY BACKGROUND `--paper`, BODY TEXT `--ink`, BASE FONT MANROPE 18PX / LINE-HEIGHT 1.7.
- HEADINGS USE `FONT-FAMILY: 'INSTRUMENT SERIF'`, `LETTER-SPACING: -0.02EM` (TRACKING-TIGHT), `FONT-WEIGHT: 400` (THE SERIF READS AS DISPLAY-WEIGHT NATURALLY).

## SCROLL-REVEAL ANIMATION (THE SIGNATURE "FADE-UP")

EVERY `[DATA-REVEAL]` ELEMENT STARTS AT:
```css
opacity: 0; transform: translateY(28px) rotate(2deg);
transition: opacity .9s var(--ease), transform .9s var(--ease);
```
AN `INTERSECTIONOBSERVER` (THRESHOLD 0.15, ROOTMARGIN `0PX 0PX -8% 0PX`) ADDS `.IN-VIEW` WHICH SETS `opacity:1; transform: translateY(0) rotate(0)`. A PER-ELEMENT `--D` CUSTOM PROPERTY DRIVES STAGGERED `TRANSITION-DELAY` (0, 80, 160, 240MS …) SO GROUPED ITEMS CASCADE. THE 2-DEGREE START ROTATION IS THE KEY EDITORIAL TELL.
