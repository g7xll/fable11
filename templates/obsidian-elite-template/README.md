# Obsidian Elite — Monochrome Invite-Only Platform Landing Page (Vanilla HTML, CSS, JS)

A dark, strictly monochromatic editorial waitlist / beta-capture landing page for a fictional invite-only platform called **Obsidian**. The palette spans obsidian black (`#080808`) to pure white with silver gradients, a 5%-opacity fractal-noise overlay, and depth built entirely from borders and backdrop blur rather than box-shadows. Three-way typography pairs DM Serif Display in italics for high-impact headlines, Geist Mono for all technical labels and buttons, and Inter for body copy at fluid 92vw editorial widths. Generated with Claude Fable 5.

## Run

This is a static project — open `index.html` in a browser, or serve the folder:

```sh
python3 -m http.server 8000
```

See `prompt.md` for the full build spec; `demo.mp4` shows it in motion.
