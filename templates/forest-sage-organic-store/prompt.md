THE TEMPLATE COMBINES BOLD INDUSTRIAL DISPLAY TYPOGRAPHY WITH A SOFT ORGANIC "FOREST AND SAGE" COLOR STORY, A PERSISTENT ANALOG NOISE OVERLAY, ULTRA-ROUNDED COLOR-BLOCKED SECTIONS, AND FLUID SCROLL-TRIGGERED REVEAL ANIMATIONS. # STYLE

THE STYLE USES A HIGH-CONTRAST 'FOREST AND SAGE' COLOR STORY. TYPOGRAPHY IS A MIX OF HEAVY, CONDENSED DISPLAY FACES (ANTON) FOR IMPACT AND CLEAN, TRACKED-OUT SANS-SERIFS (INTER) FOR UTILITY. VISUALS ARE TREATED WITH A PERSISTENT SVG FRACTAL NOISE OVERLAY (4% OPACITY) TO ADD ANALOG TEXTURE. ANIMATIONS FOCUS ON VERTICAL REVEALS USING A CUSTOM CUBIC-BEZIER FOR A 'SNAPPY YET SMOOTH' HIGH-END FEEL.

## COLOR PALETTE (EXACT)

- FOREST: `#01472e` (PRIMARY DARK / ALL DARK TEXT — NEVER USE BLACK)
- SAGE: `#ccd5ae` (HERO BACKGROUND, FOOTER ACCENT TEXT)
- OLIVE: `#e9edc9` (PRODUCT GRID BACKGROUND)
- CREAM: `#fefae0` (PAGE / NEUTRAL BACKGROUND)
- MOSS: `#a3b18a` (SECONDARY ACCENT)

## TYPOGRAPHY

- **DISPLAY / HEADINGS**: 'ANTON' (HEAVY CONDENSED). HERO SIZE: `23vw`, `line-height: 0.75`, `letter-spacing: -0.05em`, UPPERCASE. SECTION TITLES: `15vw`. ALL DISPLAY TEXT COLORED FOREST `#01472e`.
- **BODY / UTILITY**: 'INTER', WEIGHTS 400 (REGULAR) AND 700 (BOLD). ALL LABELS, NAV LINKS, AND BUTTONS ARE UPPERCASE WITH `letter-spacing` BETWEEN `0.2em` AND `0.4em`, TYPICALLY `10px`–`12px`.
- VENDOR BOTH FONTS LOCALLY (WOFF2) AND DECLARE THEM WITH `@font-face`; DO NOT HOTLINK GOOGLE FONTS.

## TEXTURE & EFFECTS

- **NOISE OVERLAY**: A `position: fixed`, FULL-VIEWPORT, `pointer-events: none`, `z-index` ABOVE CONTENT SVG `feTurbulence` FRACTAL NOISE LAYER AT `opacity: 0.04`. MUST PERSIST OVER EVERYTHING FOR THE ANALOG FEEL.
- **CORNERS**: EXTREME ROUNDING. PRIMARY SECTION CONTAINERS USE `border-radius: 5rem` (APPLIED TO THE TOP CORNERS WHERE SECTIONS STACK, SO EACH COLOR BLOCK CURVES OVER THE ONE ABOVE). CARDS / IMAGES USE `border-radius: 2.5rem`. FLOATING ORGANIC IMAGES USE `border-radius: 3rem`. NOTHING IS FULLY SHARP — EVERY ELEMENT HAS AT LEAST A SMALL RADIUS.
- **SHADOWS**: SOFT, DEEP, TINTED. FLOATING ELEMENTS USE A LARGE BLURRED SHADOW WITH A FOREST TINT: `0 25px 50px -12px rgba(1, 71, 46, 0.2)`.

## ANIMATION

- **REVEAL LOGIC**: ON-SCROLL, COMPONENTS SLIDE UP FROM `translateY(100px)` + `opacity: 0` TO `translateY(0)` + `opacity: 1` USING `cubic-bezier(0.16, 1, 0.3, 1)` OVER `1.2s`. DRIVE WITH AN `IntersectionObserver` ADDING AN `.is-visible` CLASS.
- **HERO LETTER STAGGER**: THE MASSIVE HERO WORD REVEALS LETTER-BY-LETTER ON LOAD, EACH LETTER RISING FROM `translateY(100%)` INSIDE AN `overflow: hidden` MASK WITH A `0.05s` PER-LETTER DELAY.
- **FLOATING PARALLAX**: FOREGROUND ORGANIC IMAGES RUN AN INFINITE FLOAT KEYFRAME: `@keyframes float { 0%, 100% { transform: translateY(0) rotate(0deg); } 50% { transform: translateY(-20px) rotate(5deg); } }`, PLUS A SCROLL LISTENER ADDING EXTRA `translateY` BASED ON SCROLL DEPTH (SPEED FACTOR ~`0.05`, DIFFERENT FACTOR PER IMAGE).
- **MUST USE THE EXACT `cubic-bezier(0.16, 1, 0.3, 1)`** FOR ALL TRANSITIONS TO PRESERVE THE PREMIUM MOTION SIGNATURE.

# LAYOUT & STRUCTURE

A VERTICALLY STACKED LAYOUT WITH DISTINCT COLOR-BLOCKED SECTIONS SEPARATED BY ULTRA-ROUNDED TOP CORNERS. A FIXED NAVIGATION BAR FLOATS OVER EVERYTHING. SECTION ORDER TOP-TO-BOTTOM: HEADER/NAV, HERO, MARQUEE STRIP, PRODUCT/FEATURE GRID, EDITORIAL "STORY" BAND, FOOTER.

## HEADER / NAVIGATION

`position: fixed`, FULL-WIDTH, TOP. THREE ZONES:
- **LEFT**: LOGO IN BOLD UPPERCASE WITH A HYPHEN PREFIX (E.G. `—VERDE`), FOREST COLOR.
- **CENTER**: PILL-SHAPED NAVIGATION BAR WITH `backdrop-filter: blur(20px)`, SEMI-TRANSPARENT WHITE BACKGROUND (`#ffffff1a`), FULLY ROUNDED. LINKS UPPERCASE, `font-size: 10px`, BOLD, TRACKED `0.2em` (SHOP, ABOUT, JOURNAL, CONTACT).
- **RIGHT**: CART BUTTON (UPPERCASE LABEL) WITH A NUMERIC COUNTER BADGE INSIDE A WHITE PILL.

## HERO SECTION

FULL VIEWPORT HEIGHT (`100vh`). BACKGROUND SAGE `#ccd5ae`. CENTERPIECE: A MASSIVE 'ANTON' WORD AT `23vw` WITH STAGGERED LETTER REVEAL. 2–3 FLOATING ORGANIC INGREDIENT IMAGES (E.G. FRUIT, HERBS, VEGETABLES) WITH `3rem` ROUNDED CORNERS, SOFT FOREST-TINT SHADOWS, FLOAT ANIMATION, AND PARALLAX SCROLL, POSITIONED AROUND / OVERLAPPING THE TEXT. BOTTOM ROW: DUAL-COLUMN DESCRIPTIVE TEXT ON ONE SIDE AND LOCATION / ORIGIN LABELS (E.G. "ORGANIC · SUSTAINABLE", "EST. 2024 — SONOMA, CA") ON THE OTHER, IN SMALL UPPERCASE TRACKED INTER.

## MARQUEE STRIP

A THIN FOREST `#01472e` BAND BETWEEN HERO AND GRID WITH AN INFINITE HORIZONTAL MARQUEE OF UPPERCASE SAGE TEXT (E.G. "FRESH PRODUCE — SEASONAL — FARM DIRECT — ZERO WASTE —") SEPARATED BY SMALL DOT/STAR GLYPHS.

## PRODUCT / FEATURE GRID

BACKGROUND OLIVE `#e9edc9`. TOP OF SECTION USES `border-radius: 5rem` ON THE TOP CORNERS SO IT CURVES OVER THE BAND ABOVE, WITH GENEROUS TOP PADDING. HEADING ROW: MASSIVE 'ANTON' DISPLAY TEXT (`15vw`, E.G. "SHOP") PAIRED WITH A LARGE CIRCULAR CTA BUTTON (FOREST FILL, SAGE TEXT, UPPERCASE, ROTATING/ARROW). GRID: 3-COLUMN LAYOUT (RESPONSIVE DOWN TO 1 COLUMN). EACH PRODUCT CARD: AN `aspect-ratio: 4/5` IMAGE WITH `2.5rem` RADIUS, A PRODUCT NAME (INTER BOLD UPPERCASE), AND A PRICE. ON HOVER: THE IMAGE SCALES `1.1x` (SMOOTH WITH THE SIGNATURE BEZIER) AND A "QUICK ADD" BUTTON REVEALS FROM THE BOTTOM OVER A BLURRED FOREST OVERLAY.

## EDITORIAL STORY BAND

A CREAM `#fefae0` SECTION (TOP `5rem` RADIUS) PAIRING A LARGE FLOATING ROUNDED IMAGE WITH A COLUMN OF EDITORIAL COPY — A SHORT BRAND STORY, AN `15vw`-ISH ACCENT HEADLINE, AND AN UNDERLINE TEXT-LINK CTA. REVEAL-ON-SCROLL.

## FOOTER

- **LEFT 6 COLS**: LARGE NEWSLETTER SIGNUP — AN OVERSIZED ANTON HEADLINE AND AN UPPERCASE, UNDERLINE-ONLY (NO BOX) EMAIL INPUT WITH AN ARROW SUBMIT.
- **RIGHT 6 COLS**: TWO COLUMNS OF LINKS IN BOLD, TRACKED-OUT `11px` UPPERCASE TEXT (SHOP / COMPANY GROUPS).
- **BOTTOM ROW**: COPYRIGHT AND LEGAL LINKS AT `opacity: 0.3`.
