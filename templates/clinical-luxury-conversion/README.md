# Clinical Luxury — High-Conversion Skincare Landing Page (Vanilla HTML + CSS + JS)

A high-conversion skincare landing page for the fictional brand Lumière that blends minimalist editorial design with strategic conversion triggers — urgency markers, trust signals, product comparisons, and a persistent mobile sticky cart. The palette is an "earth and stone" neutral (`#FDFCF8` background, charcoal `#1C1C1C`, sage `#E2E8E4`) with muted gold accents (`#D4AF37`) and urgency red (`#DC2626`) used sparingly. Headings mix a tight-tracked grotesque with an italic serif accent; corners are heavily rounded (32–40px) and sticky nav and overlays use 12px backdrop blur. The vertical flow builds trust before the ask: a pulsing promo bar, full-bleed hero with glass trust badges, a grayscale-to-color logo wall, a dark value-stats strip, benefit icon cards, a 50/50 product spotlight with a floating product, a feature comparison table, a social-proof grid, and a final CTA with a live monospace countdown. Generated with Claude Fable 5.

## Run

This is a static project — open `index.html` in a browser, or serve the folder:

```sh
python3 -m http.server 8000
```

See `prompt.md` for the full build spec; `demo.mp4` shows it in motion.
