## SUMMARY

COMMUNITY IS A DARK-MODE MARKETING + APP TEMPLATE FOR AN ONLINE COMMUNITY/FORUM PRODUCT. IT PAIRS A MARKETING HERO WITH A REDDIT/INDIE-HACKERS-STYLE POST FEED, A SINGLE-POST DISCUSSION VIEW WITH THREADED COMMENTS, AND SIGN-IN / JOIN-COMMUNITY AUTH SCREENS. THE UI IS BUILT WITH TAILWIND CSS UTILITY CLASSES (MINIFIED) AND ALPINE.JS FOR LIGHTWEIGHT INTERACTIVITY (DROPDOWNS, TABS, MOBILE MENU).

## STYLE

- PALETTE: NEAR-BLACK SLATE BACKGROUND (`#0B1120`-RANGE), CARD SURFACES IN LIGHTER SLATE (`SLATE-800`/`SLATE-900`), INDIGO/VIOLET ACCENT (`#6366F1`/`#818CF8` GRADIENT) FOR PRIMARY BUTTONS, LINKS, AND UPVOTE HIGHLIGHTS, MUTED SLATE-400/500 BODY TEXT, WHITE HEADINGS.
- FONT: INTER (GOOGLE FONTS), WEIGHTS 400/500/600/700/800.
- TYPE SCALE: HERO H1 ~3XL–5XL BOLD, SECTION HEADINGS ~XL–2XL SEMIBOLD, BODY ~SM–BASE, META TEXT ~XS SLATE-500.
- RADII: ROUNDED CARDS (~0.75–1REM), FULLY ROUNDED PILL BUTTONS AND AVATARS.
- SHADOWS: SOFT ELEVATED SHADOWS ON CARDS AND DROPDOWN PANELS.
- ANIMATION: ALPINE.JS `x-transition` FADE/SCALE FOR DROPDOWNS AND MOBILE NAV; HOVER STATE COLOR/OPACITY TRANSITIONS ON LINKS, BUTTONS, AND POST ROWS (~150–200MS EASE).
- BACKGROUND: RADIAL/BLURRED GRADIENT GLOW BEHIND THE HERO ILLUSTRATION (STACK OF AVATAR BUBBLES).

## LAYOUT & STRUCTURE

### `index.html` (HOME / FEED)
- STICKY HEADER: LOGO, SIGN IN LINK, "JOIN COMMUNITY" PILL BUTTON.
- HERO: TWO-LINE GRADIENT HEADLINE, SUBCOPY, "CATCH UP WITH FOUNDERS FROM" LOGO STRIP, DECORATIVE STACKED-AVATAR ILLUSTRATION WITH GLOW BACKGROUND.
- FEED SECTION (TWO-COLUMN):
  - LEFT: TAB BAR (POPULAR / NEWEST / FOLLOWING) + "CREATE POST" BUTTON, LIST OF POST ROWS (UPVOTE COUNT, TITLE, AUTHOR AVATAR STACK, DATE, COMMENT COUNT), ONE PROMOTED/SPONSORED ROW, "SHOW MORE" LINK.
  - RIGHT SIDEBAR: SEARCH INPUT, "NEW DISCUSSIONS" LIST, "LATEST STARTUPS" LIST WITH FOLLOW BUTTONS, "POPULAR POSTS" LIST, NEWSLETTER SIGNUP CARD.
- TWO PROMO CARDS: "NEWS & UPDATES" AND "OTHER RESOURCES", EACH WITH ICON, TITLE, COPY, "LEARN MORE" LINK.
- FOOTER: LOGO, NAV LINKS (ABOUT / PRIVACY & TERMS / FAQS / GET IN TOUCH), SOCIAL ICONS (X, YOUTUBE, GITHUB, TELEGRAM), COPYRIGHT.

### `post.html` (SINGLE POST / DISCUSSION THREAD)
- SAME HEADER/FOOTER CHROME AS HOME.
- BREADCRUMB BACK LINK TO FEED.
- POST DETAIL CARD: UPVOTE CONTROL, TITLE, AUTHOR + META, BODY COPY, POST IMAGE, TAG PILLS, ACTION ROW (COMMENT COUNT, SHARE).
- THREADED COMMENTS LIST: AVATAR, AUTHOR NAME, TIMESTAMP, COMMENT BODY, REPLY/UPVOTE CONTROLS, NESTED REPLIES.
- COMMENT COMPOSER BOX AT THE TOP OF THE THREAD.
- SAME RIGHT SIDEBAR AS HOME (DISCUSSIONS / STARTUPS / POPULAR POSTS / NEWSLETTER).

### `signin.html` (SIGN IN)
- MINIMAL HEADER (LOGO + "JOIN" LINK ONLY).
- CENTERED AUTH CARD: HEADING, EMAIL + PASSWORD FIELDS, "SIGN IN" PRIMARY BUTTON, DIVIDER ("OR"), "SIGN IN WITH GOOGLE" SOCIAL BUTTON, LINK TO JOIN PAGE.

### `join.html` (JOIN COMMUNITY)
- HEADER WITH LOGO, "SIGN IN" LINK, "JOIN COMMUNITY" BUTTON.
- TWO-COLUMN LAYOUT: LEFT — SIGNUP FORM CARD (NAME/EMAIL/PASSWORD FIELDS, TERMS CHECKBOX, SUBMIT BUTTON, GOOGLE SOCIAL SIGNUP, SIGN-IN LINK); RIGHT — TESTIMONIAL CARD(S) WITH MEMBER AVATAR, QUOTE, AND ATTRIBUTION.
- FOOTER SAME AS OTHER PAGES.

## INTERACTIONS

- HEADER "SIGN IN"/NAV LINKS: COLOR-TRANSITION HOVER STATES.
- FEED TABS (POPULAR/NEWEST/FOLLOWING): ACTIVE-TAB UNDERLINE STYLING.
- POST ROWS AND SIDEBAR LIST ITEMS: BACKGROUND/TEXT HOVER TRANSITIONS.
- FOLLOW BUTTONS: HOVER STATE COLOR CHANGE.
- FORM INPUTS: FOCUS RING TRANSITIONS.
- MOBILE NAV (IMPLIED BY ALPINE.JS `x-data`/`x-transition` ATTRIBUTES IN THE MARKUP) FADES/SLIDES IN.
