## SUMMARY

SINGLE-PAGE WAITLIST / COMING-SOON LANDING PAGE BUILT WITH INTER AND INTER TIGHT FONTS. THE PAGE FEATURES A FULL-VIEWPORT LAVENDER-INDIGO BACKGROUND (LIGHT MODE) OR DEEP NAVY (DARK MODE), ANIMATED BLURRED COLOR BLOBS THAT FLOAT BEHIND THE CONTENT, SUBTLE VERTICAL DECORATIVE BORDER LINES, A FLOATING ROUNDED PILL HEADER WITH LOGO, NAV LINKS, AND A DARK/LIGHT MODE TOGGLE. THE HERO SECTION CENTRES A BADGE, A LARGE GRADIENT-CAPABLE HEADLINE, A SUBTITLE, AN EMAIL CAPTURE FORM, AND A STACKED AVATAR GROUP SHOWING "2,000+ MEMBERS". A MINIMAL FOOTER CLOSES THE PAGE. ALL INTERACTIVE STATES — NAV-LINK HOVER, FORM INPUT HOVER/FOCUS, BUTTON HOVER — ARE REPRODUCED.
