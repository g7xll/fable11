> A SELF-CONTAINED, PIXEL-FAITHFUL REPRODUCTION OF THE APPY LANDING PAGE TEMPLATE BY CRUIP, BUILT AS PLAIN HTML + CSS + VANILLA JS WITH NO BUILD STEP REQUIRED.

REFERENCE: `https://cruip.com/demos/appy/`

## SUMMARY

APPY IS A MULTI-PAGE MOBILE APP LANDING PAGE HTML TEMPLATE BY CRUIP. IT FEATURES A DARK/LIGHT THEME TOGGLE, TAILWIND CSS-BASED STYLING (COMPILED AND MINIFIED WITH OBFUSCATED CLASS NAMES), ALPINEJS FOR INTERACTIVE UI COMPONENTS (DROPDOWNS, MODALS, MOBILE MENU), AOS FOR SCROLL-TRIGGERED FADE ANIMATIONS, AND SWIPER FOR A TOUCH-ENABLED IMAGE CAROUSEL. THE TEMPLATE IS DESIGNED FOR MARKETING A MOBILE APPLICATION WITH PAGES FOR HOME, ABOUT, BLOG, TESTIMONIALS, HELP CENTER, CONTACT, AND 404.

## STYLE

- **PALETTE (DARK MODE DEFAULT):**
  - BACKGROUND: #17191E (NEAR-BLACK DARK SURFACE)
  - SURFACE ELEVATED: #1E2029
  - TEAL ACCENT: #3ABAB4 (PRIMARY ACTION COLOR)
  - INDIGO ACCENT: #7F9CF5
  - PURPLE ACCENT: #9F7AEA
  - TEXT PRIMARY: #F4F4F5 (ZINC-100)
  - TEXT SECONDARY: #A1A1AA (ZINC-400 / GRAY-600 IN DARK MODE)
  - BORDER: #27272A (ZINC-800)
  - LIGHT MODE BACKGROUND: #FFFFFF
  - LIGHT MODE TEXT SECONDARY: #4B5563 (GRAY-600)

- **FONTS:**
  - RED HAT DISPLAY: WEIGHTS 500, 700, 900 — USED FOR HEADINGS
  - INTER: WEIGHTS 400, 500, 700 — USED FOR BODY TEXT

- **TYPE SCALE:**
  - H1: ~3.5REM, FONT-WEIGHT 900, RED HAT DISPLAY
  - H2: ~2.25REM, FONT-WEIGHT 900, RED HAT DISPLAY
  - H3: ~1.25REM, FONT-WEIGHT 700, RED HAT DISPLAY
  - BODY: 1REM/1.5, INTER 400
  - SMALL: 0.875REM, INTER 400/500

- **RADII:** ROUNDED-FULL FOR PILLS/BADGES, ROUNDED-LG FOR CARDS, ROUNDED-XL FOR LARGER CONTAINERS

- **ANIMATION EASINGS:**
  - AOS: EASE-OUT-QUART, DURATION 750MS, ONCE: TRUE, DISABLE ON PHONE
  - ALPINEJS TRANSITIONS: EASE-OUT-QUART FOR ENTER, LINEAR FOR LEAVE
  - DARK MODE TOGGLE: CSS TRANSITION ON TOGGLE KNOB
