> A SELF-CONTAINED, PIXEL-FAITHFUL REPRODUCTION OF THE CRUIP MOSAIC ADMIN DASHBOARD TEMPLATE — A COMPREHENSIVE MULTI-PAGE ADMIN UI KIT BUILT WITH TAILWIND CSS, ALPINE.JS, AND CHART.JS. REFERENCE: `https://cruip.com/demos/mosaic/`

## SUMMARY

MOSAIC IS A FULL-FEATURED ADMIN DASHBOARD UI KIT BY CRUIP, FEATURING 64 PAGES ACROSS DASHBOARDS, E-COMMERCE, COMMUNITY, FINANCE, JOB BOARD, TASKS, MESSAGES, CALENDAR, CAMPAIGNS, SETTINGS, AUTHENTICATION, ONBOARDING, AND UI COMPONENT PAGES. IT USES TAILWIND CSS FOR STYLING, ALPINE.JS FOR INTERACTIVE BEHAVIOR, AND CHART.JS (WITH MOMENT.JS) FOR DATA VISUALIZATION. THE DESIGN FEATURES A DARK/LIGHT MODE TOGGLE, A COLLAPSIBLE SIDEBAR NAVIGATION, AND A CONSISTENT CARD-BASED LAYOUT SYSTEM.

## STYLE

- PALETTE: DARK: GRAY-900 (#111827) BG, GRAY-800 (#1F2937) SIDEBAR, VIOLET-500 (#8B5CF6) PRIMARY ACCENT, WHITE TEXT ON DARK BG; LIGHT: WHITE BG (#FFFFFF), GRAY-100 SIDEBAR, GRAY-800 TEXT
- FONTS: INTER (SANS-SERIF), LOADED VIA CSS @IMPORT OR GOOGLE FONTS
- TYPE SCALE: BASE 14-16PX; HEADINGS 20-32PX; LABELS 12-13PX
- RADII: SM (4PX), MD (6-8PX), LG (12PX), XL (16PX)
- SHADOWS: CARD SHADOW-LG, DROPDOWN SHADOW-XL
- ANIMATIONS: SIDEBAR COLLAPSE/EXPAND WITH TRANSITION; CHART ANIMATIONS VIA CHART.JS; DROPDOWN FADE; ACCORDION SLIDE

## LAYOUT & STRUCTURE

ALL DASHBOARD PAGES SHARE A LAYOUT: FIXED LEFT SIDEBAR + MAIN CONTENT AREA WITH TOP HEADER BAR. AUTHENTICATION/ONBOARDING PAGES HAVE THEIR OWN FULL-PAGE CENTERED LAYOUT.
