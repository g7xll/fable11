# FINTECH — CRUIP TEMPLATE CLONE

> A SELF-CONTAINED, PIXEL-FAITHFUL REPRODUCTION OF THE CRUIP "FINTECH" HTML LANDING PAGE TEMPLATE, REBUILT AS PLAIN HTML + CSS + VANILLA JS (WITH AOS, SWIPER, AND ALPINE.JS AS RUNTIME LIBRARIES). EVERY PAGE, HOVER STATE, SCROLL-REVEAL ANIMATION, CAROUSEL, AND ACCORDION IS REPRODUCED SAME-TO-SAME. ALL ASSETS (FONTS, IMAGES, ICONS, VENDOR SCRIPTS) ARE VENDORED LOCALLY AND THE PROJECT RUNS OFFLINE WITH NO BUILD STEP.
>
> REFERENCE: `https://cruip.com/demos/fintech/`

## SUMMARY

A FINTECH / NEOBANK MARKETING TEMPLATE FOR A "CUSTOM CARD" PRODUCT (BRANDED "SPARK"). IT PAIRS A BRIGHT BLUE HERO WITH LARGE ASPEKTA DISPLAY TYPE, A HAND-DRAWN TEAL UNDERLINE FLOURISH, FLOATING CARD/UI ILLUSTRATIONS, AND A SEQUENCE OF ALTERNATING FEATURE SECTIONS ON SOFT RADIAL-GRADIENT BACKGROUNDS. CONTENT REVEALS ON SCROLL VIA AOS (FADE-UP, EASE-OUT-CUBIC, 700MS, ONCE). THE TEMPLATE HAS THREE PAGES: A LONG HOME LANDING PAGE, A TWO-COLUMN APPLICATION (GET YOUR CARD) FORM PAGE, AND A SUPPORT CENTER / KNOWLEDGE-BASE ARTICLE PAGE WITH A COLLAPSIBLE TOPIC SIDEBAR.

## STYLE

- FONT FAMILY: "ASPEKTA", SANS-SERIF (SELF-HOSTED WOFF2, WEIGHTS 350 / 400 / 450 / 500 / 550 / 700).
- TYPE SCALE: HERO H1 IS A LARGE CLAMPED DISPLAY SIZE (~48–72PX, WEIGHT 500, TIGHT LEADING); SECTION H2 ~40PX; H4 CARD TITLES ~20PX; BODY ~16–18PX / LINE-HEIGHT ~1.5; SMALL LABELS ~13–14PX UPPERCASE TRACKED.
- PALETTE (TOKENS):
  - BRAND BLUE PRIMARY: `#2563EB` (BUTTONS, LINKS, ACCENTS)
  - BLUE LIGHT: `#3B82F6`, SKY: `#7DD3FC`
  - TEAL FLOURISH (HERO UNDERLINE): `#2DD4BF`
  - INK / SLATE HEADINGS: `#0F172A` / `#1E293B`
  - BODY SLATE: `#334155` / `#475569`
  - MUTED: `#64748B` / `#94A3B8`
  - SURFACE WHITE: `#FFFFFF`; SOFT SLATE SURFACE: `#F8FAFC` / `#F1F5F9`
  - DARK SECTION BG (HERO BG, CTA, FOOTER): DEEP SLATE `#1E293B`–`#0F172A` WITH BLUE RADIAL GLOWS
- RADII: BUTTONS AND PILLS ARE FULLY ROUNDED (`rounded-full`); CARDS / PANELS USE LARGE RADII (~24–32PX); INPUTS ~8–12PX.
- SHADOWS: SOFT ELEVATED CARD SHADOWS ON PRICING TABLES, THE APPLY FORM PANEL, AND FLOATING HERO UI.
- ANIMATION: AOS SCROLL REVEALS — `data-aos="fade-up"`, DURATION 700MS, EASING `ease-out-cubic`, `once: true`, DISABLED ON PHONE; STAGGERED VIA `data-aos-delay` (100/200/300) AND ANCHORED PER-SECTION VIA `data-aos-anchor`. TESTIMONIAL CAROUSEL USES SWIPER FADE EFFECT WITH CLICKABLE PAGINATION BULLETS. HOVER TRANSITIONS ON BUTTONS/LINKS (COLOR + ARROW NUDGE). SUPPORT SIDEBAR TOPICS USE ALPINE.JS OPEN/CLOSE WITH A ROTATING CHEVRON AND A SLIDE/FADE MOBILE DRAWER.
