REFERENCE: `https://cruip.com/demos/quoty/`

## SUMMARY

QUOTY IS A FREELANCER QUOTE/PROPOSAL WEB APP TEMPLATE FROM CRUIP: A SPLIT-SCREEN LAYOUT WHERE THE LEFT PANEL SHOWS A STICKY "QUOTE FOR [CLIENT]" SUMMARY AND THE RIGHT PANEL SWAPS BETWEEN A QUOTE BREAKDOWN (LINE-ITEM TABLE + ACCORDION TERMS), A LINE-ITEM DETAILS PAGE, A CONTACT FORM, AND A PAYMENT FORM (CARD/PAYPAL TOGGLE). BUILT ORIGINALLY WITH TAILWIND CSS (COMPILED/OBFUSCATED CLASSES) AND ALPINE.JS FOR THE ACCORDION AND PAYMENT-METHOD TOGGLE INTERACTIVITY. SUPPORTS LIGHT AND DARK MODE VIA A HEADER TOGGLE SWITCH PERSISTED TO `localStorage`.

- TYPE SCALE: LEFT-PANEL CLIENT NAME H1 ~40-56PX BOLD DISPLAY FONT, SECTION H2 (BRIEF/DETAILS/COSTS BREAKDOWN/PROJECT TERMS/CONTACT ME/CONFIRM & PAY) ~20-24PX SEMIBOLD, BODY COPY ~14-15PX, "QUOTE FOR" EYEBROW LABEL AND DATE ~13PX UPPERCASE/MUTED.

TABLE ROWS AND NAV/BUTTON HOVER STATES USE COLOR/BACKGROUND TRANSITIONS FROM THE COMPILED TAILWIND STYLESHEET.

## LAYOUT & STRUCTURE

ALL FOUR PAGES SHARE THE SAME SHELL: A SITE HEADER (LOGO/AVATAR + "MARY RUTT" NAME LINKING HOME, LIGHT/DARK SWITCH TOGGLE ON THE RIGHT) AND A TWO-COLUMN BODY — A STICKY LEFT PANEL (BACKGROUND SVG ILLUSTRATION, "QUOTE FOR" EYEBROW, CLIENT NAME H1 "THE MARKYK CORP.", QUOTE DATE) AND A SCROLLING RIGHT PANEL THAT HOLDS THE PAGE-SPECIFIC CONTENT.

### PAGE 1 — HOME / QUOTE (`index.html`)
- RIGHT PANEL "BRIEF" SECTION: PARAGRAPH DESCRIBING THE PROJECT SCOPE.
- "DETAILS" SECTION: 3-ITEM ROW LIST (PROJECT LENGTH, START DATE, END DATE) EACH WITH A LABEL + VALUE.
- "COSTS BREAKDOWN" SECTION: LINE-ITEM TABLE WITH DESCRIPTION/COST COLUMNS — SIX ROWS (COMPETITIVE ANALYSIS $7,800; UX RESEARCH REPORTS $2,560; SITEMAP AND INFORMATION ARCHITECTURE $1,420; UX WIREFRAMES AND USER FLOWS $3,978; VISUAL DESIGN $4,476; INTERACTIVE PROTOTYPES + ASSETS EXPORTS $4,326), EACH ROW TITLE LINKS TO `details.html`, TABLE FOOTER SHOWS "TOT IN USD DOLLAR" $24,560.
- "PROJECT TERMS" SECTION: 5-ITEM ALPINE ACCORDION (PAYMENT SCHEDULE AND OPTIONS; DEADLINES AND TIMELINE; KILL FEE AND CANCELLATION TERMS; OWNERSHIP/COPYRIGHT; EQUIPMENT AND EXPENSES) EACH COLLAPSED BY DEFAULT WITH A PLUS/CROSS TOGGLE ICON.
- BOTTOM STICKY CALL-TO-ACTION BAR: "CONTACT ME" GHOST BUTTON (LINKS `contact.html`) AND "PAY - $24.560" SOLID BUTTON (LINKS `pay.html`).

### PAGE 2 — DETAILS (`details.html`)
- "BACK TO HOME" LINK WITH ARROW ICON ABOVE THE ARTICLE.
- ARTICLE TITLED "COMPETITIVE ANALYSIS": MULTI-PARAGRAPH CASE-STUDY COPY WITH BOLDED EMPHASIS PHRASES, A 2-ITEM BULLETED LIST, AND AN INLINE FIGURE (LIGHT/DARK VARIANT IMAGE SWAPPED VIA CSS VISIBILITY, CAPTION "THE THREE PILLARS OF A SUCCESSFUL COMPETITIVE ANALYSIS PHASE.").

### PAGE 3 — CONTACT (`contact.html`)
- ARTICLE TITLED "CONTACT ME" WITH INTRO COPY.
- FORM: EMAIL INPUT, DISABLED CLIENT-NAME INPUT (PREFILLED "THE MARKYK CORP."), SUBJECT `<select>` (THREE PRESET REASONS), MESSAGE TEXTAREA, "SEND" SUBMIT BUTTON.

### PAGE 4 — PAY (`pay.html`)
- ARTICLE TITLED "CONFIRM & PAY" WITH INTRO COPY.
- SEGMENTED TOGGLE: "PAY WITH CARD" / "PAY WITH PAYPAL" (ALPINE `x-data="{ card: true }"`, SLIDING PILL INDICATOR).
- CARD FORM (DEFAULT VISIBLE): EMAIL, CARD NUMBER, EXPIRY DATE + CVC (SIDE BY SIDE), CARDHOLDER NAME, COUNTRY `<select>`, OPTIONAL VAT NUMBER, "PAY - $24.560" SUBMIT BUTTON.
- PAYPAL PANEL (HIDDEN BY DEFAULT, `x-show="!card"` + `x-cloak`): SINGLE "PAY WITH PAYPAL - $24.560" BUTTON.
