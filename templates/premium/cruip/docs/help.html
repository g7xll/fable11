Frequently asked questions

Log in to your Docs account and navigate to Account → Extras → API keys. Click "Create A Key" and copy the generated key. Store it securely — it will only be shown once.
The Marketing API allows 10 simultaneous connections. Exceeding this limit returns a 429 Too Many Requests response. We recommend implementing retry logic with exponential backoff.
You can authenticate using HTTP Basic Auth (pass any string as username and your API key as password) or Bearer token authentication. OAuth 2 is available for applications that access data on behalf of other users.
All API responses are returned as JSON. Error responses include a status, code, title, and detail field to help you diagnose issues.
Your data center prefix is appended to your API key (e.g., the us6 in abc123-us6). It's also the subdomain shown in your account URL when logged in.
