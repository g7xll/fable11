## SUMMARY

MOBIT IS A CLEAN, LIGHT, WARM-BEIGE FINTECH/MONEY-MANAGEMENT MOBILE-APP MARKETING SITE BUILT WITH NEXT.JS + TAILWIND CSS 4 AND FIGTREE AS THE ONLY TYPEFACE. IT USES A CSS-VARIABLE DESIGN-TOKEN SYSTEM (`--color-primary`, `--color-secondary`, ETC.), AOS (ANIMATE-ON-SCROLL) FOR FADE-UP/FADE-UP-SM SCROLL-REVEAL ANIMATIONS, A CSS-CHECKBOX-DRIVEN MOBILE NAV (NO JS REQUIRED FOR THE HAMBURGER), AND HAND-ROLLED JS WIDGETS FOR THE FAQ ACCORDION, PRICING MONTHLY/YEARLY TOGGLE, AND TABS. THE SITE HAS A HOME PAGE, A FEATURES PAGE, A PRICING PAGE, A REVIEWS PAGE, A CONTACT PAGE, A FULL BLOG (3-PAGE PAGINATED INDEX + 13 INDIVIDUAL POST PAGES), A KITCHEN-SINK "ELEMENTS"/STYLE-GUIDE PAGE, PRIVACY POLICY, TERMS & CONDITIONS, AND A 404 PAGE. SHARED CHROME = A TOP ANNOUNCEMENT BAR, A STICKY HEADER (LOGO, "ALL PAGES" DROPDOWN NAV, FEATURES/REVIEWS/PRICING LINKS, "GET THIS TEMPLATE" CTA BUTTON, CSS-CHECKBOX MOBILE HAMBURGER) AND A DARK FOOTER (BRAND COLUMN + NEWSLETTER SIGNUP + 3 LINK COLUMNS + BOTTOM BAR WITH SOCIAL ICONS), PLUS A REPEATED "BUILD BEAUTIFUL WEBSITE" CTA BAND WITH ANIMATED NESTED CONCENTRIC CIRCLES ABOVE THE FOOTER ON EVERY PAGE.
