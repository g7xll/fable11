## SUMMARY

DEXLER IS A CLEAN, LIGHT, CORPORATE "HR / WORKFORCE-MANAGEMENT SAAS" LANDING TEMPLATE BUILT BY THEMEFISHER ON NEXT.JS + TAILWIND CSS. IT USES SOFT PASTEL GRADIENT BLOBS BEHIND WHITE SECTIONS, A SERIF DISPLAY FONT FOR HEADLINES, ROUNDED PILL BUTTONS, AND A DARK FOOTER. THE SITE HAS ELEVEN TOP-LEVEL PAGES PLUS A BLOG-POST DETAIL TEMPLATE.

## STYLE

- **PALETTE (LIGHT):** BACKGROUND `#FFFFFF`, INK/HEADING `#0B1220`-ISH DARK NAVY (`#111827`), BODY TEXT GREY `#4B5563`, PRIMARY BRAND BLUE `#2F6BFF`/`#3B82F6` (BUTTONS, LINKS, ICON ACCENTS), SOFT PASTEL GRADIENT BLOBS (MINT `#B7F0D8`, LAVENDER `#D9CCFB`, PEACH `#FBE0C4`, SKY `#CDE7FB`) USED AS LARGE BLURRED BACKGROUND SHAPES, CARD BORDER `#E5E7EB`, DARK FOOTER `#0B0B14`/`#111111` WITH LIGHT-GREY TEXT.
- **PALETTE (DARK — DERIVED, TEMPLATE HAS NO NATIVE TOGGLE):** BACKGROUND `#0B0F19`, SURFACE `#111827`, BORDER `#1F2937`, INK `#F3F4F6`, BODY TEXT `#9CA3AF`, PRIMARY BLUE STAYS `#3B82F6`, GRADIENT BLOBS DIMMED/DESATURATED FOR DARK.
- **FONTS:** HEADLINES USE A SERIF DISPLAY FACE ("LIBRE BASKERVILLE", SERIF), BODY/UI TEXT USES "INTER", SANS-SERIF. GOOGLE FONTS, SELF-HOSTED LOCALLY IN THE CLONE.
- **TYPE SCALE:** HERO H1 ~44-56PX/1.15 SERIF, SECTION H2 ~32-36PX SERIF, CARD TITLES ~18-20PX SANS SEMIBOLD, BODY ~15-16PX SANS REGULAR, SMALL/LABEL ~13-14PX.
- **RADII:** PILL BUTTONS (FULLY ROUNDED, ~9999PX), CARDS ~16-24PX, IMAGE FRAMES ~16-24PX.
- **SHADOWS:** SOFT, LARGE-BLUR DROP SHADOWS ON CARDS AND FLOATING IMAGE PANELS (`0 20px 60px rgba(0,0,0,0.08)`), NO HARD BORDERS ON HERO IMAGERY.
- **ANIMATION/EASING:** SCROLL-REVEAL FADE-UP ON SECTION ENTRY (OPACITY 0→1, TRANSLATEY 24PX→0, ~600-800MS EASE-OUT, STAGGERED PER CARD), BUTTON HOVER = SLIGHT DARKEN/LIFT WITH 150-200MS EASE TRANSITION, NAV LINK HOVER = COLOR SHIFT TO BRAND BLUE, TESTIMONIAL CAROUSEL WITH DOT PAGINATION AND SLIDE/FADE TRANSITION, MOBILE HAMBURGER MENU SLIDE/FADE.
