## ANIMATIONS AND INTERACTIONS

### SCROLL ENTRANCE ANIMATIONS (AOS LIBRARY FROM CDN)
- LIBRARY: AOS (ANIMATE ON SCROLL) — LOADED FROM CDN, INITIALIZED WITH DURATION 800MS, ONCE:TRUE.
- COMMON ANIMATIONS:
  - FADE-UP-SM: FADE IN + TRANSLATE-Y FROM 20PX TO 0. USED ON SECTION HEADINGS, CARDS, TEXT BLOCKS.
  - FADE-UP: STANDARD AOS FADE-UP ON SUPPORTING CONTENT.
  - FADE-LEFT / FADE-RIGHT: USED ON TWO-COLUMN ALTERNATING LAYOUT IMAGES AND TEXT.
- STAGGER DELAY: 100MS INCREMENTS PER CARD (DATA-AOS-DELAY="100", "200", "300").

### STICKY HEADER / REVEAL
- HEADER IS INITIALLY TRANSPARENT OR LIGHTLY STYLED OVER THE HERO.
- ON SCROLL PAST 80PX: JAVASCRIPT ADDS CLASS "HEADER-REVEAL" WHICH ADDS POSITION:FIXED, SEMI-TRANSPARENT WHITE BACKGROUND (RGBA(255,255,255,0.9)), BACKDROP-FILTER:BLUR(16PX), AND A SLIDE-DOWN ANIMATION.

### BRANDS MARQUEE (INFINITE HORIZONTAL SCROLL)
- IMPLEMENTATION: CSS-ONLY MARQUEE USING FLEXBOX ROW, DUPLICATED BRAND LOGOS IN A .BRANDS-TRACK CONTAINER.
- ANIMATION: @KEYFRAMES MARQUEE TRANSLATES X FROM 0 TO -50% ON A 25S LINEAR INFINITE LOOP.
- PAUSES ON HOVER VIA animation-play-state:paused.

### HAMBURGER MOBILE MENU
- THREE-BAR ICON BUTTON TRANSFORMS VIA CSS CLASS .OPEN: TOP/BOTTOM BARS ROTATE TO FORM AN "X", MIDDLE BAR FADES OUT.
- MOBILE NAVBAR SLIDES DOWN BELOW HEADER ON .OPEN STATE.

### DROPDOWN NAVIGATION
- DESKTOP: DROPDOWN MENUS APPEAR ON HOVER USING CSS :HOVER PSEUDO-CLASS WITH OPACITY AND TRANSLATEY TRANSITION.
- MOBILE: DROPDOWNS TOGGLE VIA JAVASCRIPT CLICK HANDLERS, EXPANDING INLINE.

### FAQ ACCORDION
- JAVASCRIPT CLICK HANDLERS TOGGLE THE .ACTIVE CLASS ON EACH FAQ ITEM.
- ANSWER PANEL ANIMATES MAX-HEIGHT FROM 0 TO AUTO (VIA COMPUTED SCROLLHEIGHT) FOR SMOOTH EXPAND/COLLAPSE.
- ONLY ONE ITEM OPEN AT A TIME; CLICKING AN OPEN ITEM CLOSES IT.

### PROGRESS BARS (PROGRAMS SECTION)
- FILLED VIA INLINE style="width: XX%" ON THE .PROGRESS-FILL ELEMENT.
- COLORS: PRIMARY CORAL, SECONDARY GREEN, AND TERTIARY YELLOW ALTERNATING.

### SWIPER.JS CAROUSELS
- LOADED FROM CDN. USED FOR TESTIMONIALS SLIDER (AUTOPLAY, LOOP, NAVIGATION ARROWS, PAGINATION DOTS).

### HOVER EFFECTS
- CARDS: TRANSLATEY(-4PX) ON HOVER WITH BOX-SHADOW INCREASE AND 0.3S CUBIC-BEZIER TRANSITION.
- BUTTONS: BRIGHTEN/DARKEN BACKGROUND ON HOVER; OUTLINE BUTTON FILLS WITH PRIMARY COLOR.
- NAV LINKS: COLOR TRANSITIONS TO PRIMARY ON HOVER.
- PROGRAM CARD IMAGES: SCALE(1.05) ON HOVER WITHIN OVERFLOW:HIDDEN WRAPPER.
