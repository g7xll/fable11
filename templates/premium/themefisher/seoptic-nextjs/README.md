# Seoptic — SEO Analytics SaaS Template (Themefisher)

[![Watch Demo](./poster.jpg)](./demo.mp4)

A pixel-faithful same-to-same HTML/CSS/JS clone of the **Seoptic** premium Next.js template by Themefisher — a modern SEO analytics SaaS landing page built for keyword tracking, backlink monitoring, and competitor analysis tools.

![Seoptic Template Preview](poster.jpg)

## Overview

Seoptic is a full-featured SaaS marketing template covering every page a real product needs: homepage with animated hero and feature showcases, pricing with monthly/annual toggle, integrations grid, blog archive, team pages, legal pages, and more. The design system uses a vibrant orange (`#ff5a03`) primary accent, Inter + Inter Tight fonts, and a dark/light theme toggle.

## Design System

- **Primary color:** `#ff5a03` (orange)
- **Secondary color:** `#ffd350` (yellow)
- **Tertiary color:** `#ff4ec5` (pink)
- **Fonts:** Inter (body), Inter Tight (headings)
- **Animations:** AOS scroll reveals, CSS gradient border animations, integration logo entrance animations

## Run Locally

No build step required — open any HTML file directly in a browser or serve with a static file server:

```bash
cd templates/premium/themefisher/seoptic-nextjs
npx serve .
# or
python3 -m http.server 8080
```

Then open `http://localhost:8080` in your browser.
