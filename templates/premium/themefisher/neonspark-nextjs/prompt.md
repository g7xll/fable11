## SUMMARY

NEONSPARK IS A DARK-THEME DIGITAL AGENCY / SAAS LANDING PAGE TEMPLATE BY THEMEFISHER. IT FEATURES AN ALL-BLACK AESTHETIC WITH SUBTLE NEON GLOW EFFECTS, SCROLL-TRIGGERED ENTRANCE ANIMATIONS, A MARQUEE TICKER, AND TYPOGRAPHIC HEADINGS THAT MIX REGULAR AND ITALIC WEIGHTS. THE LIVE DEMO RUNS AT https://neonspark-nextjs.vercel.app. THE CLONE REPRODUCES ALL 13 DISCOVERED PAGES: HOME, ABOUT, BLOG, BLOG POST, CAREER, CONTACT, ELEMENTS, FAQ, PRICING, PRIVACY POLICY, SERVICES, SERVICE DETAIL, TEAMS, TERMS AND CONDITION, AND WORK.

- MONO: `ui-monospace, SFMono-Regular, Menlo, Monaco, Consolas, "Liberation Mono", "Courier New", monospace` — CODE BLOCKS

### TYPE SCALE

- HERO H1: `clamp(2.5rem, 5vw, 4.5rem)` BOLD + ITALIC MIX
- SECTION H2: `clamp(1.75rem, 3vw, 3rem)` WITH ITALIC COUNTERPART
- CARD TITLE: `1.125rem` / `600` WEIGHT
- BODY: `1rem` / `400`
- SMALL / META: `0.875rem` / `400`
- NAV LINK: `0.75rem` UPPERCASE TRACKING-WIDER

### RADII & SPACING

- CARD RADIUS: `1rem` (16PX)
- BUTTON RADIUS: `9999PX` (PILL)
- NAVBAR RADIUS (DESKTOP): `9999PX`
- BASE SPACING UNIT: `0.25REM` (TAILWIND DEFAULT)

### ANIMATION EASINGS & KEYFRAMES

- `@keyframes marquee` — `transform: translate(0) → translate(-100%)` CONTINUOUS LOOP, 20S LINEAR INFINITE
- `@keyframes rotateBorder` — `--gradient-angle: 0deg → 360deg` ROTATING CONIC GRADIENT BORDER
- `@keyframes glowPulse` — `drop-shadow(0 0 20px rgba(255,255,255,0.3)) → drop-shadow(0 0 30px rgba(255,255,255,0.4))` 3S EASE-IN-OUT INFINITE
- `@keyframes dramatic-light-entrance` — `opacity:0 scale(0.5) rotate(-90deg) → opacity:1 scale(1) rotate(0)` 1S
- `@keyframes light-horizontal-flow` — TRANSLATE X 0→40PX→-40PX LOOP
- `@keyframes drawPath` — SVG STROKE DASHOFFSET DRAW-IN
- SCROLL REVEALS: `IntersectionObserver` — ELEMENTS START `opacity:0 translateY(30px)` AND TRANSITION TO `opacity:1 translateY(0)` OVER `0.6s ease`
