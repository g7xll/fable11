Getting Started

The fastest way to begin is by following our Quick Start Guide. It walks you through installation, project structure, and your first page — all in under 10 minutes.

ZeroIndex is built on top of Astro, a modern static site builder optimized for content-heavy sites. You get the flexibility of a component framework without the overhead of a full SPA — pages ship as static HTML by default.

Prerequisites

Before you begin, make sure you have the following installed on your machine:

  • Node.js 18 or higher
  • A code editor (VS Code is recommended)
  • Basic knowledge of HTML, CSS, and JavaScript
  • Familiarity with Markdown or MDX (optional but helpful)

Technology Stack

ZeroIndex is built with a carefully chosen set of modern tools:

  • Astro — Static site framework for content-heavy sites
  • Tailwind CSS — Utility-first CSS for rapid UI development
  • Shiki — Syntax highlighter for beautiful code blocks
  • MDX — Markdown with JSX component support

Project Philosophy

ZeroIndex is designed around a few core principles:

  • Developer Experience First — Clear conventions, minimal config
  • Performance by Default — Static output, zero JS where possible
  • Accessible Out of the Box — Semantic HTML, ARIA attributes, keyboard navigation
  • Dark Mode Ready — System-aware theme with manual override
