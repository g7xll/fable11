## SUMMARY

CARRIERA IS A JOB BOARD / CAREER PLATFORM UI TEMPLATE BY LEXINGTON THEMES. IT IS BUILT WITH ASTRO AND TAILWIND CSS V4 AND FEATURES A MINIMAL, TYPOGRAPHY-DRIVEN DESIGN WITH THREE SWITCHABLE COLOR THEMES (LIGHT, DARK, AND GREEN ACCENT). THE TEMPLATE COVERS THE FULL LIFECYCLE OF A JOB BOARD: HOMEPAGE WITH JOB LISTINGS AND CATEGORIES, BLOG, HELP CENTER, PRICING WITH MONTHLY/ANNUAL TOGGLE AND FAQ ACCORDION, SIGN IN / SIGN UP FORMS, JOB DETAIL PAGES, COMPANY PROFILE PAGES, AND CANDIDATE PROFILE PAGES. A FUSE.JS POWERED GLOBAL SEARCH MODAL AND A FIXED THEME-SWITCHER WIDGET ARE PRESENT ON EVERY PAGE.

## STYLE

- **FONT**: GEIST (SANS-SERIF) + GEIST MONO (MONOSPACE) — LOADED FROM GOOGLE FONTS
- **PALETTE (LIGHT THEME)**:
  - `--color-primary`: `oklch(.187 .015 266.77)` — NEAR-BLACK NAVY
  - `--color-secondary`: `oklch(100% 0 0)` — WHITE (PAGE BACKGROUND)
  - `--color-accent`: `oklch(71.7% .157 53)` — AMBER/ORANGE
- **PALETTE (DARK THEME)** (`[data-theme="dark"]`):
  - `--color-primary`: `oklch(100% 0 0)` — WHITE
  - `--color-secondary`: `oklch(39.6% .076 150.1)` — DARK GREEN (`#101319` EFFECTIVELY)
  - `--color-accent`: `oklch(71.7% .157 53)` — AMBER/ORANGE (UNCHANGED)
- **PALETTE (ACCENT/GREEN THEME)** (`[data-theme="accent"]`):
  - `--color-primary`: `oklch(66.7% 0 0)` — MID GRAY
  - `--color-secondary`: `oklch(18.7% .015 266.77)` — VERY DARK NAVY
  - `--color-accent`: UNCHANGED AMBER
- **TYPE SCALE**: XS (0.75REM) → 9XL (8REM); DISPLAY HEADINGS USE `TEXT-8XL` (6REM) WITH `TRACKING-TIGHTER` (−0.05EM) AND `FONT-SEMIBOLD`
- **RADII**: MD=0.375REM, LG=0.5REM, XL=0.75REM
- **TRANSITIONS**: 150MS EASE-IN-OUT DEFAULT; THEME SWITCH USES 250MS TRANSITION ON BG/COLOR/BORDER/FILL/STROKE
- **CONTAINER**: `MAX-W-SCREEN 2XL:MAX-W-[1400PX] MX-AUTO PX-8`
- **HOVER STATES**:
  - NAV LINKS: `HOVER:TEXT-PRIMARY/40` (FADE)
  - BUTTONS: ACCENT BUTTON `HOVER:BG-ACCENT/80`; GHOST BUTTON `HOVER:BG-PRIMARY/20`
  - CARDS/ARTICLES: HOVER TITLE TURNS ACCENT COLOR VIA `GROUP-HOVER:TEXT-ACCENT`
- **THEME SWITCHER**: FIXED BOTTOM-RIGHT, 3 CIRCULAR BUTTONS (WHITE=LIGHT, `#101319`=DARK, `#245230`=GREEN/ACCENT); PERSISTED IN `LOCALSTORAGE`; APPLIED ON BOOT VIA INLINE `<SCRIPT>` TO PREVENT FLASH

## INTERACTIONS

1. **MOBILE MENU**: HAMBURGER "MENU" BUTTON OPENS FULL-SCREEN FIXED OVERLAY; "CLOSE" BUTTON OR RESIZE TO LG CLOSES IT
2. **SEARCH MODAL**: "SEARCH" BUTTON IN NAV OPENS MODAL WITH FUSE.JS FUZZY SEARCH ACROSS JOBS, BLOG POSTS, COMPANIES, CANDIDATES, HELP CENTER ARTICLES
3. **THEME SWITCHER**: 3 CIRCULAR BUTTONS (WHITE/DARK/GREEN) IN FIXED BOTTOM-RIGHT; SETS `DATA-THEME` ON `<HTML>` + SAVES TO `LOCALSTORAGE`; BRIEF CSS TRANSITION ANIMATION ON SWITCH; NO-FLASH BOOT SCRIPT
4. **PRICING TOGGLE**: MONTHLY/ANNUAL BILLING SWITCH UPDATES PRICE DISPLAY
5. **FAQ ACCORDION**: CLICK QUESTION TO EXPAND/COLLAPSE ANSWER
6. **HOVER STATES**: ALL INTERACTIVE ELEMENTS (CARDS, BUTTONS, LINKS, NAV) TRANSITION ON HOVER
