CLI

Master the command-line interface and available commands for efficient development.

Installation

Install the Plasma CLI globally to access all commands from anywhere:

npm install -g plasma-cli

Commands

The Plasma CLI provides a focused set of commands for running, watching, and building your project.

plasma run

Execute a workflow or script once:

plasma run ./workflows/sync.ts

plasma watch

Watch files and re-run on change:

plasma watch ./workflows --ignore node_modules

plasma build

Produce an optimized production bundle:

plasma build --minify --target es2020

plasma init

Scaffold a new Plasma project in the current directory:

plasma init my-app --template starter

Flags

FlagTypeDescription
--configstringPath to a custom config file.
--watch, -wbooleanRe-run when source files change.
--minifybooleanMinify the output bundle.
--targetstringCompilation target (e.g. es2020).
--ignorestring[]Glob patterns to exclude.
--verbose, -vbooleanPrint detailed logging output.

Examples

Run a workflow in watch mode with verbose logging:

plasma run ./workflows/sync.ts --watch --verbose
Tip: Combine plasma watch with a custom --config to run different pipelines per environment.
