# PLASMA — TEMPLATE CLONE

> A SELF-CONTAINED, PIXEL-FAITHFUL REPRODUCTION OF THE "PLASMA" SHADCN/UI TEMPLATE, REBUILT AS PLAIN HTML + CSS + VANILLA JS (NO BUILD STEP). EVERY PAGE, HOVER STATE, DROPDOWN, ACCORDION, TAB, THEME TOGGLE, AND SCROLL-ENTRANCE ANIMATION IS RECREATED SAME-TO-SAME, IN BOTH DARK AND LIGHT MODE.

REFERENCE: `https://www.shadcnblocks.com/template/plasma`

(LIVE DEMO USED FOR RECON: `https://plasma-nextjs-template.vercel.app/`)

## SUMMARY

PLASMA IS A DARK-FIRST SAAS / DEVELOPER-TOOL MARKETING TEMPLATE BUILT ON SHADCN/UI + TAILWIND, SHIPPED BY SHADCNBLOCKS.COM. ITS SIGNATURE IS A DEEP NEAR-BLACK INDIGO BACKGROUND WITH A GLOWING PURPLE→MAGENTA "PLASMA" GRADIENT USED IN HERO HEADLINES, THE FOOTER CTA BAND, PRIMARY BUTTONS, AND CHART ACCENTS. THE SITE SPANS A MARKETING SURFACE (HOME, PRODUCT, PRICING, ABOUT, CHANGELOG, DOWNLOAD), AN AUTH SURFACE (SIGN IN / SIGN UP WITH A SPLIT GRADIENT-ARTWORK CARD), AND A FUMADOCS-STYLE DOCUMENTATION SURFACE (DOCS HUB + SIX SUB-PAGES) WITH A COLLAPSIBLE SIDEBAR, SEARCH, AND "ON THIS PAGE" TOC. INTER IS THE SOLE TYPEFACE. EVERY COLOR IS DRIVEN THROUGH SHADCN THEME TOKENS THAT CARRY BOTH A DARK AND A LIGHT VALUE, SO A THEME TOGGLE RECOLORS THE WHOLE CLONE.
