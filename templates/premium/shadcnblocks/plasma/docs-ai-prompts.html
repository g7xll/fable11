AI Prompts

Discover how to leverage AI capabilities and create effective prompts for your use cases.

Getting Started with AI

Plasma integrates AI capabilities to accelerate development and improve code quality through intelligent assistance.

1. Setup AI Configuration

First, configure the AI settings in the Plasma configuration file: 

export default {
  ai: {
    provider: 'openai',
    model: 'gpt-4',
    apiKey: process.env.PLASMA_AI_KEY,
  },
}

2. Create Your First Prompt

Use the AI prompt system to generate code or get suggestions:

import { ai } from 'plasma';

const result = await ai.prompt('Generate a React component for user profile');

3. Customize Prompts

Enhance your prompts with context and examples for better results.

Prompt Types

Generate code components, functions, and entire modules using AI. 

// Component generation
const component = await ai.generateComponent({
  name: 'UserCard',
  props: ['name', 'email', 'avatar'],
});

Automatically generate documentation from your source code.

const docs = await ai.generateDocs({ source: './src/utils.ts' });

Generate test suites and edge cases for your components.

const tests = await ai.generateTests({ target: './src/UserCard.tsx' });

File Structure

Here's how AI-generated files are organized in your project:

src/
├── ai-generated/
│   ├── components/
│   └── utils/
├── prompts/
│   ├── codegen.ts
│   └── docs.ts
└── ai.config.ts

Advanced Configuration

Configure custom AI models and providers to suit your team's needs. Plasma supports OpenAI, Anthropic, and self-hosted models.
Create reusable prompt templates with variables to standardize AI output across your project.
Provide relevant codebase context to the AI so generated output matches your conventions.

API Reference

Prop Type Description
prompt string The prompt text to send to the AI model.
context object Additional context to improve generation accuracy.
model? 'gpt-4' | 'gpt-3.5-turbo' | 'claude' Override the configured model for this call.

Best Practices

Pro Tip: Always provide specific context and examples in your prompts for better AI responses.
  • Be specific about the desired output format
  • Include relevant project context
  • Test prompts with different inputs
  • Review and refine AI-generated code
