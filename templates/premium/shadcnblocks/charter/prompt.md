# CHARTER — SHADCNBLOCKS PREMIUM TEMPLATE (CLONE)

> A SELF-CONTAINED, PIXEL-FAITHFUL REPRODUCTION OF THE "CHARTER" FINTECH / DEVELOPER-API LANDING TEMPLATE FROM SHADCNBLOCKS, REBUILT AS PLAIN HTML + CSS + VANILLA JS WITH NO BUILD STEP. EVERY PAGE, HOVER STATE, ACCORDION, DROPDOWN, THEME TOGGLE, AND SCROLL/ENTRANCE REVEAL IS REPRODUCED SAME-TO-SAME ACROSS BOTH LIGHT AND DARK MODE.
>
> REFERENCE: `https://www.shadcnblocks.com/template/charter`
>
> LIVE DEMO MIRRORED: `https://charter-nextjs-template.vercel.app/`

## SUMMARY

CHARTER IS A MODERN FINTECH / "DEVELOPER API FOR BUILDING FINANCE PRODUCTS" MARKETING TEMPLATE. THE TONE IS CLEAN AND CORPORATE-TECH: A NEAR-WHITE BACKGROUND (DEEP NAVY IN DARK MODE), DEEP INDIGO PRIMARY COLOR, OVERSIZED CONDENSED-FEEL DISPLAY HEADINGS (HUBOT SANS) WITH NEGATIVE LETTER-SPACING, INTER BODY TEXT, AND IBM PLEX MONO FOR CODE / EYEBROW LABELS. SUBTLE DOTTED-GRID BACKGROUND TEXTURE, SOFT GRADIENT GLOWS BEHIND HEROES, ROUNDED CARDS WITH HAIRLINE BORDERS, AND SCROLL-TRIGGERED FADE/RISE ENTRANCE ANIMATIONS. SHIPS WITH A FULL MARKETING SITE: HOME, ABOUT, PRICING, FAQ, CONTACT (BOOK A DEMO), LOGIN, SIGN UP, TERMS, AND PRIVACY.

- `--primary`: `240 77% 25%` (DEEP INDIGO) / `227 81% 67%` (BRIGHT INDIGO)
- `--primary-gradient`: `240 77% 25%` / `227 81% 67%`
- `--secondary`: `0 0% 96%` / `240 6% 10%`
- `--secondary-foreground`: `240 100% 31%` / `227 81% 67%`
- `--secondary-gradient`: `125 51% 53%` (GREEN ACCENT)
- `--muted`: `0 0% 96%` / `240 6% 10%`
- `--muted-foreground`: `0 0% 49%` / `0 0% 63%`
- `--accent`: `0 0% 96%` / `240 6% 10%`
- `--accent-foreground`: `240 26% 46%` / `227 81% 67%`

- ANNOUNCEMENT BAR: SOLID DEEP NAVY/INDIGO (`primary`) WITH LIGHT TEXT.

### FONTS
- DISPLAY / HEADINGS: **HUBOT SANS** (HEAVY WEIGHTS, TIGHT `-0.02em`..`-0.04em` TRACKING).
- BODY / UI: **INTER** (VARIABLE, OPTICAL SIZING).
- MONO / EYEBROW LABELS / CODE: **IBM PLEX MONO** (UPPERCASE, WIDE TRACKING FOR EYEBROWS LIKE "WHY CHARTER?", "MORE COMPUTER FUGAZI").
- SECONDARY DISPLAY: **MONA SANS** (USED FOR SOME STAT / PRICE NUMERALS).
- LOAD FONTS LOCALLY-VENDORED OR VIA GOOGLE FONTS; FALL BACK TO SYSTEM SANS.

### TYPE SCALE
- HERO H1: ~72–96PX, WEIGHT 800–900, LINE-HEIGHT ~1.0, TRACKING TIGHT (E.G. "SECURE. COMPOSABLE. BANKABLE.", "FREQUENTLY ASKED QUESTIONS", "BOOK A DEMO").
- SECTION H2: ~40–56PX BOLD.
- EYEBROW: ~12–13PX IBM PLEX MONO, UPPERCASE, LETTER-SPACING ~0.15EM, INDIGO.
- BODY: 16–18PX INTER, MUTED-FOREGROUND FOR SUB-COPY.

### RADII / SHADOWS / MOTION
- RADII: `--radius-xs .125rem`, `sm .25rem`, `md .375rem`, `lg .5rem`, `xl .75rem`. CARDS USE LG/XL.
- BORDERS: HAIRLINE 1PX AT ~9% BLACK (LIGHT) / 10% WHITE (DARK).
- SHADOWS: SOFT, LOW-SPREAD CARD SHADOWS; HERO DASHBOARD CARD HAS A LARGER ELEVATED SHADOW.
- BACKGROUND TEXTURE: SUBTLE DOTTED-GRID PATTERN + RADIAL GRADIENT GLOW (INDIGO→GREEN) BEHIND HEROES.
- MOTION: SCROLL-TRIGGERED ENTRANCE (OPACITY 0→1 + TRANSLATE-Y ~16–24PX) VIA INTERSECTIONOBSERVER, STAGGERED; SMOOTH EASE-OUT ~0.5–0.7S. BUTTON / LINK / CARD HOVER TRANSITIONS ~150–200MS.
