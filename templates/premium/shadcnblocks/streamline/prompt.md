IT MARKETS A FICTIONAL ISSUE-TRACKING / PROJECT-MANAGEMENT PRODUCT ("STREAMLINE") WITH A LIGHT, WARM-NEUTRAL EDITORIAL AESTHETIC, A CONDENSED DISPLAY TYPEFACE (INTER TIGHT) FOR HEADINGS, AND PLENTY OF SCROLL-TRIGGERED REVEAL ANIMATIONS. ## STYLE

### PALETTE (SHADCN HSL TOKENS — LIGHT / DARK)
- `--background`: `0 0% 100%` (WHITE) / `240 10% 4%` (NEAR-BLACK)
- `--foreground`: `240 10% 4%` / `0 0% 100%`
- `--muted`: `0 0% 96%` / `240 10% 10%`
- `--muted-foreground`: `0 0% 49%` / `240 5% 65%`
- `--card` / `--popover`: WHITE / NEAR-BLACK
- `--primary`: `240 10% 4%` / `0 0% 100%` (BUTTON FILL = NEAR-BLACK ON LIGHT)
- `--border`: `0 0% 0% / .09` / `0 0% 100% / .1`
- `--input`: `0 0% 89.8%` / `240 10% 10%`
- SIGNATURE WARM SECTION BACKGROUND: `rgb(238, 236, 231)` (`#EEECE7`) — HERO, AUTH PAGES, FAQ HERO
- DARK CTA / FOOTER SECTION: `#0A0A0B` (NEAR-BLACK, `240 10% 4%`)
- BLACK `#000`, WHITE `#FFF`

### FONTS
- HEADINGS / DISPLAY: `Inter Tight` (CONDENSED, WEIGHTS 500–700) — USED FOR H1/H2 HERO & SECTION TITLES
- BODY / UI: `Inter` (WEIGHTS 400–600)
- LOADED VIA GOOGLE FONTS CDN; FALLBACK `ui-sans-serif, system-ui, sans-serif`

### TYPE SCALE
- HERO H1 ≈ 3REM–3.75REM, INTER TIGHT, TIGHT LEADING (~1.05), LETTER-SPACING ~ -0.02EM
- SECTION H2 ≈ 2.25REM–3REM
- BODY ≈ 1REM–1.125REM, MUTED SUBTEXT IN `--muted-foreground`
- SMALL LABELS ≈ 0.875REM

### RADII
- BUTTONS / INPUTS / CARDS: `--radius-lg` 0.5REM, `--radius-xl` 0.75REM
- LARGE CARDS / PRICING / AUTH PANELS: 1REM–1.5REM

### ANIMATION / EASINGS
- SCROLL-REVEAL: SECTIONS START `opacity:0` + SLIGHT `translateY`, REVEAL ON INTERSECTION (~0.5–0.7S EASE-OUT) — PORTED VIA INTERSECTIONOBSERVER
- ACCORDION: `accordion-down/up .2s ease-out` (HEIGHT)
- LOGO MARQUEE: `marquee 25s linear infinite`
- BUTTON / LINK HOVER: ~150MS COLOR/BG/OPACITY TRANSITIONS
