### TYPOGRAPHY
- FAMILY: "SF PRO DISPLAY" (`sfProDisplay`) — VENDORED AS `assets/fonts/sf-pro-display-{300,400,500,700}.otf`; FALLBACK ARIAL / SYSTEM SANS. SMALL SPEC LABELS USE A MONO FALLBACK.
- BODY: 16PX / 1.5, WEIGHT 400, COLOR MUTED-FOREGROUND.
- DISPLAY HEADINGS: LARGE CENTERED (≈44–72PX DESKTOP), TIGHT NEGATIVE LETTER-SPACING, WEIGHT 500–700.
- CARD/SECTION TITLES: ~20PX, WEIGHT 500–600.
- NAV / BUTTON / SMALL LABELS: 14PX, WEIGHT 500.

### RADII / SHADOWS / EASING
- RADII: SM .25REM / MD .375REM / LG .5REM / XL .75REM / 3XL 1.5REM. IMAGE & CARD CORNERS ≈24PX. PRIMARY CTA BUTTONS ARE `ROUNDED-FULL` (H-14); SECONDARY/ICON BUTTONS ROUNDED-MD.
- SHADOWS: SOFT, LOW-OPACITY; CARDS LIFT SUBTLY ON HOVER.
- THEME TOGGLE: CIRCULAR CLIP-PATH VIEW-TRANSITION MASK (`@keyframes theme-mask`) EXPANDING FROM THE TOGGLE; PERSISTED TO `localStorage`; HONORS `prefers-color-scheme` ON FIRST LOAD WITH A NO-FLASH BOOT SCRIPT.
- ENTRANCE ANIMATION: INTERSECTIONOBSERVER-DRIVEN — OPACITY 0→1 WITH `scale(0.9→1)` / SLIGHT TRANSLATEY SETTLE, ~0.6S EASE-OUT. HERO SCALES DOWN SLIGHTLY ON SCROLL.
- HOVER TRANSITIONS: ~0.2S EASE. PRIMARY BUTTONS `hover:bg-primary/95` + `active:scale-95`; NAV LINKS COLOR/UNDERLINE SHIFT; CARDS MINOR BG/TRANSFORM SHIFT.
