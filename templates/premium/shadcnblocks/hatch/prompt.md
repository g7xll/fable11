## SUMMARY

HATCH IS A FREELANCE PRODUCT-DESIGNER PORTFOLIO/AGENCY TEMPLATE BY SHADCNBLOCKS, ORIGINALLY BUILT WITH NEXT.JS + TAILWIND + SHADCN/UI. THE LIVE PREVIEW RUNS AT `https://hatch-nextjs-template.vercel.app/`. THE DESIGN IS EDITORIAL AND CALM: A LARGE CASTORO SERIF DISPLAY HEADLINE, INTER BODY TEXT, IBM PLEX MONO / CAVEAT ACCENTS, GENEROUS WHITESPACE, A SIGNATURE RED-ORANGE CTA COLOR (`#fb001a`), AND A NATIVE LIGHT/DARK THEME TOGGLE PERSISTED TO LOCALSTORAGE.
