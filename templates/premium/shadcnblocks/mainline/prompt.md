## SUMMARY

MAINLINE IS A CLEAN, MODERN SAAS / PRODUCT-MANAGEMENT MARKETING TEMPLATE BUILT ON SHADCN/UI + TAILWIND 4. IT USES A NEUTRAL GRAYSCALE PALETTE WITH A WARM GOLD/TAN ACCENT (THE PRIMARY BUTTON COLOR) AND A FAINT TAN "MAINLINE" WORDMARK WATERMARK AT THE BOTTOM OF EVERY PAGE. THE CHROME IS A FLOATING ROUNDED-PILL NAVBAR (DARK IN HERO) AND A SHARED LINK FOOTER WITH A LARGE "START YOUR FREE TRIAL TODAY" CTA BLOCK. THE TEMPLATE SHIPS WITH A FULL LIGHT + DARK THEME (TOGGLED VIA A SUN/MOON BUTTON, PERSISTED, RESPECTING `prefers-color-scheme`).

### TYPOGRAPHY
- BODY / UI: `Inter` (SANS).
- DISPLAY HEADINGS: `DM Sans` (TIGHT, BOLD — USED FOR HERO + SECTION TITLES + "START YOUR FREE TRIAL TODAY").
- MONO: SYSTEM `ui-monospace` STACK (LABELS LIKE "MEASURE TWICE. CUT ONCE.").
- TYPE SCALE: TAILWIND DEFAULT (`text-xs`…`text-7xl`); HERO ≈ `text-5xl`/`text-6xl` BOLD, TIGHT TRACKING.

### RADII / SHADrows
- `--radius: 8px` (sm/md/lg/xl derived). CARDS USE `rounded-xl`/`rounded-2xl`, BUTTONS `rounded-md`, NAVBAR/PILLS FULLY ROUNDED.
- SOFT SUBTLE SHADOWS ON CARDS, PRICING TILES, AND THE FLOATING NAVBAR.

### MOTION
- RADIX-STYLE TRANSITIONS: ACCORDION HEIGHT/CHEVRON ROTATE, TABS CROSSFADE, DROPDOWN FADE+SLIDE, SWITCH THUMB SLIDE.
- TESTIMONIAL CAROUSEL: HORIZONTAL TRACK WITH PREV/NEXT BUTTONS.
- HOVER: BUTTON/LINK COLOR + BACKGROUND TRANSITIONS (~150MS EASE), CARD BORDER/SHADOW LIFTS.
- THEME TOGGLE: SUN/MOON ICON SCALE+ROTATE SWAP (`.dark` CLASS ON `<html>`).
