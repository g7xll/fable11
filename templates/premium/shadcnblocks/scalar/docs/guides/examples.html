Examples

Practical, copy-paste code examples and starter projects to help you build with Scalar CMS faster.

Starter templates

Clone one of the official starter projects to skip the boilerplate and get a working setup in seconds:

Next.js Blog

A fully wired blog with content models, GraphQL queries, and incremental static regeneration. Run npx create-scalar@latest --template blog to scaffold it.

Astro Docs Site

A documentation site powered by Scalar's rich-text and reference fields, complete with search and a live preview pane. Run npx create-scalar@latest --template docs.

Fetching a single entry

Use findUnique to retrieve one entry by its slug or id:

post.ts
// fetch a single blog post by slug const post = await client.findUnique({ model: "blogPost", where: { slug: "hello-scalar" }, include: ["author"], }); if (!post) throw new Error("Not found");

Paginated queries

Page through large collections with cursor-based pagination:

paginate.ts
const page = await client.query({ model: "blogPost", take: 20, cursor: searchParams.get("cursor"), }); // page.items, page.nextCursor console.log(page.nextCursor);
Always prefer cursor pagination over offset for large datasets — it stays fast and stable even when entries are added or removed between requests.

Filtering & sorting

Combine where filters with orderBy to narrow and arrange results:

filter.ts
const featured = await client.query({ model: "blogPost", where: { status: "published", tags: { has: "featured" }, }, orderBy: { publishedAt: "desc" }, take: 5, });

Real-time subscriptions

Subscribe to content changes and react to updates as they happen:

subscribe.ts
// listen for changes to published posts const unsubscribe = client.subscribe({ model: "blogPost", on: ["create", "update"], }, (event) => { console.log(event.type, event.entry.id); }); // later: unsubscribe();
