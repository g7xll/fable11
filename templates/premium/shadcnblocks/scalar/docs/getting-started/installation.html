Installation

Get Scalar CMS up and running in your project in under five minutes.

Requirements

Before installing Scalar, make sure your environment meets the following requirements:

  • Node.js 18.17 or later
  • A package manager (npm, pnpm, yarn, or bun)
  • A supported database (PostgreSQL 14+, MySQL 8+, or SQLite)

Install the CLI

The fastest way to scaffold a new project is with the create command:

npm pnpm yarn terminal
# scaffold a new Scalar project npx create-scalar@latest my-cms # move into the directory cd my-cms npm run dev

Manual installation

If you'd rather add Scalar to an existing project, install the packages directly:

terminal
npm install @scalar/core @scalar/sdk npm install -D @scalar/cli

Then initialize the configuration file with npx scalar init.

Verify your install

Start the development server and open the admin panel to confirm everything works:

terminal
npm run dev # → Scalar Studio ready at http://localhost:3000/admin
If port 3000 is already in use, set a different port with PORT=4000 npm run dev. The admin URL will update automatically.

Next steps

With Scalar installed, head to the configuration guide to connect your database and define environment variables.
