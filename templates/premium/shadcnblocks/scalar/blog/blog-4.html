Introducing Scalar Cloud

We are launching Scalar Cloud — everything you love about Scalar, now hosted with autoscale, team features, and zero setup.

Yassine Zaanouni · 4 min read

Today we're thrilled to announce Scalar Cloud — a fully managed home for your Scalar projects. Everything you love about Scalar, now with zero infrastructure to run yourself.

What's included

  • Autoscaling infrastructure that handles traffic spikes automatically.
  • Built-in CDN for fast media delivery worldwide.
  • Realtime collaboration and autosave out of the box.
  • Team permissions and SSO for organizations.

Getting started

Connect your existing project with one command and Scalar Cloud provisions everything for you:

terminal
# link your project to the cloud $ scalar cloud link # deploy $ scalar deploy

Your API endpoint goes live instantly with an SSL certificate and global edge caching.

Pricing

Scalar Cloud starts at $29/month for small teams and scales with usage. The open-source core remains free to self-host forever.

Migration

Self-hosted projects migrate with no schema changes — your models, content, and API all move across unchanged.

Wrap-up

Scalar Cloud is the fastest way to run Scalar in production. Spin up a project today and let us handle the infrastructure while you focus on content.
