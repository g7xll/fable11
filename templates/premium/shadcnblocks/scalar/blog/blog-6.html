Building a blog with Scalar, Next.js and Tailwind

Follow this step-by-step guide to launch a performant, developer-friendly blog using Scalar CMS and your favorite tools.

Yassine Zaanouni · 8 min read

In this tutorial we'll build a fast, modern blog from scratch using Scalar for content, Next.js for rendering, and Tailwind for styling. By the end you'll have a deployable blog wired to a structured CMS.

1. Model your content

Start by defining a blogPost model with a title, slug, body, and author reference, then push it to Scalar.

models/blogPost.ts
export const blogPost = defineModel({ name: "blogPost", fields: { title: fields.string(), slug: fields.slug({ from: "title" }), body: fields.richText(), }, });

2. Set up Next.js

Create a Next.js app and install the Scalar SDK. Add your project ID and API key as environment variables.

3. Fetch posts

Use the SDK in a server component to pull published posts at build time:

app/page.tsx
const posts = await client.query({ model: "blogPost" }); return posts.map((p) => <PostCard key={p.id} {...p} />);

4. Style with Tailwind

Add Tailwind's typography plugin and wrap your rich text in a prose container for beautiful defaults with zero custom CSS.

  • Use prose-lg for comfortable reading width.
  • Lazy-load images for fast first paint.
  • Generate static pages with generateStaticParams.

5. Deploy

Push to your host of choice. Static generation plus Scalar's edge cache means your blog loads in milliseconds.

Wrap-up

With Scalar handling content and Next.js handling rendering, you get a fast, type-safe blog you can extend however you like. Clone the starter and ship today.
