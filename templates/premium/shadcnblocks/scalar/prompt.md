## SUMMARY

SCALAR IS A CLEAN, MONOCHROME SHADCN/UI MARKETING + DOCUMENTATION TEMPLATE FOR A FICTIONAL "MODERN HEADLESS CONTENT MANAGEMENT SYSTEM". IT PAIRS A NEAR-BLACK / NEAR-WHITE NEUTRAL PALETTE (OKLCH GREY RAMP) WITH A SINGLE BLUE ACCENT (OKLCH 57% .2328 264 ≈ #2563EB), INTER FOR TEXT AND IBM PLEX MONO FOR CODE. EVERY PAGE SHARES A STICKY HEADER (LOGO, NAV WITH A "PRODUCT" DROPDOWN, GITHUB STAR COUNT, THEME TOGGLE, LOG IN, START FREE TRIAL), A DISMISSIBLE TOP PROMO BAR, A DARK "START FREE. SCALE CONFIDENTLY." OPEN-SOURCE-VS-CLOUD CTA BLOCK, A FOOTER WITH SOCIAL ICONS + "ALL SYSTEMS OPERATIONAL", AND A GIANT FAINT "SCALAR" WORDMARK BLEEDING OFF THE BOTTOM.

## STYLE

- PALETTE (LIGHT `:root` / DARK `.dark`), OKLCH NEUTRAL RAMP:
  - `--background` 99% / 18% · `--foreground` 18% / 99%
  - `--card` 100% / 5% · `--popover` 100% / 5%
  - `--primary` 18% / 99% (near-black / near-white) · `--primary-foreground` 99% / 9%
  - `--secondary` (BLUE ACCENT) `oklch(57% .2328 264.07)` BOTH MODES · `--secondary-foreground` 99%
  - `--muted` 90% / 20% · `--muted-foreground` 60% / 60%
  - `--accent` #f7f7f7 / #080808 · `--border` 89% / 37% · `--input` 89% / 37% · `--ring` 70.8% / 29.2%
  - `--destructive` `oklch(57.7% .245 27.325)`
  - GREEN STATUS DOT `#22c55e`-ish FOR "ALL SYSTEMS OPERATIONAL"
- FONTS: INTER (BODY/HEADINGS, WEIGHTS 400–700), IBM PLEX MONO (CODE BLOCKS / INLINE CODE).
- TYPE: LARGE TIGHT HERO HEADINGS (~clamp 2.5–3.5rem, FONT-WEIGHT 600, TRACKING-TIGHT), MUTED SUB-COPY, SMALL UPPERCASE-ISH LABELS, 14PX BODY.
- RADII: `--radius` ~0.625rem; CARDS/INPUTS ROUNDED, BUTTONS PILL-ISH (rounded-full FOR CTA PILLS, rounded-md FOR FORM BUTTONS).
- BORDERS: HAIRLINE 1PX `--border`; SECTIONS FRAMED IN A CENTERED MAX-W CONTAINER WITH VERTICAL BORDER GUIDES.
- MOTION: SUBTLE — IntersectionObserver FADE/RISE REVEALS, 150–200MS COLOR/BG TRANSITIONS ON HOVER, ACCORDION HEIGHT ANIMATIONS, THEME-TOGGLE ICON CROSSFADE. EASING ~ease / cubic-bezier defaults.
