## SUMMARY

LUMEN IS A MODERN, MINIMAL SAAS MARKETING TEMPLATE BY **SHADCNBLOCKS** (BUILT ON SHADCN/UI + TAILWIND V4). IT PAIRS A CLEAN NEAR-WHITE SURFACE WITH SOFT MULTI-COLOR AURORA GRADIENTS, GENEROUS WHITESPACE, INTER TYPOGRAPHY, AND TIGHT, LOW-CONTRAST BORDERS. THE PRODUCT NARRATIVE IS A "FEATURE MANAGEMENT / WORKFLOW" SAAS, WITH A LANDING PAGE, ABOUT, PRICING, BLOG (INDEX + POSTS), FAQ, AND AUTH + LEGAL PAGES.

THE CLONE IS FULLY THEME-DEPENDENT: ALL COLORS FLOW THROUGH LIGHT/DARK SHADCN TOKENS (OKLCH CUSTOM PROPERTIES). IT HONORS `prefers-color-scheme`, PERSISTS THE USER'S CHOICE IN `localStorage`, BOOTS WITH NO FLASH, AND REPRODUCES THE HEADER THEME TOGGLE.

### TYPOGRAPHY

- PRIMARY FONT: **INTER** (VARIABLE, `opsz 14..32`, `wght 100..900`), `ui-sans-serif` FALLBACK STACK. MONO: **IBM PLEX MONO** / AZERET MONO FOR SMALL LABELS. PLAYFAIR DISPLAY + CAVEAT LOADED BY SOURCE (USED SPARINGLY).
- TYPE SCALE (TAILWIND): `xs .75rem` → `sm .875` → `base 1` → `lg 1.125` → `xl 1.25` → `2xl 1.5` → `3xl 1.875` → `4xl 2.25` → `5xl 3` → `6xl 3.75` → `7xl 4.5`. HEADING LINE-HEIGHTS TIGHTEN AT `5xl+` TO `1`.
- WEIGHTS: NORMAL 400 (BODY / `--text-weight`), MEDIUM 500, SEMIBOLD 600 (DISPLAY / `--display-weight`), BOLD 700.
- TRACKING: HEADLINES `tracking-tight (-.025em)` → `tracking-tighter (-.05em)`.

### RADII / SHADOWS / EASINGS

- `--radius: .625rem` (BASE), `--radius-2xl: 1rem`, `--radius-3xl: 1.5rem`. CARDS USE `xl`/`2xl` ROUNDING.
- SHADOWS: SOFT, LOW-ALPHA `shadow-sm`/`shadow` ON CARDS; ELEVATED HOVER ON BUTTONS/CARDS.
- EASINGS: `--ease-out: cubic-bezier(0,0,.2,1)`, `--ease-in-out: cubic-bezier(.4,0,.2,1)`. DEFAULT TRANSITION `.15s`.
- BLURS: `--blur-sm 8px`, `--blur-md 12px`, `--blur-3xl 64px` (AURORA BLOBS).

### INTERACTIONS

- **THEME TOGGLE** (HEADER): SWITCHES `<html>` CLASS `light` ↔ `dark`; PERSIST IN `localStorage`; NO-FLASH BOOT SCRIPT; HONOR `prefers-color-scheme` ON FIRST VISIT.
- **HEADER**: STICKY, `--header-height: calc(spacing*14)`. FEATURES NAV ITEM OPENS A RADIX-STYLE DROPDOWN/NAVIGATION MENU. MOBILE HAMBURGER OPENS A SLIDE/SHEET MENU.
- **FAQ ACCORDION**: RADIX-STYLE; `aria-expanded` `false`→`true`, PANEL HEIGHT ANIMATES OPEN, CHEVRON ROTATES.
- **PRICING**: MONTHLY/ANNUAL SWITCH RE-RENDERS THE PRICE FIGURES; HERO PLAN CARDS + FULL FEATURE COMPARISON TABLE.
- **BLOG INDEX**: FEATURED POST CARD, CATEGORY FILTER TABS (ALL / DESIGN / PRODUCT / SOFTWARE ENGINEERING / CUSTOMER SUCCESS), "NEWEST FIRST" SORT DROPDOWN, PAGINATED CARD GRID (PAGES 1–2).
- **HOVER**: CARDS LIFT/BORDER-HIGHLIGHT; BUTTONS DARKEN/INVERT; NAV LINKS + FOOTER LINKS COLOR-SHIFT; LOGO MARQUEE.
- **SCROLL/ENTRANCE**: SECTIONS FADE/RISE ON SCROLL (INTERSECTION-OBSERVER), LOGO BAR MARQUEE, FEATURE CAROUSEL.
