# METAFI — SHADCNBLOCKS TEMPLATE CLONE

> A SELF-CONTAINED, PIXEL-FAITHFUL REPRODUCTION OF THE "METAFI" SAAS PAYMENTS TEMPLATE FROM SHADCNBLOCKS, BUILT AS PLAIN HTML + CSS + VANILLA JS WITH NO BUILD STEP. EVERY PAGE, HOVER STATE, AND ENTRANCE ANIMATION IS RECREATED SAME-TO-SAME ACROSS BOTH LIGHT AND DARK THEMES.

REFERENCE: `https://www.shadcnblocks.com/template/metafi`

## SUMMARY

METAFI IS A FINTECH / PAYMENTS SAAS MARKETING TEMPLATE BUILT ON THE SHADCN/UI DESIGN SYSTEM (TAILWIND V4 TOKENS, OKLCH COLORS, INTER TYPEFACE). IT IS A MULTI-PAGE SITE: A LANDING PAGE PLUS FEATURES, INTEGRATIONS, ABOUT, PRICING, BLOG (INDEX + POST), CONTACT, LOGIN, SIGN UP, CAREERS, AND LEGAL PAGES (PRIVACY, TERMS, COOKIE POLICY). THE CLONE IS THEME-DEPENDENT WITH A WORKING LIGHT/DARK TOGGLE THAT PERSISTS TO LOCALSTORAGE AND HONORS `prefers-color-scheme`.

## STYLE

- TYPEFACE: INTER (SANS), IBM PLEX MONO / UI-MONO FOR CODE, SYSTEM FALLBACKS.
- COLOR SYSTEM: SHADCN TOKENS DRIVEN BY CSS CUSTOM PROPERTIES IN OKLCH, WITH A LIGHT BASE AND A `.dark` OVERRIDE.
  - LIGHT: `--background: oklch(1 0 89.88)`, `--foreground: oklch(.172 .006 286.3)`, `--primary: oklch(.172 .006 286.3)`, `--muted: oklch(.9702 0 89.88)`, `--border: oklch(.924 .003 255.9)`, `--accent: oklch(.982 .004 230)`.
  - DARK: `--background: oklch(.23 .02 265)`, `--foreground: oklch(.98 .01 255)`, `--primary: oklch(.98 .01 255)`, `--card: oklch(.26 .02 265)`, `--muted: oklch(.28 .015 265)`, `--border: oklch(.36 .01 265)`.
  - CHART / ACCENT COLORS: `--chart-1..5` (TEAL/AMBER/SLATE PALETTE) FOR THE DASHBOARD MOCKUP.
- RADII: `--radius-sm .25rem`, `--radius-md .375rem`, `--radius-lg .5rem`, `--radius-xl .75rem`.
- TYPE SCALE: LARGE TIGHT-TRACKED DISPLAY HEADINGS (HERO ~3–3.75REM), SECTION HEADINGS ~2–2.5REM, BODY ~1REM/1.5 LINE-HEIGHT, MUTED SUBTEXT.
- MOTION: SUBTLE FADE/RISE ENTRANCE ON SCROLL, SMOOTH COLOR/BG TRANSITIONS ON BUTTONS, CARDS, NAV LINKS (≈150–200MS EASE).
