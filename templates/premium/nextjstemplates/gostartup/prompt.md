# GOSTARTUP — NEXT.JS STARTUP BOILERPLATE (SAME-TO-SAME CLONE)

> A SELF-CONTAINED, PIXEL-FAITHFUL REPRODUCTION OF THE "GOSTARTUP" NEXT.JS BUSINESS/STARTUP BOILERPLATE, REBUILT AS PLAIN HTML + CSS + VANILLA JS WITH NO BUILD STEP. EVERY PAGE, THE FULL LOOK & FEEL, HOVER STATES, LIGHT/DARK THEME TOGGLE, AND SCROLL/ENTRANCE ANIMATIONS ARE REPRODUCED FROM CAPTURED REFERENCE ARTIFACTS. ALL ASSETS ARE VENDORED LOCALLY AND IT RUNS OFFLINE.

REFERENCE: `https://go.demo.nextjstemplates.com`

## SUMMARY

GOSTARTUP IS A DARK-MODE-FIRST SAAS/STARTUP MARKETING SITE (WITH A FULL LIGHT-MODE TOGGLE) BUILT WITH TAILWIND CSS. IT USES A DEEP NAVY-BLACK BACKGROUND, AN INDIGO/BLUE ACCENT ("STARTUP BLUE"), INTER FOR BODY TEXT AND LEXEND FOR HEADINGS. THE SITE HAS A ONE-PAGE MARKETING HOME (HERO, FEATURES, ABOUT, TEAM, PORTFOLIO, TESTIMONIALS, LOGO CLOUD, PRICING, BLOG PREVIEW, CONTACT FORM, FINAL CTA) PLUS A FULL BLOG (INDEX + 6 POST DETAIL PAGES + A TAG-FILTER PAGE), A DOCS SECTION (INDEX + 5 SUB-DOC PAGES WITH A SIDEBAR), A SUPPORT/CONTACT PAGE, SIGN-IN AND SIGN-UP AUTH PAGES, AND A 404 ERROR PAGE. SHARED CHROME = A STICKY HEADER (LOGO, NAV, "PAGES" DROPDOWN, SIGN IN / SIGN UP BUTTONS, SEARCH ICON, LIGHT/DARK TOGGLE) AND A 4-COLUMN FOOTER (COMPANY / SUPPORT / GET IN TOUCH / NEWSLETTER) WITH A BOTTOM BAR.
