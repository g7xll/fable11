## SUMMARY

TAXONOMY IS THE OFFICIAL DEMO APP BUILT BY SHADCN (THE CREATOR OF SHADCN/UI) TO SHOWCASE NEXT.JS 13 APP ROUTER, SERVER COMPONENTS, AND MODERN WEB PATTERNS INCLUDING AUTHENTICATION (NEXTAUTH.JS), DATABASE (PRISMA + PLANETSCALE), AND SUBSCRIPTIONS (STRIPE). IT IS AN OPEN-SOURCE REFERENCE PROJECT FOR DEVELOPERS BUILDING MODERN NEXT.JS APPLICATIONS.

THE SITE FEATURES:
- A MINIMAL BLACK/WHITE DESIGN SYSTEM WITH NEUTRAL COLOR PALETTE AND HSL CSS VARIABLES
- CALSANS-SEMIBOLD AS THE DISPLAY/HEADING FONT, INTER AS THE BODY FONT
- DARK MODE AND LIGHT MODE WITH SYSTEM-PREFERENCE DETECTION AND MANUAL TOGGLE
- RESPONSIVE LAYOUT THAT COLLAPSES TO MOBILE WITH HAMBURGER MENU
- SUBTLE BORDER-BASED CARD DESIGNS, ROUNDED CORNERS (0.5REM), NO SHADOW HEAVY AESTHETICS
