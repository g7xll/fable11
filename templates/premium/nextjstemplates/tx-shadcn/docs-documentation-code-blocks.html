Code Blocks

Taxonomy uses Shiki for syntax highlighting. Shiki is a beautiful yet powerful syntax highlighter, with the same level of accuracy and aesthetics as VS Code's syntax highlighting.

Usage

To add a code block to your MDX file, use the standard Markdown code fence syntax with a language identifier:

```tsx import { Button } from "@/components/ui/button" export default function Page() { return <Button>Click me</Button> } ```

TypeScript Example

import { type ClassValue, clsx } from "clsx" import { twMerge } from "tailwind-merge" export function cn(...inputs: ClassValue[]) { return twMerge(clsx(inputs)) }

Shell Commands

npm install @radix-ui/react-dialog npm install class-variance-authority npm install clsx tailwind-merge
