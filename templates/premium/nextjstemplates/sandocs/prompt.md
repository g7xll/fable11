REFERENCE: `https://sandocs.vercel.app`

## SUMMARY

SANDOCS IS A NEXT.JS + TAILWIND DOCUMENTATION-SITE TEMPLATE BY WEB3TEMPLATES. IT HAS A MARKETING LANDING PAGE (`/`) AND A FULL DOCS SECTION (`/docs/...`) WITH A LEFT SIDEBAR NAV, MARKDOWN-STYLE CONTENT, AND A RIGHT "ON THIS PAGE" TABLE OF CONTENTS. THE CLONE REPRODUCES THE LANDING PAGE PLUS ALL 20 DOCS PAGES DISCOVERED BY CRAWLING THE SIDEBAR.

## STYLE

- FONT: `Inter` (UI/BODY TEXT), MONOSPACE (`ui-monospace, SFMono-Regular, Menlo, Monaco, Consolas`) FOR CODE/TERMINAL BLOCKS.
- PALETTE (LIGHT / DEFAULT):
  - ACCENT/BRAND VIOLET: `#754ffe` (LOGO BADGE), TAILWIND `violet-500` `#8b5cf6`, `violet-600` `#7c3aed`, `violet-100` `#ede9fe`.
  - TEXT: BODY `slate-900`/`black`, MUTED PARAGRAPH `gray-500` `#6b7280`, NAV LINKS `slate-900` `#0f172a`.
  - SURFACES: PAGE BACKGROUND WHITE, HEADER `bg-white` WITH `border-b`, CODE BLOCKS LIGHT GRAY `bg-slate-50`/`gray-100` WITH SYNTAX TOKENS (PINK `#d946ef` CLASS/FUNCTION, CYAN `#06b6d4` KEYWORD, INDIGO `#4f46e5` STRING, GRAY `#64748b` PUNCTUATION, GREEN/RED FOR DIFF INSERT/DELETE).
  - LOGO WORDMARK: ROUNDED VIOLET BADGE (`#754ffe`) WITH WHITE "SAN" + DARK "DOCS" (`#18113c`).
- TYPE SCALE: HERO H1 ~48-56PX BOLD, SECTION H2 ~28-32PX BOLD, DOCS H1 ~34PX BOLD, DOCS H2 ~22PX BOLD, BODY 16-18PX REGULAR, NAV/TOC 14PX.
- RADII: FULL PILLS FOR BADGES/BUTTONS (`rounded-full`), `rounded-lg`/`rounded-xl` FOR CARDS AND CODE WINDOWS.
- SHADOWS: SOFT CARD SHADOWS ON FEATURE CARDS AND THE CODE-WINDOW MOCKUP (TRAFFIC-LIGHT DOTS HEADER).
- ANIMATION/EASING: SIMPLE COLOR/BORDER TRANSITIONS ON LINK AND BUTTON HOVER (`transition-colors`), NO SCROLL-TRIGGERED ANIMATIONS OBSERVED ON THE SOURCE — STICKY HEADER AND STICKY SIDEBAR/TOC ON DOCS PAGES.
- THEME: SOURCE HAS NO DARK MODE. CLONE ADDS A LIGHT/DARK TOKEN SYSTEM (CSS CUSTOM PROPERTIES) HONORING `prefers-color-scheme`, WITH THE SOURCE'S PALETTE AS THE LIGHT DEFAULT AND A FAITHFUL DARK-SLATE COUNTERPART, SO NO PAGE HARDCODES RAW COLORS.
