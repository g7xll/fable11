Prerequisites

  • Node.js 18.0 or higher
  • npm or yarn package manager
  • OpenAI API key
  • A database (PostgreSQL recommended)

Installation Steps

Follow these steps to get your AI Tool project up and running:

git clone https://github.com/your-repo/ai-tool.git

cd ai-tool

npm install

npm run dev

Environment Variables

Configure your contact form by setting the following environment variables in your .env file:

OPENAI_API_KEY=your_openai_api_key

NEXTAUTH_SECRET=your_nextauth_secret

DATABASE_URL=your_database_url

SMTP_HOST=your_smtp_host
