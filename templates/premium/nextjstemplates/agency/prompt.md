## SUMMARY

AGENCY IS A MULTI-PAGE MARKETING TEMPLATE FOR A DESIGN/DEV AGENCY, BUILT WITH NEXT.JS + TAILWIND CSS (V4 UTILITY LAYER) ON THE "GRAYGRIDS/STARTUP" TEMPLATE FAMILY. IT SHIPS A ONE-PAGE MARKETING HOME (HERO, ABOUT, SERVICES, PORTFOLIO GRID, CLIENT LOGOS, NEWSLETTER CTA, TEAM, PRICING, TESTIMONIALS, BLOG PREVIEW, CONTACT FORM) PLUS A FULL SET OF INNER PAGES: SERVICE HUB + 3 SERVICE DETAIL PAGES, PORTFOLIO HUB + 4 PORTFOLIO DETAIL PAGES, BLOG GRID + 7 BLOG POST PAGES, A CONTACT PAGE, A DOCS PAGE, A 404 PAGE, AND SIGN-IN / SIGN-UP AUTH PAGES.
