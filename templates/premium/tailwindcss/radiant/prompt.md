# RADIANT — TAILWIND PLUS SAAS / MARKETING TEMPLATE — STUDY / CLONE

> A SELF-CONTAINED, PIXEL-FAITHFUL REPRODUCTION OF THE TAILWIND PLUS "RADIANT" SAAS MARKETING TEMPLATE, REBUILT AS PLAIN HTML + CSS + VANILLA JS WITH NO BUILD STEP. EVERY PAGE, SECTION, HOVER STATE, AND SCROLL/ENTRANCE BEHAVIOUR IS RECREATED FROM RECON ARTIFACTS. ALL ASSETS (THE SWITZER FONT FAMILY, LOGOS, TEAM/TESTIMONIAL/INVESTOR PORTRAITS, MAP TILES, APP SCREENSHOT, AND BLOG IMAGERY) ARE VENDORED LOCALLY SO THE CLONE RUNS OFFLINE. THIS IS A STUDY OF SOMEONE ELSE'S DESIGN, BUILT FOR LEARNING — ALL CREDIT FOR THE ORIGINAL GOES TO TAILWIND PLUS (TAILWIND LABS).

REFERENCE: `https://tailwindcss.com/plus/templates/radiant/preview`

## SUMMARY

RADIANT IS A PREMIUM SAAS / STARTUP MARKETING TEMPLATE FROM TAILWIND PLUS. IT MARKETS A FICTIONAL B2B SALES-INTELLIGENCE PRODUCT ("RADIANT HELPS YOU SELL MORE BY REVEALING SENSITIVE INFORMATION ABOUT YOUR CUSTOMERS") WITH A BOLD, BRIGHT, GRADIENT-DRIVEN AESTHETIC: A SIGNATURE WARM-TO-MAGENTA-TO-VIOLET LINEAR GRADIENT (`#fff1be → #ee87cb → #b060ff`) FRAMED INSIDE A LARGE ROUNDED `rounded-4xl` (2REM) PANEL, A TIGHT GROTESQUE DISPLAY FACE (SWITZER), DARK NEAR-BLACK CONTRAST SECTIONS, AND FINELY HAIRLINED "PLUS-MARK" GRID DECORATIONS ON THE NAV AND FOOTER.

THE LIVE TEMPLATE IS A NEXT.JS APP SERVED FROM `radiant.tailwindui.com`; THIS CLONE PRESERVES ITS COMPILED TAILWIND CSS V4 STYLESHEET (~59KB OF OBFUSCATED UTILITY CLASSES, OKLCH/LAB COLOR RAMP, GRADIENT + MASK UTILITIES) AND ITS EXACT RENDERED MARKUP, SELF-HOSTS THE SWITZER 400/500/600/700 WOFF2 WEIGHTS, VENDORS ALL IMAGERY LOCALLY, AND REPLACES THE PROPRIETARY HEADLESS-UI + NEXT + FRAMER-MOTION RUNTIME WITH A SMALL VANILLA-JS SHIM THAT REIMPLEMENTS THE SAME BEHAVIOURS (MOBILE NAV DISCLOSURE, FAQ/SECTION ANIMATIONS-ON-SCROLL, HOVER/FOCUS `data-*` STATES, GRADIENT/PARALLAX DECOR).

THE BUILD SPANS 16 PAGES — THE MARKETING **HOME**, A **PRICING** PAGE (PLAN CARDS + FEATURE-COMPARISON TABLE + TESTIMONIAL + FAQ), A **COMPANY** / ABOUT PAGE (STATS, TEAM, INVESTORS, OPEN-ROLES), A **BLOG** INDEX (FEATURED ROW + POST LIST + PAGINATION), A **LOGIN** PAGE, AND 11 INDIVIDUAL **BLOG POST** ARTICLE PAGES (ALL SHARING ONE ARTICLE LAYOUT).

## STYLE

### PALETTE
- SIGNATURE GRADIENT: `bg-linear-115 from-[#fff1be] from-28% via-[#ee87cb] via-70% to-[#b060ff]` (HERO PANEL, FOOTER CTA, LOGIN BG GLOW); `sm:bg-linear-145`
- WARM YELLOW `#fff1be` · PINK `#ee87cb` · VIOLET `#b060ff` — GRADIENT STOPS
- WHITE `#fff` — PAGE / CARD BACKGROUND
- GRAY-950 / NEAR-BLACK `#0a0a0a`-ish (LAB ~8) — DARK "CUSTOMER OUTREACH" + TESTIMONIAL SECTIONS, BUTTONS, INK HEADINGS
- GRAY-600 / GRAY-500 — BODY + META TEXT
- HAIRLINES: `black/5` / `black/10` (LIGHT GRIDS, PLUS-MARK DECOR) · `white/10`–`white/15` (DARK SECTIONS)
- ACCENT CHIPS / GRADIENT TAGLINE TEXT REUSE THE PINK→VIOLET RANGE

### TYPOGRAPHY
- DISPLAY / UI / BODY: **SWITZER** (`Switzer, system-ui, sans-serif`) — SELF-HOSTED WOFF2 AT 400 / 500 / 600 / 700
- MONO FALLBACK: `ui-monospace, SFMono-Regular, Menlo, …` (BADGES / SMALL LABELS)
- HUGE TIGHT DISPLAY HEADINGS (`text-6xl`–`text-8xl`, `tracking-tight`, `font-medium`) FOR HERO "CLOSE EVERY DEAL." AND SECTION TITLES; UPPERCASE EYEBROW LABELS `text-sm/6 font-medium` WITH GRADIENT/INK; BODY `text-base`/`text-lg`, MUTED GRAY

### SPACING / SHAPE
- RADII: HERO / MAJOR PANELS `rounded-4xl` (2REM); CARDS `rounded-3xl` (1.5REM) / `rounded-2xl` (1REM); IMAGES `rounded-xl`; PILLS/BUTTONS FULLY ROUNDED `rounded-full`; MENU `rounded-lg`
- LAYOUT: CENTERED `max-w-2xl lg:max-w-7xl` CONTAINER, `px-6 lg:px-8` GUTTERS; HERO BG GRADIENT INSET `inset-2 bottom-0` WITH `ring-1 ring-black/5`
- "PLUS-MARK" NAV/FOOTER DECOR: 15×15 SVG PLUS GLYPHS AT GRID INTERSECTIONS, `fill-black/10`, WITH DOUBLE HAIRLINE BORDERS
- DARK SECTIONS USE INNER `rounded-3xl/4xl` CARDS WITH SUBTLE RINGS AND BENTO GRID LAYOUTS

### ANIMATION / EASINGS / INTERACTION
- DEFAULT TAILWIND TRANSITION `.15S CUBIC-BEZIER(.4,0,.2,1)`; HOVER COLOR/OPACITY SHIFTS ON NAV LINKS, PLAN CARDS, BLOG CARDS, BUTTONS (DARK PILL BUTTONS DARKEN / LIGHTEN ON HOVER, `data-hover` HEADLESS-UI STATES)
- **ENTRANCE / SCROLL REVEALS** — SECTIONS (LOGO CLOUD, BENTO CARDS, TESTIMONIAL, FAQ) FADE/RISE INTO VIEW VIA INTERSECTIONOBSERVER (REPRODUCING FRAMER-MOTION'S `whileInView` FADE-UP); STAGGERED PER ITEM
- **GRADIENT / DECOR** — STATIC ANGLED LINEAR GRADIENTS WITH MASKS; LOGO-TIMELINE + LOGO-CLUSTER + MAP SECTIONS USE LAYERED ABSOLUTE-POSITIONED IMAGERY
- **MOBILE NAV** — HAMBURGER DISCLOSURE TOGGLES AN OVERLAY/EXPANDED MENU BELOW THE `lg` BREAKPOINT (HEADLESS-UI DISCLOSURE → VANILLA TOGGLE)
- **PRICING FAQ** — "YOUR QUESTIONS ANSWERED" STACKED Q&A LIST (STATIC OPEN IN SOURCE; HOVER/FOCUS STYLING PRESERVED)
- **BLOG** — CARD HOVER LIFT / IMAGE TREATMENT; PAGINATION PREV/NEXT + NUMBERED PAGES; CATEGORY FILTER DROPDOWN
- **LOGIN** — CENTERED CARD ON A GRADIENT-GLOW BACKGROUND; FOCUS RING ON INPUTS; REMEMBER-ME CHECKBOX
