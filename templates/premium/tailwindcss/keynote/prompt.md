## SUMMARY

KEYNOTE IS A PREMIUM SINGLE-PAGE CONFERENCE / EVENT MARKETING TEMPLATE FROM TAILWIND PLUS, SHOWCASED WITH THE FICTIONAL "DECEPTICONF" — "A DESIGN CONFERENCE FOR THE DARK SIDE" HELD 04–06 APRIL 2022 IN LOS ANGELES, CA. IT IS A LIGHT, AIRY, INDIGO/BLUE-TINTED LAYOUT BUILT ON A SOFT `bg-indigo-50` FIELD WITH A LARGE FAINT GRID-PATTERN BACKGROUND IMAGE BLEEDING BEHIND THE HERO AND SPEAKERS SECTIONS, A BOLD DM SANS DISPLAY FACE FOR HEADINGS, INTER FOR BODY, AND A MONOSPACE ACCENT FOR DATES/TIMES. THE LIVE TEMPLATE IS A NEXT.JS APP SERVED FROM `keynote.tailwindui.com`; THIS CLONE PRESERVES ITS EXACT MARKUP/COPY/IMAGERY, SELF-HOSTS THE INTER + DM SANS VARIABLE FONTS, VENDORS ALL SPEAKER PHOTOS / SPONSOR SVGS / BACKGROUND JPEGS LOCALLY, AND REPLACES THE PROPRIETARY HEADLESS-UI TAB GROUPS + NEXT RUNTIME WITH A SMALL HAND-WRITTEN CSS (TAILWIND-EQUIVALENT TOKENS) PLUS A VANILLA-JS SHIM THAT REIMPLEMENTS THE SAME BEHAVIOURS (DATE/LOCATION HEADER TABS, SPEAKERS DAY TAB GROUP, SCHEDULE DAY TAB GROUP, HOVER STATES).

THE BUILD IS A SINGLE PAGE (`index`) WITH: HEADER, HERO, SPEAKERS (DAY-TABBED PHOTO GRID), SCHEDULE (DAY-TABBED TALK LISTS), SPONSORS LOGO CLOUD, NEWSLETTER CTA, AND FOOTER. NO OTHER ROUTES EXIST — ALL NAV LINKS ARE IN-PAGE `#` ANCHORS.
