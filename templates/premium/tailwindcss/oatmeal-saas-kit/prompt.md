# OATMEAL — TAILWIND CSS SAAS MARKETING KIT (OLIVE / INSTRUMENT THEME) — STUDY / CLONE

> A SELF-CONTAINED, PIXEL-FAITHFUL REPRODUCTION OF THE TAILWIND PLUS "OATMEAL" SAAS MARKETING KIT (OLIVE + INSTRUMENT SERIF THEME), REBUILT AS PLAIN HTML + CSS + VANILLA JS WITH NO BUILD STEP. EVERY PAGE, SECTION, HOVER STATE, AND SCROLL/ENTRANCE ANIMATION IS RECREATED FROM RECON ARTIFACTS. ALL ASSETS (FONTS, IMAGES, LOGOS, AVATARS, SCREENSHOTS) ARE VENDORED LOCALLY SO THE CLONE RUNS OFFLINE. THIS IS A STUDY OF SOMEONE ELSE'S DESIGN, BUILT FOR LEARNING — ALL CREDIT FOR THE ORIGINAL GOES TO TAILWIND PLUS (TAILWIND LABS).

REFERENCE: `https://tailwindcss.com/plus/kits/oatmeal/preview?theme=olive_instrument`

## SUMMARY

A PREMIUM SAAS MARKETING KIT FOR A FICTIONAL CUSTOMER-SUPPORT INBOX PRODUCT ("OATMEAL"). THE KIT IS A WARM, QUIET, EDITORIAL DESIGN BUILT ON A SOFT OLIVE / CREAM PALETTE WITH AN ELEGANT "INSTRUMENT SERIF" DISPLAY FACE PAIRED WITH INTER FOR BODY/UI. THE BUILD SPANS THIRTEEN PAGES — THREE HOME LAYOUTS, THREE ABOUT LAYOUTS, THREE PRICING LAYOUTS, TWO PRIVACY-POLICY LAYOUTS, AND TWO 404 PAGES — ALL SHARING A CONSISTENT TOP NAV (PRICING · ABOUT · DOCS · CENTERED "OATMEAL." WORDMARK · LOG IN · BLACK "GET STARTED" PILL) AND A LIGHT FOUR-COLUMN FOOTER (PRODUCT / COMPANY / RESOURCES / LEGAL + "STAY IN THE LOOP" EMAIL CAPTURE + X / GITHUB / YOUTUBE SOCIAL ICONS + "© 2025 OATMEAL, INC.").

THE ORIGINAL IS GENERATED BY TAILWIND PLUS AS STATIC HTML WITH AN INLINED, COMPILED TAILWIND CSS V4.1.16 STYLESHEET (OBFUSCATED UTILITY CLASS NAMES), AN OKLCH "OLIVE" COLOR RAMP, GOOGLE-HOSTED INTER + INSTRUMENT SERIF, AND THE PROPRIETARY `@tailwindplus/elements` MODULE FOR INTERACTIVITY (MOBILE MENU DIALOG, FAQ DISCLOSURES, PRICING MONTHLY/YEARLY TABS, COPY-TO-CLIPBOARD). THE CLONE PRESERVES THE ORIGINAL COMPILED STYLESHEET AND CLASS-DRIVEN MARKUP FOR FIDELITY, SELF-HOSTS THE FONTS AND VENDORS ALL IMAGERY, AND REPLACES THE PROPRIETARY ELEMENTS LIBRARY WITH A SMALL VANILLA-JS SHIM THAT REIMPLEMENTS THE SAME `command` / `commandfor` INVOKER BEHAVIORS (DIALOG SHOW-MODAL/CLOSE, `--toggle` DISCLOSURES, TAB GROUPS, `--copy`) PLUS AN INTERSECTIONOBSERVER ENTRANCE-REVEAL SYSTEM.

## STYLE

### PALETTE (OKLCH "OLIVE" RAMP)
- OLIVE-100 `oklch(96.6% .005 106.5)` — LIGHTEST CREAM (PAGE / SECTION BG)
- OLIVE-200 `oklch(93% .007 106.5)` — CARD / FOOTER SURFACE, HAIRLINE BORDERS
- OLIVE-300 `oklch(88% .011 106.6)` — BORDERS / DIVIDERS
- OLIVE-400 `oklch(73.7% .021 106.9)` — MUTED ICONS / PLACEHOLDERS
- OLIVE-500 `oklch(58% .031 107.3)` — SECONDARY TEXT
- OLIVE-600 `oklch(46.6% .025 107.3)` — BODY / MUTED LABELS
- OLIVE-700 `oklch(39.4% .023 107.4)` — STRONG BODY
- OLIVE-800 `oklch(28.6% .016 107.4)` — NEAR-INK
- OLIVE-950 `oklch(15.3% .006 107.1)` — INK / HEADINGS / BLACK BUTTONS
- BLACK `#000`, WHITE `#fff`

### TYPOGRAPHY
- DISPLAY: **INSTRUMENT SERIF** (`--font-display`, `font-variation-settings:"wdth" 112.5`) — HERO HEADLINES, SECTION TITLES, BIG "PAGE NOT FOUND"
- SANS / UI / BODY: **INTER** (VARIABLE, OPSZ 14..32, WGHT 100..900, + ITALIC)
- MONO: UI-MONOSPACE STACK (CODE SNIPPET ON HOME-03)
- TYPE SCALE: XS .75 / SM .875 / BASE 1 / LG 1.125 / 2XL 1.5 / 3XL 1.875 / 5XL 3REM (DISPLAY HEROES SCALE LARGER VIA RESPONSIVE UTILITIES)
- WEIGHTS: 400 / 500 (MEDIUM) / 600 (SEMIBOLD); HEADING TRACKING `--tracking-tight:-.025em`

### SPACING / SHAPE
- RADII: SM .25 / MD .375 / LG .5 / XL .75REM; PILL BUTTONS FULLY ROUNDED
- SOFT HAIRLINE BORDERS (OLIVE-200/300), GENEROUS WHITESPACE, CENTERED MAX-WIDTH CONTAINERS
- SUBTLE CARD SURFACES (WHITE / OLIVE-100) ON THE CREAM PAGE

### ANIMATION / EASINGS
- DEFAULT TRANSITION `.15S CUBIC-BEZIER(.4,0,.2,1)`; HOVER COLOR/OPACITY SHIFTS ON LINKS, BUTTONS, CARDS, LOGOS
- ENTRANCE: FADE + RISE ON SCROLL VIA INTERSECTIONOBSERVER (REPLACES ANY ORIGINAL MOTION)
- FAQ DISCLOSURE: TOGGLING `hidden` ON THE ANSWER PANEL + 180° CHEVRON/PLUS-TO-MINUS ICON ROTATION
- PRICING TABS: MONTHLY ↔ YEARLY CROSS-SWAP OF PRICE PANELS VIA `aria-selected` TABS
- MOBILE MENU: NATIVE `<dialog>` SHOW-MODAL SLIDE-IN PANEL WITH A CIRCULAR CLOSE BUTTON
- HOME-03 SNIPPET: CLICK-TO-COPY `npm install oatmeal` (`command="--copy"`)
- SCROLLING SVG TURBULENCE GRAIN OVERLAY ON SOME SECTIONS (DATA-URI feTurbulence)
