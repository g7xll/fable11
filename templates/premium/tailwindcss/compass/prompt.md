# COMPASS — TAILWIND PLUS COURSE / PERSONAL-WEBSITE TEMPLATE — STUDY / CLONE

> A SELF-CONTAINED, PIXEL-FAITHFUL REPRODUCTION OF THE TAILWIND PLUS "COMPASS" COURSE TEMPLATE, REBUILT AS PLAIN HTML + CSS + VANILLA JS WITH NO BUILD STEP. EVERY PAGE, SECTION, HOVER STATE, AND SCROLL/ENTRANCE BEHAVIOUR IS RECREATED FROM RECON ARTIFACTS. ALL ASSETS (FONTS, IMAGES, VIDEO POSTERS, LESSON CLIPS) ARE VENDORED LOCALLY SO THE CLONE RUNS OFFLINE. THIS IS A STUDY OF SOMEONE ELSE'S DESIGN, BUILT FOR LEARNING — ALL CREDIT FOR THE ORIGINAL GOES TO TAILWIND PLUS (TAILWIND LABS).

REFERENCE: `https://tailwindcss.com/plus/templates/compass/preview`

## SUMMARY

COMPASS IS A PREMIUM "ONLINE COURSE" / PERSONAL-WEBSITE TEMPLATE FROM TAILWIND PLUS. IT PRESENTS A FICTIONAL SELF-HELP COURSE ("A COMPREHENSIVE JOURNEY THAT HELPS YOU NAVIGATE UNCERTAINTY AND MAKE CHOICES ALIGNED WITH YOUR VALUES AND GOALS") AS A LIGHT, EDITORIAL, TYPOGRAPHY-FIRST READING EXPERIENCE. THE LIVE TEMPLATE IS A NEXT.JS APP SERVED FROM `compass.tailwindui.com`; THIS CLONE PRESERVES ITS COMPILED TAILWIND CSS V4 STYLESHEET (OBFUSCATED UTILITY CLASSES, OKLCH GRAY RAMP) AND ITS EXACT RENDERED MARKUP, SELF-HOSTS THE INTER + GEIST MONO FONTS, VENDORS ALL IMAGERY/VIDEO LOCALLY, AND REPLACES THE PROPRIETARY HEADLESS-UI + NEXT RUNTIME WITH A SMALL VANILLA-JS SHIM THAT REIMPLEMENTS THE SAME BEHAVIOURS (SIDEBAR COLLAPSE, MOBILE MENU, ACCOUNT DROPDOWN MENU, SCROLL-SPY TABLE OF CONTENTS, HOVER/FOCUS `data-*` STATES, NATIVE VIDEO PLAYBACK).

THE BUILD SPANS 23 PAGES — THE COURSE **OVERVIEW** HOME, 20 LESSON **ARTICLE** PAGES (ALL SHARING ONE ARTICLE LAYOUT, EACH WITH ITS OWN VIDEO + PROSE + TABLE OF CONTENTS), AN **INTERVIEWS** GRID, AND A **RESOURCES** PAGE.

## STYLE

### PALETTE (TAILWIND V4 OKLCH GRAY RAMP, LIGHT-FIRST WITH DARK VARIANTS)
- WHITE `#fff` — PAGE BACKGROUND (LIGHT)
- GRAY-50 / GRAY-100 — SUBTLE SURFACES, CODE-LINE HIGHLIGHTS
- GRAY-400 `oklch(.704 .04 256.8)` — MUTED ICONS / SECONDARY TEXT
- GRAY-600 `oklch(.446 .043 257.3)` — BODY / META TEXT
- GRAY-700 `oklch(.373 .039 257.4)` — SIDEBAR LINK TEXT
- GRAY-950 `oklch(.13 .028 261.7)` — INK / HEADINGS / ACTIVE LINKS (AND DARK-MODE PAGE BG)
- BLUE-500 `oklch(.623 .214 259.8)` — MENU-ITEM ACTIVE HIGHLIGHT
- BLUE-700 — FOCUS OUTLINE
- HAIRLINES: `gray-950/10` (LIGHT) · `white/10` (DARK)

### TYPOGRAPHY
- SANS / UI / BODY: **INTER VARIABLE** (`InterVariable`, OPSZ + WGHT, PLUS ITALIC) — SELF-HOSTED WOFF2
- MONO / CODE: **GEIST MONO VARIABLE** — SELF-HOSTED WOFF2 (SHIKI-HIGHLIGHTED CODE BLOCKS)
- PROSE BODY `text-base/7` (16/28) STEPPING TO `sm:text-sm/6`; HEADINGS SEMIBOLD, TIGHT TRACKING
- BREADCRUMB / NAV / META `text-sm/6` (14/24); VIDEO DURATION + TAGS `text-xs`

### SPACING / SHAPE
- RADII: CARDS / VIDEO `rounded-2xl` (1REM); IMAGES `rounded-xl`; MENU `rounded-lg`; CODE LINES `rounded-md`
- LAYOUT: FIXED LEFT COURSE SIDEBAR `w-2xs` (18REM) ON `xl+`, CENTERED `max-w-7xl/6xl/5xl` CONTENT, RIGHT "ON THIS PAGE" TOC ON `lg+`
- HERO BACKGROUND IMAGE WITH `mask-b-from-60%` + `mask-l-from-60%` GRADIENT MASKS AT 40% OPACITY
- GENEROUS WHITESPACE, SOFT HAIRLINE DIVIDERS, NO HEAVY SHADOWS (ONLY MENU/MINI-PLAYER `shadow-sm/lg`)

### ANIMATION / EASINGS / INTERACTION
- DEFAULT TAILWIND TRANSITION `.15S CUBIC-BEZIER(.4,0,.2,1)`; HOVER COLOR SHIFTS ON SIDEBAR LINKS (TEXT → INK + LEFT-BORDER → GRAY-400), NAV LINKS, CARDS, BUTTONS
- HEADLESS-UI-STYLE `data-hover` / `data-focus` / `data-active` STATES ON ICON BUTTONS (FROSTED PILL HALO `before:` ON HOVER, BLUE FOCUS RING)
- **SIDEBAR COLLAPSE** — ICON BUTTON TOGGLES `data-sidebar-collapsed` ON THE `.group` WRAPPER; COLLAPSED HIDES THE ASIDE (`group-data-sidebar-collapsed:hidden`) AND REMOVES THE CONTENT LEFT-MARGIN; A SECOND COLLAPSE BUTTON IN THE TOP BAR RE-EXPANDS
- **ACCOUNT DROPDOWN** — `Account ▾` BUTTON OPENS A FROSTED `rounded-lg` MENU (SETTINGS / SUPPORT / SIGN OUT) ANCHORED BOTTOM-END; `data-focus` ITEM HIGHLIGHTS BLUE-500/WHITE; ESC / OUTSIDE-CLICK CLOSES
- **MOBILE MENU + MOBILE SIDEBAR** — HAMBURGER + KEBAB ICON BUTTONS OPEN OVERLAY PANELS BELOW `xl`/`lg`
- **TABLE OF CONTENTS SCROLL-SPY** — RIGHT "ON THIS PAGE" LINKS GET `aria-current="location"` (INK TEXT + INK LEFT-BORDER) AS THEIR SECTION SCROLLS INTO VIEW (INTERSECTIONOBSERVER)
- **VIDEO** — NATIVE `<video controls poster>`; ON LESSON PAGES THE PLAYER DOCKS TO A FIXED BOTTOM-RIGHT MINI-PLAYER WHEN IT SCROLLS OFF-SCREEN WHILE PLAYING (`group-data-offscreen:data-playing:fixed …`)
- **INTERVIEW CARDS** — HOVER LIFTS THE THUMBNAIL / REVEALS PLAY AFFORDANCE; DURATION BADGE OVERLAY

## LAYOUT & STRUCTURE

### SHARED CHROME
- **COURSE SIDEBAR (HOME + ARTICLES, `xl+`)** — FIXED LEFT `w-2xs` NAV LISTING THE FOUR CHAPTERS AS `<h2>` HEADINGS, EACH WITH A LEFT-BORDERED LIST OF LESSON LINKS; ACTIVE LESSON GETS `aria-current=page` (MEDIUM INK TEXT + INK LEFT-BORDER); A COLLAPSE ICON BUTTON STICKS TO THE TOP.
