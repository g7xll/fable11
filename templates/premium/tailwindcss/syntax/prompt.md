# SYNTAX — TAILWIND PLUS MARKDOC DOCUMENTATION TEMPLATE — STUDY / CLONE

> A SELF-CONTAINED, PIXEL-FAITHFUL REPRODUCTION OF THE TAILWIND PLUS "SYNTAX" DOCUMENTATION TEMPLATE, REBUILT AS PLAIN HTML + CSS + VANILLA JS WITH NO BUILD STEP. EVERY PAGE, SECTION, HOVER STATE, AND SCROLL/ENTRANCE BEHAVIOUR IS RECREATED FROM RECON ARTIFACTS. ALL ASSETS (FONTS, COMPILED STYLESHEET, IMAGES) ARE VENDORED LOCALLY SO THE CLONE RUNS OFFLINE. THIS IS A STUDY OF SOMEONE ELSE'S DESIGN, BUILT FOR LEARNING — ALL CREDIT FOR THE ORIGINAL GOES TO TAILWIND PLUS (TAILWIND LABS).

REFERENCE: `https://tailwindcss.com/plus/templates/syntax/preview`

## SUMMARY

SYNTAX IS A PREMIUM SOFTWARE-DOCUMENTATION TEMPLATE FROM TAILWIND PLUS, BUILT (IN THE ORIGINAL) WITH NEXT.JS + MARKDOC + HEADLESS UI AND SERVED FROM `syntax.tailwindui.com`. IT DOCUMENTS A FICTIONAL CACHING LIBRARY CALLED **CACHEADVANCE** ("NEVER MISS THE CACHE AGAIN.") AS A CLASSIC THREE-ZONE DOCS LAYOUT: A STICKY TOP HEADER (LOGO, COMMAND-K SEARCH PILL, THEME LISTBOX, GITHUB ICON), A FIXED LEFT SIDEBAR NAVIGATION GROUPED INTO FIVE SECTIONS WITH A SLIDING ACTIVE-SECTION HIGHLIGHT AND A SKY-500 ACTIVE-LINK DOT, AND A CENTERED `prose` CONTENT COLUMN WITH A STICKY "ON THIS PAGE" TABLE OF CONTENTS ON THE RIGHT.

THE HOME PAGE OPENS WITH A FULL-WIDTH DARK SLATE-900 HERO — A GRADIENT WORDMARK HEADLINE ("NEVER MISS THE CACHE AGAIN."), A LEAD PARAGRAPH, A "GET STARTED" / "VIEW ON GITHUB" BUTTON PAIR, A DECORATIVE SKEWED DOT-GRID SVG WITH CYAN/INDIGO BLUR GLOWS, AND A SYNTAX-HIGHLIGHTED `cache-advance.config.js` CODE PANEL. BELOW THE HERO IS THE SHARED DOCS SHELL SHOWING THE "GETTING STARTED" ARTICLE.

THIS CLONE PRESERVES THE COMPILED TAILWIND CSS V4 STYLESHEET (OBFUSCATED UTILITY CLASSES, SLATE RAMP, SKY ACCENT) AND THE EXACT RENDERED MARKUP, SELF-HOSTS ITS INTER + LEXEND FONTS, AND REPLACES THE PROPRIETARY NEXT + HEADLESS-UI + FRAMER-MOTION RUNTIME WITH A SMALL VANILLA-JS SHIM THAT REIMPLEMENTS THE SAME BEHAVIOURS (THEME LISTBOX WITH `localStorage`, MOBILE NAV SLIDE-OVER, SEARCH MODAL OVERLAY, SIDEBAR SCROLL-SPY WITH SLIDING SECTION HIGHLIGHT, TOC SCROLL-SPY).

THE BUILD SPANS 21 PAGES — THE **GETTING STARTED** HOME PLUS 20 `/docs/*` ARTICLE PAGES ACROSS FIVE NAV SECTIONS (INTRODUCTION, CORE CONCEPTS, ADVANCED GUIDES, API REFERENCE, CONTRIBUTING). EACH DOC PAGE SHARES THE SAME SHELL, DIFFERING IN TITLE, ACTIVE NAV STATE, AND PREV/NEXT PAGER.

### ANIMATION / EASINGS / INTERACTION
- DEFAULT TAILWIND TRANSITION `.15S CUBIC-BEZIER(.4,0,.2,1)`; HOVER COLOR SHIFTS ON NAV LINKS (SLATE-500 → SLATE-600 + DOT REVEAL), TOC LINKS, BUTTONS, SEARCH PILL RING
- **THEME LISTBOX** — A SUN/MOON ICON BUTTON OPENS A SMALL DROPDOWN WITH LIGHT / DARK / SYSTEM OPTIONS; SELECTING FLIPS `class="dark"` ON `<html>`, PERSISTS TO `localStorage["theme"]`, RESPECTS `prefers-color-scheme` ON FIRST LOAD VIA AN INLINE BOOT SCRIPT (NO FLASH)
- **SIDEBAR ACTIVE HIGHLIGHT** — THE ACTIVE LINK GETS SKY-500 TEXT + A SKY-500 `before:` DOT; A SOFT FRAMER-MOTION-STYLE SECTION HIGHLIGHT BAND TRACKS THE ACTIVE SECTION; INACTIVE LINKS REVEAL A SLATE DOT ON HOVER
- **SEARCH MODAL** — CLICKING THE "SEARCH DOCS" PILL (OR `Ctrl/⌘ K`) OPENS A CENTERED WHITE DIALOG OVER A `bg-slate-900/50 backdrop-blur-sm` BACKDROP WITH A "FIND SOMETHING…" SEARCH FIELD; ESC / OUTSIDE-CLICK CLOSES
- **MOBILE NAV** — BELOW `lg` A HAMBURGER OPENS A LEFT SLIDE-OVER DRAWER (CLOSE-X + LOGO AT TOP, FULL SIDEBAR NAV) OVER A BLURRED BACKDROP; ESC / BACKDROP CLOSES
- **TOC SCROLL-SPY** — THE "ON THIS PAGE" LINKS HIGHLIGHT (SKY-500) THE IN-VIEW HEADING AS YOU SCROLL
- **PROSE LINK UNDERLINE** — `inset` BOX-SHADOW UNDERLINE THAT THICKENS ON HOVER (`--tw-prose-underline-size: 4px → 6px`)
- **QUICK-LINK CARDS** — HOVER REVEALS A GRADIENT BORDER (INDIGO→CYAN→SKY) + TINTED BG
- **HERO BACKGROUND** — SKEWED ROTATED DOT-GRID SVG (SLATE-700 STROKE + SKY-500 HIGHLIGHT DOTS) AT 40% OPACITY, MASKED, WITH CYAN/INDIGO BLUR-PNG GLOWS BEHIND A GRADIENT-RING CODE PANEL (HOME ONLY)

## LAYOUT & STRUCTURE

- **LEFT SIDEBAR (`lg+`)** — FIXED `w-64`/`xl:w-72`, STICKY `top-19`, SLATE-50 SURFACE STRIPE; FIVE NAV GROUPS, EACH A LEXEND `<h2>` LABEL OVER A LEFT-BORDERED LINK LIST. ACTIVE LINK = SKY-500 TEXT + SKY-500 DOT; HOVER REVEALS A SLATE DOT.
