# COMMIT — TAILWIND PLUS CHANGELOG TEMPLATE — STUDY / CLONE

> A SELF-CONTAINED, PIXEL-FAITHFUL REPRODUCTION OF THE TAILWIND PLUS "COMMIT" CHANGELOG / RELEASE-NOTES TEMPLATE, REBUILT AS PLAIN HTML + CSS + VANILLA JS WITH NO BUILD STEP. EVERY SECTION, HOVER STATE, AND THE THEME-TOGGLE INTERACTION IS RECREATED FROM RECON ARTIFACTS. ALL ASSETS (FONTS, COMPILED STYLESHEET, CHANGELOG SCREENSHOTS, FAVICON) ARE VENDORED LOCALLY SO THE CLONE RUNS OFFLINE. THIS IS A STUDY OF SOMEONE ELSE'S DESIGN, BUILT FOR LEARNING — ALL CREDIT FOR THE ORIGINAL GOES TO TAILWIND PLUS (TAILWIND LABS).

REFERENCE: `https://tailwindcss.com/plus/templates/commit/preview`

## SUMMARY

COMMIT IS A PREMIUM CHANGELOG / RELEASE-NOTES TEMPLATE FROM TAILWIND PLUS. IT MARKETS A FICTIONAL "OPEN-SOURCE GIT CLIENT FOR macOS MINIMALISTS" CALLED COMMIT AND PRESENTS ITS RELEASE HISTORY AS A SINGLE, ELEGANT SCROLLING PAGE. THE LAYOUT IS A SPLIT TWO-ZONE DESIGN: A FIXED, ALWAYS-DARK LEFT INTRO PANEL (LOGO, MARKETING HEADLINE, DESCRIPTION, EMAIL-SIGNUP FORM, SOCIAL LINKS, AND A "BROUGHT TO YOU BY" CREDIT) PINNED TO THE VIEWPORT ON `lg+`, AND A SCROLLABLE RIGHT CONTENT COLUMN THAT LISTS FOUR DATED CHANGELOG ENTRIES — EACH WITH A PRODUCT SCREENSHOT CARD, A DATE STAMP, A TITLE, PROSE, AND AN "IMPROVEMENTS" BULLET LIST. A VERTICAL DOTTED RULE RUNS DOWN THE SEAM BETWEEN THE TWO COLUMNS, AND A FAINT SKEWED "CONSTELLATION" LINE-ART SVG SITS BEHIND THE LOGO.

THE LIVE TEMPLATE IS A NEXT.JS APP SERVED FROM `commit.tailwindui.com`. THIS CLONE PRESERVES ITS COMPILED TAILWIND CSS V4 STYLESHEET (OBFUSCATED UTILITY CLASSES, OKLCH GRAY RAMP, SKY-BLUE ACCENT), ITS EXACT RENDERED MARKUP (INCLUDING THE DECORATIVE GRADIENT + CONSTELLATION + DOTTED-SEAM SVGS), SELF-HOSTS ITS INTER + MONA-SANS FONTS, AND REPLACES THE PROPRIETARY NEXT + NEXT-THEMES RUNTIME WITH A SMALL VANILLA-JS SHIM THAT REIMPLEMENTS THE SAME BEHAVIOURS (THEME TOGGLE WITH `localStorage` + `prefers-color-scheme`, AND A DECORATIVE EMAIL-SIGNUP FORM).

THE BUILD IS A **SINGLE PAGE** (THE CHANGELOG HOME). ALL IN-TEMPLATE LINKS ARE EITHER ANCHOR JUMPS TO THE FOUR ENTRIES (`#commit-message-suggestions`, `#project-configuration-files`, `#dark-mode-support`, `#commit-v010`) OR PLACEHOLDER `#` LINKS (DOCUMENTATION / GITHUB / RSS / AUTHOR) — THERE ARE NO ADDITIONAL ROUTES.

- GRAY-50 / GRAY-900 — SCREENSHOT-CARD SURFACE (LIGHT / DARK)
- GRAY-300 — LEFT-PANEL DESCRIPTION TEXT (ON DARK)
- GRAY-400 — DATE TICK MARK
- GRAY-500 — "BROUGHT TO YOU BY" + DATE STAMPS + EMAIL PLACEHOLDER
- WHITE/30 — REST STATE OF SOCIAL / AUTHOR LINK LABELS (LEFT PANEL)
- SKY-300 `#7dd3fc` — "for macOS minimalists" ACCENT, LINK HOVER COLOR, FORM FOCUS RING, IMPROVEMENTS-ICON HIGHLIGHT
- SKY-900 `#0c4a6e` — DOTTED-SEAM RULE (LIGHT)
- HERO RADIAL GLOW: `rgba(56,189,248,0.3) → rgba(0,71,255,0.09) → transparent` BEHIND LEFT PANEL
- IMPROVEMENTS SPARKLE ICON: SKY-400 `#38bdf8` + SKY-300 `#7dd3fc` TWO-STAR GLYPH
- HAIRLINES: `gray-900/10` (LIGHT) · `white/10` (DARK) — SCREENSHOT-CARD RING; `white/15` — FORM RING

### TYPOGRAPHY
- DISPLAY / HEADLINE: `monaSans` (font-display) — H1 `text-4xl/tight font-light`, H2 changelog titles
- SANS / UI / BODY / PROSE: INTER (`font-sans`) — DESCRIPTION `text-sm/6`, BODY PROSE `typography` (mdx) STYLES, META / DATES `text-2xs`/`text-[0.8125rem]`
- MONO / CODE: `ui-monospace` STACK FOR `<code>` (`.commitrc`) AND `<kbd>` (`Cmd` / `Enter`)

### SPACING / SHAPE
- RADII: SCREENSHOT CARDS `rounded-xl`; FORM SUBMIT BUTTON `rounded-md`; SOCIAL/AUTHOR LINK CHIPS + FORM OUTLINE `rounded-lg`
- LAYOUT: LEFT INTRO PANEL FIXED `lg:inset-0` AT `lg:w-96` (24REM) INSIDE A `max-w-[38rem]+40rem` GUTTER SYSTEM; RIGHT COLUMN OFFSET BY `lg:ml-96 lg:pl-32`, ENTRIES IN A `max-w-xl` PROSE MEASURE; VERTICAL SECTION RHYTHM `space-y-20 sm:space-y-32`, `py-20 sm:py-32`
- MINIMAL SHADOWS; SCREENSHOT CARDS USE INSET RING HAIRLINES, NOT DROP SHADOWS

### ANIMATION / EASINGS / INTERACTION
- DEFAULT TAILWIND TRANSITION `.15S CUBIC-BEZIER(.4,0,.2,1)`
- **THEME TOGGLE** — SUN/MOON-STYLE HALF-FILLED CIRCLE BUTTON (TOP-RIGHT) FLIPS `class="dark"`/`"light"` ON `<html>`, PERSISTS TO `localStorage["theme"]`, RESPECTS `prefers-color-scheme` ON FIRST LOAD VIA AN INLINE BOOT SCRIPT (NEXT-THEMES BEHAVIOUR, NO FLASH); ICON OPACITY 50 → 100 ON HOVER; ICON IS WHITE OVER THE DARK LEFT PANEL, GRAY-900 (LIGHT) / WHITE (DARK) OVER THE RIGHT COLUMN
- **SOCIAL / AUTHOR LINK CHIPS** — REST LABEL `text-white` WITH A SCALED-DOWN INVISIBLE `bg-white/5` PILL; HOVER SCALES THE PILL TO FULL (`scale-75 → scale-100`, `opacity-0 → 100`) AND SHIFTS THE WHOLE CHIP TO `text-sky-300`
- **EMAIL-SIGNUP FORM** — FOCUSING THE INPUT RINGS THE OUTLINE SKY-300 (`peer-focus:ring-sky-300` + A 4PX `sky-300/15` GLOW RING); SUBMIT BUTTON HAS LAYERED GRADIENT + INSET-HIGHLIGHT OVERLAYS THAT BRIGHTEN ON HOVER (DECORATIVE — `preventDefault`)
- **CHANGELOG ANCHORS** — EACH ENTRY'S DATE AND TITLE LINK TO ITS `#id`; SMOOTH-SCROLL JUMP
- **DECORATIVE SVGS** — A SKEWED, ROTATED "CONSTELLATION" LINE-ART SVG (FADED, BLURRED DOTS) BEHIND THE LOGO; A RADIAL SKY→BLUE GLOW GRADIENT FILLING THE LEFT PANEL; A 6×8 DOTTED PATTERN RULE DOWN THE COLUMN SEAM
