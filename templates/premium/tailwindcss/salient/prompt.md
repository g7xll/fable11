# SALIENT — TAILWIND PLUS SAAS MARKETING TEMPLATE — STUDY / CLONE

> A SELF-CONTAINED, PIXEL-FAITHFUL REPRODUCTION OF THE TAILWIND PLUS "SALIENT" SAAS MARKETING TEMPLATE, REBUILT AS PLAIN HTML + CSS + VANILLA JS WITH NO BUILD STEP. EVERY PAGE, SECTION, HOVER STATE, AND SCROLL/ENTRANCE BEHAVIOUR IS RECREATED FROM RECON ARTIFACTS. ALL ASSETS (FONTS, SCREENSHOTS, AVATARS, LOGO SVGS, BACKGROUND IMAGES) ARE VENDORED LOCALLY SO THE CLONE RUNS OFFLINE. THIS IS A STUDY OF SOMEONE ELSE'S DESIGN, BUILT FOR LEARNING — ALL CREDIT FOR THE ORIGINAL GOES TO TAILWIND PLUS (TAILWIND LABS).

REFERENCE: `https://tailwindcss.com/plus/templates/salient/preview`

## SUMMARY

SALIENT IS A PREMIUM SAAS / STARTUP MARKETING TEMPLATE FROM TAILWIND PLUS, SHOWCASED WITH THE FICTIONAL ACCOUNTING PRODUCT "TAXPAL" ("ACCOUNTING MADE SIMPLE FOR SMALL BUSINESSES"). IT IS A CLASSIC LIGHT, FRIENDLY, ROUNDED MARKETING LANDING PAGE: BIG LEXEND DISPLAY HEADLINES, A BLUE ACCENT, SOFT SLATE TEXT, HAND-DRAWN SQUIGGLE UNDERLINES, AND FULL-BLEED ANGLED BLUE GRADIENT BACKGROUND BANDS BEHIND THE FEATURE/CTA SECTIONS. THE LIVE TEMPLATE IS A NEXT.JS APP SERVED FROM `salient.tailwindui.com`; THIS CLONE PRESERVES ITS EXACT MARKUP/COPY/IMAGERY, SELF-HOSTS THE INTER + LEXEND VARIABLE FONTS, VENDORS ALL SCREENSHOTS / AVATARS / LOGO SVGS / GRADIENT BACKGROUNDS LOCALLY, AND REPLACES THE PROPRIETARY HEADLESS-UI + NEXT RUNTIME WITH A SMALL HAND-WRITTEN CSS (TAILWIND-EQUIVALENT TOKENS) PLUS A VANILLA-JS SHIM THAT REIMPLEMENTS THE SAME BEHAVIOURS (MOBILE NAV POPOVER, AUTO-ROTATING PRIMARY-FEATURE TAB GROUP, SECONDARY-FEATURE TAB GROUP, HOVER/FOCUS STATES).

THE BUILD SPANS 3 PAGES — THE MARKETING **HOME** PAGE (HERO, LOGO CLOUD, PRIMARY FEATURES, SECONDARY FEATURES, CTA, TESTIMONIALS, PRICING, FAQ, FOOTER), A SPLIT-SCREEN **LOGIN** PAGE, AND A SPLIT-SCREEN **REGISTER** PAGE.

### ANIMATION / EASINGS / INTERACTION
- DEFAULT TRANSITION `.15S CUBIC-BEZIER(.4,0,.2,1)` ON COLORS / TRANSFORMS
- BUTTON HOVERS: DARK BUTTON → SLATE-700; BLUE BUTTON → BLUE-500; OUTLINE "WATCH VIDEO" → SLATE-50 RING; NAV LINKS → SLATE-900 + ROUNDED HOVER BG
- **PRIMARY FEATURES TAB GROUP** (HEADLESS-UI STYLE) — FOUR TABS (PAYROLL / CLAIM EXPENSES / VAT HANDLING / REPORTING) ON A BLUE BAND; SELECTING A TAB SETS `aria-selected`, SWAPS THE FROSTED TAB CARD HIGHLIGHT, AND CROSS-FADES THE ASSOCIATED PRODUCT SCREENSHOT; **AUTO-ROTATES** ON AN INTERVAL UNTIL THE USER INTERACTS; ON MOBILE IT BECOMES A STACKED/HORIZONTAL-SCROLL TAB LIST
- **SECONDARY FEATURES TAB GROUP** — THREE TABS (REPORTING / INVENTORY / CONTACTS) ON A LIGHT SURFACE; CLICK SWITCHES THE ICON-TITLE CARD HIGHLIGHT + SCREENSHOT; ON MOBILE IT FALLS BACK TO STACKED CARDS EACH WITH ITS OWN IMAGE
- **MOBILE NAV POPOVER** — BELOW `lg`, A HAMBURGER ICON TOGGLES TO AN `X` AND OPENS A WHITE `rounded-lg shadow-xl` PANEL (FEATURES / TESTIMONIALS / PRICING, HAIRLINE, SIGN IN) WITH A DIMMED/BLURRED BACKDROP; OUTSIDE-CLICK / ESC CLOSES
- HERO + CTA HEADLINES CARRY HAND-DRAWN SVG SQUIGGLE UNDERLINES UNDER THE BLUE "MADE SIMPLE" / "TODAY" WORDS
- SCROLL: STICKY HEADER STAYS PINNED; FULL-BLEED GRADIENT BACKGROUND BANDS ARE ABSOLUTELY POSITIONED BEHIND THEIR SECTIONS
