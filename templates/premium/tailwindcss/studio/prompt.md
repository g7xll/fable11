# STUDIO — TAILWIND PLUS AGENCY TEMPLATE — STUDY / CLONE

> A SELF-CONTAINED, PIXEL-FAITHFUL REPRODUCTION OF THE TAILWIND PLUS "STUDIO" AGENCY TEMPLATE, REBUILT AS PLAIN HTML + CSS + VANILLA JS WITH NO BUILD STEP. EVERY PAGE, SECTION, HOVER STATE, AND SCROLL/ENTRANCE ANIMATION IS RECREATED FROM RECON ARTIFACTS. ALL ASSETS (THE MONA SANS VARIABLE FONT, EVERY PHOTO, EVERY CLIENT LOGO SVG) ARE VENDORED LOCALLY SO THE CLONE RUNS OFFLINE. THIS IS A STUDY OF SOMEONE ELSE'S DESIGN, BUILT FOR LEARNING — ALL CREDIT FOR THE ORIGINAL GOES TO TAILWIND PLUS (TAILWIND LABS).

REFERENCE: `https://tailwindcss.com/plus/templates/studio/preview`

## SUMMARY

STUDIO IS A PREMIUM "AGENCY / STUDIO" MARKETING TEMPLATE FROM TAILWIND PLUS. IT PRESENTS A FICTIONAL (TONGUE-IN-CHEEK) DANISH DEVELOPMENT STUDIO — "AWARD-WINNING DEVELOPMENT STUDIO BASED IN DENMARK" — ACROSS A BOLD, TYPOGRAPHY-FIRST, LIGHT-ON-WHITE EDITORIAL LAYOUT BUILT AROUND THE MONA SANS VARIABLE TYPEFACE. THE LIVE TEMPLATE IS A NEXT.JS + MDX + FRAMER MOTION APP SERVED FROM `studio.tailwindui.com`; THIS CLONE PRESERVES ITS COMPILED TAILWIND CSS V4.3.1 STYLESHEET (UTILITY CLASSES, OKLCH NEUTRAL RAMP), ITS EXACT RENDERED MARKUP AND COPY, SELF-HOSTS THE MONA SANS WOFF2, VENDORS ALL IMAGERY + CLIENT-LOGO SVGS LOCALLY, AND REPLACES THE NEXT/FRAMER-MOTION RUNTIME WITH A SMALL VANILLA-JS LAYER THAT REIMPLEMENTS THE SAME BEHAVIOURS (EXPANDING NAVIGATION FLYOUT, FADE-IN/FADE-UP ENTRANCE ANIMATIONS, GRID-PATTERN BACKGROUNDS, HOVER STATES, CONTACT FORM).

THE BUILD SPANS 12 PAGES — THE **HOME** PAGE, **ABOUT**, **PROCESS**, THE **WORK** INDEX, THREE WORK **CASE STUDIES** (FAMILYFUND, UNSEAL, PHOBIA), THE **BLOG** INDEX, THREE **BLOG ARTICLES** (FUTURE OF WEB DEVELOPMENT, 3 LESSONS GOING BACK TO THE OFFICE, A SHORT GUIDE TO COMPONENT NAMING), AND **CONTACT**. ALL PAGES SHARE ONE EXPANDING HEADER + ONE FAT FOOTER.

## STYLE

### PALETTE (TAILWIND V4 NEUTRAL RAMP — LIGHT, NEAR-MONOCHROME)
- WHITE `#fff` — PAGE / CARD BACKGROUND
- NEUTRAL-950 `#0a0a0a` — INK / HEADINGS / DARK FLYOUT + FOOTER BACKGROUND / PRIMARY BUTTON FILL
- NEUTRAL-800 `#262626` — BUTTON HOVER FILL, DARK-PANEL HAIRLINES, SECONDARY DARK SURFACES
- NEUTRAL-700 `#404040` — STRONG BODY TEXT
- NEUTRAL-600 `#525252` — BODY / PARAGRAPH TEXT
- NEUTRAL-300 `#d4d4d4` — MUTED TEXT ON DARK, FOOTER SECONDARY TEXT
- NEUTRAL-200 / NEUTRAL-100 / NEUTRAL-50 — HAIRLINES, SUBTLE SURFACES, GRID-PATTERN STROKES
- ACCENT: ESSENTIALLY MONOCHROME — NO BRAND HUE; EMPHASIS COMES FROM TYPE WEIGHT/SIZE AND BLACK↔WHITE INVERSION

### TYPOGRAPHY
- ALL TEXT: **MONA SANS** VARIABLE (`font-weight:200 900; font-stretch:75% 125%`) — SELF-HOSTED WOFF2, `font-display:block`
- DISPLAY HEADINGS: HUGE, TIGHT-TRACKED (`text-5xl`→`text-7xl`, `font-medium`, `tracking-tight`, `text-balance`) — E.G. THE HERO "AWARD-WINNING DEVELOPMENT STUDIO BASED IN DENMARK."
- SECTION EYEBROWS: SMALL-CAPS-LIKE LABEL ROWS WITH A LEADING HAIRLINE RULE (`font-semibold text-neutral-950` + `h-px flex-none bg-neutral-950`)
- BODY: `text-base`/`text-lg`/`text-xl` `text-neutral-600`, RELAXED LEADING
- STAT NUMBERS / META: MONA SANS, MEDIUM

### SPACING / SHAPE
- RADII: PAGE TOP CORNERS `rounded-t-4xl` (THE WHITE PAGE SITS ON A DARK BODY, ITS TOP CORNERS ROUNDED ~2.5REM); CARDS / IMAGES `rounded-3xl`/`rounded-4xl`; BUTTONS / PILLS `rounded-full`
- LAYOUT: CENTERED `max-w-7xl` CONTAINER, `px-6 lg:px-8`, INNER `max-w-2xl lg:max-w-none`; GENEROUS VERTICAL RHYTHM (`mt-24 sm:mt-32 lg:mt-40` BETWEEN SECTIONS)
- GRID-PATTERN BACKGROUNDS: AN OFFSET SQUARES / SKEWED-GRID SVG PATTERN BEHIND HERO + SECTION BANDS (FAINT NEUTRAL STROKES, A FEW FILLED CELLS), CLIPPED INTO ROUNDED BANDS
- FAT FOOTER ON A `bg-neutral-950` PANEL WITH `rounded-t-4xl`, WHITE TEXT, NAV COLUMNS + NEWSLETTER FORM

### ANIMATION / EASINGS / INTERACTION
- **EXPANDING NAVIGATION FLYOUT** — THE HAMBURGER TOGGLES A FULL-WIDTH `bg-neutral-950` OVERLAY THAT GROWS DOWNWARD FROM THE TOP (FRAMER-MOTION LAYOUT ANIMATION IN THE ORIGINAL). HEADER INVERTS TO DARK: LOGO + "CONTACT US" GO WHITE, HAMBURGER MORPHS INTO AN X IN A CIRCULAR `neutral-800` BUTTON. PANEL = 2×2 GRID OF HUGE NAV LINKS (OUR WORK · ABOUT US · OUR PROCESS · BLOG) WITH `neutral-800` HAIRLINE DIVIDERS, THEN AN "OUR OFFICES" + "FOLLOW US" ROW. ESC / OUTSIDE-CLICK / RE-CLICK COLLAPSES.
- **ENTRANCE FADE-INS** (`FadeIn`/`FadeInStagger`) — SECTIONS, LIST ITEMS, CLIENT LOGOS, CASE-STUDY ROWS, AND TEAM CARDS FADE UP (OPACITY 0→1, TRANSLATEY ~24PX→0, ~0.5S EASE) ON LOAD AND AS THEY ENTER THE VIEWPORT, STAGGERED VIA INTERSECTIONOBSERVER
- **LOGO HOVER** — THE LOGOMARK'S INNER RECT ANIMATES WIDTH 0→8 (FILLS THE ARROW GLYPH) OVER 300MS ON HOVER
- **BUTTONS** — `bg-neutral-950 hover:bg-neutral-800` (LIGHT) / INVERTED ON DARK; `transition`
- **LINKS / LIST ROWS** — CASE-STUDY + BLOG ROWS AND TEAM/CONTACT LINKS GET SUBTLE COLOR / UNDERLINE / TRANSLATE SHIFTS ON HOVER
- **CONTACT FORM** — FLOATING-LABEL TEXT FIELDS, RADIO "BUDGET" GROUP, FULL-WIDTH ROUNDED SUBMIT (CLIENT-SIDE ONLY)
