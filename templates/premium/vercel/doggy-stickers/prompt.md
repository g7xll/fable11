> SELF-CONTAINED, PIXEL-FAITHFUL REPRODUCTION OF AN EXISTING SITE — BUILT AS PLAIN HTML/CSS/JS (NO BUILD STEP), INCLUDING HOVER STATES AND INTERACTIVE BEHAVIOR.

REFERENCE: `https://doggystickers.vercel.app/`

## SUMMARY

DOGGY STICKERS IS A SMALL NEXT.JS-POWERED E-COMMERCE DEMO STORE (BUILT BY BILAL TAHIR) SELLING CARTOON DOG STICKERS. IT HAS THREE DISTINCT TEMPLATES: A HOME/CATALOG PAGE LISTING 11 STICKER PRODUCTS IN A CARD GRID, A DYNAMIC PRODUCT-DETAIL PAGE (ONE PER PRODUCT, SAME LAYOUT REUSED FOR ALL 11 SLUGS) WITH AN IMAGE GALLERY/CAROUSEL, QUANTITY AND SIZE SELECTORS, AND AN "ADD TO CART" ACTION, AND A CART PAGE LISTING ITEMS ADDED, WITH QUANTITY CONTROLS, PER-ROW REMOVE, AND A CHECKOUT CTA. THE ORIGINAL WIRES "ADD TO CART" / CHECKOUT TO A PRIVATE SHOPIFY STOREFRONT GRAPHQL API THAT CANNOT BE REPRODUCED STATICALLY — THE CLONE APPROXIMATES THIS WITH A LOCALSTORAGE-BACKED CART SO THE UI AND INTERACTIONS (ADD, UPDATE QTY, REMOVE, CART BADGE COUNT, EMPTY-STATE) BEHAVE IDENTICALLY.

## LAYOUT & STRUCTURE

### 1. HOME / CATALOG PAGE (`/`)
- STICKY HEADER: PAW-PRINT LOGO SVG + "DOGGY STICKERS" WORDMARK (LINKS HOME) ON THE LEFT; SHOPPING-CART ICON BUTTON (LINKS TO `/CART`) ON THE RIGHT, WITH A SMALL BADGE SHOWING THE NUMBER OF ITEMS IN THE CART WHEN NON-EMPTY.
- HERO: CENTERED H1 "GET DOGGY STICKERS!", SUBTITLE "TIMES ARE TOUGH. LIVEN UP YOUR HOME WITH SOME CUTE DOGGY STICKERS.", AND A SMALL DOG-PAW/EMOJI ACCENT BELOW IT.
- PRODUCT GRID: 11 CARDS (THE UNICORN, THE FASHIONISTA, THE DROOLER, THE PLAYER, SCAREDY DOG, THE PEE-ER, THE BRAWL, TIP TOE PUB, DOG BATH, TONGUE WAGGER, ANGRY DOG) EACH WITH A SQUARE PRODUCT IMAGE, BOLD VIOLET TITLE, GRAY TAGLINE, AND A "$9.99" PRICE PILL BOTTOM-RIGHT WITH A NOTCHED FLAG EDGE. WHOLE CARD IS A LINK TO ITS PRODUCT PAGE. CARDS HAVE A HOVER LIFT/SHADOW STATE.
- FOOTER: CENTERED "BUILT WITH ♥ BY BILAL TAHIR" (HEART IN RED, NAME IN BOLD VIOLET, LINKING OUT).

### 2. PRODUCT DETAIL PAGE (`/PRODUCTS/<SLUG>` — ONE SHARED TEMPLATE, 11 DATA VARIANTS)
- SAME STICKY HEADER/FOOTER AS HOME.
- LEFT COLUMN: LARGE SQUARE HERO IMAGE OF THE SELECTED PRODUCT PHOTO ABOVE A HORIZONTAL THUMBNAIL STRIP (STICKER SHOT, LAPTOP LIFESTYLE SHOT, DESK/PLANT LIFESTYLE SHOT) WITH LEFT/RIGHT ARROW BUTTONS TO SCROLL THE STRIP AND SWAP THE HERO IMAGE ON THUMBNAIL CLICK.
- RIGHT COLUMN: "← BACK TO ALL PRODUCTS" OUTLINE BUTTON (VIOLET BORDER/TEXT, HOVER FILLS LIGHT VIOLET), BOLD VIOLET PRODUCT TITLE, GRAY TAGLINE/DESCRIPTION, "$9.99" PRICE IN VIOLET, A "QTY." NUMBER INPUT (DEFAULT 1) AND A "SIZE" DROPDOWN (OPTIONS INCLUDE "3 X 3" ETC.) SIDE BY SIDE, AND A FULL-WIDTH SOLID VIOLET "ADD TO CART" BUTTON WITH A CART ICON THAT DARKENS ON HOVER.
- "ADD TO CART" PUSHES THE SELECTED PRODUCT/QTY/SIZE INTO THE LOCALSTORAGE CART AND UPDATES THE HEADER BADGE.

### 3. CART PAGE (`/CART`)
- SAME STICKY HEADER/FOOTER.
- CENTERED "YOUR CART" H1.
- TABLE WITH COLUMN HEADERS "PRODUCT / QUANTITY / PRICE / REMOVE" UNDER A VIOLET-LIGHTER DIVIDER RULE; EACH ROW SHOWS THE PRODUCT THUMBNAIL + NAME, A QUANTITY STEPPER, THE LINE PRICE, AND A REMOVE (✕) CONTROL. EMPTY CART SHOWS JUST THE HEADER ROW WITH NO ITEMS.
- BELOW THE TABLE: FULL-WIDTH SOLID VIOLET "CHECK OUT →" BUTTON AND AN OUTLINE "← BACK TO ALL PRODUCTS" BUTTON (SAME HOVER TREATMENT AS THE PRODUCT PAGE'S BACK BUTTON). CHECKOUT IS A NO-OP/PLACEHOLDER SINCE THE ORIGINAL'S CHECKOUT IS BACKED BY A PRIVATE SHOPIFY STORE.
