# SAASSPACE - PREMIUM MODERN SAAS LANDING PAGE TEMPLATE DESIGN SYSTEM

> THIS IS A SELF-CONTAINED, PIXEL-FAITHFUL REPRODUCTION OF THE SAASSPACE TEMPLATE FROM TAILGRIDS, REBUILT AS PLAIN HTML + CSS + VANILLA JS WITH NO BUILD STEP. EVERY PAGE, THE FULL LOOK & FEEL, HOVER STATES, AND SCROLL/ENTRANCE ANIMATIONS ARE REPRODUCED FROM CAPTURED REFERENCE ARTIFACTS. ALL ASSETS (FONTS, IMAGES, LOGOS, ICONS) ARE VENDORED LOCALLY AND IT RUNS OFFLINE.
>
> REFERENCE: `https://saasspace.demos.tailgrids.com`

# SUMMARY

A PREMIUM, MODERN, HIGH-CONVERTING SAAS MARKETING LANDING PAGE TEMPLATE DESIGN SYSTEM. IT FEATURES A CLEAN LANDING PAGE, AN ABOUT PAGE, A FEATURES PAGE, A PRICING PAGE, A BLOG DIRECTORY AND DETAIL PAGES, A CONTACT PAGE, AND A COMPLETE DUAL-MODE THEMING SYSTEM. THE SYSTEM USES A MODERN DESIGN WITH VIBRANT BLUE ACCENTS (#3758F9), SHADOW-HERO GRAPHICS, SMOOTH HOVER STATES, BOLD TYPOGRAPHY, AND CLEAN COMPONENT CARDS.

# STYLE

THE STYLE RELIES ON THE 'DM SANS' FONT FOR ALL INTERFACES. IT USES TAILGRIDS BLUE (#3758F9) AS THE ACCENT PRIMARY SHADE, STYLED OVER CLEAN LIGHT SURFACES IN LIGHT MODE, AND DEEP BLUE-BLACK (#090D1A / #0E1326) IN DARK MODE. ALL BORDERS ARE THIN (#E5E7EB IN LIGHT, #1F2937 IN DARK), AND EDGES ARE ROUNDED (LARGE CARDS AT 16PX-24PX).

# SPECIAL COMPONENTS

- **THEME SWITCH TOGGLE:** APPLIES CLASS `.dark` TO `<html>` OR `<body>` ELEMENT AND PRESERVES PREFERENCE IN `localStorage`.
