> A SELF-CONTAINED, PIXEL-FAITHFUL REPRODUCTION OF THE FUNDEX PERSONAL FINANCE DASHBOARD TEMPLATE FROM TAILGRIDS, REBUILT AS PLAIN HTML + CSS + VANILLA JS WITH NO BUILD STEP. EVERY PAGE, THE FULL LOOK & FEEL, HOVER STATES, AND SCROLL/ENTRANCE ANIMATIONS ARE REPRODUCED FROM CAPTURED REFERENCE ARTIFACTS. ALL ASSETS (FONTS, IMAGES, LOGOS, ICONS) ARE VENDORED LOCALLY AND IT RUNS OFFLINE.

REFERENCE: `https://fundex.demos.tailgrids.com/`

## SUMMARY

FUNDEX IS A MODERN, SLEEK PERSONAL FINANCE DASHBOARD DESIGNED ON A CLEAN, TWO-TONE SCHEME (LIGHT MODE AND DARK MODE). THE PALETTE REVOLVES AROUND A PRIMARY VIOLET/PURPLE EMISSION COLOR, WITH NEUTRAL GRAYS AND SHARP STATE ACCENTS (GREEN FOR INCOME/SUCCESS, RED/ROSE FOR EXPENSES/WARNINGS, ORANGE FOR PENDING/ALERT). THE TYPEFACE IS "GEIST", A MINIMALIST AND GEOMETRIC SANS-SERIF. THE TEMPLATE CONTAINS FIVE KEY VIEWS: DASHBOARD (HOME), TRANSACTIONS, ANALYTICS, BUDGETS, AND SETTINGS. ALL VIEWS SHARE A DOUBLE-BAR LAYOUT CONSISTING OF A FIXED SIDE NAVIGATION (SIDEBAR) AND A STICKY TOP HEADER WITH SEARCH, NOTIFICATION, AND THEME TOGGLE CAPABILITIES.

## LAYOUT & STRUCTURE

### SHARED CHROME
- SIDEBAR (ASIDE): LEFT-DOCKED NAVIGATION WITH LOGO HEADER, NAVIGATION LINK ITEM LIST (DASHBOARD, TRANSACTIONS, ANALYTICS, BUDGETS, SETTINGS), "UPGRADE TO PRO" MARKETING BOX GRADIENT, AND A "LOG OUT" RED BUTTON AT THE FOOT. RESPONSIVE COLLAPSIBLE DRAWER ON MOBILE.
- HEADER: STICKY TOP BAR CONTAINING THE MOBILE MENU TOGGLE BUTTON, A SEARCH FIELD WITH SHORTHAND "CMD + K" KEY HINT, THEME MODE SWITCH TOGGLE (SUN/MOON INTERACTIVE ICONS), A NOTIFICATION BUTTON WITH A RED ALERT PIN, AND USER PROFILE AVATAR WITH DISPLAY NAME ("GUSTAVO").

### PAGE: DASHBOARD (HOME)
1. TITLE BAR: "Dashboard Overview" HEADING WITH DESCRIPTIVE SUBTEXT.
2. STAT CARDS ROW: TWO-COLUMN BLOCK FEATURING "Monthly Income" CARD (GREEN STAT ACCENT, EMERALD DECORATION ICON) AND "Monthly Expense" CARD (RED STAT ACCENT, ROSE DECORATION ICON).
3. CASH FLOW WIDGET: LARGE LINE/AREA GRAPH (RECHARTS-STYLE SVG CHART) COMPARING INCOME TREND OVER MONTHS WITH FILTER SELECT FOR RANGE (WEEKLY, MONTHLY, YEARLY).
4. RECENT TRANSACTIONS TABLE: TABLE DISPLAYING TRANSACTION ENTRIES (SALARY PAYMENT, CHATGPT, NETFLIX, TRANSFER) WITH THE CORRESPONDING ACCENT CIRCLE AVATARS/BRAND LOGOS AND STATUS BADGES.
5. BALANCE WIDGET: SOLID VIOLET-GRADIENT DECORATIVE CARD SHOWING "Total balance" ($124,580.45), USD CURRENCY ACCENT FLAG, AND DUAL PILL BUTTONS ("Send", "Received").
6. MY CARD WIDGET: VISA DEBIT CARD PRESETS SWITCHER WITH LEFT/RIGHT CAROUSEL NAVIGATION CONTROLS.
7. BUDGET PROGRESS: WIDGET DEPICTING DOUGHNUT CHART WIDGET AND CATEGORIZED LEGENDS ("Daily Needs", "Savings", "Shopping", "Others") FOR EXPENSES BREAKDOWN.

### PAGE: TRANSACTIONS
1. ACTION HEADER: TITLE "Transactions" WITH DESCRIPTIVE CAPTION AND DUAL BUTTON ACTIONS FOR "Export CSV" AND "Download PDF".
2. FILTER CONTROL CARD: FULL CONTAINER INCORPORATING INPUT SEARCH BY ID/DESCR, AND RANGE SELECTORS (DATE RANGE, ALL CATEGORIES, ALL TYPES, ALL STATUSES).
3. MASTER TRANSACTION LIST: EXTENDED LIST OF RECENT TRANSACTIONS WITH COLUMNS FOR Description, Category, Transaction ID, Amount, Status (Completed, Pending, Failed), Order Date, and Action.
4. TABLE PAGINATION: FOOTER BAR CONTROLLING PAGE NUMBERS (PREVIOUS, PAGE 1, 2, ..., NEXT).

### PAGE: ANALYTICS
1. TITLE & INSIGHT CARD ROW: THREE AVERAGE STATISTICS FOR "Avg Monthly Income", "Avg Monthly Expense", AND "Average Savings Rate".
2. CASH FLOW ANALYTICS GRAPH: EXTENDED RECHARTS CHART FOR ACCUMULATIVE DATA VISUALIZATION.
3. BUDGET STATUS PROGRESS BAR: LINE PROGRESSION BAR INDICATING PERCENTAGE TARGET REMAINING OUT OF MONTHLY EXPENSES.

### PAGE: BUDGETS
1. BUDGET ANALYSIS ROW: STAT CARDS DOCKING PIE CHART DATA (SPENT VS AVAILABLE) ALONGSIDE BUDGET INSIGHT LOG ITEMS.
2. CATEGORY BUDGETS: REUSABLE LIST ITEMS SHOWING INDIVIDUAL CATEGORIES (FOOD & DINING, TRANSPORT, HOUSE RENT) WITH STRIPED PROGRESS PERCENTAGE BAR, VALUE REMAINING, STATUS ACCENT BADGES (SAFE, NEAR LIMIT, EXCEEDED) AND ACTION CONTROLS (EDIT, VIEW TRANSACTIONS, RESET, DELETE).

### PAGE: SETTINGS
1. SETTINGS MAIN NAV BAR: HORIZONTAL SIDE NAV TAB SELECTOR FOR Profile, Cards & Accounts, Security, Notifications, Data Management, App Preferences, Legal and support.
2. FORM WIDGETS: DOCKED PROFILE FORM HANDLING USER AVATAR IMAGE UPLOADER, First Name, Last Name, Email, Phone (with US country prefix and flag dropdown), Address, Currency, Language, and Timezone dropdowns.
3. FOOTER CONTROL: "Save Changes" VIOLET PILL ACTION BUTTON.

## INTERACTIONS
- SIDEBAR COLLAPSE DRAWER: SLIDES OUT FROM LEFT ON MOBILE SCREEN TRIGGERS (`-translate-x-full` TO `translate-x-0`).
- THEME TOGGLE: MANAGES SYSTEM THEME BY WRITING CLASS `.dark` ON THE `<html>` ELEMENT AND STORING PREFERENCE TO `localStorage`.
- CAROUSEL WIDGET: SLIDES PRESETS CARDS SMOOTHLY LEFT/RIGHT IN "MY CARD" AREA.
- CHART INTERACTIVITY: SIMULATED SVG TOOLTIPS REVEALING SPECIFIC DATA ON HOVER.
- FORM SELECTORS: DROPDOWN MENU SELECTION AND ACTIVE TABS SEAMLESS SELECTION STATE TOGGLE.
