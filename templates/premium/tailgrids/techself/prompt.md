> BUILD A MULTI-PAGE, SELF-CONTAINED ELECTRONICS AND GADGET E-COMMERCE SYSTEM, REPRODUCED SAME-TO-SAME (PIXEL / LAYOUT PARITY) FROM THE REFERENCE PREVIEW. SHIP IT AS STATIC HTML + CSS + VANILLA JS, RUNNABLE OFFLINE WITH THE DM SANS FONT VENDORED LOCALLY. NO BUILD STEP REQUIRED.

# SUMMARY

A PREMIUM, HIGH-CONVERTING E-COMMERCE DESIGN SYSTEM TAILORED FOR GADGETS AND ELECTRONICS RETAILERS. IT FEATURES A CLEAN LANDING PAGE, A RICH SHOPPING DIRECTORY WITH ADVANCED SIDEBAR FILTERS, AN INTERACTIVE PRODUCT DETAILS PAGE WITH TABS AND GALLERY, A MINIMAL CHECKOUT INTERFACE, AND A SUCCESS CONFIRMATION SCREEN. THE DESIGN USES VIBRANT VIOLET ACCENTS, LIGHT GRADIENTS, SHARP PRODUCT TILES, AND A DETAILED DUAL-MODE THEMING SYSTEM.

# STYLE

THE STYLE RELIES ON THE 'DM SANS' FONT FOR ALL INTERFACES. IT USES VIOLET (#7C3AED / #8B5CF6) AS THE ACCENT SHADE, STYLED OVER LIGHT SURFACES AND GRADIENTS IN LIGHT MODE, AND DEEP SLATE/ZINC GRAYS IN DARK MODE. ALL BORDERS ARE THIN (#E5E7EB), AND EDGES ARE HEAVILY ROUNDED (CONTAINERS AT 12PX-24PX).

# LAYOUT & STRUCTURE

THE CLONE LIVES IN A MULTI-PAGE STRUCTURE WITH A SHARED HEADER, TOP INFO BAR, NAVIGATION BAR, MEGA-MENU PANEL, AND FOOTER. THE WIDTH IS CAPPED AT 1280PX (MAX-W-7XL) WITH NATIVE RESPONSIVE BREAKPOINTS (SM, MD, LG, XL).

## HEADER & NAVIGATION (SHARED)

- **TOP NOTIFICATION BAR:** LIGHT GRAY BAR DISPLAYING LANGUAGE/CURRENCY POPUP BUTTONS, CENTRAL RUNNING BANNER "FLASH SALE LIVE – 30% OFF EVERYTHING", AND THE RIGHT-HAND "SIGN IN / REGISTER" GUEST NAVIGATION LINK.
- **MAIN HEADER:** STICKY WHITE OR DARK NAV WITH LOGO ("TECHSELF"), GADGET SEARCH BAR WITH INTEGRATED SEARCH ICON, MIDDLE LINKS (HOME, SHOP, PRODUCTS MEGA MENU, SALE), AND RIGHT INTERACTIVE BUTTONS (WISHLIST COUNT, SHOPPING CART DRAWER / TOGGLE WITH LIVE ITEM COUNTER, AND THE DARK MODE TOGGLE SWITCH).
- **MEGA MENU DRAWER:** TRIGGERED BY "PRODUCTS" LINK HOVER/CLICK. CONTAINS GRID COLUMNS OF CATEGORIES ("SMART DEVICES", "AUDIO & ENTERTAINMENT", "ACCESSORIES") WITH ASSOCIATED PRODUCT SUB-LINKS, AND A BANNER SIDEBAR IMAGE WITH "BEST SELLER" PROMO CALL-TO-ACTION.

## HOME PAGE (`index.html`)

- **HERO MASONRY GRID:** 3-COLUMN LAYOUT. LEFT CARD IS A MASSIVE IMMERSIVE PROMO PANEL ("IMMERSIVE VR EXPERIENCE" WITH HIGH-RES HERO BACKGROUND IMAGE, TAG, HEADER, AND VIOLET "DISCOVER COLLECTION" BUTTON). MIDDLE COLUMN FEATURES VERTICAL STACK OF GAILY COLORED CARDS ("STYLE THAT MOVES WITH YOU" ON VIOLET-100 AND "TABLET - A BIGGER SCREEN" ON SKY-100). RIGHT COLUMN DETAILS SMARTPHONES ON BLUE-100.
- **CATEGORY CAROUSEL:** CURATED SLIDER WITH ROUNDED CARDS REPRESENTING PRODUCT GROUPS (SMARTPHONE, LAPTOP, WATCH, TV, CAMERA, SOUND BOX). NAVIGATION CONTAINS LEFT/RIGHT CIRCULAR ICON BUTTONS.
- **TRENDING PRODUCTS GRID:** 8-ITEM PRODUCT GRID (4 COLUMNS). EACH PRODUCT DISPLAY CONTAINS A "NEW" OR "DISCOUNT" ACCENT BADGE, HIGH-QUALITY RENDERED IMAGE, OVERLAY CONTROL BUTTONS (WISHLIST, DETAIL VIEW, COMPARE) THAT SLIDE/FADE IN ON HOVER, TITLE, CURRENT PRICE, STRIKETHROUGH PAST PRICE, AND COLOR SELECTOR CIRCLES.
- **PROMOTIONAL MIDDLE BANNER:** DUAL PROMO BLOCK SPLIT FOR ADVERTISING ACCESSORIES AND ELECTRONICS WITH BOLD CTA BUTTONS.
- **CLIENT LOGOS:** GRAYSCALE GRID OF GADGET BRAND LABELS.
- **LATEST BLOGS:** 3-COLUMN RECENT GADGET REVIEWS AND NEWS.

## SHOP DIRECTORY (`shop.html`)

- **BREADCRUMBS BAR:** TOP NAVIGATION HISTORY TRACKER.
- **SHOPPING GRID WRAPPER:** 2-COLUMN BENTO-LIKE SYSTEM.
  - **LEFT FILTER SIDEBAR (WIDTH: ~300PX):** INCLUDES FILTER ACCORDIONS FOR CATEGORIES (COLLAPSIBLE TABS), PRICE RANGE TRACKER, BRAND CHECKBOXES, AND RATING STAR FILTERS.
  - **RIGHT PRODUCT DIRECTORY:** TOP TOOLBAR WITH ACTIVE RESULTS COUNT AND SORTING COMBOBOX. BOTTOM GRID DISPLAYS 9 TO 12 DETAILED GADGET TILES WITH COMPREHENSIVE E-COMMERCE CARDS.

## PRODUCT DETAILS (`product-detail.html`)

- **2-COLUMN DETAILS SPLIT:**
  - **LEFT COLUMN (GALLERY):** LARGE ACTIVE PRODUCT IMAGE WITH OVERLAY CONTROLS AND HORIZONTAL SLIDER BAR OF SECONDARY THUMBNAIL PREVIEWS.
  - **RIGHT COLUMN (ATTRIBUTES):** GADGET METADATA, TITLE, RATING STAR RATING WITH NUMERICAL COUNT, PRICING (CURRENT VS OLD), COLOR CIRCLES, SIZE SELECTOR, QUANTITY INCREMENTOR, AND PRIMARY ACTION BUTTONS ("ADD TO CART", "BUY NOW") PLUS SOCIAL SHARING LINKS.
- **INFORMATION TABS PANEL:** THREE ACCORDION TABS ("DESCRIPTION", "SPECIFICATIONS", "REVIEWS") DISPLAYING DETAILED RAW DATA SHEETS.
- **RECOMMENDED CAROUSEL:** SWIPING GRID OF 4 RELATED GADGETS.

## CHECKOUT INTERFACE (`checkout.html`)

- **BILLING & SHIPPING FORMS:** CLEAN INPUT TEXTFIELDS ARRANGED IN GRID SECTIONS (FIRST/LAST NAME, EMAIL, PHONE, ADRESS, STATE, COUNTRY, POSTCODE).
- **SHIPPING METHODS:** SELECTABLE CARRIER CARDS WITH PRICING AND TRANSIT DAYS.
- **PAYMENT OPTIONS:** SELECTABLE SECTIONS WITH CREDIT CARD FORM AND ALTERNATIVE OPTIONS.
- **ORDER SUMMARY CARD:** BOX TO THE RIGHT OF FORMS WITH LINE ITEMS, PRICE SUMMARIES, PROMO INPUT FIELD, TAX/SHIPPING SUMS, AND BOLD "PLACE ORDER" CTA.

## THANK YOU SCREEN (`thank-you.html`)

- **SUCCESS CHECKMARK COMPONENT:** CENTRED VIOLET ROUNDED ICON INDICATING ORDER CONFIRMATION.
- **SUCCESS DETAILS:** DETAILED MASSAGE WITH AN ORDER REFERENCE ID AND BOLD "CONTINUE SHOPPING" BUTTON ROUTING BACK TO `shop.html`.
