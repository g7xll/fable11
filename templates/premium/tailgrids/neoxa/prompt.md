> THIS IS A SELF-CONTAINED, PIXEL-FAITHFUL REPRODUCTION OF THE NEOXA TEMPLATE.
> REFERENCE: `https://neoxa.demos.tailgrids.com`

## SUMMARY

NEOXA IS A MODERN REACT SAAS DASHBOARD TEMPLATE BUILT WITH TAILWIND CSS. IT INCLUDES A FULL SET OF COMPREHENSIVE SAAS DASHBOARD PAGES, COMPONENT VIEWS, USER MANAGEMENT, ORDER/INVOICE VIEWS, AUTHENTICATION PAGES, AND CORE SETTINGS. THE DESIGN FEATURES A PREMIUM MINIMALIST CARD-BASED LAYOUT, SMOOTH SIDEBAR NAVIGATION, EXPANDABLE SUBMENUS, INTERACTIVE CHARTS, DATA TABLES, MODALS, AND FULL LIGHT/DARK MODE SUPPORT.

## LAYOUT & STRUCTURE

THE APPLICATION CONSISTS OF 10 MAIN PAGES SHARING A PERSISTENT SIDEBAR NAVIGATION WRAPPER (ON DESKTOP) AND A TOP ACTIONS HEADER:

1. **DASHBOARD (`index.html`)**:
   - HEADER: BREADCRUMBS, COLLAPSIBLE SIDEBAR TOGGLE, SEARCH BAR WITH SHORTCUT (⌘K), NOTIFICATIONS TOGGLE, USER PROFILE EXPANDABLE DROPDOWN.
   - SIDEBAR: BRAND LOGO, COLLAPSIBLE TOGGLE, MENU SECTIONS (DASHBOARD, USERS, ORDERS, INVOICES), SUPPORT & SETTINGS (SUPPORT TICKETS, SETTINGS), AUTH (SIGN IN, SIGN UP, RESET PASSWORD, TWO STEP VERIFICATION).
   - STAT CARDS: 4 METRIC BLOCKS (USER SIGNUPS, CHURN RATE, SUPPORT TICKETS, ACTIVE SUBSCRIPTIONS) WITH TREND METRICS AND Sparkline INDICATORS.
   - CHARTS ROW: REVENUE AREA CHART (MONTHLY/ANNUALLY/LIFETIME CHANGER) AND TRAFFIC/TRANSACTION HISTORY LIST WITH FILTERS.
   - DONUT/PIE CHART: USER RETENTION PROFILE & DEVICE DISTRIBUTION BREAKDOWN.
2. **USERS (`users.html`)**:
   - SIDEBAR & HEADER WRAPPER.
   - MAIN CONTENT: INTERACTIVE SEARCHABLE AND FILTERABLE USER DATA TABLE (AVATAR, NAME, EMAIL, ROLE, STATUS, ACTIONS DROPDOWN).
   - PAGINATION CONTROLS AND BULK SELECTION CHECKBOXES.
3. **ORDERS (`orders.html`)**:
   - SIDEBAR & HEADER WRAPPER.
   - MAIN CONTENT: ORDER MANAGEMENT INTERFACE WITH LIST OF RECENT TRANSACTIONS, ITEM DETAIL OVERVIEWS, ORDER STATUS BADGES (COMPLETED, PENDING, CANCELLED), TOTAL REVENUE METRIC SUMMARIES.
4. **INVOICES (`invoices.html`)**:
   - SIDEBAR & HEADER WRAPPER.
   - MAIN CONTENT: INVOICE TABLE WITH BILLING HISTORY, PDF DOWNLOAD LINK INVOCATIONS, STATUS LABELS (PAID, UNPAID, OVERDUE), INVOICE DETAIL SHEET / DRAWER PREVIEWS.
5. **SUPPORT (`support.html`)**:
   - SIDEBAR & HEADER WRAPPER.
   - MAIN CONTENT: SUPPORT TICKETS BOARD WITH COMPREHENSIVE CARD LISTING STATUS (OPEN, PROGRESS, CLOSED), PRIORITY METRICS (HIGH, MEDIUM, LOW), ASSIGNED AGENTS.
6. **SETTINGS (`settings.html`)**:
   - SIDEBAR & HEADER WRAPPER.
   - MAIN CONTENT: USER PROFILE PRESET MANAGEMENT, CONFIGURATION FOR NOTIFICATIONS, SECURITY TAB (PASSWORD UPDATES, 2FA STATUS), APP DISPLAY (LIGHT/DARK PREFERENCES).
7. **SIGN IN (`signin.html`)**:
   - FULL-SCREEN AUTHENTICATION LAYOUT.
   - LEFT HAND CARD: BRAND MARK, FORM FIELDS (EMAIL, PASSWORD), "REMEMBER ME" AND "FORGOT PASSWORD" OPTIONS, BLUE CALL-TO-ACTION BUTTON, LINK TO SIGN UP.
   - RIGHT HAND CARD: PROMOTIONAL SPLASH ARTWORK & FEATURES LISTING.
8. **SIGN UP (`signup.html`)**:
   - FULL-SCREEN AUTHENTICATION LAYOUT (MATCHING SIGN IN SPLIT STRUCTURE).
   - REGISTRATION FORM (FULL NAME, EMAIL, PASSWORD, TERMS AGREEMENT CHECKBOX, CTA BUTTON).
9. **RESET PASSWORD (`resetpassword.html`)**:
   - FULL-SCREEN AUTHENTICATION LAYOUT.
   - SINGLE CENTERED FORM CONTAINER FOR INITIATING PASSWORD RECOVERY WITH EMAIL SUBMISSION AND BACK-TO-LOGIN NAVIGATION.
10. **TWO STEP VERIFICATION (`twostepverification.html`)**:
    - FULL-SCREEN AUTHENTICATION LAYOUT.
    - SECURE OTP INPUT INTERFACE WITH MULTIPLE DIGIT CODES, BACK TO LOGIN LINKS, RESEND TIMERS.
