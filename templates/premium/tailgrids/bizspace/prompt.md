# BIZSPACE - PREMIUM PROFESSIONAL AGENCY & STARTUP TEMPLATE DESIGN SYSTEM

> THIS IS A SELF-CONTAINED, PIXEL-FAITHFUL REPRODUCTION OF THE BIZSPACE TEMPLATE FROM TAILGRIDS, REBUILT AS PLAIN HTML + CSS + VANILLA JS WITH NO BUILD STEP. EVERY PAGE, THE FULL LOOK & FEEL, HOVER STATES, AND SCROLL/ENTRANCE ANIMATIONS ARE REPRODUCED FROM CAPTURED REFERENCE ARTIFACTS. ALL ASSETS (FONTS, IMAGES, LOGOS, ICONS) ARE VENDORED LOCALLY AND IT RUNS OFFLINE.
>
> REFERENCE: `https://bizspace.demos.tailgrids.com`

# SUMMARY

A PREMIUM, HIGH-CONVERTING PROFESSIONAL AGENCY AND STARTUP WEBSITE DESIGN SYSTEM. IT FEATURES A CLEAN LANDING PAGE, AN ABOUT PAGE WITH STORIES AND TEAM GRIDS, A DEDICATED SERVICES PAGE, PORTFOLIO DIRECTORY AND DETAIL PAGES, A BLOG SECTION AND ARTICLE READ PAGES, A CONTACT PAGE WITH INTEGRATED FORM, AND A 404 ERROR INTERFACE. THE SYSTEM USES A MODERN DESIGN WITH VIBRANT BLUE ACCENTS (#3758F9), ELEGANT CARD LAYOUTS, BOLD TYPOGRAPHY, AND A COMPREHENSIVE DUAL-MODE THEMING SYSTEM.

# STYLE

THE STYLE RELIES ON THE 'DM SANS' FONT FOR ALL INTERFACES. IT USES TAILGRIDS BLUE (#3758F9) AS THE ACCENT PRIMARY SHADE, STYLED OVER CLEAN LIGHT SURFACES IN LIGHT MODE, AND DEEP BLUE-BLACK (#030712 / #111827) IN DARK MODE. ALL BORDERS ARE THIN (#E5E7EB IN LIGHT, #1F2937 IN DARK), AND EDGES ARE ROUNDED (LARGE CARDS AT 16PX-24PX).

# SPECIAL COMPONENTS

- **THEME SWITCH TOGGLE:** FLOATING IN NAV. APPLIES CLASS `.dark` TO `<html>` OR `<body>` ELEMENT AND PRESERVES PREFERENCE IN `localStorage`.
