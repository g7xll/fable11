## SUMMARY

A FULL-FEATURED CMS (CONTENT MANAGEMENT SYSTEM) ADMIN DASHBOARD TEMPLATE. THE LAYOUT CONSISTS OF A FIXED LEFT SIDEBAR WITH NAVIGATION, A STICKY TOP HEADER WITH SEARCH AND ACTIONS, AND A SCROLLABLE MAIN CONTENT AREA WITH A ROUNDED CARD CONTAINER. THE TEMPLATE INCLUDES SEVEN DISTINCT PAGES: DASHBOARD (OVERVIEW STATS + TRAFFIC CHART + RECENT POSTS), ANALYTICS (METRICS + LINE CHART + TOP POSTS + TRAFFIC SOURCE PIE CHART), ALL POSTS (SORTABLE TABLE WITH STATUS BADGES), CREATE POST (RICH EDITOR + PUBLISHING SIDEBAR), DRAFTS (UNPUBLISHED POSTS TABLE), SCHEDULED POSTS (UPCOMING POSTS TABLE), MEMBERS (SUBSCRIBER TABLE WITH PLAN TIERS), AND REVENUE (MRR CHART + SUBSCRIPTION BREAKDOWN). FULLY SUPPORTS LIGHT AND DARK MODES.
