## SUMMARY

NEXSTUDIO IS A CLEAN, MONOCHROME-DOMINANT MODERN AGENCY/CREATIVE STUDIO WEBSITE BUILT ON A HIGH-CONTRAST PALETTE (WHITE BACKGROUND, RICH BLACK ELEMENTS, LIGHT GRAY BORDERS AND BACKDROPS). THE TEMPLATE USES "INTER" AS ITS CORE SANS-SERIF FONT. THE DESIGN FEATURES A CLEAN NAV BAR WITH INTERACTIVE TWO-WAY SLIDING NAV LINK TEXT ANIMATIONS, A HERO SECTION WITH A VERY CLEAN H1 ("WE BUILD DIGITAL PRODUCTS THAT DRIVE GROWTH.") AND AN ITALIC SPAN, AN INFINITE CLIENT LOGO MARQUEE, A COVER IMAGE CARD, STATS COUNTERS, A CARD GRID DETAIL SECTION, A CLIENT REVIEWS SLIDER (SWIPER.JS-BASED SWIPING LAYOUT), RECENT BLOG ARTICLES, AND A CALL-TO-ACTION PANEL. TEN PAGES TOTAL: HOME, PORTFOLIO, PORTFOLIO DETAILS 1 & 2, ABOUT, SERVICES, BLOGS, BLOG DETAILS 1 & 2, CONTACT. SHARED CHROME = STICKY HEADER AND FOOTER WITH LINKS, SOCIALS AND LARGE BRAND TEXT MASK.
