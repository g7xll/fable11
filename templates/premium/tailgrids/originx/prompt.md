## SUMMARY

ORIGINX IS A CLEAN, MINIMALIST STARTUP AND BUSINESS WEBSITE TEMPLATE BY TAIlGRIDS. THE DESIGN IS BUILT ON A MODERN DIGITALLY-ORIENTED LAYOUT WITH AN OFF-WHITE/LIGHT-GREY BODY BACKGROUND (#F5F5F5) THAT GIVES A PREMIUM, CALM AND CLEAN FEEL. IT USES FIGTREE AS ITS PRIMARY FONT AND DRIVES BRAND IDENTITY THROUGH A VIBRANT ORANGE-CORAL PRIMARY ACCENT COLOR (#FF7C61) AND A SHADOW-PRIMARY DECORATIVE ACCENT. EACH SECTION IN THE TEMPLATE FEATURES CAREFULLY PROPORTIONED PADDING, GRID CARD SYSTEMS, SCROLL-TRIGGERED ENTRANCE ANIMATIONS, TEXT CAROUSELS, AND ACCORDIONS. THE CLONE REPRODUCES ALL KEY TEMPLATE PAGES AS A SELF-CONTAINED PLAIN HTML/CSS/JS PROJECT WITH A CUSTOM DARK MODE TOGGLE AND LOCALLY VENDORED ASSETS.

- `--COLOR-DARK-BORDER`: `#262626` — DARK MODE BORDER

### FONTS
- PRIMARY: `FIGTREE, SANS-SERIF` — USED FOR ALL TEXT, WEIGHTS 300 TO 900

### TYPE SCALE
- HERO TITLE: `TEXT-4XL LG:TEXT-6XL` — WEIGHT `semibold` (600), LINE-HEIGHT `leading-17`, LETTER-SPACING `-1.8px`
- SECTION HEADINGS: `TEXT-3XL MD:TEXT-4XL LG:TEXT-5XL` — WEIGHT `semibold` (600), LETTER-SPACING `-1.44px`
- CARD TITLE: `TEXT-2XL` — WEIGHT `semibold` (600)
- BODY TEXT: `TEXT-BASE` OR `TEXT-LG` — WEIGHT `normal` (400) OR `medium` (500)
- NAV LINKS: `TEXT-BASE` — WEIGHT `medium` (500)

### LAYOUT TOKENS
- SPACING UNIT: `--SPACING: 0.25REM` (4PX)
- CONTAINER: MAX-WIDTH `MAX-W-7XL` (1280PX) WITH `PX-4` OR `PX-6` GUTTERS
- SECTION PADDING: `PY-10 LG:PY-20` (40PX TO 80PX)
- HEADER HEIGHT: 64PX

### ANIMATION / INTERACTION
- ENTER-SCROLL ANIMATION: CARDS FADE AND SLIDE UP (`opacity: 0; transform: translateY(50px)` -> `opacity: 1; transform: translateY(0)`) ON SCROLL REVEAL.
- BUTTON SLIDE-UP EFFECT: CTAS USE A TRANSITION THAT SHIFTS AN ABSOLUTE ELEMENT TO SIMULATE A TEXT SWIPE/SLIDE EFFECT ON HOVER.
- SLIDER/CAROUSEL: CLIENT LOGOS MARQUEE AND TESTIMONIALS SLIDER WITH PAGINATION AND NAVIGATION CONTROLS.
- ACCORDION TRANSITIONS: FAQS FLIP OPEN WITH SMOOTH ROTATION OF THE EXPAND ICON AND HEIGHT TRANSITIONS.

### BORDER / RADIUS
- RADIUS-CARD: `ROUNDED-[20PX]` FOR HERO CARD CONTAINER; `ROUNDED-LG` OR `ROUNDED-FULL` FOR OTHERS.
- BORDERS: `BORDER-12` ON HERO IMAGE CONTAINER; `BORDER-WHITE/40` WITH `BG-CLIP-PADDING`.
- CARD ANCHORS: LITTLE SQUARE CORNER BLOCKS (#CBCBCB, size 5px x 5px) AT ALL FOUR CORNERS OF THE BORDERED CARDS (`-TOP-0.5 -LEFT-0.5`, ETC.).
