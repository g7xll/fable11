> A SELF-CONTAINED, PIXEL-FAITHFUL REPRODUCTION OF THE MAGIC UI "CHANGELOG" TEMPLATE, REBUILT AS PLAIN HTML + CSS + VANILLA JS WITH NO BUILD STEP. EVERY PAGE, COLOR, FONT, HOVER STATE, AND ENTRANCE/INTERACTION ANIMATION IS MATCHED SAME-TO-SAME.

REFERENCE: `https://changelog-magicui.vercel.app/`

## SUMMARY

A MINIMAL, EDITORIAL CHANGELOG / RELEASE-NOTES PAGE BUILT ON THE SHADCN/UI DESIGN LANGUAGE. A SINGLE-PAGE APP: A STICKY-COLUMN VERTICAL TIMELINE OF RELEASE ENTRIES, EACH WITH A DATE + VERSION BADGE ON THE LEFT AND A CONTENT BLOCK ON THE RIGHT (TITLE, CATEGORY PILLS, MEDIA, FEATURE BULLETS, OPTIONAL SYNTAX-HIGHLIGHTED CODE BLOCK, AND COLLAPSIBLE "FEATURES" / "BUG FIXES" ACCORDIONS). A HEADER WITH THE "CHANGELOG" WORDMARK AND A LIGHT/DARK THEME TOGGLE.
