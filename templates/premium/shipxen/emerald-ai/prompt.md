## SUMMARY

A SINGLE-PAGE DARK AI SAAS PRODUCT LANDING PAGE. THE TEMPLATE PROMOTES AN AI PRODUCTIVITY APP ("EMERALD AI"). IT FEATURES A STICKY HEADER WITH A LOGO AND NAV, A TOP ANNOUNCEMENT/LOGO STRIP, A SPLIT HERO WITH A PRODUCT SCREENSHOT AND PRICING BADGE, ALTERNATING FEATURE SECTIONS WITH MEDIA, A TESTIMONIAL WALL, A BENTO/GRID FEATURE LIST, AN FAQ ACCORDION, AND A RICH MULTI-COLUMN FOOTER. THE DESIGN LANGUAGE IS GLASSY DARK PANELS WITH VIOLET + FUCHSIA GRADIENT ACCENTS AND A TEAL/EMERALD "GLASS" GLOW.

THE SOURCE IS A NEXT.JS + SHADCN UI / PAGE UI BUILD. ALL IN-TEMPLATE NAVIGATION IS EITHER ON-PAGE ANCHORS OR LINKS OUT TO SHIPIXEN'S OWN MARKETING SITE (NOT PART OF THE TEMPLATE), SO THE CLONE IS A SINGLE PAGE: `index.html`. THE PREVIEW HOST CHROME (TOOLBAR, VIEWPORT TOGGLES, CART/BUY BUTTONS) IS NOT REPRODUCED.
