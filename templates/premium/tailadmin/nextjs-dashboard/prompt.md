Build a complete pixel-faithful HTML/CSS/JS clone of the TailAdmin Next.js dashboard template (https://nextjs-demo.tailadmin.com/).

Features:
- Full sidebar navigation with collapsible sections and accordion submenus
- Topbar with search, notifications, dark mode toggle, and user profile
- Dark mode support via [data-theme="dark"] on <html>, persisted to localStorage
- Responsive design (mobile sidebar overlay)
- ApexCharts integration for all charts
- 10 full-content pages: index.html, analytics.html, calendar.html, profile.html, signin.html, signup.html, form-elements.html, basic-tables.html, error-404.html, chat.html
- 60+ stub pages with full sidebar/topbar layout
- Outfit font from Google Fonts
- Design tokens: brand #465fff, sidebar 290px, topbar 64px
