> A SELF-CONTAINED, PIXEL-FAITHFUL REPRODUCTION OF THE "MINIMAL PORTFOLIO TEMPLATE" — A CLEAN, NEUTRAL-TONED PERSONAL PORTFOLIO SITE FOR A SOFTWARE ENGINEER, COVERING HOME, ABOUT, PROJECTS, BLOG (INDEX + POSTS), AND CONTACT PAGES, WITH LIGHT/DARK THEME SUPPORT.

REFERENCE: `https://ui.aceternity.com/template-preview/minimal-portfolio-template`

## SUMMARY

A MINIMAL, TYPOGRAPHY-FIRST PERSONAL PORTFOLIO/BLOG TEMPLATE FOR A SOFTWARE ENGINEER PERSONA ("JOHN MACTAVISH", DESIGN ENGINEER). THE SITE USES A FLOATING AVATAR IN THE TOP-LEFT AND A FLOATING PILL-SHAPED NAVIGATION BAR IN THE TOP-RIGHT (WITH A THEME TOGGLE ICON), A SINGLE-COLUMN CENTERED CONTENT LAYOUT CAPPED AT ROUGHLY 820PX, AND SECTIONS SEPARATED BY THIN HORIZONTAL RULES. THE SITE IS BUILT ON A NEUTRAL GRAY PALETTE WITH NO ACCENT COLOR — HIERARCHY COMES FROM TYPE WEIGHT/SIZE AND SUBTLE BACKGROUND TINTS ONLY. IT SUPPORTS LIGHT AND DARK MODE VIA A SUN/MOON TOGGLE, PERSISTED ACROSS VISITS.

## STYLE

### PALETTE (CSS CUSTOM PROPERTIES, LIGHT DEFAULT / DARK OVERRIDE)

- `--bg`: LIGHT `#ffffff` / DARK `#0a0a0a` (NEUTRAL-950)
- `--bg-subtle`: LIGHT `#fafafa` (NEUTRAL-50) / DARK `#141414`
- `--fg`: LIGHT `#09090b` (NEUTRAL-950/ZINC-950) / DARK `#fafafa`
- `--fg-muted`: LIGHT `#71717a` (NEUTRAL-500) / DARK `#a1a1aa` (NEUTRAL-400)
- `--fg-subtle`: LIGHT `#a1a1aa` / DARK `#71717a`
- `--border`: LIGHT `rgba(228,228,231,0.6)` (NEUTRAL-200/60) / DARK `rgba(63,63,70,0.5)` (NEUTRAL-700/50)
- `--card-bg`: LIGHT `rgba(255,255,255,0.9)` / DARK `rgba(24,24,27,0.9)`
- `--tag-bg`: LIGHT NEUTRAL-100 / DARK NEUTRAL-800
- `--pill-bg`: LIGHT `rgba(255,255,255,0.9)` WITH BACKDROP BLUR / DARK `rgba(24,24,27,0.85)`
- `--code-bg`: `#0f172a` (SLATE-900) IN BOTH MODES (CODE BLOCKS STAY DARK)
- `--danger/success/info` NOT USED ON-PAGE (RESERVED TOAST TOKENS, NOT NEEDED)

NO SATURATED ACCENT COLOR IS USED ANYWHERE — LINKS, BUTTONS, AND HIGHLIGHTS ARE ALL NEUTRAL GRAYSCALE WITH OPACITY/WEIGHT FOR EMPHASIS. COMPANY LOGOS (GOOGLE, MICROSOFT, AIRBNB, SHOPIFY, ADOBE) KEEP THEIR OWN BRAND COLORS AS IMAGE ASSETS.

### TYPOGRAPHY

- FONT: `Inter` (GOOGLE FONT / VARIABLE, WITH SYSTEM SANS FALLBACK STACK: `ui-sans-serif, system-ui, sans-serif, "Apple Color Emoji", "Segoe UI Emoji", "Segoe UI Symbol", "Noto Color Emoji"`)
- MONOSPACE (CODE BLOCKS): `ui-monospace, SFMono-Regular, Menlo, Monaco, Consolas, "Liberation Mono", "Courier New", monospace`
- BASE BODY: 16PX / 24PX LINE-HEIGHT / 400 WEIGHT
- H1 (PAGE TITLES, e.g. "JOHN MACTAVISH", "ABOUT ME", "PROJECTS", "ALL BLOGS", "CONTACT ME"): ~36–40PX, WEIGHT 800/900 (EXTRABOLD/BLACK), TIGHT LEADING, WITH A SUBTLE SOFT DROP-SHADOW BEHIND THE TEXT (BLURRED DUPLICATE LAYER)
- SECTION LABELS (e.g. "I LOVE BUILDING THINGS", "WORKED AT REPUTED FIRMS"): 14PX, WEIGHT 500, ON A LIGHT PILL/TAG BACKGROUND (`--tag-bg`), ROUNDED
- CARD/ITEM TITLES: 16–18PX, WEIGHT 600
- BODY/DESCRIPTION TEXT: 14–16PX, WEIGHT 400, `--fg-muted`
- NAV LINKS: 14PX, WEIGHT 500

### RADII, SHADOWS, SPACING

- PILL NAV / AVATAR / TAGS / BUTTONS: FULLY ROUNDED (`border-radius: 9999px`)
- CARDS / IMAGES / CODE BLOCKS / TIMELINE BADGES: `border-radius: 0.5rem–0.75rem` (8–12PX)
- CONTENT MAX-WIDTH: ~820PX, CENTERED, WITH ~24PX SIDE GUTTERS ON MOBILE
- SECTION VERTICAL RHYTHM: GENEROUS (48–64PX BETWEEN SECTIONS), THIN 1PX BORDER (`--border`) AS SECTION DIVIDER
- SHADOWS: VERY SUBTLE — `0 1px 2px rgba(0,0,0,0.04)` ON CARDS/PILLS; NO HEAVY ELEVATION ANYWHERE

### MOTION / EASING

- DEFAULT TRANSITION: `150ms cubic-bezier(0.4, 0, 0.2, 1)` FOR HOVER COLOR/BACKGROUND CHANGES ON NAV LINKS, BUTTONS, CARDS
- THEME TOGGLE: INSTANT CLASS SWAP (`.dark` ON `<html>`) WITH THE SAME 150MS TRANSITION APPLIED TO COLOR/BACKGROUND PROPERTIES FOR A SMOOTH CROSSFADE, NO FLASH ON LOAD (INLINE BOOT SCRIPT READS `localStorage` BEFORE PAINT)
- PROJECT/TESTIMONIAL CARDS: SUBTLE SCALE/BORDER-COLOR SHIFT ON HOVER (`transform: translateY(-2px)` OPTIONAL, KEPT MINIMAL PER THE "MINIMAL" BRAND OF THE TEMPLATE)
- TESTIMONIALS ROW ON HOME: HORIZONTALLY SCROLLABLE STRIP WITH A FADE MASK ON THE LEFT/RIGHT EDGES
