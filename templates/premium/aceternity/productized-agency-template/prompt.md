# PRODUCTIZED AGENCY TEMPLATE — ACETERNITY (STUDY / CLONE)

> A SELF-CONTAINED, PIXEL-FAITHFUL REPRODUCTION OF THE ACETERNITY "PRODUCTIZED AGENCY" TEMPLATE, REBUILT AS PLAIN HTML + CSS + VANILLA JS WITH NO BUILD STEP. EVERY PAGE, SECTION, HOVER STATE, AND SCROLL/ENTRANCE ANIMATION IS RECREATED FROM RECON ARTIFACTS. ALL ASSETS (FONTS, IMAGES) ARE VENDORED LOCALLY SO THE CLONE RUNS OFFLINE. THIS IS A STUDY OF SOMEONE ELSE'S DESIGN, BUILT FOR LEARNING — ALL CREDIT FOR THE ORIGINAL GOES TO ACETERNITY.

REFERENCE: `https://ui.aceternity.com/template-preview/productized-agency-template`

## SUMMARY

A PREMIUM "PRODUCTIZED AGENCY" MARKETING SITE FOR A FICTIONAL DESIGN & DEVELOPMENT STUDIO ("ACETERNITY LABS"). THE SITE SELLS A PRODUCTIZED SERVICE — A SUBSCRIPTION-STYLE DESIGN/ENGINEERING TEAM — WITH A WARM, LIGHT, EDITORIAL AESTHETIC ACCENTED BY A SIGNATURE YELLOW (`#FFCC00`) AND A DARK, GLOWING HERO. THE BUILD SPANS ELEVEN PAGES: A LONG HOME PAGE, A WORK/PROJECTS GALLERY, A PRODUCTS PAGE, A PRICING PAGE, A BLOG INDEX, AND SIX BLOG ARTICLE PAGES. SHARED CHROME (STICKY NAV WITH "CHAT WITH ALEX" CTA, AND A DARK "MAKE YOUR WEBSITE A SALES MACHINE" FOOTER WITH AN "ACETERNITY" WATERMARK) IS CONSISTENT ACROSS EVERY PAGE.

THE ORIGINAL IS A NEXT.JS + TAILWIND CSS V4 APP USING FRAMER MOTION FOR ENTRANCE/SCROLL ANIMATION. THE CLONE PRESERVES THE ORIGINAL TAILWIND-COMPILED STYLESHEET AND CLASS-DRIVEN MARKUP FOR FIDELITY, SELF-HOSTS THE FONTS, AND REPLACES FRAMER MOTION WITH AN INTERSECTIONOBSERVER-BASED REVEAL SYSTEM PLUS NATIVE CSS FOR ACCORDIONS, MARQUEES, TABS, AND HOVER STATES.

- WEIGHTS: 400 / 500 / 600 / 700; HEADINGS TIGHT TRACKING (`-TRACKING-XS`)

### SPACING / SHAPE
- SPACING UNIT: `.25REM`; CONTAINER MAX WIDTHS 72REM (6XL) / 80REM (7XL); NAV MAX-WIDTH `CALC(100% - 4REM)`
- RADII: SM .25REM, MD .375REM, LG .5REM, XL .75REM, 2XL 1REM, 3XL 1.5REM, 4XL 2REM (HERO CARD USES 3XL/4XL ROUNDING)
- CARD SHADOW: `0 2PX 6PX 0 #0000001F` (SHADOW-CARD-MD)
- GENEROUS WHITESPACE; CARDS ON BONE BACKGROUND, MANY WITH SUBTLE BORDERS / WHITE FILLS

### ANIMATION / EASINGS
- DEFAULT TRANSITION: `.15S CUBIC-BEZIER(.4,0,.2,1)`; EASE-OUT `CUBIC-BEZIER(0,0,.2,1)`
- ENTRANCE: FADE + RISE (TRANSLATE-Y) ON SCROLL VIA INTERSECTIONOBSERVER, STAGGERED FOR GRIDS/LISTS (REPLACES FRAMER MOTION)
- ACCORDION (FAQ): CSS `ACCORDION-DOWN` / `ACCORDION-UP` KEYFRAMES ON `--RADIX-ACCORDION-CONTENT-HEIGHT`
- LOGO MARQUEE: HORIZONTAL AUTO-SCROLL OF TRUSTED-BY LOGOS
- HOVER: CARDS LIFT / BORDER + SHADOW SHIFTS; BUTTONS DARKEN/BRIGHTEN; LOGOS FADE; NAV LINKS COLOR SHIFT (ALL 150–300MS)
- HERO: ANIMATED WARM RADIAL GLOW / SUN-ARC OVER BLACK, WITH LARGE "ACETERNITY" WATERMARK
