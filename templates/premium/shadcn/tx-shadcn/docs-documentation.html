Documentation

This section contains documentation for all the features in Taxonomy.

What's in the documentation?

The documentation is written using ContentLayer and MDX. It is automatically validated and transformed into type-safe JSON data.

Features

  • Components — Documentation for all the UI components used in Taxonomy.
  • Code Blocks — How to use Shiki for code highlighting in your documentation.
  • Style Guide — Typography and styling guidelines for the documentation.

Adding documentation

To add new documentation pages, create a new .mdx file in the content/docs directory.

content/ docs/ getting-started/ introduction.mdx documentation/ introduction.mdx components.mdx

ContentLayer will automatically pick up the new file and make it available in your app.
