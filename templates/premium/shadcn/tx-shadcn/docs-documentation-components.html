Components

Taxonomy uses a set of components built on top of Radix UI and styled with Tailwind CSS. All components are accessible and follow the WAI-ARIA patterns.

Button

A button component with variants: default, outline, and ghost.

<Button>Default</Button> <Button variant="outline">Outline</Button> <Button variant="ghost">Ghost</Button>

Badge

A badge component to display short text labels.

Default Secondary Outline
<Badge>Default</Badge> <Badge variant="secondary">Secondary</Badge> <Badge variant="outline">Outline</Badge>

Card

A card component to display content in a contained box.
