Introduction

This is a Next.js documentation site built with shadcn/ui components and ContentLayer.

About this documentation

This documentation site is built using ContentLayer which is a content SDK that validates and transforms your content into type-safe JSON data you can easily import into your application.

Contentlayer allows you to write your content in MDX, which is a format that allows you to write Markdown with embedded JSX components. This is a great way to write documentation that includes interactive examples.

Features

  • Write your content in MDX with embedded React components.
  • Type-safe content using the ContentLayer SDK.
  • Syntax highlighting using Shiki.
  • Automatic table of contents.
  • Full-text search with Algolia.
  • Dark mode support.

Getting Started

To get started with Taxonomy, you can view the source code on GitHub.

npx create-next-app@latest --example taxonomy

This will create a new Next.js app with Taxonomy pre-configured.
