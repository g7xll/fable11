## INTERACTIONS TO REPRODUCE

- **THEME TOGGLE SWITCH** (NAVBAR, DESKTOP + MOBILE) — CLICKING FLIPS `documentElement` BETWEEN `light-theme` AND `dark-theme` CLASSES, RECOLORING THE ENTIRE PAGE VIA CSS CUSTOM PROPERTIES (BACKGROUND WHITE→BLACK, TEXT DARK→WHITE, BORDERS/SHADOWS INVERTED). MUST HONOR `prefers-color-scheme` ON FIRST LOAD AND PERSIST CHOICE (LOCALSTORAGE), WITH A NO-FLASH BOOT SCRIPT.
- **FEATURES DROPDOWN** — CLICKING THE "FEATURES" NAV TRIGGER (CHEVRON ICON) OPENS A POPOVER PANEL LISTING 5 ICON+TITLE+DESCRIPTION ITEMS; TRIGGER GETS A PRESSED STATE (`opacity:0.7`, `scale(0.97)`) AND `aria-expanded` TOGGLES; CLICKING OUTSIDE OR PRESSING ESCAPE CLOSES IT.
- **MOBILE HAMBURGER MENU** — BELOW THE NAV BREAKPOINT, A HAMBURGER BUTTON REPLACES THE INLINE LINKS; CLICKING IT OPENS A FULL-SCREEN OVERLAY NAV (BLURRED BACKDROP) LISTING ALL NAV LINKS + LEGAL + GITHUB ICON + LOGIN + START FREE TRIAL, WITH A CLOSE (X) BUTTON.
- **HOVER STATES** — NAV LINKS (COLOR SHIFT TO PRIMARY BLUE, UNDERLINE), PRIMARY/GHOST BUTTONS (BACKGROUND/BORDER DARKEN), PRICING CARD "GET STARTED" BUTTONS, FOOTER LINKS, SOCIAL ICONS — ALL WITH SMOOTH COLOR TRANSITIONS PER `source.css`.
- **SCROLL** — STICKY NAVBAR REMAINS PINNED TO TOP THROUGHOUT SCROLL (NO SHRINK/HIDE BEHAVIOR OBSERVED IN SOURCE).
