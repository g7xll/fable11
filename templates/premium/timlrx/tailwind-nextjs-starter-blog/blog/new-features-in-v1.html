Saturday, August 7, 2021

New features in v1

Tails Azimuth

@TailsAzimuth

An overview of the new features released in v1 — code block copy, multiple authors, frontmatter layout and more.

Performance Improvements

The initial load JS has been reduced from 43kB to 39kB despite the addition of many new features. Theme colors can now be easily customized through the Tailwind configuration file, allowing site owners to switch away from the default green color scheme.

MDX Compiler Upgrade

The template has migrated from next-mdx-remote to mdx-bundler, utilizing xdm under the hood with micromark 3 and updated remark/rehype libraries. This enables direct component imports within MDX files. 

import CustomComponent from './CustomComponent'

<CustomComponent />

This is some regular markdown content after the component.

Layout System

Content can now map to custom React layout components via frontmatter configuration, allowing flexible post templates and multiple author support. 

---
title: My Post
layout: PostBanner
authors: ['default', 'sparrowhawk']
---

Code Block Improvements

Added code block copy buttons, line highlighting, line numbers, and improved syntax highlighting.

// Example with line highlighting
const greet = (name) => {
  return `Hello, ${name}!` // highlight-line
}

console.log(greet('world'))

Analytics and Comments

Added integration with multiple analytics providers (Plausible, Simple Analytics, Google Analytics) and comment systems (Giscus, Utterances, Disqus).

Self-Hosted Fonts

Self-hosted fonts from Fontsource replaced Google Fonts, improving performance and privacy while maintaining version control.
