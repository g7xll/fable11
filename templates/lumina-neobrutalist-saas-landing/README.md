# Lumina — Neo-Brutalist SaaS Landing Page (Cabinet Grotesk, Satoshi, Hard Offset Shadows)

A vibrant neo-brutalist landing page for a fictional creator-OS SaaS, **Lumina**, built with dominant yellow (`#FFE17C`), deep charcoal (`#171E19`), and sage (`#B7C6C2`) accents. High-contrast 2px black borders, hard offset shadows (zero blur), a 32px radial dot pattern on yellow backgrounds, and bold geometric type — Cabinet Grotesk extrabold/black for headings, Satoshi for body — define the aesthetic. The vertically stacked page runs through a fixed yellow nav, a two-column hero with a text-stroke "CHAOS" keyword and a browser-mockup dashboard, a dark social-proof marquee, a problem-vs-solution card pair, a 3-column feature grid, a dark "How It Works" 3-step flow, a bento use-case personas section, asymmetric-corner testimonial cards, and a final CTA with footer. The signature neo-brutalist push button translates `4px, 4px` on hover using a bouncy `cubic-bezier(0.175, 0.885, 0.32, 1.275)` to simulate physical press. Generated with Claude Fable 5.

## Run

This is a static project — open `index.html` in a browser, or serve the folder:

```sh
python3 -m http.server 8000
```

See `prompt.md` for the full build spec; `demo.mp4` shows it in motion.
