# VERDANT — STRATEGIC GROWTH ADVISORY HERO

## AESTHETIC IDENTITY

BUILD A REFINED, EDITORIAL HERO SECTION FOR A HIGH-END STRATEGY ADVISORY FIRM NAMED **VERDANT**. THE DESIGN LANGUAGE IS "QUIET LUXURY CONSULTING": A CALM WHITE CANVAS, A SINGLE OVERSIZED FOREST-GREEN HERO CARD, SERIF DISPLAY TYPOGRAPHY PAIRED WITH A NEUTRAL GROTESQUE, AND GENEROUS NEGATIVE SPACE. THE MOOD IS COMPOSED, EXPENSIVE, AND TRUSTWORTHY — THE OPPOSITE OF FLASHY SAAS. EVERY ELEMENT FEELS DELIBERATELY PLACED, WITH SOFT ROUNDED CORNERS AND UNDERSTATED MOTION.

## COLOR PALETTE

- PAGE BACKGROUND: PURE WHITE `RGB(255, 255, 255)`.
- BRAND PRIMARY (HERO CARD): DEEP FOREST GREEN `RGB(31, 81, 76)` / `#1F514C`.
- BRAND GREEN DARKER ACCENT: `#163C38` FOR SUBTLE INNER GRADIENT DEPTH.
- BRAND DARK (TEXT ON WHITE): NEAR-BLACK `RGB(20, 20, 20)` / `#141414`.
- ON-CARD TEXT: WHITE `#FFFFFF` FOR HEADLINES, WHITE AT 80% OPACITY FOR BODY.
- MUTED LOGO TEXT: BRAND DARK AT 55–60% OPACITY.
- WARM PAPER TINT FOR SECONDARY SECTIONS: `#F6F4EF`.

## TYPOGRAPHY

- DISPLAY / HEADLINES: **HEDVIG LETTERS SERIF** (SERIF, 400). USED FOR THE WORDMARK, THE HERO H1, STAT NUMBERS, AND LOGO MARQUEE NAMES. TIGHT NEGATIVE TRACKING (`-1.6PX` AT LARGE SIZES).
- UI / BODY: **INTER** (400/500/600). USED FOR NAV LINKS, BUTTONS, BODY COPY, EYEBROWS, AND LABELS, WITH SLIGHTLY TIGHT TRACKING (`-0.2` TO `-0.6PX`).
- VENDOR BOTH FONT FAMILIES LOCALLY (WOFF2 + @FONT-FACE), NO REMOTE CDN AT RUNTIME.

## MOTION / ANIMATION / INTERACTION SPEC

- **ENTRANCE**: ELEMENTS WITH `data-animation-on-scroll` START AT OPACITY 0 / TRANSLATEY(20PX) AND ANIMATE TO VISIBLE VIA AN INTERSECTIONOBSERVER, USING `CUBIC-BEZIER(0.2, 0.8, 0.2, 1)` OVER 0.8S. STAGGER WITH DELAYS (200/300/400/600/800MS) SO THE EYEBROW, HEADLINE, SUBHEAD, BUTTONS, AND IMAGE CASCADE IN.
- **MARQUEE**: CONTINUOUS `TRANSLATEX(0 → -50%)` LINEAR INFINITE OVER ~22S; PAUSES ON HOVER.
- **HERO IMAGE**: A SLOW VERTICAL FLOAT (±8PX, 6S EASE-IN-OUT) AND THE GLASS STAT CHIP FLOATS ON AN OFFSET LOOP.
- **BUTTON HOVER**: ARROW INSIDE THE CIRCLE TRANSLATES RIGHT ~8PX WITH A SPRINGY `CUBIC-BEZIER(0.34, 1.56, 0.64, 1)`; PILL DARKENS/LIGHTENS SLIGHTLY.
- **NAV LINKS**: OPACITY DIP TO 70% ON HOVER.
- **STATS**: COUNT-UP FROM 0 TO TARGET WHEN THE STRIP SCROLLS INTO VIEW.
- RESPECT `PREFERS-REDUCED-MOTION` BY DISABLING LOOPS AND COUNT-UP.

## RESPONSIVE BEHAVIOR

- DESKTOP (≥1024PX): TWO-COLUMN HERO, CENTERED NAV LINKS VISIBLE, FOUR-UP STATS, THREE-UP FEATURES.
