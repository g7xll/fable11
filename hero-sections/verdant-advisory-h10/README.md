# Verdant — Quiet-Luxury Strategy Advisory Hero Section (Vanilla HTML + CSS + JS)

[![Watch Demo](./poster.jpg)](./demo.mp4)

A refined, editorial hero section for a fictional high-end strategy advisory firm, **Verdant**, built in a "Quiet Luxury Consulting" design language — a calm white canvas with a single oversized forest-green hero card (`#1F514C`), Hedvig Letters Serif display type paired with Inter grotesque, and generous negative space. Beyond the two-column hero card, the page includes a trust marquee, a stats strip, an approach/feature trio, and a footer CTA band. All fonts and assets are vendored locally for offline use with accessible, semantic markup throughout. Generated with Claude Fable 5.

## Run

This is a static project — open `index.html` in a browser, or serve the folder:

```sh
python3 -m http.server 8000
```

See `prompt.md` for the full build spec; `demo.mp4` shows it in motion.
