# UI Desgin

4 **UI Desgin** experiments generated with Claude Fable 5 — part of the [claude-directory](../README.md).

| Project | Description | Stack |
|---------|-------------|-------|
| [industrial-skeuomorphism](./industrial-skeuomorphism/) | "SCHEMATIC" Industrial Skeuomorphism design-system showcase — a tactile hardware-control landing built on centralized neumorphic tokens, with an all-CSS hero device mockup (scanline screen + live dashboard), glowing LED status indicators, corner screws & vent slots on every panel, a cylindrical How-It-Works pipe connector, push-pinned testimonials with masking tape, punched-hole pricing tags, a conic-gradient radar sweep, recessed "data slot" inputs, and a matte-plastic noise overlay; vendored Inter/JetBrains Mono and all assets, fully offline & responsive | React, TypeScript, Vite, Tailwind CSS v4, Motion, Lucide |
| [landing-page-copy-machine](./landing-page-copy-machine/) | Exact-spec Vite + React + TypeScript landing-page scaffold for a strict "copy machine" workflow — `@/`→`/src/` alias wired in both `vite.config.ts` and `tsconfig.json`, a `cn()` (`twMerge(clsx())`) utility, Tailwind base styles, a `src/components/ui` drop zone, and an empty `relative min-h-screen flex flex-col` `App.tsx` stacking shell; deps (Framer Motion, Lucide, clsx, tailwind-merge) installed, builds clean, fully self-contained | React, TypeScript, Vite, Tailwind CSS, tailwind-merge, clsx, Framer Motion, Lucide |
| [landing-page-scaffold](./landing-page-scaffold/) | Pre-component landing-page scaffold built to an exact multi-turn spec — Vite + React + TypeScript, Tailwind CSS with tailwind-merge/clsx (`cn()` helper), Framer Motion and Lucide React installed, a verbatim `@/`→`/src/` path alias in vite/tsconfig, and the mandated `/src/components/ui` + `/src/lib` folder layout with an empty `App.tsx` shell ready for components to stack into | React, TypeScript, Vite, Tailwind CSS, tailwind-merge, clsx, Framer Motion, Lucide |
| [serif-editorial-design-system](./serif-editorial-design-system/) | "Roman & Quill" Serif design-system showcase — a private-publishing-house editorial landing (hero, stats, drop-cap features, asymmetric 1.3fr/0.7fr benefits, decorative-quote testimonials, gold-tinted featured pricing, journal grid, ARIA FAQ, inverted correspond form, footer) with vendored Playfair/Source Sans 3/IBM Plex Mono, a rule-line + small-caps label system, bespoke SVG engraving plates, burnished-gold accent, paper-grain overlay, and ambient glow — all on channel-format design tokens, fully offline | React, TypeScript, Vite, Tailwind CSS, Lucide |
